<div id="sync-web-data-in-flowise-workflows">
  ## Sincronize dados da web em fluxos de trabalho do Flowise
</div>

O Firecrawl pode ser usado dentro do [Flowise, o construtor de Chatflows](https://flowiseai.com/). Esta página explica como configurar e usar um bloco do Firecrawl no Flowise.

<img
  className="block"
  src="/images/fc_flowise_block.png"
  alt="Bloco do Firecrawl no Flowise"
/>

<div id="crawling-with-firecrawl-blocks">
  ### Rastreamento com blocos do Firecrawl
</div>

1. Faça login na sua conta Firecrawl, obtenha sua chave de API e insira-a no bloco.
2. (Opcional) Conecte o Text Splitter.
3. Selecione o modo de rastreamento para coletar páginas abaixo da URL de destino.
4. Insira a URL de destino a ser rastreada.
5. Use os documentos resultantes nos seus fluxos de trabalho.

<div id="scraping-with-firecrawl-blocks">
  ### Extraindo com blocos do Firecrawl
</div>

1. Faça login na sua conta do Firecrawl, obtenha sua chave de API e insira-a no bloco.
2. (Opcional) Conecte o Text Splitter.
3. Selecione o modo de extração para processar uma única página.
4. Insira a URL de destino a ser extraída.
5. Use os documentos resultantes nos seus fluxos de trabalho.
