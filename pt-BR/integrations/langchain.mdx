> Observação: esta integração ainda usa a [versão v0 da API do Firecrawl](/pt-BR/v0/introduction). Você pode instalar a versão 0.0.20 do SDK para Python ou a 0.0.36 do SDK para Node.

<div id="installation">
  ## Instalação
</div>

```bash
pip install firecrawl-py==0.0.20
```

<div id="usage">
  ## Uso
</div>

Você precisará obter sua própria chave de API. Consulte https://firecrawl.dev

```python
from langchain_community.document_loaders import FireCrawlLoader

loader = FireCrawlLoader(
    api_key="SUA_API_KEY", url="https://firecrawl.dev", mode="crawl"
)

docs = loader.load()
```

<div id="modes">
  ### Modos
</div>

Scrape: Raspa uma única URL e retorna o markdown.
Crawl: Rastreia a URL e todas as subpáginas acessíveis, retornando o markdown de cada uma.

```python
loader = FireCrawlLoader(
    api_key="SUA_API_KEY",
    url="https://firecrawl.dev",
    mode="scrape",
)

data = loader.load()
```

<div id="crawler-options">
  ### Opções do Crawler
</div>

Você também pode passar parâmetros para o loader. Isso é um dicionário de opções a serem repassadas ao crawler. Consulte a documentação da API do Firecrawl para mais informações.

<div id="langchain-js">
  ## Langchain JS
</div>

Para usar no Langchain JS, você pode instalar via npm:

```bash
npm install @mendableai/firecrawl-js
```

Depois, você pode usá-lo assim:

```typescript
import { FireCrawlLoader } from "langchain/document_loaders/web/firecrawl";

const loader = new FireCrawlLoader({
  url: "https://firecrawl.dev", // A URL a ser extraída
  apiKey: process.env.FIRECRAWL_API_KEY, // Opcional; por padrão usa `FIRECRAWL_API_KEY` do seu ambiente.
  mode: "scrape", // Modo de execução do crawler. Use "scrape" para uma única URL ou "crawl" para todas as subpáginas acessíveis
  params: {
    // parâmetros opcionais conforme a documentação da API do Firecrawl
    // Para ver a documentação da API, visite https://docs.firecrawl.dev
  },
});

const docs = await loader.load();
```
