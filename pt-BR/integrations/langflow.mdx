  ## Sincronize dados da web em fluxos do Langflow
</div>

<div id="scraping-with-firecrawl-blocks">
  ### Fazer scraping com blocos do Firecrawl
</div>

1. Faça login na sua conta do Firecrawl, obtenha sua chave de API e insira-a no bloco ou passe-a a partir de outra etapa do fluxo de trabalho.
2. (Opcional) Conecte o Text Splitter.
3. Selecione o modo de scraping para capturar uma única página.
4. Insira a URL de destino a ser coletada ou passe-a a partir de outra etapa do fluxo de trabalho.
5. Configure as Page Options e as Extractor Options conforme o site e os dados que você deseja obter. Você também pode passar essas opções a partir de outra etapa do fluxo de trabalho.
6. Use os dados nos seus fluxos de trabalho.
