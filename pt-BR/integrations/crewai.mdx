<div id="using-firecrawl-with-crewai">
  ## Usando o Firecrawl com o CrewAI
</div>

O Firecrawl está integrado ao [CrewAI, o framework para orquestração de agentes de IA](https://www.crewai.com/). Esta página apresenta todas as ferramentas do Firecrawl adicionadas ao framework.

<div id="installing-firecrawl-tools-inside-of-crewai">
  ### Instalando as ferramentas do Firecrawl no CrewAI
</div>

* Obtenha uma chave de API no seu [painel do firecrawl.dev](https://firecrawl.dev) e defina-a nas variáveis de ambiente (`FIRECRAWL_API_KEY`).
* Instale o [SDK do Firecrawl](https://github.com/mendableai/firecrawl) junto com o pacote `crewai[tools]`:

```
pip install firecrawl-py 'crewai[tools]'
```

  ## Ferramentas

<div id="firecrawlcrawlwebsitetool">
  ### Ferramenta FirecrawlCrawlWebsite
</div>

<div id="example">
  #### Exemplo
</div>

Use o FirecrawlScrapeFromWebsiteTool da seguinte forma para permitir que seu agente carregue sites:

```python
from crewai_tools import FirecrawlCrawlWebsiteTool

tool = FirecrawlCrawlWebsiteTool(url='firecrawl.dev')
```


<div id="arguments">
  #### Argumentos
</div>

- `api_key`: Opcional. Especifica a chave de API do Firecrawl. O padrão é a variável de ambiente `FIRECRAWL_API_KEY`.
- `url`: A URL base de onde o rastreamento deve começar.
- `page_options`: Opcional.
  - `onlyMainContent`: Opcional. Retorna apenas o conteúdo principal da página, excluindo cabeçalhos, barras de navegação, rodapés etc.
  - `includeHtml`: Opcional. Inclui o HTML bruto da página. Irá gerar uma chave html na resposta.
- `crawler_options`: Opcional. Opções para controlar o comportamento do rastreamento.
  - `includes`: Opcional. Padrões de URL para incluir no rastreamento.
  - `exclude`: Opcional. Padrões de URL para excluir do rastreamento.
  - `generateImgAltText`: Opcional. Gera texto alternativo para imagens usando LLMs (requer um plano pago).
  - `returnOnlyUrls`: Opcional. Se true, retorna apenas as URLs como uma lista no status do rastreamento. Observação: a resposta será uma lista de URLs dentro de data, não uma lista de documentos.
  - `maxDepth`: Opcional. Profundidade máxima de rastreamento. Profundidade 1 é a URL base, profundidade 2 inclui a URL base e seus filhos diretos, e assim por diante.
  - `mode`: Opcional. Modo de rastreamento a ser usado. O modo Fast rastreia 4x mais rápido em sites sem sitemap, mas pode não ser tão preciso e não deve ser utilizado em sites com forte renderização em JavaScript.
  - `limit`: Opcional. Número máximo de páginas a rastrear.
  - `timeout`: Opcional. Tempo limite, em milissegundos, para a operação de rastreamento.

<div id="firecrawlscrapewebsitetool">
  ### FirecrawlScrapeWebsiteTool
</div>

<div id="example">
  #### Exemplo
</div>

Use o FirecrawlScrapeWebsiteTool da seguinte forma para permitir que seu agente carregue sites:

```python
from crewai_tools import FirecrawlScrapeWebsiteTool

tool = FirecrawlScrapeWebsiteTool(url='firecrawl.dev')
```


<div id="arguments">
  #### Argumentos
</div>

- `api_key`: Opcional. Especifica a chave da API do Firecrawl. O padrão é a variável de ambiente `FIRECRAWL_API_KEY`.
- `url`: A URL a ser coletada.
- `page_options`: Opcional.
  - `onlyMainContent`: Opcional. Retorna apenas o conteúdo principal da página, excluindo cabeçalhos, navegação, rodapés etc.
  - `includeHtml`: Opcional. Inclui o HTML bruto da página. Gera uma chave html na resposta.
- `extractor_options`: Opcional. Opções para extração com LLM de informações estruturadas do conteúdo da página
  - `mode`: Modo de extração a ser usado; atualmente oferece suporte a 'llm-extraction'
  - `extractionPrompt`: Opcional. Um prompt descrevendo quais informações extrair da página
  - `extractionSchema`: Opcional. O esquema para os dados a serem extraídos
- `timeout`: Opcional. Tempo limite, em milissegundos, para a requisição

<div id="firecrawlsearchtool">
  ### FirecrawlSearchTool
</div>

<div id="example">
  #### Exemplo
</div>

Use o FirecrawlSearchTool da seguinte maneira para permitir que seu agente carregue sites:

```python
from crewai_tools import FirecrawlSearchTool

tool = FirecrawlSearchTool(query='o que é o Firecrawl?')
```


<div id="arguments">
  #### Argumentos
</div>

- `api_key`: Opcional. Especifica a chave da API do Firecrawl. O padrão é a variável de ambiente `FIRECRAWL_API_KEY`.
- `query`: A string de consulta a ser usada na pesquisa.
- `page_options`: Opcional. Opções para formatação do resultado.
  - `onlyMainContent`: Opcional. Retorna apenas o conteúdo principal da página, excluindo cabeçalhos, menus de navegação, rodapés etc.
  - `includeHtml`: Opcional. Inclui o HTML bruto da página. Gera uma chave html na resposta.
  - `fetchPageContent`: Opcional. Busca o conteúdo completo da página.
- `search_options`: Opcional. Opções para controlar o comportamento de rastreamento.
  - `limit`: Opcional. Número máximo de páginas a rastrear.
