<div id="how-it-works">
  ## Como Funciona
</div>

Transforme sites de e-commerce em dados estruturados de produtos. Monitore os preços da concorrência em tempo real, acompanhe os níveis de estoque entre fornecedores e migre catálogos de produtos entre plataformas sem complicações.

<div id="what-you-can-extract">
  ## O que você pode extrair
</div>

* **Dados do produto**: Título, SKU, especificações, descrições, categorias
* **Preços**: Preço atual, descontos, frete, impostos
* **Estoque**: Níveis de estoque, disponibilidade, prazos de entrega
* **Avaliações**: Classificações, avaliações de clientes, seções de Perguntas e Respostas

<div id="faqs">
  ## FAQs
</div>

<AccordionGroup>
  <Accordion title="Como posso acompanhar mudanças nos preços dos concorrentes?">
    Construa um sistema de monitoramento usando a API do Firecrawl para extrair preços em intervalos regulares. Compare os dados extraídos ao longo do tempo para identificar tendências de preços, promoções e posicionamento competitivo.
  </Accordion>

  <Accordion title="Posso extrair variantes de produto (tamanho, cor etc.)?">
    Sim. O Firecrawl pode extrair todas as variantes de produto, incluindo tamanho, cor e outras opções. Estruture os dados com esquemas personalizados para capturar todas as informações de variantes.
  </Accordion>

  <Accordion title="Como lidar com preços dinâmicos ou específicos por usuário?">
    Para preços dinâmicos, você pode usar a renderização de JavaScript do Firecrawl para capturar os preços após o carregamento. Para preços específicos do usuário, configure cabeçalhos de autenticação nas suas requisições.
  </Accordion>

  <Accordion title="Posso extrair dados de diferentes plataformas de e-commerce?">
    Sim. O Firecrawl pode extrair dados de qualquer site de e-commerce publicamente acessível. Usuários têm extraído com sucesso de Shopify, WooCommerce, Magento, BigCommerce e lojas personalizadas.
  </Accordion>

  <Accordion title="O Firecrawl consegue lidar com paginação e rolagem infinita?">
    Sim. O Firecrawl pode navegar por listas de produtos paginadas e lidar com mecanismos de rolagem infinita para extrair catálogos completos, garantindo que nenhum produto seja deixado de fora durante a extração.
  </Accordion>
</AccordionGroup>
