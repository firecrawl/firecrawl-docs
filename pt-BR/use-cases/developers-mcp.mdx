</div>

Integre o Firecrawl diretamente ao seu fluxo de desenvolvimento com IA. Pesquise documentação, obtenha especificações de API e acesse dados da web sem sair do seu ambiente de desenvolvimento por meio do Model Context Protocol.

<div id="faqs">
  ## FAQs
</div>

<AccordionGroup>
  <Accordion title="Quais assistentes de IA são compatíveis com MCP?">
    Atualmente, o Claude Desktop e o Cursor têm suporte nativo ao MCP. Outros assistentes de IA vêm adicionando suporte regularmente. Você também pode usar o MCP SDK para criar integrações personalizadas.
  </Accordion>

  <Accordion title="Posso usar MCP no VS Code ou em outras IDEs?">
    O VS Code e outras IDEs podem usar MCP por meio de extensões da comunidade ou integrações via terminal. O suporte nativo varia conforme a IDE. Confira nosso [repositório no GitHub](https://github.com/mendableai/firecrawl-mcp-server) para guias de configuração específicos por IDE.
  </Accordion>

  <Accordion title="Como faço cache de documentos acessados com frequência?">
    O servidor MCP armazena respostas em cache automaticamente por 15 minutos. Você pode configurar a duração do cache nas configurações do servidor MCP ou implementar uma lógica de cache personalizada.
  </Accordion>

  <Accordion title="Existe limite de taxa para solicitações MCP?">
    As solicitações MCP usam seus limites padrão da API do Firecrawl. Recomendamos agrupar solicitações relacionadas e usar cache para documentação acessada com frequência.
  </Accordion>

  <Accordion title="Como configuro o MCP com minha chave de API do Firecrawl?">
    Siga nosso [guia de configuração](https://github.com/mendableai/firecrawl-mcp-server#installation) para configurar o MCP. Você precisará adicionar sua chave de API do Firecrawl ao arquivo de configuração do MCP. O processo leva apenas alguns minutos.
  </Accordion>
</AccordionGroup>
