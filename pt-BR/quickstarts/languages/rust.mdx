<div id="overview">
  ## Visão geral
</div>

Use o SDK Rust do Firecrawl para chamar a API v2 em código Rust assíncrono.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Toolchain do Rust com tokio

<div id="install">
  ## Instalação
</div>

```bash
cargo add firecrawl tokio --features tokio/full
```

<div id="configure-your-api-key">
  ## Configure sua chave de API
</div>

Exponha sua chave de API por meio de uma variável de ambiente.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Faça sua primeira requisição
</div>

```rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    let api_key = std::env::var("FIRECRAWL_API_KEY").expect("chave de API ausente");
    let app = FirecrawlApp::new(api_key).expect("Falha ao inicializar o Firecrawl");

    let doc = app
        .scrape_url("https://example.com", None)
        .await
        .expect("Falha no scraping");

    println!("{}", doc.markdown.unwrap_or_default());
}
```

<div id="complete-example">
  ## Exemplo completo
</div>

```rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    let api_key = std::env::var("FIRECRAWL_API_KEY").expect("Missing chave de API");
    let app = FirecrawlApp::new(api_key).expect("Failed to init Firecrawl");

    let doc = app
        .scrape_url("https://example.com", None)
        .await
        .expect("Scrape failed");

    if let Some(metadata) = doc.metadata {
        println!("Title: {}", metadata.title.unwrap_or_default());
    }
    println!("{}", doc.markdown.unwrap_or_default());
}
```
