<div id="overview">
  ## Visão geral
</div>

Use o SDK oficial do Firecrawl para Node.js para transformar uma URL em conteúdo limpo e pronto para LLM.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Node.js 18+
* npm, pnpm ou yarn

<div id="install">
  ## Instalação
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-your-api-key">
  ## Configure sua chave de API
</div>

Armazene sua chave de API em uma variável de ambiente à qual apenas seu servidor tenha acesso.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Faça sua primeira requisição
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

(async () => {
  const doc = await app.scrape('https://example.com', { formats: ['markdown'] });
  console.log(doc.markdown);
})();
```

<div id="complete-example">
  ## Exemplo completo
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

async function main() {
  const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
  const doc = await app.scrape('https://example.com', { formats: ['markdown', 'html'] });
  console.log(doc.metadata?.title);
  console.log(doc.markdown);
}

main().catch((error) => {
  console.error(error);
  process.exit(1);
});
```
