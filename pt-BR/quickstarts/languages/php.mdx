<div id="overview">
  ## Visão geral
</div>

Use a extensão cURL do PHP para enviar uma solicitação de scraping ao Firecrawl.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* PHP 8.1+ com cURL habilitado

<div id="install">
  ## Instale
</div>

Não é necessário instalar pacotes extras; este exemplo usa a extensão cURL nativa.

<div id="configure-your-api-key">
  ## Configure sua chave de API
</div>

Salve sua chave de API em uma variável de ambiente no servidor.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Envie sua primeira requisição
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
```

<div id="complete-example">
  ## Exemplo completo
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown', 'html'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
if ($response === false) {
    throw new RuntimeException(curl_error($ch));
}
curl_close($ch);

echo $response;
```
