<div id="overview">
  ## Visão geral
</div>

Use um endpoint `+server.ts` do SvelteKit para fazer chamadas ao Firecrawl no servidor.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Um app SvelteKit
* Node.js 18+

<div id="install">
  ## Instalação
</div>

Nenhum pacote extra é necessário; use `fetch` no endpoint do servidor.

<div id="configure-environment">
  ## Configurar o ambiente
</div>

Armazene a chave de API em `.env` e carregue-a de `$env/static/private`.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

src/routes/api/firecrawl/+server.ts

```ts
import { json } from '@sveltejs/kit';
import { FIRECRAWL_API_KEY } from '$env/static/private';

export async function POST({ request }) {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return json(data);
}
```

<div id="call-it-from-your-app">
  ## Chame-o no seu app
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Mantenha a chave de API em variáveis de ambiente privadas; evite expô-la por meio de `$env/static/public`.
