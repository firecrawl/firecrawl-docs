<div id="overview">
  ## Visão geral
</div>

Use o `fetch` nativo do Bun para chamar a API HTTP do Firecrawl.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Bun 1.x

<div id="install">
  ## Instale
</div>

Nenhum pacote extra é necessário; o Bun já inclui `fetch` nativamente.

<div id="configure-environment">
  ## Configurar o ambiente
</div>

Defina a chave de API nas variáveis de ambiente antes de executar o Bun.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

server.ts

```ts
const server = Bun.serve({
  port: 3000,
  async fetch(req) {
    const url = new URL(req.url);
    if (url.pathname != '/api/firecrawl' || req.method != 'POST') {
      return new Response('Not found', { status: 404 });
    }

    const { url: target } = await req.json();
    const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ url: target, formats: ['markdown'] }),
    });

    return new Response(await response.text(), {
      headers: { 'Content-Type': 'application/json' },
    });
  },
});

console.log(`Listening on http://localhost:${server.port}`);
```

<div id="call-it-from-your-app">
  ## Faça a chamada no seu app
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Observações
</div>

* Mantenha a chave de API no servidor e proxie as requisições pelo seu handler do Bun.
