<div id="overview">
  ## Visão geral
</div>

Use uma Edge Function com Deno para chamar a API HTTP do Firecrawl.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Um projeto Supabase
* Supabase CLI

<div id="install">
  ## Instale
</div>

Nenhum pacote extra é necessário; as Edge Functions do Supabase são executadas no Deno com fetch.

<div id="configure-environment">
  ## Configurar ambiente
</div>

Defina a chave de API como um segredo no Supabase.

```bash
supabase secrets set FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

supabase/functions/firecrawl/index.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

<div id="call-it-from-your-app">
  ## Faça a chamada no seu aplicativo
</div>

```js
const response = await fetch('https://project.supabase.co/functions/v1/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* As Edge Functions usam Deno, então evite pacotes exclusivos do Node.
