<div id="overview">
  ## Visão geral
</div>

Use uma Vercel Function para atuar como proxy das chamadas ao Firecrawl sem expor sua chave de API.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API da Firecrawl
* Um projeto na Vercel
* Node.js 18+

<div id="install">
  ## Instalação
</div>

Nenhum pacote adicional é necessário; use `fetch` dentro da função.

<div id="configure-environment">
  ## Configure o ambiente
</div>

Defina `FIRECRAWL_API_KEY` em `.env.local` e nas configurações do seu projeto na Vercel.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

api/firecrawl.ts

```ts
import type { VercelRequest, VercelResponse } from '@vercel/node';

export default async function handler(req: VercelRequest, res: VercelResponse) {
  if (req.method !== 'POST') {
    res.status(405).send('Method not allowed');
    return;
  }

  const { url } = req.body;
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  res.status(200).json(data);
}
```

<div id="call-it-from-your-app">
  ## Faça a chamada no seu aplicativo
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

  ## Observações
</div>

* Configure variáveis de ambiente na Vercel para manter a chave de API privada.
