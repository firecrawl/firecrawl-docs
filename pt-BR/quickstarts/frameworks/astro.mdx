<div id="overview">
  ## Visão geral
</div>

Use uma rota de API do Astro para chamar o Firecrawl no servidor.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Um projeto Astro
* Node.js 18+

<div id="install">
  ## Instalação
</div>

Não é necessário nenhum pacote extra; use `fetch` na rota da API.

<div id="configure-environment">
  ## Configure o ambiente
</div>

Adicione a chave de API ao `.env` e acesse-a no código do servidor.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

src/pages/api/firecrawl.ts

```ts
import type { APIRoute } from 'astro';

export const POST: APIRoute = async ({ request }) => {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${import.meta.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
};
```

<div id="call-it-from-your-app">
  ## Chame-o a partir do seu app
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Use rotas no servidor para chamadas ao Firecrawl; evite requisições do lado do cliente com a chave de API.
