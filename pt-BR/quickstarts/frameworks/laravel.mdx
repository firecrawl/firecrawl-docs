<div id="overview">
  ## Visão geral
</div>

Use o cliente HTTP do Laravel para fazer chamadas à API do Firecrawl.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Aplicação Laravel
* PHP 8.1+

<div id="install">
  ## Instalação
</div>

Não é necessário instalar pacotes extras; o Laravel já inclui um cliente HTTP.

<div id="configure-environment">
  ## Configure o ambiente
</div>

Adicione a chave de API no arquivo `.env`.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

routes/api.php

```php
<?php

use Illuminate\Support\Facades\Http;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Route;

Route::post('/firecrawl', function (Request $request) {
    $response = Http::withToken(env('FIRECRAWL_API_KEY'))
        ->post('https://api.firecrawl.dev/v2/scrape', [
            'url' => $request->input('url'),
            'formats' => ['markdown'],
        ]);

    return $response->json();
});
```

<div id="call-it-from-your-app">
  ## Chame a partir do seu app
</div>

```bash
curl -X POST http://localhost:8000/api/firecrawl               -H 'Content-Type: application/json'               -d '{"url": "https://example.com"}'
```

<div id="notes">
  ## Observações
</div>

* Mantenha a chave de API apenas em variáveis de ambiente do servidor.
