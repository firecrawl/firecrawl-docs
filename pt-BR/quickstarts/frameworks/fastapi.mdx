<div id="overview">
  ## Visão geral
</div>

Use o SDK Python do Firecrawl em um handler de rota do FastAPI.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Aplicação FastAPI
* Python 3.8+

<div id="install">
  ## Instalação
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configure o ambiente
</div>

Defina a chave de API no ambiente do servidor.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Adicione uma rota ao servidor
</div>

app/main.py

```python
import os
from fastapi import FastAPI
from firecrawl import Firecrawl

app = FastAPI()
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def scrape(body: dict):
    doc = client.scrape(body["url"], formats=["markdown"])
    return {"markdown": doc.markdown}
```

<div id="call-it-from-your-app">
  ## Chame a partir do seu aplicativo
</div>

```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Observações
</div>

* Mantenha o cliente do Firecrawl inicializado uma única vez para reutilizar conexões HTTP.
