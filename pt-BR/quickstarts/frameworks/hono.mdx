<div id="overview">
  ## Visão geral
</div>

Use a API HTTP do Firecrawl diretamente em uma rota do Hono para funcionar em diferentes runtimes.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Aplicação Hono
* Um runtime compatível (Node, Bun, Cloudflare, Deno)

<div id="install">
  ## Instalação
</div>

```bash
npm install hono
```

<div id="configure-environment">
  ## Configure o ambiente
</div>

Exponha a chave de API por meio dos bindings de ambiente do seu runtime.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota no servidor
</div>

src/index.ts

```ts
import { Hono } from 'hono';

type Bindings = { FIRECRAWL_API_KEY: string };

const app = new Hono<{ Bindings: Bindings }>();

app.post('/api/firecrawl', async (c) => {
  const { url } = await c.req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${c.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return c.json(data);
});

export default app;
```

<div id="call-it-from-your-app">
  ## Faça a chamada no seu aplicativo
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Configure os bindings de ambiente para o runtime escolhido (Wrangler, Deno Deploy etc.).
