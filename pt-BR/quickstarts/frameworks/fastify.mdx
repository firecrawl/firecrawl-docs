<div id="overview">
  ## Visão geral
</div>

Use o SDK Node.js do Firecrawl em uma rota de servidor Fastify.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Aplicação Fastify
* Node.js 18+

<div id="install">
  ## Instalar
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-environment">
  ## Configure o ambiente
</div>

Adicione a chave de API ao ambiente do seu servidor.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota ao servidor
</div>

server.js

```js
import Fastify from 'fastify';
import Firecrawl from '@mendable/firecrawl-js';

const app = Fastify();
const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

app.post('/api/firecrawl', async (request) => {
  const { url } = request.body as { url: string };
  const doc = await client.scrape(url, { formats: ['markdown'] });
  return { markdown: doc.markdown };
});

app.listen({ port: 3000 });
```

<div id="call-it-from-your-app">
  ## Faça a chamada no seu aplicativo
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

<div id="notes">
  ## Observações
</div>

* Mantenha as requisições ao Firecrawl no servidor para proteger sua chave de API.
