<div id="overview">
  ## Visão geral
</div>

Use o SDK Python do Firecrawl em um endpoint do servidor Flask.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* Aplicação Flask
* Python 3.8+

<div id="install">
  ## Instalação
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configure o ambiente
</div>

Defina sua chave de API nas variáveis de ambiente antes de iniciar o Flask.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Adicione uma rota de servidor
</div>

app.py

```python
import os
from flask import Flask, request, jsonify
from firecrawl import Firecrawl

app = Flask(__name__)
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def firecrawl():
    url = request.json["url"]
    doc = client.scrape(url, formats=["markdown"])
    return jsonify(markdown=doc.markdown)
```

<div id="call-it-from-your-app">
  ## Faça a chamada pelo seu app
</div>

```python
import requests

response = requests.post(
    "http://localhost:5000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Notas
</div>

* Mantenha o cliente Firecrawl inicializado uma única vez e reutilize-o em todas as requisições.
