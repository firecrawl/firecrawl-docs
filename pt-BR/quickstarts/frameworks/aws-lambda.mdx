<div id="overview">
  ## Visão geral
</div>

Use um handler do Lambda para encaminhar solicitações ao Firecrawl com segurança.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Uma chave de API do Firecrawl
* AWS Lambda (ambiente de execução Node.js 18)

<div id="install">
  ## Instalação
</div>

Nenhum pacote extra é necessário; use o fetch no Node.js 18.

<div id="configure-environment">
  ## Configure o ambiente
</div>

Defina `FIRECRAWL_API_KEY` nas variáveis de ambiente da sua função Lambda.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Adicione uma rota do servidor
</div>

index.mjs

```js
export const handler = async (event) => {
  const { url } = JSON.parse(event.body || '{}');

  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return {
    statusCode: 200,
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data),
  };
};
```

<div id="call-it-from-your-app">
  ## Faça a chamada no seu app
</div>

```js
const response = await fetch('https://your-api-id.execute-api.region.amazonaws.com/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Configure o API Gateway (ou as URLs de função do Lambda) para expor o endpoint da sua função Lambda.
