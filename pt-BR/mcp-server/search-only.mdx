O [Firecrawl MCP Server](/pt-BR/mcp-server) em `https://mcp.firecrawl.dev/v2/mcp` é o endpoint padrão e fornece o conjunto completo de ferramentas do Firecrawl.

<div id="endpoint">
  ## Endpoint
</div>

```text
https://mcp.firecrawl.dev/v2/mcp-search
```

Toda requisição exige autenticação: conecte-se com OAuth diretamente pelo endpoint ou envie uma Firecrawl API key no cabeçalho `Authorization`.

```json
{
  "mcpServers": {
    "firecrawl-search": {
      "url": "https://mcp.firecrawl.dev/v2/mcp-search",
      "headers": {
        "Authorization": "Bearer fc-YOUR_API_KEY"
      }
    }
  }
}
```

<div id="available-tools">
  ## Ferramentas disponíveis
</div>

O endpoint de busca expõe exatamente estas seis ferramentas somente leitura:

| Ferramenta                          | Finalidade                                                    |
| ----------------------------------- | ------------------------------------------------------------- |
| `firecrawl_search`                  | Fazer uma busca na web e retornar resultados ranqueados       |
| `firecrawl_research_search_papers`  | Buscar artigos de pesquisa                                    |
| `firecrawl_research_inspect_paper`  | Obter metadados de um artigo                                  |
| `firecrawl_research_related_papers` | Encontrar artigos relacionados por meio de links de citação   |
| `firecrawl_research_read_paper`     | Ler trechos do texto completo de um artigo                    |
| `firecrawl_research_search_github`  | Buscar o histórico de repositórios públicos e arquivos README |

<div id="differences-from-the-full-mcp-endpoint">
  ## Diferenças em relação ao endpoint MCP completo
</div>

* O conjunto de ferramentas é fixo e de somente leitura.
* As ferramentas de scraping, mapeamento, rastreamento, extração, agente, interação, parse, monitoramento e feedback não estão disponíveis.
* `firecrawl_search` não aceita `scrapeOptions` e não permite solicitar a obtenção do conteúdo da página.
