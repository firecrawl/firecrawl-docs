O formato `menu` extrai um cardápio estruturado de restaurante **de forma determinística** — o mesmo tipo de resultado estruturado do [`formato `json&#96;&#96;](/pt-BR/features/scrape#extract-structured-data), mas sem uma chamada de LLM nem um schema definido por você, projetado especificamente para páginas de cardápio de restaurante. Se você vem extraindo campos de cardápio com um schema `json`, use `formats: ["menu"]` no lugar — é mais rápido e mais barato, mas limitado a cardápios.

Ele retorna um objeto `menu` com o estabelecimento, a moeda e uma lista de seções — em que cada seção contém itens, e cada item contém descrição, imagens, preço, disponibilidade, tags dietéticas, calorias e grupos de opções — útil para ingestão de cardápios, monitoramento de preços ou ferramentas de delivery e pedidos de comida.

<div id="menu-object-structure">
  ## Estrutura do objeto menu
</div>

O objeto `menu` contém as seguintes propriedades:

* `isMenu`: Indica se a página foi identificada como um menu
* `confidence`: Uma pontuação de confiança entre `0` e `1` para a extração do menu
* `merchant`: O estabelecimento ao qual o menu pertence (opcional):
  * `name`: O nome do estabelecimento
  * `type`: O tipo de estabelecimento, por exemplo `"restaurant"` (opcional)
* `currency`: O código da moeda do menu, informado apenas quando a página fornece essa informação (opcional)
* `sections`: Array de seções do menu. Cada seção tem:
  * `id`: O identificador da seção (opcional)
  * `name`: O nome da seção, por exemplo `"Appetizers"`
  * `description`: A descrição da seção (opcional)
  * `items`: Array de itens do menu. Cada item tem:
    * `id`: O identificador do item (opcional)
    * `name`: O nome do item
    * `description`: A descrição do item (opcional)
    * `images`: array de imagens do item, cada uma com uma `url` e um texto `alt` opcional (opcional)
    * `price`: o objeto de preço (opcional):
      * `amount`: O valor numérico do preço
      * `currency`: O código da moeda, informado apenas quando a página fornece essa informação (opcional)
      * `formatted`: O preço conforme exibido na página (opcional)
    * `availability`: informações de disponibilidade (opcional):
      * `inStock`: Indica se o item está disponível
      * `text`: O texto bruto de disponibilidade extraído da página (opcional)
    * `dietary`: array de tags alimentares, por exemplo `["vegetarian"]` (opcional)
    * `calories`: A quantidade de calorias do item (opcional)
    * `optionGroups`: Array de grupos de opções/modificadores do item (opcional)
    * `identifiers`: identificadores específicos do estabelecimento para o item (opcional):
      * `merchantItemId`: O ID próprio do item no estabelecimento (opcional)
    * `url`: A URL canônica do item (opcional)
    * `sourceUrl`: A URL da qual o item foi extraído (opcional)
* `sourceUrl`: A URL da qual o menu foi extraído (opcional)

<div id="how-menu-extraction-works">
  ## Como a extração de menu funciona
</div>

O formato `menu` extrai o menu por meio de uma chain determinística, executada na seguinte ordem: **adaptadores de plataforma (por exemplo, Toast) &gt; JSON-LD do schema.org &gt; microdados / estado JSON incorporado &gt; um parser de texto visível/markdown**. Ele retorna o resultado mais completo produzido pela chain e só recorre a uma etapa final com LLM quando a chain determinística não encontra nada ou encontra um resultado de baixa qualidade. Os nomes dos itens e os preços são sempre baseados na página — a extração nunca inventa itens. A moeda só é informada quando a página a fornece.

<Note>
  A extração de menu falha em modo fechado: páginas que não são menus não retornam menu. Em uma página sem menu extraível, a resposta omite o objeto `menu` e adiciona um `warning`.
</Note>

<Note>
  **Auto-hospedagem:** o formato `menu` depende de um serviço dedicado de extração de menu. No Firecrawl Cloud, ele funciona imediatamente. Se você fizer auto-hospedagem, defina `MENU_EXTRACTION_SERVICE_URL` para apontar para esse serviço — quando ela não estiver definida, solicitar o formato `menu` retorna um aviso &quot;Menu format is not available&quot; e nenhum campo `menu` (o scraping ainda é concluído com sucesso).
</Note>
