<div id="generate-llmstxt">
  ## generate-llmstxt
</div>

Um pacote simples do NPX que gera arquivos LLMs.txt via CLI usando a API do Firecrawl. Este pacote cria dois arquivos no diretório de saída especificado (por padrão, a pasta 'public'):

- `llms.txt`: Contém um resumo do conteúdo relacionado a LLMs
- `llms-full.txt`: Contém o texto completo

<div id="usage">
  ## Uso
</div>

Você pode executar este pacote com NPX, sem precisar instalá-lo. Existem duas maneiras de fornecer sua chave de API do Firecrawl:

<div id="1-using-command-line-arguments">
  ### 1. Usando argumentos de linha de comando
</div>

```bash
npx generate-llmstxt --api-key SUA_CHAVE_DA_API_DA_FIRECRAWL
```

<div id="2-using-environment-variables">
  ### 2. Usando variáveis de ambiente
</div>

Crie um arquivo `.env` na raiz do seu projeto e adicione sua chave de API:

```env
FIRECRAWL_API_KEY=your_api_key_here
```

Em seguida, execute o comando sem a opção --api-key:

```bash
npx generate-llmstxt
```


<div id="options">
  ### Opções
</div>

- `-k, --api-key <key>` (opcional se definido em .env): Sua chave de API do Firecrawl
- `-u, --url <url>` (opcional): URL a ser analisada (padrão: https://example.com)
- `-m, --max-urls <number>` (opcional): Número máximo de URLs a serem analisadas (padrão: 50)
- `-o, --output-dir <path>` (opcional): Caminho do diretório de saída (padrão: 'public')

<div id="examples">
  ### Exemplos
</div>

```bash
# Usando argumento de linha de comando com diretório de saída padrão
npx generate-llmstxt -k your_api_key -u https://your-website.com -m 20

# Usando arquivo .env com diretório de saída padrão
npx generate-llmstxt -u https://your-website.com -m 20

# Definindo um diretório de saída personalizado
npx generate-llmstxt -k your_api_key -u https://your-website.com -o custom/path/to/output

# Usando arquivo .env e diretório de saída personalizado
npx generate-llmstxt -u https://your-website.com -o content/llms
```

<div id="requirements">
  ## Requisitos
</div>

- Node.js 14 ou superior
- Uma chave de API Firecrawl válida (pela linha de comando ou em um arquivo .env)

<div id="output">
  ## Saída
</div>

O pacote criará dois arquivos no diretório de saída especificado (o padrão é 'public'):

1. `llms.txt`: Contém um resumo do conteúdo relacionado a LLMs
2. `llms-full.txt`: Contém o texto completo
