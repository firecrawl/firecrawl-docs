A API de pesquisa do Firecrawl permite realizar buscas na web e, opcionalmente, fazer scraping dos resultados em uma única operação.

* Escolha formatos de saída específicos (markdown, HTML, links, screenshots)
* Pesquise na web com parâmetros personalizáveis (localização, etc.)
* Opcionalmente recupere o conteúdo dos resultados de pesquisa em vários formatos
* Controle a quantidade de resultados e defina tempos limite

Para mais detalhes, consulte a [Referência da API do endpoint /search](https://docs.firecrawl.dev/api-reference/endpoint/search).

</div>

Valores comuns de `tbs`:

- `qdr:h` - Última hora
- `qdr:d` - Últimas 24 horas
- `qdr:w` - Última semana
- `qdr:m` - Último mês
- `qdr:y` - Último ano

Para um filtro temporal mais preciso, você pode especificar intervalos de datas exatos usando o formato de intervalo personalizado:

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp

# Inicialize o cliente com sua chave de API
app = FirecrawlApp(api_key="fc-YOUR_API_KEY")

# Pesquisar resultados de dezembro de 2024
search_result = app.search(
    "firecrawl updates",
    limit=10,
    tbs="cdr:1,cd_min:12/1/2024,cd_max:12/31/2024"
)
```

```js JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

// Inicialize o cliente com sua chave de API
const app = new FirecrawlApp({apiKey: "fc-YOUR_API_KEY"});

// Pesquisar resultados de dezembro de 2024
app.search("firecrawl updates", {
  limit: 10,
  tbs: "cdr:1,cd_min:12/1/2024,cd_max:12/31/2024"
})
.then(searchResult => {
  console.log(searchResult.data);
});
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer fc-YOUR_API_KEY" \
  -d '{
    "query": "firecrawl updates",
    "limit": 10,
    "tbs": "cdr:1,cd_min:12/1/2024,cd_max:12/31/2024"
  }'
```

</CodeGroup>

<div id="scraping-options">
  ## Opções de scraping
</div>

Ao fazer scraping de resultados de pesquisa, você pode especificar múltiplos formatos de saída e opções avançadas de scraping:

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp, ScrapeOptions

# Inicialize o cliente com sua chave de API
app = FirecrawlApp(api_key="fc-YOUR_API_KEY")

# Obtenha resultados de pesquisa com vários formatos
search_result = app.search(
    "firecrawl features",
    limit=3,
    scrape_options=ScrapeOptions(formats=["markdown", "html", "links", "screenshot"])
)
```

```js JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

// Inicialize o cliente com sua chave de API
const app = new FirecrawlApp({apiKey: "fc-YOUR_API_KEY"});

// Obtenha resultados de pesquisa com vários formatos
app.search("firecrawl features", {
  limit: 3,
  scrapeOptions: {
    formats: ["markdown", "html", "links", "screenshot"]
  }
})
.then(searchResult => {
  // Processe os resultados em vários formatos
  console.log(searchResult.data);
});
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer fc-YOUR_API_KEY" \
  -d '{
    "query": "firecrawl features",
    "limit": 3,
    "scrapeOptions": {
      "formats": ["markdown", "html", "links", "screenshot"]
    }
  }'
```

</CodeGroup>

Formatos disponíveis:

- `markdown`: Conteúdo em Markdown limpo e formatado
- `html`: Conteúdo HTML processado
- `rawHtml`: Conteúdo HTML sem modificações
- `links`: Lista de links encontrados na página
- `screenshot`: Captura de tela da página
- `screenshot@fullPage`: Captura de tela da página inteira
- `extract`: Extração de dados estruturados

<div id="cost-implications">
  ## Implicações de custo
</div>

Quando os resultados de busca não são coletados (nenhuma opção de scraping especificada), o custo é de 2 créditos a cada 10 resultados de busca. Quando o scraping está habilitado, não há cobrança adicional por coletas básicas de cada resultado de busca além dos custos padrão de scraping.

No entanto, fique atento a estes fatores de custo:

- **Parse de PDF**: 1 crédito por página de PDF (pode aumentar significativamente os custos em PDFs com várias páginas)
- **Modo de proxy stealth**: +4 créditos adicionais por resultado de busca
- ***modo JSON***: +4 créditos adicionais por resultado de busca

Para controlar os custos:

- Defina `parsePDF: false` se você não precisar de conteúdo em PDF
- Use `proxy: "basic"` em vez de `"stealth"` quando possível
- Limite a quantidade de resultados de busca com o parâmetro `limit`

<div id="advanced-scraping-options">
  ## Opções avançadas de scraping

Para mais detalhes sobre as opções de scraping, consulte a [documentação do recurso de Scrape](https://docs.firecrawl.dev/features/scrape). Tudo, exceto o FIRE-1 (Agente) e o recurso de rastreioDeMudanças, é compatível com este endpoint de Search.
