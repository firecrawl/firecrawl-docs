<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## Pesquisando na web e fazendo scraping dos resultados com o Firecrawl
</div>

O Firecrawl integra sua API de busca à sua robusta infraestrutura de scraping para oferecer uma experiência contínua de busca e extração por meio de um único endpoint. Veja por que:

1. **Consulta de busca unificada:**
   Os usuários enviam uma consulta de busca pelo endpoint de busca.

2. **Scraping automatizado dos resultados:**
   O Firecrawl processa automaticamente os resultados da busca e utiliza seus recursos de scraping para extrair dados de cada página de resultado.

3. **Entrega de dados:**
   Os dados extraídos de todas as páginas de resultados são compilados e entregues em markdown limpo — pronto para uso.

Essa integração permite realizar buscas na web com eficiência e obter dados abrangentes, extraídos de múltiplas fontes, com esforço mínimo.

Para mais detalhes, consulte a [documentação do endpoint /search](https://docs.firecrawl.dev/api-reference/endpoint/search).

<div id="search-endpoint">
  ### endpoint /search
</div>

Usado para pesquisar na web, obter os resultados mais relevantes, extrair o conteúdo de cada página e retornar o markdown.

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // false para uma API SERP mais rápida
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Conteúdo em Markdown",
      "provider": "web scraper",
      "metadata": {
        "title": "Firecrawl | Faça scraping da web com confiabilidade para seus LLMs",
        "description": "IA para CX e Vendas"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="with-python-sdk">
  ### Usando o SDK em Python
</div>

<div id="installing-python-sdk">
  #### Instalando o SDK do Python
</div>

```bash
pip install firecrawl-py
```

#### Fazer uma busca

```python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

result = app.search(query="What is firecrawl?")
```

A resposta será semelhante à mostrada no comando curl acima.

go get github.com/firecrawl/firecrawl-go

<div id="search-a-query">
  #### Pesquisar uma query
</div>

```go
import (
  "fmt"
  "log"

  "github.com/firecrawl/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
  }

  query := "O que é o Firecrawl?"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("Failed to search: %v", err)
  }
  fmt.Println(searchResult)
}
```


<div id="with-rust-sdk">
  ### Com o SDK em Rust
</div>

<div id="installing-rust-sdk">
  #### Instalando o SDK do Rust
</div>

Adicione o seguinte ao seu `Cargo.toml`:

```toml
[dependencies]
firecrawl = "^0.1"
tokio = { version = "^1", features = ["full"] }
serde = { version = "^1.0", features = ["derive"] }
serde_json = "^1.0"
uuid = { version = "^1.10", features = ["v4"] }

[build-dependencies]
tokio = { version = "1", features = ["full"] }
```

#### Realizar uma busca

```rust
async fn main() {
  let api_key = "YOUR_API_KEY";
  let api_url = "https://api.firecrawl.dev";
  let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

  let query = "What is firecrawl?";
  let search_result = app.search(query).await;

  match search_result {
    Ok(data) => println!("Search Result: {}", data),
    Err(e) => eprintln!("Failed to search: {}", e),
  }
}
```
