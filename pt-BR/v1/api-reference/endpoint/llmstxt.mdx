---
title: "Gerar LLMs.txt"
openapi: "v1-openapi POST /llmstxt"
---

O endpoint de geração de LLMs.txt cria os arquivos LLMs.txt e LLMs-full.txt para qualquer site. Esses arquivos oferecem um formato estruturado e otimizado para LLMs do conteúdo do site, facilitando para os modelos de linguagem entender e processar as informações.

Procurando o endpoint de status? Confira o endpoint [Status do LLMs.txt](/pt-BR/api-reference/endpoint/llmstxt-get).

<div id="response-structure">
  ### Estrutura da resposta
</div>

A resposta inclui:

* **success**: Booleano que indica se a solicitação foi bem-sucedida
* **id**: Identificador exclusivo do job de geração

<div id="output-format">
  ### Formato de saída
</div>

1. **LLMs.txt**

   * Resumo conciso e estruturado do site
   * Contém links e descrições principais
   * Formatado em Markdown para facilitar a análise
   * Exemplo:

     ```markdown
     # http://example.com llms.txt

     - [Título da página](https://example.com/page): Descrição breve
     - [Outra página](https://example.com/another): Outra descrição
     ```

2. **LLMs-full.txt** (quando showFullText é true)

   * Contém o conteúdo completo das páginas processadas
   * Mantém a estrutura hierárquica
   * Inclui informações mais detalhadas
   * Exemplo:

     ```markdown
     # http://example.com llms-full.txt

     ## Título da página

     Conteúdo completo da página...

     ## Outra página

     Conteúdo mais detalhado...
     ```

<div id="billing">
  ### Cobrança
</div>

A cobrança é baseada em chamadas à API e URLs processadas:

* Custo base: 1 crédito por chamada à API
* Adicional: 1 crédito por URL processada
* Controle os custos por URL com o parâmetro `maxUrls`
