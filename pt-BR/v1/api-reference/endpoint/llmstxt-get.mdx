---
title: "Status do LLMs.txt"
openapi: "v1-openapi GET /llmstxt/{id}"
---

Obtenha o status e os resultados de uma tarefa de geração do LLMs.txt. Este endpoint permite verificar se a geração foi concluída e recuperar o conteúdo gerado.

<div id="response-structure">
  ### Estrutura da resposta
</div>

A resposta inclui:

- **success**: Booleano que indica se a solicitação foi bem-sucedida
- **status**: Status atual do job de geração:

  - `processing`: O job ainda está em execução
  - `completed`: O job foi concluído com sucesso
  - `failed`: O job encontrou um erro

- **data**: Conteúdo gerado (quando o status é `completed`):

  - `llmstxt`: O conteúdo gerado do LLMs.txt
  - `llmsfulltxt`: Texto completo (se showFullText for true)

- **expiresAt**: Carimbo de data/hora em formato ISO indicando quando os resultados expiram

<div id="status-examples">
  ### Exemplos de status
</div>

1. **Status de processamento**

   ```json
   {
     "success": true,
     "status": "processing",
     "data": {
       "llmstxt": "# Firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/)...",
       "llmsfulltxt": "# Firecrawl.dev llms-full.txt\n\n"
     },
     "expiresAt": "2025-03-03T23:19:18.000Z"
   }
   ```

2. **Status concluído**
   ```json
   {
     "success": true,
     "status": "completed",
     "data": {
       "llmstxt": "# http://firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/): Transform websites into clean, LLM-ready data effortlessly.\n- [Flexible Web Scraping Pricing](https://www.firecrawl.dev/pricing): Flexible pricing plans for web scraping and data extraction.",
       "llmsfulltxt": "# http://firecrawl.dev llms-full.txt\n\n## Web Data Extraction Tool\nIntroducing /extract - Get web data with a prompt..."
     },
     "expiresAt": "2025-03-03T22:45:50.000Z"
   }
   ```

<div id="error-handling">
  ### Tratamento de erros
</div>

Se o trabalho de geração falhar ou não for encontrado, você receberá uma resposta de erro apropriada:

```json
{
  "success": false,
  "error": "Tarefa de geração de LLMs.txt não encontrada"
}
```


<div id="polling-recommendations">
  ### Recomendações de polling
</div>

1. Faça polling a cada 2–3 segundos inicialmente
2. Aumente o intervalo se ainda estiver em processamento após 30 segundos
3. Interrompa o polling se o status for `completed` ou `failed`
4. Implemente backoff exponencial para evitar limites de taxa
