<div id="better-sitemap-control">
  ### Melhor controle de sitemaps
</div>

Três maneiras de lidar com sitemaps:

* `"include"` - Usar o sitemap + encontrar outras páginas (padrão)
* `"skip"` - Ignorar o sitemap completamente
* `"only"` - Retornar apenas as URLs do sitemap

```json
{
  "url": "https://exemplo.com",
  "sitemap": "only"
}
```

<div id="smarter-url-discovery">
  ### Descoberta mais inteligente de URLs
</div>

- Encontra todas as páginas de um site
- Sem limite de resultados (v1 limitava a 500 resultados)
- Mais rápido e mais preciso

Mesma API da v1, mas encontra mais páginas e funciona melhor em sites complexos.
