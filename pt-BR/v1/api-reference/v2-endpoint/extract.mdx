<div id="whats-new-in-v2">
  ## Novidades na v2
</div>

<div id="smarter-extraction">
  ### Extração mais inteligente
</div>

O FIRE-1 (Agente) lida com sites complexos:

- Clica automaticamente em páginas e na paginação
- Funciona bem em sites pesados em JavaScript
- É melhor para encontrar e extrair dados

<div id="no-url-required">
  ### Nenhuma URL necessária
</div>

Extraia dados sem saber a URL exata:

- Descreva o que você precisa; a v2 encontra
- Faz buscas na web e extrai das melhores páginas
- Mais preciso do que a v1

<div id="better-performance">
  ### Melhor desempenho
</div>

- Processamento mais rápido
- Tratamento de erros mais robusto
- Maior precisão

Mesma API do v1, fácil de atualizar.
