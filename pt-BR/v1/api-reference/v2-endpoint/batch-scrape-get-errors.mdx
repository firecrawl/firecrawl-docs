<div id="whats-new-in-v2">
  ## Novidades na v2
</div>

<div id="better-batch-error-messages">
  ### Mensagens de erro em lote aprimoradas
</div>

- Erros claros para problemas de processamento de formatos em todas as URLs
- Mensagens específicas para falhas de PDF e do parser
- Melhor rastreamento de erros para diferentes modos de proxy
- Erros claros para problemas de raspagem com base em localização

<div id="more-error-details">
  ### Mais detalhes sobre erros
</div>

- Informações de depuração para problemas de rastreioDeMudanças
- Relatórios de erro para falhas na automação do navegador
- Melhor acompanhamento de erros de créditos e limites de recursos
- Erros específicos para problemas no processamento em lote

<div id="faster-and-easier-debugging">
  ### Depuração mais rápida e fácil
</div>

- Recuperação de erros mais ágil em lotes grandes
- Agrupamento de erros aprimorado
- Contexto de erros mais útil

Mesma API do v1, com relatórios de erros muito melhores para os recursos do v2 em lotes.
