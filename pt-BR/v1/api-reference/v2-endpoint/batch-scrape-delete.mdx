<div id="whats-new-in-v2">
  ## O que há de novo na v2
</div>

A API de cancelamento de lote de scrape da v2 mantém a mesma funcionalidade da v1, com confiabilidade aprimorada:

<div id="enhanced-cancellation-process">
  ### Processo de cancelamento aprimorado
</div>

- Limpeza e gerenciamento de recursos otimizados para operações em lote
- Melhor tratamento da interrupção de lotes ativos
- Tratamento de erros aprimorado para operações de cancelamento em lote

Os parâmetros da API e o formato de resposta permanecem inalterados em relação à v1, garantindo uma migração tranquila e um cancelamento em lote mais confiável.
