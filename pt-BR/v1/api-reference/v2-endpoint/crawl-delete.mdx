<div id="whats-new-in-v2">
  ## Novidades na v2
</div>

A API de cancelamento de crawl da v2 mantém a mesma funcionalidade da v1, com confiabilidade aprimorada:

<div id="enhanced-cancellation-process">
  ### Processo de cancelamento aprimorado
</div>

- Limpeza e gerenciamento de recursos melhorados
- Melhor tratamento do encerramento de crawls ativos
- Manipulação de erros aprimorada para operações de cancelamento

Os parâmetros da API e o formato de resposta permanecem inalterados em relação ao v1, garantindo uma migração tranquila e um cancelamento de crawl mais confiável.
