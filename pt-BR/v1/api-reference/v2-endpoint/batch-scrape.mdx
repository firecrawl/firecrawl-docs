<div id="whats-new-in-v2">
  ## Novidades na v2
</div>

<div id="all-new-formats-work-in-batches">
  ### Todos os novos formatos funcionam em lotes
</div>

- Use capturas de tela personalizadas, extração em modo JSON e rastreioDeMudanças em todas as URLs
- Processe PDFs em lotes com o novo sistema de análise
- Aplique as mesmas configurações de formato a todas as URLs do lote

<div id="better-batch-processing">
  ### Melhor processamento em lote
</div>

- Raspe a partir de diferentes países/idiomas para lotes inteiros
- Use os modos de proxy basic, stealth ou auto em todas as URLs
- Todas as novas opções de raspagem funcionam no modo em lote

<div id="faster-and-more-reliable">
  ### Mais rápido e mais confiável
</div>

- Melhor gerenciamento de recursos para lotes complexos
- Acompanhamento de progresso aprimorado
- Tratamento de erros aprimorado

Mesma API do v1; todos os recursos do v2 funcionam em todo o lote.
