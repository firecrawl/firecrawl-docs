<div id="whats-new-in-v2">
  ## Novidades na v2
</div>

<div id="better-error-messages">
  ### Mensagens de erro aprimoradas
</div>

- Mensagens claras quando prompts em linguagem natural não funcionam
- Mensagens específicas para problemas de sitemap e limite de profundidade
- Mensagens mais úteis para problemas de rastreamento em todo o domínio

<div id="more-error-details">
  ### Mais detalhes sobre erros
</div>

- Erros detalhados para problemas no processamento de formatos
- Mensagens específicas para falhas em PDFs e no parser
- Depuração aprimorada para crawling baseado em localização
- Erros claros para problemas de rastreio de mudanças

<div id="faster-and-easier-debugging">
  ### Depuração mais rápida e simples
</div>

- Obtenção de erros mais ágil em crawls grandes
- Agrupamento de erros aprimorado
- Contexto de erros mais útil

Mesma API do v1, mas com relatórios de erro muito melhores para os recursos do v2.
