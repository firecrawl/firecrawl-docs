<div id="whats-new-in-v2">
  ## Novidades da v2
</div>

<div id="tell-the-crawler-what-you-want">
  ### Diga ao crawler o que você quer
</div>

Descreva o que rastrear em inglês simples:

```json
{
  "url": "https://example.com",
  "prompt": "Rasteie apenas posts do blog e documentação; ignore páginas de marketing"
}
```

<div id="better-sitemap-control">
  ### Melhor controle de sitemap
</div>

v1: opção ligar/desligar para sitemap. v2: três opções:

- `"include"` - Usar o sitemap e encontrar outras páginas (padrão)
- `"skip"` - Ignorar completamente o sitemap  
- `"only"` - Rastrear apenas URLs do sitemap

<div id="new-crawling-options">
  ### Novas opções de rastreamento
</div>

* `crawlEntireDomain` - Rastreia todo o domínio, não apenas páginas filhas
* `maxDiscoveryDepth` - Controla a profundidade do rastreamento (substitui `maxDepth`)

```json
{
  "url": "https://example.com/features",
  "crawlEntireDomain": true,
  "maxDiscoveryDepth": 2,
  "sitemap": "include"
}
```
