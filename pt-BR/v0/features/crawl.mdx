O Firecrawl rastreia sites de forma minuciosa, garantindo extração de dados abrangente enquanto contorna mecanismos de bloqueio. Veja como funciona:

1. **Análise da URL:**
   Começa com uma URL especificada, identifica links consultando o sitemap e, em seguida, rastreia o site. Se nenhum sitemap for encontrado, rastreia o site seguindo os links.

2. **Percurso recursivo:**
   Segue recursivamente cada link para descobrir todas as subpáginas.

3. **Extração de conteúdo:**
   Coleta conteúdo de cada página visitada, lidando com complexidades como renderização em JavaScript e limites de taxa.

4. **Compilação de resultados:**
   Converte os dados coletados em markdown limpo ou em uma saída estruturada, ideal para processamento por LLMs ou outras tarefas.

Esse método garante um rastreamento e uma coleta de dados exaustivos a partir de qualquer URL inicial.

<div id="crawling">
  ## Rastreamento
</div>

<div id="crawl-endpoint">
  ### endpoint /crawl
</div>

Usado para rastrear uma URL e todas as subpáginas acessíveis. Isso envia uma tarefa de rastreamento e retorna um ID de tarefa para verificar o status do rastreamento.

<div id="usage">
  ### Uso
</div>

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

crawl_result = app.crawl_url('mendable.ai', {'crawlerOptions': {'excludes': ['blog/*']}})

# Obter o markdown
for result in crawl_result:
    print(result['markdown'])
```

```js JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

// Inicialize o FirecrawlApp com sua chave de API
const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

// Faça o crawl de um site
const crawlResult = await app.crawlUrl('mendable.ai', { crawlerOptions: { excludes: ['blog/*'] } });

// Registre o markdown no console
console.log(crawlResult.map(result => result.markdown));
```

```go Go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
    log.Fatalf("Falha ao inicializar o FirecrawlApp: %v", err)
  }

  crawlResult, err := app.CrawlURL("mendable.ai", nil)
  if err != nil {
    log.Fatalf("Falha ao executar o crawl da URL: %v", err)
  }

  fmt.Println(crawlResult)
}
```

```rust Rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
  // Inicialize o FirecrawlApp com a chave de API
  let api_key = "YOUR_API_KEY";
  let api_url = "https://api.firecrawl.dev";
  let app = FirecrawlApp::new(api_key, api_url).expect("Falha ao inicializar o FirecrawlApp");

  // Execute o crawl da URL
  let crawl_params = json!({
    "crawlerOptions": {
        "excludes": ["blog/*"]
    }
  });

  let crawl_result = app
      .crawl_url("https://example.com", Some(crawl_params), true, 2, None)
      .await;

  // Imprima o resultado do crawl
  match crawl_result {
      Ok(data) => println!("Resultado do crawl:\n{}", data),
      Err(e) => eprintln!("Falha no crawl: {}", e),
  }
}
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v0/crawl \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "url": "https://mendable.ai"
    }'
```

<div id="job-id-response">
  ### Resposta de Job ID
</div>

Se você não estiver usando o SDK ou preferir usar webhook ou outro método de polling, defina `wait_until_done` como `false`.
Isso retornará um jobId.

No cURL, o endpoint /crawl sempre retornará um jobId que você pode usar para verificar o status do crawl.

```json
{ "jobId": "1234-5678-9101" }
```


<div id="check-crawl-job">
  ### Verificar tarefa de crawl
</div>

Usado para checar o status de uma tarefa de crawl e obter seu resultado.

<CodeGroup>

```python Python
status = app.check_crawl_status(job_id)
```

```js JavaScript
const status = await app.checkCrawlStatus(jobId);
```

```go Go
status, err := app.CheckCrawlStatus(jobId)
if err != nil {
  log.Fatalf("Failed to check crawl status: %v", err)
}
```

```rust Rust
let status = match app.check_crawl_status(jobId).await {
    Ok(status) => status,
    Err(e) => panic!("Failed to check crawl status: {:?}", e),
};

println!("Crawl Status: {:?}", status);
```

```bash cURL
curl -X GET https://api.firecrawl.dev/v0/crawl/status/1234-5678-9101 \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```
