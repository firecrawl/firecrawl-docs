O endpoint de busca combina uma API de pesquisa com o poder do Firecrawl para oferecer uma experiência de busca robusta para qualquer consulta.

Ele pesquisa automaticamente na web pela consulta e retorna os resultados mais relevantes das páginas principais em formato markdown. A vantagem deste endpoint é que ele de fato faz scraping de cada site no topo dos resultados, garantindo que você sempre receba o conteúdo completo.

Este endpoint está em beta e pode sofrer alterações.
