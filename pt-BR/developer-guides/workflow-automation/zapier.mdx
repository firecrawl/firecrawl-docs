<div id="firecrawl-actions">
  ## Ações do Firecrawl
</div>

| Ação | Caso de uso |
|--------|----------|
| **Raspar URL** | Captura rápida de dados de uma única página |
| **Rastrear site** | Raspagem completa do site em várias páginas |
| **Extrair dados estruturados** | Extração com IA usando esquemas personalizados |
| **Pesquisar na web** | Automação de pesquisa com busca + raspagem |
| **Mapear site** | Análise de SEO e mapeamento da estrutura do site |

<div id="quick-reference">
  ## Referência rápida
</div>

<CardGroup cols={2}>
  <Card title="Primeiros passos" icon="play">
    1. Inscreva-se em [firecrawl.dev](https://firecrawl.dev)
    2. Obtenha sua chave de API
    3. Crie um Zap no Zapier
    4. Conecte o Firecrawl com sua chave de API
    5. Escolha seu fluxo de trabalho e ative
  </Card>

  <Card title="Boas práticas" icon="lightbulb">
    - Use `/Scrape URL` para páginas individuais (mais rápido)
    - Programe estrategicamente (de hora em hora/diariamente/semanalmente)
    - Teste primeiro no playground do Firecrawl
    - Adicione tratamento de erros para coletas com falha
    - Use filtros para evitar execuções desnecessárias
  </Card>
</CardGroup>

<div id="zapier-vs-n8n">
  ## Zapier vs n8n
</div>

| Recurso | Zapier | n8n |
|---------|--------|-----|
| **Configuração** | No-code, na nuvem | Autogerenciado ou na nuvem |
| **Preço** | Por tarefa | Mensal fixo |
| **Integrações** | 8.000+ apps | 400+ integrações |
| **Melhor para** | Automação rápida, usuários não técnicos | Lógica personalizada, desenvolvedores |
