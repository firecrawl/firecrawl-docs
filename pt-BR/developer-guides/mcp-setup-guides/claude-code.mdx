Adicione recursos de scraping e busca na web ao Claude Code com o Firecrawl MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Inscreva-se em [firecrawl.dev/app](https://firecrawl.dev/app) e copie sua chave de API.

<div id="2-add-firecrawl-mcp-server">
  ### 2. Adicionar o Firecrawl MCP Server
</div>

```bash
claude mcp add firecrawl -e FIRECRAWL_API_KEY=sua-chave-de-api -- npx -y firecrawl-mcp
```

Substitua `your-api-key` pela sua chave de API do Firecrawl.

Pronto! Agora você pode pesquisar e extrair dados da web pelo Claude Code.

<div id="quick-demo">
  ## Demonstração rápida
</div>

Experimente o seguinte no Claude Code:

**Buscar na web:**

```
Search for the latest Next.js 15 features
```

**Fazer scraping de uma página:**

```
Scrape firecrawl.dev and tell me what it does
```

**Acessar a documentação:**

```
Encontre e extraia a documentação da API do Stripe sobre payment intents
```

Claude usará automaticamente as ferramentas de busca e scraping do Firecrawl para obter as informações.
