</Warning>

Adicione recursos de scraping e busca na web ao ChatGPT com o Firecrawl MCP.

<div id="quick-setup">
  ## Configuração rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenha sua chave de API
</div>

Crie uma conta em [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys) e copie sua chave de API.

<div id="2-enable-developer-mode">
  ### 2. Ative o Modo Desenvolvedor
</div>

Abra as configurações do ChatGPT clicando no seu nome de usuário no canto inferior esquerdo ou acesse diretamente [chatgpt.com/#settings](https://chatgpt.com/#settings).

Na janela de configurações, role até a parte inferior e selecione **Advanced Settings**. Ative o **Developer mode**.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/enable-developer-mode.gif" alt="Configurações do ChatGPT mostrando Advanced Settings com a opção Developer mode ativada" />
</Frame>

<div id="3-create-the-connector">
  ### 3. Criar o Conector
</div>

Com o modo Developer habilitado, vá até a aba **Apps & Connectors** no mesmo modal de configurações.

Clique no botão **Create** no canto superior direito.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/apps-connectors-create.png" alt="Aba Apps & Connectors com o botão Create em destaque" />
</Frame>

Preencha os detalhes do conector:

- **Name:** `Firecrawl MCP`
- **Description:** `Web scraping, crawling, search, and content extraction` (opcional)
- **MCP Server URL:** `https://mcp.firecrawl.dev/YOUR_API_KEY_HERE/v2/mcp`
- **Authentication:** `None`

Substitua `YOUR_API_KEY_HERE` na URL pela sua [chave de API Firecrawl](https://www.firecrawl.dev/app/api-keys) real.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/connector-form-filled.png" alt="Formulário New Connector preenchido com os detalhes do Firecrawl MCP" />
</Frame>

Marque a caixa de seleção **"I understand and want to continue"** e, em seguida, clique em **Create**.

<div id="4-verify-setup">
  ### 4. Verificar a configuração
</div>

Volte para a interface principal do ChatGPT. Você deve ver **Modo Desenvolvedor** exibido, indicando que os conectores MCP estão ativos.

Se você não vir o Modo Desenvolvedor, recarregue a página. Se ainda assim ele não aparecer, abra as configurações novamente e verifique se o Modo Desenvolvedor está ativado em Configurações Avançadas.

<div id="5-access-firecrawl-tools">
  ### 5. Acesse as ferramentas do Firecrawl
</div>

Para usar o Firecrawl em uma conversa, clique no botão **+** no campo de entrada do chat, depois selecione **More** e escolha **Firecrawl MCP**.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/select-firecrawl-mcp.png" alt="Campo de entrada de chat do ChatGPT mostrando o menu + expandido com o submenu More e a opção Firecrawl MCP" />
</Frame>

<div id="quick-demo">
  ## Demonstração rápida
</div>

Com o Firecrawl MCP selecionado, experimente estes prompts:

**Busca:**

```
Search for the latest React Server Components updates
```

**Scrape:**

```
Scrape firecrawl.dev and tell me what it does
```

**Acessar documentação:**

```
Extraia a documentação da Vercel sobre edge functions e resuma-a
```


<div id="tool-confirmation">
  ## Confirmação da ferramenta
</div>

Quando o ChatGPT utiliza as ferramentas MCP do Firecrawl, você verá um prompt de confirmação pedindo a sua aprovação.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/tool-confirmation.png" alt="Janela de confirmação de ferramenta do ChatGPT exibindo uma solicitação do Firecrawl MCP" />
</Frame>

Você pode marcar **"Lembrar desta conversa"** para evitar confirmações repetidas durante a mesma sessão de chat. Essa medida de segurança é implementada pela OpenAI para garantir que as ferramentas MCP não executem ações não intencionais.

Depois de confirmar, o ChatGPT executará a solicitação e retornará os resultados.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/search-results-example.png" alt="Exemplo de resultados de pesquisa do Firecrawl exibidos no ChatGPT" />
</Frame>
