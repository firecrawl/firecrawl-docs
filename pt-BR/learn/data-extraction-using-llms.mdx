<div id="load-website-with-firecrawl">
  ## Carregar site com o Firecrawl
</div>

Para obter todos os dados de uma página e garantir o formato mais limpo possível, usaremos o [Firecrawl](https://firecrawl.dev). Ele contorna sites bloqueados por JS, extrai o conteúdo principal e gera uma saída em um formato legível por LLM para maior precisão.

Veja como faremos o scrape de uma URL usando o Firecrawl. Também definiremos `pageOptions` para extrair apenas o conteúdo principal (`onlyMainContent: True`) da página — excluindo navegações, rodapés etc.

```python
from firecrawl import FirecrawlApp  # Importando o FirecrawlApp

url = "https://about.fb.com/news/2024/04/introducing-our-open-mixed-reality-ecosystem/"

firecrawl = FirecrawlApp(
    api_key="fc-YOUR_FIRECRAWL_API_KEY",
)
page_content = firecrawl.scrape_url(url=url,  # URL de destino para rastrear
    params={
        "pageOptions":{
            "onlyMainContent": True # Ignorar navegação, rodapés etc.
        }
    })
print(page_content)
```

Perfeito, agora temos dados limpos do site — prontos para serem enviados ao LLM para extração de dados.

<div id="extraction-and-generation">
  ## Extração e Geração
</div>

Agora que temos os dados do site, vamos usar o Groq para extrair as informações necessárias. Usaremos o modelo Llama 3 do Groq no modo JSON e selecionaremos campos específicos do conteúdo da página.

Estamos usando o modelo Llama 3 8B neste exemplo. Sinta-se à vontade para usar modelos maiores para obter resultados melhores.

```python
import json
from groq import Groq

client = Groq(
    api_key="gsk_YOUR_GROQ_API_KEY",  # Observação: substitua 'API_KEY' pela sua chave de API do Groq
)

# Aqui definimos os campos que queremos extrair do conteúdo da página
extract = ["summary","date","companies_building_with_quest","title_of_the_article","people_testimonials"]

completion = client.chat.completions.create(
    model="llama3-8b-8192",
    messages=[
        {
            "role": "system",
            "content": "Você é um(a) assessor(a) jurídico(a) que extrai informações de documentos em JSON."
        },
        {
            "role": "user",
            # Aqui passamos o conteúdo da página e os campos que queremos extrair
            "content": f"Extraia as seguintes informações da documentação fornecida:\nConteúdo da página:\n\n{page_content}\n\nInformações a serem extraídas: {extract}"
        }
    ],
    temperature=0,
    max_tokens=1024,
    top_p=1,
    stream=False,
    stop=None,
    # Definimos o formato da resposta como objeto JSON
    response_format={"type": "json_object"}
)


# Exibe a resposta JSON de forma legível
dataExtracted = json.dumps(str(completion.choices[0].message.content), indent=4)

print(dataExtracted)
```
