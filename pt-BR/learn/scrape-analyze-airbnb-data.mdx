> Observação: este exemplo usa a [versão v0 da API do Firecrawl](/pt-BR/v0/introduction). Você pode instalar a versão 0.0.20 para o SDK de Python ou 0.0.36 para o SDK de Node.

> [Repositório no GitHub (https://github.com/e2b-dev/e2b-cookbook) ](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)

Este exemplo demonstra como raspar dados do Airbnb e analisá-los usando o [Firecrawl](https://www.firecrawl.dev/) e o [Code Interpreter SDK](https://github.com/e2b-dev/code-interpreter) da E2B.

Não escreveremos nenhum código de análise; em vez disso, deixaremos a IA da Anthropic escrevê-lo para nós. Usaremos o Code Interpreter SDK da E2B para executar o código com segurança e obter os resultados.

<div id="prerequisites">
  ## Pré-requisitos
</div>

* Node.js instalado na sua máquina
* Uma chave de API da E2B
* Uma chave de API do Firecrawl
* Uma chave de API da Anthropic

<div id="setup-run">
  ## Configuração e execução
</div>

<div id="1-clone-the-repository">
  ### 1. Clone o repositório
</div>

Aqui está o [repositório do GitHub](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl) deste exemplo.

```
git clone https://github.com/e2b-dev/e2b-cookbook.git
cd e2b-cookbook/examples/scrape-and-analyze-airbnb-data-with-firecrawl
```

<div id="2-install-dependencies">
  ### 2. Instale as dependências
</div>

```
npm install
```

<div id="3-set-up-env">
  ### 3. Configure o `.env`
</div>

1. Copie `.env.template` para `.env`
2. Obtenha a [chave de API da E2B](https://e2b.dev/docs/getting-started/api-key)
3. Obtenha a [chave de API do Firecrawl](https://firecrawl.dev)
4. Obtenha a [chave de API da Anthropic](https://anthropic.com)

<div id="4-run-the-example">
  ### 4. Execute o exemplo
</div>

```
npm run start
```
