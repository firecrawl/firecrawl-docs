> Observação: este exemplo usa a [versão v0 da Firecrawl API](/pt-BR/v0/introduction). Você pode instalar a versão 0.0.20 do SDK de Python ou a 0.0.36 do SDK de Node.

Com o Firecrawl, você pode enviar cookies nos cabeçalhos para contornar a autenticação em um site. Isso pode ser útil quando você precisa extrair conteúdo de um site que exige que você esteja conectado.

<div id="example">
  ### Exemplo
</div>

Para ignorar a autenticação, inclua o parâmetro `headers` na sua requisição e defina o cabeçalho `Cookies` com os valores apropriados, que você pode copiar dos cookies do seu navegador. Se precisar de ajuda para obter os cookies, veja este artigo: [Como capturar cookies](https://cookie-script.com/documentation/how-to-check-cookies-on-chrome-and-firefox)

<div id="usage">
  ### Uso
</div>

<CodeGroup>
  ```python Python
  from firecrawl import FirecrawlApp

  app = FirecrawlApp(api_key="YOUR_API_KEY")

  app.scrape_url("https://docs.firecrawl.dev", {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  })
  ```

  ```javascript JavaScript
  import { FirecrawlApp } from 'firecrawl-js';

  const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

  app.scrapeUrl('https://docs.firecrawl.dev', {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  });
  ```

  ```go Go
  import (
    "fmt"
    "log"

    "github.com/mendableai/firecrawl-go"
  )

  func main() {
    // Inicialize o FirecrawlApp com sua chave de API
    app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
    if err != nil {
      log.Fatalf("Falha ao inicializar o FirecrawlApp: %v", err)
    }

    params := map[string]any{
      "pageOptions": map[string]any{
        "headers": map[string]any{
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      },
    }
    scrapedData, err := app.ScrapeURL("docs.firecrawl.dev", params)
    if err != nil {
      log.Fatalf("Erro ao fazer scraping: %v", err)
    }
    fmt.Println(scrapedData)
  }
  ```

  ```rust Rust
  use firecrawl::FirecrawlApp;

  #[tokio::main]
  async fn main() {
      let api_key = "YOUR_API_KEY";
      let api_url = "https://api.firecrawl.dev";
      let app = FirecrawlApp::new(api_key, api_url).expect("Falha ao inicializar o FirecrawlApp");

      let scrape_params = json!({
          "pageOptions": {
              "headers": {
                  "Cookie": "cookie1=value1; cookie2=value2"
              }
          }
      });

      let scrape_result = app.scrape_url("https://docs.firecrawl.dev", Some(scrape_params)).await;
      match scrape_result {
          Ok(data) => println!("Resultado do scraping:\n{}", data),
          Err(e) => eprintln!("Falha no scraping: {}", e),
      }
  }
  ```

  ```curl cURL
  curl -X POST https://api.firecrawl.dev/v0/scrape \

      -d '{
        "url": "https://example.com",
        "headers": {
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      }'
  ```
