<div id="inline-code">
  ### Código inline
</div>

Para indicar uma `palavra` ou `frase` como código, envolva-a com crases (`).

```
Para indicar uma `palavra` ou `frase` como código, coloque-a entre crases (`).
```


<div id="code-block">
  ### Bloco de código
</div>

Use [blocos de código com cercas](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks), envolvendo o código com três crases, e indique a linguagem de programação do trecho logo após as crases iniciais para obter realce de sintaxe. Opcionalmente, você também pode escrever o nome do seu código após a linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Olá, mundo!");
    }
}
```
