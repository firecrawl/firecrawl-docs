  ## Básico

<div id="inline-code">
  ### Código inline
</div>

Para indicar uma `palavra` ou `frase` como código, envolva-a com crases (`).

```
Para indicar uma `palavra` ou `frase` como código, coloque-a entre crases (`).
```

<div id="code-block">
  ### Bloco de código
</div>

Use [blocos de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) envolvendo o código em três crases e, em seguida, indicando a linguagem de programação do seu trecho logo após as crases iniciais para obter realce de sintaxe. Opcionalmente, você também pode escrever o nome do seu trecho de código depois da linguagem de programação.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
