<Info>
  **Pré-requisito** Você deve ter o Node.js (versão 18.10.0 ou
  superior) instalado.
</Info>

Passo 1. Instale o Mintlify no seu sistema operacional:

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

Passo 2. Vá até a pasta onde a documentação está (onde você encontra o `mint.json`) e execute o seguinte comando:

```bash
mintlify dev
```

O site da documentação agora está disponível em `http://localhost:3000`.

<div id="custom-ports">
  ### Portas personalizadas
</div>

O Mintlify usa a porta 3000 por padrão. Você pode usar a opção `--port` para personalizar a porta em que o Mintlify será executado. Por exemplo, use este comando para rodar na porta 3333:

```bash
mintlify dev --port 3333
```

Você verá um erro como este se tentar executar o Mintlify em uma porta que já está ocupada:

```md
Erro: listen EADDRINUSE: endereço já está em uso :::3000
```

<div id="mintlify-versions">
  ## Versões do Mintlify
</div>

Cada CLI está vinculada a uma versão específica do Mintlify. Atualize a CLI se o seu site local estiver diferente do ambiente de produção.

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

<div id="troubleshooting">
  ## Solução de problemas
</div>

Veja como resolver alguns problemas comuns ao trabalhar com a CLI.

<AccordionGroup>
  <Accordion title="Mintlify não está carregando">
    Atualize para o Node v18. Execute `mintlify install` e tente novamente.
  </Accordion>

  <Accordion title="Nenhum arquivo ou diretório no Windows">
    Vá até o diretório `C:/Users/Username/.mintlify/` e remova a pasta `mint`.
    Depois, abra o Git Bash nesse local e execute `git clone
    https://github.com/mintlify/mint.git`.

    Repita o passo 3.
  </Accordion>

  <Accordion title="Recebendo um erro desconhecido">
    Tente ir até a raiz do seu dispositivo e excluir a pasta ~/.mintlify.
    Em seguida, execute `mintlify dev` novamente.
  </Accordion>
</AccordionGroup>
