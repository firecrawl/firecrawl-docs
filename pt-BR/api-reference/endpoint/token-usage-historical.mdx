title: "Uso Histórico de Tokens"
openapi: '/pt-BR/api-reference/v2-openapi.json GET /team/token-usage/historical'
---

Retorna o uso histórico de tokens mês a mês. O endpoint também pode, opcionalmente, detalhar o uso por chave de API.

<Info>Simplificamos a cobrança para que o Extract agora use créditos, assim como todos os outros endpoints. Cada crédito equivale a 15 tokens. O uso de tokens informado agora inclui o consumo de todos os endpoints.</Info>
