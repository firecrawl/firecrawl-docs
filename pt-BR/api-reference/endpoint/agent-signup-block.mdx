---
title: "Bloquear cadastro de agente"
description: "Bloqueie e desative uma API key provisionada por um agente."
---

Bloqueia um cadastro de agente pendente, desativando imediatamente a API key e banindo a conta em sandbox. Use isto se o agente criou uma chave sem autorização. Nenhuma autenticação é necessária — o token de verificação serve como comprovação.

<div id="headers">
  ## Cabeçalhos
</div>

| Cabeçalho      | Valor              |
| -------------- | ------------------ |
| `Content-Type` | `application/json` |

<div id="request-body">
  ## Corpo da requisição
</div>

| Parâmetro            | Tipo   | Obrigatório | Descrição                                                         |
| -------------------- | ------ | ----------- | ----------------------------------------------------------------- |
| `agent_signup_token` | string | **Sim**     | O token de verificação do link de confirmação enviado por e-mail. |

<div id="response">
  ## Resposta
</div>

| Campo | Tipo | Descrição |
|-------|------|-------------|
| `success` | boolean | Indica se o bloqueio foi bem-sucedido |
| `message` | string | Mensagem de resultado legível por humanos |

<div id="example-request">
  ### Exemplo de requisição
</div>

```bash
curl -X POST "https://api.firecrawl.dev/v2/agent-signup/block" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_signup_token": "a1b2c3d4..."
  }'
```


<div id="example-response">
  ### Exemplo de resposta
</div>

```json
{
  "success": true,
  "message": "Agent key has been blocked and disabled."
}
```


<div id="what-happens-on-block">
  ## O que acontece em caso de bloqueio
</div>

* A API key do agente é **excluída**.
* A equipe em sandbox é **banida**.
* O registro do patrocinador é marcado como `"blocked"`, impedindo novos cadastros de agentes para esse email.
* Outras requisições de API feitas com a chave bloqueada retornarão `403`.

<div id="error-responses">
  ## Respostas de erro
</div>

| Status | Erro                                   | Quando                                                  |
| ------ | -------------------------------------- | ------------------------------------------------------- |
| `400`  | Requisição inválida                    | `agent_signup_token` ausente                            |
| `404`  | Token de verificação inválido          | Token não encontrado                                    |
| `409`  | Esta chave do agente já foi confirmada | Não é possível bloquear uma chave que já foi confirmada |
