---
title: "Inspecionar ou ler artigo"
openapi: "/pt-BR/api-reference/v2-openapi.json GET /search/research/papers/{id}"
---

Inspecione os metadados do artigo pelo ID ou adicione um parâmetro `query` para ler os trechos mais relevantes do texto completo para responder a uma pergunta.

Os IDs aceitos incluem valores canônicos de `paperId` e valores de `primaryId` específicos da fonte (por exemplo, `arxiv:1706.03762`).

Para uma visão geral do fluxo de trabalho, consulte o [guia do Research Index](/pt-BR/features/research).
