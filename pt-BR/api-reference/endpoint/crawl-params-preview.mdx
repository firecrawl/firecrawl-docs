<div id="whats-new-in-v2">
  ## Novidades da v2
</div>

<div id="crawl-params-preview">
  ### Prévia dos parâmetros de crawl
</div>

```json
{
  "url": "https://example.com",
  "prompt": "Rastreie todo o site, mas exclua /admin e /api"
}   
```


<div id="response">
  ### Resposta
</div>

```json
{
  "url": "https://example.com",
  "prompt": "Rastreie todo o site",
  "crawlEntireDomain": true,
  "excludePaths": ["/admin/.*", "/api/.*"],
  "sitemap": "incluir"
}
```
