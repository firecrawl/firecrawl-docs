Converta qualquer documento local ou acessível publicamente em dados limpos para agentes de IA via API.

* Transforme arquivos PDF, DOCX e XLSX em Markdown ou JSON
* Preserva a ordem de leitura e as tabelas
* Suporte a Zero Data Retention

<div id="how-to-use-document-parsing">
  ## Como usar a análise de documentos
</div>

A análise de documentos no Firecrawl funciona de duas formas:

1. **Análise baseada em URL (`/v2/scrape`)**: forneça uma URL que aponte para um tipo de documento compatível. Confira o endpoint [Scrape](/pt-BR/api-reference/endpoint/scrape) para mais detalhes.
2. **Análise por upload de arquivo (`/v2/parse`)**: envie os bytes do arquivo diretamente com `multipart/form-data`.

Na análise baseada em URL, o Firecrawl detecta automaticamente o tipo de arquivo pela extensão ou pelo tipo de conteúdo.
