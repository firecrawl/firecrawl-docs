---
title: "Pesquisar histórico do GitHub"
openapi: "/pt-BR/api-reference/v2-openapi.json GET /search/research/github"
---

Pesquise o histórico de issues/PRs, discussões e READMEs de repositórios no GitHub. Os resultados incluem metadados do repositório, URLs, trechos e conteúdo em Markdown correspondente, quando disponível.

Para uma visão geral do fluxo de trabalho, consulte o [guia do Research Index](/pt-BR/features/research).
