---
title: "Cadastro de agente"
description: "Crie uma chave de API do Firecrawl em sandbox em nome de um patrocinador humano."
---

Provisiona imediatamente uma nova chave de API, em sandbox, com 50 créditos. Um email de verificação é enviado ao patrocinador. Não requer autenticação.

Para uma visão geral completa do fluxo, consulte o [guia de cadastro de agente](/pt-BR/features/agent-signup).

<div id="headers">
  ## Cabeçalhos
</div>

| Cabeçalho      | Valor              |
| -------------- | ------------------ |
| `Content-Type` | `application/json` |

<div id="request-body">
  ## Corpo da requisição
</div>

| Parâmetro      | Tipo    | Obrigatório | Descrição                                                                                            |
| -------------- | ------- | ----------- | ---------------------------------------------------------------------------------------------------- |
| `email`        | string  | **Sim**     | Endereço de email do patrocinador humano. Deve ser um email válido.                                  |
| `agent_name`   | string  | **Sim**     | Nome do agente que cria a chave (1–100 caracteres). Exibido no email de verificação.                 |
| `accept_terms` | boolean | **Sim**     | Deve ser `true`. Indica a aceitação dos [Termos de Serviço](https://firecrawl.dev/terms-of-service). |

<div id="response">
  ## Resposta
</div>

| Campo                      | Tipo    | Descrição                                                                             |
| -------------------------- | ------- | ------------------------------------------------------------------------------------- |
| `success`                  | boolean | Indica se a chave foi criada                                                          |
| `api_key`                  | string  | A API key provisionada (começa com `fc-`)                                             |
| `sponsor_status`           | string  | Sempre `"pending"` na criação                                                         |
| `credit_limit`             | number  | Número máximo de créditos disponíveis antes da verificação (50)                       |
| `credits_remaining`        | number  | Créditos restantes (50 na criação)                                                    |
| `verification_deadline_at` | string  | Timestamp no formato ISO 8601. O patrocinador deve confirmar até este prazo (5 dias). |
| `tos_url`                  | string  | URL dos Termos de Serviço aceitos                                                     |

<div id="example-request">
  ### Exemplo de requisição
</div>

```bash
curl -X POST "https://api.firecrawl.dev/v2/agent-signup" \
  -H "Content-Type: application/json" \
  -d '{
    "email": "user@company.com",
    "agent_name": "Research Assistant",
    "accept_terms": true
  }'
```

<div id="example-response">
  ### Exemplo de resposta
</div>

```json
{
  "success": true,
  "api_key": "fc-abc123...",
  "sponsor_status": "pending",
  "credit_limit": 50,
  "credits_remaining": 50,
  "verification_deadline_at": "2026-03-23T12:00:00.000Z",
  "tos_url": "https://firecrawl.dev/terms-of-service"
}
```

<div id="error-responses">
  ## Respostas de erro
</div>

| Status | Erro                                                          | Quando                                                        |
| ------ | ------------------------------------------------------------- | ------------------------------------------------------------- |
| `400`  | Corpo da requisição inválido                                  | Campos ausentes ou inválidos                                  |
| `403`  | Este email bloqueou cadastros de agente                       | O patrocinador bloqueou anteriormente os cadastros de agente  |
| `403`  | A verificação de um cadastro anterior de agente expirou       | Um cadastro anterior expirou; o patrocinador deve fazer login |
| `409`  | Uma confirmação pendente de cadastro de agente já foi enviada | Já existe uma chave pendente para este email                  |
