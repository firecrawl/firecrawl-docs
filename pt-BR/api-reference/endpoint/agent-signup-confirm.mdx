---
title: "Confirmar cadastro de agente"
description: "Verifique e vincule uma chave de API provisionada por um agente à conta do patrocinador."
---

Confirma um cadastro de agente pendente. Se o patrocinador já tiver uma conta Firecrawl, a chave será mesclada a ela. Caso contrário, a conta em sandbox será promovida para uma conta completa no email do patrocinador. Não é necessária autenticação — o token de verificação serve como comprovante.

<div id="headers">
  ## Cabeçalhos
</div>

| Cabeçalho      | Valor              |
| -------------- | ------------------ |
| `Content-Type` | `application/json` |

<div id="request-body">
  ## Corpo da requisição
</div>

| Parâmetro | Tipo | Obrigatório | Descrição |
|-----------|------|-------------|-----------|
| `agent_signup_token` | string | **Sim** | O token de verificação do link de confirmação do email. |

<div id="response">
  ## Resposta
</div>

| Campo     | Tipo    | Descrição                                        |
| --------- | ------- | ------------------------------------------------ |
| `success` | boolean | Indica se a confirmação foi bem-sucedida         |
| `message` | string  | Mensagem de resultado compreensível para humanos |

<div id="example-request">
  ### Exemplo de requisição
</div>

```bash
curl -X POST "https://api.firecrawl.dev/v2/agent-signup/confirm" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_signup_token": "a1b2c3d4..."
  }'
```

<div id="example-response">
  ### Exemplo de Resposta
</div>

```json
{
  "success": true,
  "message": "Agent key confirmed and linked to your account."
}
```

<div id="what-happens-on-confirm">
  ## O que acontece ao confirmar
</div>

* **Existing account**: A API key do agente é movida para a equipe do patrocinador. A account em sandbox é desativada.
* **No account**: O email da account em sandbox é atualizado para o email do patrocinador, tornando-se a account real dele.
* Em ambos os casos, o limite de 50 créditos é removido, e a chave herda o plan completo do patrocinador.

<div id="error-responses">
  ## Respostas de erro
</div>

| Status | Erro                                      | Quando                                     |
| ------ | ----------------------------------------- | ------------------------------------------ |
| `400`  | Requisição inválida                       | `agent_signup_token` ausente               |
| `403`  | O prazo de verificação expirou            | O período de confirmação de 5 dias expirou |
| `403`  | Este cadastro de agente foi bloqueado     | O patrocinador já bloqueou esta chave      |
| `404`  | Token de verificação inválido ou expirado | Token não encontrado ou já utilizado       |
