<div id="base-url">
  ## URL base
</div>

Todas as requisições usam a seguinte URL base:

```bash
https://api.firecrawl.dev 
```

<div id="authentication">
  ## Autenticação
</div>

Para autenticar, inclua o cabeçalho Authorization. Ele deve conter `Bearer fc-123456789`, onde `fc-123456789` é a sua chave de API.

```bash
Authorization: Bearer fc-123456789
```

​

<div id="response-codes">
  ## Códigos de resposta
</div>

O Firecrawl utiliza códigos de status HTTP convencionais para indicar o resultado das suas solicitações.

Em geral, códigos HTTP 2xx indicam sucesso, 4xx representam erros relacionados ao cliente e 5xx sinalizam problemas de infraestrutura.

Status | Descrição
--- | ---
200 | Solicitação bem-sucedida.
400 | Verifique se os parâmetros estão corretos.
401 | A chave da API não foi fornecida.
402 | Pagamento necessário.
404 | O recurso solicitado não foi encontrado.
429 | O limite de requisições foi excedido.
5xx | Indica um erro de servidor no Firecrawl.

Consulte a seção Códigos de erro para uma explicação detalhada de todos os possíveis erros da API.

<div id="firecrawl-error-codes-5xx">
  ### Códigos de erro do Firecrawl (5xx)
</div>

Quando ocorre um erro 5xx, o Firecrawl fornece códigos mais específicos para esclarecer o que deu errado.​

Error Code | Status | Description\
--- | --- | ---\
`SCRAPE_ALL_ENGINES_FAILED` | 500 | Todos os mecanismos de scraping falharam.\
`SCRAPE_SSL_ERROR` | 500 | O certificado SSL da página é inválido. Você pode usar `skipTlsVerification:true` para ignorar essa verificação.\
`SCRAPE_SITE_ERROR` | 500 | Erro irrecuperável no site.\
`SCRAPE_DNS_RESOLUTION_ERROR` | 500 | Falha na resolução de DNS.\
`SCRAPE_ACTION_ERROR` | 500 | Erro ao executar uma ação na página.\
`SCRAPE_PDF_PREFETCH_FAILED` | 500 | Falha ao pré-carregar o PDF.\
`SCRAPE_PDF_INSUFFICIENT_TIME_ERROR` | 500 | Tempo insuficiente para processar o PDF.\
`SCRAPE_PDF_ANTIBOT_ERROR` | 500 | PDF bloqueado por mecanismos antibot.\
`SCRAPE_ZDR_VIOLATION_ERROR` | 500 | Conflito de Zero Data Retention: ocorre quando `zeroDataRetention:true`, mas outra opção (por exemplo, `screenshot`) requer armazenamento temporário.\
`SCRAPE_UNSUPPORTED_FILE_ERROR` | 500 | Tipo de arquivo não compatível.\
`UNKNOWN_ERROR` | 500 | Erro genérico ou inesperado.

<div id="rate-limit">
  ## Limite de taxa
</div>

A API do Firecrawl possui um limite de taxa para garantir a estabilidade e confiabilidade do serviço. O limite é aplicado a todos os endpoints e se baseia no número de requisições feitas em um período específico.

Ao exceder o limite, você receberá o código de status 429.
