<div id="discovery">
  ## Descoberta
</div>

As etapas de descoberta estão na Etapa 1 de [`auth.md`](https://www.firecrawl.dev/auth.md). Se você já tem um link direto para [`auth.md`](https://www.firecrawl.dev/auth.md), acesse-o — você não precisa desta página.

<div id="what-happens-next">
  ## O que acontece a seguir
</div>

Siga as etapas 1–4 em [`auth.md`](https://www.firecrawl.dev/auth.md): descobrir → registrar-se em `POST /agent/auth` → usar a chave de API retornada como token Bearer nas requisições para `https://api.firecrawl.dev`.
