Use um SDK para fazer scraping, rastreamento e extração de dados estruturados de sites sem precisar gerenciar solicitações HTTP brutas. Cada SDK encapsula a API do Firecrawl com helpers idiomáticos para autenticação, consulta e tratamento de erros.
