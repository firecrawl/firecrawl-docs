<div id="firecrawl-nodejs-source-of-truth">
  # Fonte canônica do Firecrawl Node.js
</div>

Este arquivo é a fonte pública canônica para agentes. Ele é gerado a partir do código-fonte do SDK JS do Firecrawl e da especificação OpenAPI v2.



<div id="install">
  ## Instalar
</div>

```bash
npm install @mendable/firecrawl-js
```



<div id="authenticate">
  ## Autenticação
</div>

```ts
import Firecrawl from "@mendable/firecrawl-js";

const client = new Firecrawl({
  apiKey: process.env.FIRECRAWL_API_KEY,
  // apiUrl: "https://api.firecrawl.dev" // substituição opcional para self-hosted
});
```



<div id="when-to-use-what">
  ## Quando usar cada um
</div>

* `search`: comece com uma consulta para encontrar URLs relevantes, incluindo restrições `site:`.
* `scrape`: busque e extraia conteúdo de uma URL conhecida.
* `interact`: execute ações no navegador ou código quando uma página exigir cliques, formulários ou etapas após o scraping.



<div id="search">
  ## Search
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use a busca quando precisar descobrir resultados a partir de uma consulta e quiser que o Firecrawl retorne URLs (e, opcionalmente, realize scraping de cada resultado). Você pode restringir os resultados a um domínio com o operador padrão `site:`, por exemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`client.search(query, options?)`

<div id="simple-example">
  ### Exemplo simples
</div>

```ts
const results = await client.search("site:docs.firecrawl.dev webhook retries")
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```ts
const results = await client.search("site:docs.firecrawl.dev crawl webhooks", {
  sources: ["web", "news"],
  categories: ["research"],
  limit: 10,
  tbs: "qdr:m",
  location: "San Francisco,California,United States",
  ignoreInvalidURLs: true,
  timeout: 60000,
  scrapeOptions: {
    formats: [
      "markdown",
      "links",
      { type: "json", prompt: "Extract title, summary, and key endpoints." }
    ],
    onlyMainContent: true,
    includeTags: ["main", "article"],
    excludeTags: ["nav", "footer"],
    waitFor: 1000,
    actions: [
      { type: "click", selector: "button#accept" },
      { type: "wait", milliseconds: 750 },
      { type: "scrape" }
    ]
  }
})
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro                   | Tipo / Opções   | Descrição                                                               |
| --------------------------- | --------------- | ----------------------------------------------------------------------- | -------- | ---------------- | ---------- | ------------ | ----------------------------- |
| `query`                     | `string`        | Consulta de busca. Use `site:domain` para restringir os resultados a um site. |
| `options.sources`           | `Array<"web"    | "news"                                                                  | "images" | { type: "web"    | "news"     | "images" }>` | Fontes a serem pesquisadas.   |
| `options.categories`        | `Array<"github" | "research"                                                              | "pdf"    | { type: "github" | "research" | "pdf" }>`    | Categorias para filtrar os resultados. |
| `options.limit`             | `number`        | Número máximo de resultados a retornar.                                 |
| `options.tbs`               | `string`        | Filtro de busca baseado em tempo (por exemplo `qdr:d`, `qdr:w`, `sbd:1,qdr:m`). |
| `options.location`          | `string`        | String de localização para os resultados de busca.                      |
| `options.ignoreInvalidURLs` | `boolean`       | Exclui URLs inválidas que outros endpoints do Firecrawl não conseguem processar via scraping. |
| `options.timeout`           | `number`        | Tempo limite em milissegundos.                                          |
| `options.scrapeOptions`     | `ScrapeOptions` | Opções de scraping aplicadas a cada resultado (mesmos campos que `scrape`). |
| `options.integration`       | `string`        | Somente para uso interno. Reservado para instrumentação do Firecrawl.   |
| `options.origin`            | `string`        | Somente para uso interno. Reservado para instrumentação do Firecrawl.   |


<div id="scrape">
  ## Scrape
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use o scraping quando você já tiver uma URL e quiser que o Firecrawl retorne conteúdo estruturado e formatos opcionais.

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`client.scrape(url, options?)`

<div id="simple-example">
  ### Exemplo simples
</div>

```ts
const doc = await client.scrape("https://docs.firecrawl.dev", {
  formats: ["markdown"]
});
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```ts
const doc = await client.scrape("https://example.com/pricing", {
  formats: [
    "markdown",
    "links",
    { type: "json", prompt: "Extract plan names and prices." },
    { type: "screenshot", fullPage: true, quality: 80, viewport: { width: 1280, height: 720 } },
    { type: "changeTracking", modes: ["git-diff"], tag: "pricing" }
  ],
  headers: {
    "User-Agent": "FirecrawlDocsBot/1.0"
  },
  includeTags: ["main", "article"],
  excludeTags: ["nav", "footer"],
  onlyMainContent: true,
  waitFor: 1000,
  mobile: false,
  parsers: [{ type: "pdf", mode: "auto", maxPages: 5 }],
  actions: [
    { type: "click", selector: "#accept" },
    { type: "write", text: "firecrawl" },
    { type: "press", key: "Enter" },
    { type: "wait", milliseconds: 1500 },
    { type: "scrape" }
  ],
  location: { country: "US", languages: ["en-US"] },
  skipTlsVerification: true,
  removeBase64Images: true,
  fastMode: true,
  blockAds: true,
  proxy: "auto",
  maxAge: 86400000,
  storeInCache: true,
  profile: { name: "docs-session", saveChanges: true }
});
```

<div id="interact">
  ## Interagir
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use `interact` quando uma página exigir ações no navegador ou execução de código após o início do scraping (cliques, formulários ou etapas com script).

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`client.interact(jobId, args)`

<div id="simple-example">
  ### Exemplo simples
</div>

```ts
const doc = await client.scrape("https://example.com", { formats: ["markdown"] });
const jobId = doc.metadata?.scrapeId;
if (!jobId) throw new Error("Missing scrapeId from scrape response");

const result = await client.interact(jobId, {
  prompt: "Click the pricing tab and summarize the plans.",
  language: "node"
});
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```ts
const result = await client.interact("<scrapeJobId>", {
  code: "console.log(await page.title());",
  language: "node",
  timeout: 60
});
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro       | Tipo / Opções       | Descrição                                                                                           |                   |                                                                |
| --------------- | ------------------- | --------------------------------------------------------------------------------------------------- | ----------------- | -------------------------------------------------------------- |
| `jobId`         | `string`            | ID do job de scraping em `document.metadata.scrapeId`.                                              |                   |                                                                |
| `args.code`     | `string`            | Código a ser executado na sessão do navegador. Obrigatório se `prompt` não for fornecido.           |                   |                                                                |
| `args.prompt`   | `string`            | Instrução em linguagem natural para o agente do navegador. Obrigatório se `code` não for fornecido. |                   |                                                                |
| `args.language` | `&quot;python&quot; | &quot;node&quot;                                                                                    | &quot;bash&quot;` | Linguagem de execução. O valor padrão é `node` quando omitido. |
| `args.timeout`  | `number`            | Tempo limite de execução em segundos.                                                               |                   |                                                                |
| `args.origin`   | `string`            | Apenas para uso interno. Reservado para a instrumentação do Firecrawl.                              |                   |                                                                |



<div id="notes">
  ## Observações
</div>

* Aliases descontinuados: `scrapeExecute`, `stopInteractiveBrowser` e `deleteScrapeBrowser` apontam para `interact` e `stopInteraction`.
* O cliente `Firecrawl` de nível superior expõe os métodos da v2 diretamente; a v1 legada está disponível em `client.v1`.
