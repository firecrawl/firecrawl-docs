<div id="firecrawl-curl-source-of-truth">
  # Fonte oficial do curl do Firecrawl
</div>

Este arquivo é a fonte pública canônica para agentes. Ele é gerado a partir da especificação OpenAPI v2 do Firecrawl e das interfaces dos SDKs.

<div id="authenticate">
  ## Autenticação
</div>

Todas as requisições usam um token Bearer no cabeçalho `Authorization`.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="when-to-use-what">
  ## Quando usar cada um
</div>

* `search`: comece com uma consulta para descobrir URLs relevantes, incluindo restrições `site:`.
* `scraping`: obtenha e extraia conteúdo de uma URL conhecida.
* `interagir`: execute ações do navegador ou código quando uma página exigir cliques, formulários ou etapas após o scraping.

<div id="search">
  ## Busca
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use a busca quando precisar descobrir resultados a partir de uma consulta e quiser que o Firecrawl retorne URLs (e, opcionalmente, faça scraping de cada resultado). Você pode restringir os resultados a um domínio com o operador padrão `site:`, por exemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`POST https://api.firecrawl.dev/v2/search`

<div id="simple-example">
  ### Exemplo simples
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev webhook retries"
  }'
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev crawl webhooks",
    "sources": ["web", "news"],
    "categories": ["research"],
    "limit": 10,
    "tbs": "qdr:m",
    "location": "San Francisco,California,United States",
    "country": "US",
    "ignoreInvalidURLs": true,
    "timeout": 60000,
    "scrapeOptions": {
      "formats": [
        "markdown",
        "links",
        { "type": "json", "prompt": "Extract title and key endpoints." }
      ],
      "onlyMainContent": true,
      "includeTags": ["main", "article"],
      "excludeTags": ["nav", "footer"],
      "waitFor": 1000,
      "actions": [
        { "type": "click", "selector": "button#accept" },
        { "type": "wait", "milliseconds": 750 },
        { "type": "scrape" }
      ]
    }
  }'
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro           | Tipo / Opções                                       | Descrição                                                                             |
| ------------------- | --------------------------------------------------- | ------------------------------------------------------------------------------------- |
| `query`             | `string`                                            | Consulta de busca. Use `site:domain` para restringir os resultados a um site.         |
| `sources`           | `array of source names or typed source objects`     | Fontes para pesquisar.                                                                |
| `categories`        | `array of category names or typed category objects` | Categorias para filtrar os resultados.                                                |
| `limit`             | `integer`                                           | Número máximo de resultados a retornar.                                               |
| `tbs`               | `string`                                            | Filtro de busca baseado em tempo (por exemplo, `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).      |
| `location`          | `string`                                            | String de localização para os resultados de busca.                                    |
| `country`           | `string`                                            | Código de país ISO para segmentação geográfica dos resultados de busca.               |
| `timeout`           | `integer`                                           | Tempo limite em milissegundos.                                                        |
| `ignoreInvalidURLs` | `boolean`                                           | Exclui URLs inválidas que outros endpoints do Firecrawl não conseguem fazer scraping. |
| `enterprise`        | `array of enterprise modes such as anon or zdr`     | Opções de ZDR Enterprise quando habilitadas para sua equipe.                          |
| `scrapeOptions`     | `ScrapeOptions`                                     | Opções de scraping aplicadas a cada resultado (mesmos campos que `scrape`).           |

<div id="scrape">
  ## Scraping
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use o scraping quando você já tiver uma URL e quiser que o Firecrawl retorne conteúdo estruturado e formatos opcionais.

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`POST https://api.firecrawl.dev/v2/scrape`

<div id="simple-example">
  ### Exemplo simples
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://docs.firecrawl.dev",
    "formats": ["markdown"]
  }'
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/pricing",
    "formats": [
      "markdown",
      "links",
      { "type": "json", "prompt": "Extract plan names and prices." },
      { "type": "screenshot", "fullPage": true, "quality": 80, "viewport": { "width": 1280, "height": 720 } },
      { "type": "changeTracking", "modes": ["git-diff"], "tag": "pricing" }
    ],
    "headers": { "User-Agent": "FirecrawlDocsBot/1.0" },
    "includeTags": ["main", "article"],
    "excludeTags": ["nav", "footer"],
    "onlyMainContent": true,
    "waitFor": 1000,
    "mobile": false,
    "parsers": [{ "type": "pdf", "mode": "auto", "maxPages": 5 }],
    "actions": [
      { "type": "click", "selector": "#accept" },
      { "type": "write", "text": "firecrawl" },
      { "type": "press", "key": "Enter" },
      { "type": "wait", "milliseconds": 1500 },
      { "type": "scrape" }
    ],
    "location": { "country": "US", "languages": ["en-US"] },
    "skipTlsVerification": true,
    "removeBase64Images": true,
    "blockAds": true,
    "proxy": "auto",
    "maxAge": 86400000,
    "storeInCache": true,
    "profile": { "name": "docs-session", "saveChanges": true }
  }'
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro             | Tipo / Opções             | Descrição                                                                                                                                         |
| --------------------- | ------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- |
| `url`                 | `string`                  | URL para scraping.                                                                                                                                |
| `formats`             | `Array`                   | Formatos de resultado a incluir. Strings ou objetos como `{type: "json", prompt, schema}` ou `{type: "screenshot", fullPage, quality, viewport}`. |
| `onlyMainContent`     | `boolean`                 | Mantenha apenas o conteúdo principal.                                                                                                             |
| `includeTags`         | `Array<string>`           | Tags a incluir.                                                                                                                                   |
| `excludeTags`         | `Array<string>`           | Tags a excluir.                                                                                                                                   |
| `maxAge`              | `integer`                 | Prefira dados em cache até esta idade.                                                                                                            |
| `minAge`              | `integer`                 | Retorne dados em cache apenas se atenderem à idade mínima.                                                                                        |
| `headers`             | `object`                  | Cabeçalhos personalizados.                                                                                                                        |
| `waitFor`             | `integer`                 | Tempo de espera em milissegundos antes do scraping.                                                                                               |
| `mobile`              | `boolean`                 | Emule um dispositivo móvel.                                                                                                                       |
| `skipTlsVerification` | `boolean`                 | Ignore a verificação do certificado TLS.                                                                                                          |
| `timeout`             | `integer`                 | Tempo limite em milissegundos.                                                                                                                    |
| `parsers`             | `Array`                   | Configuração do parser (com suporte a opções de PDF).                                                                                             |
| `actions`             | `Array`                   | Ações do navegador a executar antes do scraping.                                                                                                  |
| `location`            | `object`                  | Configurações de localização para a requisição.                                                                                                   |
| `location.country`    | `string`                  | Código ISO do país.                                                                                                                               |
| `location.languages`  | `Array<string>`           | Idiomas preferidos.                                                                                                                               |
| `removeBase64Images`  | `boolean`                 | Remova imagens em base64 da saída em markdown.                                                                                                    |
| `blockAds`            | `boolean`                 | Ative o bloqueio de anúncios e pop-ups de cookies.                                                                                                |
| `proxy`               | `basic, enhanced ou auto` | Modo do proxy.                                                                                                                                    |
| `storeInCache`        | `boolean`                 | Armazene resultados em cache.                                                                                                                     |
| `profile`             | `object`                  | Perfil persistente do navegador com `name` e `saveChanges`.                                                                                       |
| `zeroDataRetention`   | `boolean`                 | Ative a retenção zero de dados quando sua conta permitir.                                                                                         |

<div id="interact">
  ## Interagir
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use o recurso de interagir quando uma página exigir ações no navegador ou execução de código após o início de um scraping (cliques, formulários ou etapas com script).

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`POST https://api.firecrawl.dev/v2/scrape/{jobId}/interact`

<div id="simple-example">
  ### Exemplo simples
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "console.log(document.title)",
    "language": "node"
  }'
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Click the pricing tab and summarize the plans.",
    "language": "node",
    "timeout": 60
  }'
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro  | Tipo / Opções           | Descrição                                                                                           |
| ---------- | ----------------------- | --------------------------------------------------------------------------------------------------- |
| `jobId`    | `string`                | ID do job de scraping em `data.metadata.scrapeId`.                                                  |
| `code`     | `string`                | Código a ser executado na sessão do navegador. Obrigatório se `prompt` não for fornecido.           |
| `prompt`   | `string`                | Instrução em linguagem natural para o agente do navegador. Obrigatório se `code` não for fornecido. |
| `language` | `python, node, ou bash` | Linguagem de execução. O padrão é `node` quando omitido.                                            |
| `timeout`  | `integer`               | Tempo limite de execução em segundos.                                                               |
| `origin`   | `string`                | Apenas para uso interno. Reservado para a instrumentação do Firecrawl.                              |

<div id="notes">
  ## Notas
</div>

* Atualmente, a especificação OpenAPI marca `code` como obrigatório para interagir, mas os SDKs aceitam `prompt` como alternativa. Na prática, use `code` ou `prompt`.
