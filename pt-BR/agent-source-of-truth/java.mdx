<div id="firecrawl-java-source-of-truth">
  # Fonte oficial do Firecrawl Java
</div>

Este arquivo é a fonte pública oficial para agentes. Ele é gerado a partir do código-fonte do SDK Java do Firecrawl e da especificação OpenAPI v2.

<div id="install">
  ## Instalação
</div>

Maven:

```xml
<dependency>
  <groupId>com.firecrawl</groupId>
  <artifactId>firecrawl-java</artifactId>
  <version>1.2.0</version>
</dependency>
```

Gradle:

```gradle
implementation("com.firecrawl:firecrawl-java:1.2.0")
```

<div id="authenticate">
  ## Autentique-se
</div>

```java
import com.firecrawl.client.FirecrawlClient;

FirecrawlClient client = FirecrawlClient.builder()
    .apiKey(System.getenv("FIRECRAWL_API_KEY"))
    .build();
```

<div id="when-to-use-what">
  ## Quando Usar Cada Um
</div>

* `search`: comece com uma consulta para descobrir URLs relevantes, incluindo restrições `site:`.
* `scrape`: obtenha e extraia conteúdo de uma URL conhecida.
* `interact`: execute ações no navegador ou código quando uma página exigir cliques, formulários ou etapas pós-scraping.

<div id="search">
  ## Busca
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use a busca quando precisar descobrir resultados a partir de uma consulta e quiser que o Firecrawl retorne URLs (e, opcionalmente, faça o scraping de cada resultado). Você pode restringir os resultados a um domínio com o operador padrão `site:`, por exemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`client.search(query, options)`

<div id="simple-example">
  ### Exemplo simples
</div>

```java
SearchData results = client.search("site:docs.firecrawl.dev webhook retries");
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```java
import com.firecrawl.models.SearchOptions;
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;

SearchOptions options = SearchOptions.builder()
    .sources(List.of("web", "news"))
    .categories(List.of("research"))
    .limit(10)
    .tbs("qdr:m")
    .location("San Francisco,California,United States")
    .ignoreInvalidURLs(true)
    .timeout(60000)
    .scrapeOptions(
        ScrapeOptions.builder()
            .formats(List.of(
                "markdown",
                "links",
                JsonFormat.builder().prompt("Extract title and key endpoints.").build()
            ))
            .onlyMainContent(true)
            .waitFor(1000)
            .build()
    )
    .build();

SearchData results = client.search("site:docs.firecrawl.dev crawl webhooks", options);
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro                   | Tipo / Opções                                                          | Descrição                                                                                     |
| --------------------------- | ---------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| `query`                     | `String`                                                               | Consulta de busca. Use `site:domain` para restringir os resultados a um site específico.      |
| `options.sources`           | `List<Object>` (`"web"`, `"news"`, `"images"` ou maps `{type: "web"}`) | Fontes a serem pesquisadas.                                                                   |
| `options.categories`        | `List<Object>` (`"github"`, `"research"`, `"pdf"`)                     | Categorias para filtrar os resultados.                                                        |
| `options.limit`             | `Integer`                                                              | Número máximo de resultados retornados.                                                       |
| `options.tbs`               | `String`                                                               | Filtro de busca por período (por exemplo, `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).                   |
| `options.location`          | `String`                                                               | Localização usada nos resultados de busca.                                                    |
| `options.ignoreInvalidURLs` | `Boolean`                                                              | Exclui URLs inválidas que outros endpoints do Firecrawl não conseguem processar com scraping. |
| `options.timeout`           | `Integer`                                                              | Tempo limite em milissegundos.                                                                |
| `options.scrapeOptions`     | `ScrapeOptions`                                                        | Opções de scraping aplicadas a cada resultado (mesmos campos de `scrape`).                    |
| `options.integration`       | `String`                                                               | Apenas para uso interno. Reservado para instrumentação do Firecrawl.                          |

<div id="scrape">
  ## Scraping
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use scraping quando você já tiver uma URL e quiser que o Firecrawl retorne conteúdo estruturado e formatos opcionais.

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`client.scrape(url, options)`

<div id="simple-example">
  ### Exemplo simples
</div>

```java
Document doc = client.scrape(
    "https://docs.firecrawl.dev",
    ScrapeOptions.builder()
        .formats(List.of("markdown"))
        .build()
);
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```java
import com.firecrawl.models.ScrapeOptions;

ScrapeOptions options = ScrapeOptions.builder()
    .formats(List.of(
        "markdown",
        "links",
        Map.of("type", "json", "prompt", "Extract plan names and prices."),
        Map.of("type", "screenshot", "fullPage", true, "quality", 80, "viewport", Map.of("width", 1280, "height", 720)),
        Map.of("type", "changeTracking", "modes", List.of("git-diff"), "tag", "pricing")
    ))
    .headers(Map.of("User-Agent", "FirecrawlDocsBot/1.0"))
    .includeTags(List.of("main", "article"))
    .excludeTags(List.of("nav", "footer"))
    .onlyMainContent(true)
    .waitFor(1000)
    .mobile(false)
    .parsers(List.of(Map.of("type", "pdf", "mode", "auto", "maxPages", 5)))
    .actions(List.of(
        Map.of("type", "click", "selector", "#accept"),
        Map.of("type", "write", "text", "firecrawl"),
        Map.of("type", "press", "key", "Enter"),
        Map.of("type", "wait", "milliseconds", 1500),
        Map.of("type", "scrape")
    ))
    .location(LocationConfig.builder().country("US").languages(List.of("en-US")).build())
    .skipTlsVerification(true)
    .removeBase64Images(true)
    .blockAds(true)
    .proxy("auto")
    .maxAge(86400000L)
    .storeInCache(true)
    .build();

Document doc = client.scrape("https://example.com/pricing", options);
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro                     | Tipo / Opções                                                                      | Descrição                                                                                                                                                     |
| ----------------------------- | ---------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `url`                         | `String`                                                                           | URL para scraping.                                                                                                                                            |
| `options.formats`             | `List<Object>`                                                                     | Formatos de resultado a incluir. Strings ou objetos de formato, como `{type: "json", prompt, schema}` ou `{type: "screenshot", fullPage, quality, viewport}`. |
| `options.headers`             | `Map<String, String>`                                                              | Cabeçalhos personalizados.                                                                                                                                    |
| `options.includeTags`         | `List<String>`                                                                     | Tags a incluir.                                                                                                                                               |
| `options.excludeTags`         | `List<String>`                                                                     | Tags a excluir.                                                                                                                                               |
| `options.onlyMainContent`     | `Boolean`                                                                          | Manter apenas o conteúdo principal.                                                                                                                           |
| `options.timeout`             | `Integer`                                                                          | Tempo limite em milissegundos.                                                                                                                                |
| `options.waitFor`             | `Integer`                                                                          | Tempo de espera em milissegundos antes do scraping.                                                                                                           |
| `options.mobile`              | `Boolean`                                                                          | Emular um dispositivo móvel.                                                                                                                                  |
| `options.parsers`             | `List<Object>`                                                                     | Configuração do parser (com suporte a opções de PDF).                                                                                                         |
| `options.actions`             | `List<Map<String, Object>>`                                                        | Ações do navegador a executar antes do scraping.                                                                                                              |
| `options.location`            | `LocationConfig`                                                                   | Configurações de localização para a solicitação.                                                                                                              |
| `options.skipTlsVerification` | `Boolean`                                                                          | Ignorar a verificação do certificado TLS.                                                                                                                     |
| `options.removeBase64Images`  | `Boolean`                                                                          | Remover imagens em base64 do resultado em markdown.                                                                                                           |
| `options.blockAds`            | `Boolean`                                                                          | Ativar o bloqueio de anúncios e pop-ups de cookies.                                                                                                           |
| `options.proxy`               | `String` (`"basic"`, `"stealth"`, `"enhanced"`, `"auto"` ou uma URL personalizada) | Modo de proxy.                                                                                                                                                |
| `options.maxAge`              | `Long`                                                                             | Preferir dados em cache com até esta idade.                                                                                                                   |
| `options.storeInCache`        | `Boolean`                                                                          | Armazenar resultados em cache.                                                                                                                                |
| `options.integration`         | `String`                                                                           | Apenas para uso interno. Reservado para a instrumentação do Firecrawl.                                                                                        |

<div id="interact">
  ## Interagir
</div>

<div id="why-use-it">
  ### Por que usar
</div>

Use interagir quando uma página exigir ações no navegador ou execução de código após o início do scraping (cliques, formulários ou etapas com script).

<div id="preferred-sdk-method">
  ### Método recomendado do SDK
</div>

`client.interact(jobId, code, language, timeout, origin)`

<div id="simple-example">
  ### Exemplo simples
</div>

```java
Document doc = client.scrape("https://example.com");
String jobId = doc.getMetadata() != null ? String.valueOf(doc.getMetadata().get("scrapeId")) : null;
if (jobId == null || jobId.isBlank()) {
    throw new IllegalStateException("Missing scrapeId from scrape response");
}

BrowserExecuteResponse result = client.interact(jobId, "console.log(document.title)");
```

<div id="complex-example">
  ### Exemplo complexo
</div>

```java
BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "console.log(await page.title());",
    "node",
    60,
    null
);
```

<div id="parameters">
  ### Parâmetros
</div>

| Parâmetro  | Tipo / Opções                             | Descrição                                                             |
| ---------- | ----------------------------------------- | --------------------------------------------------------------------- |
| `jobId`    | `String`                                  | ID do job de scraping em `document.metadata.scrapeId`.                |
| `code`     | `String`                                  | Código a ser executado na sessão do navegador.                        |
| `language` | `String` (`"python"`, `"node"`, `"bash"`) | Linguagem de execução. Usa `node` por padrão quando omitido.          |
| `timeout`  | `Integer`                                 | Tempo limite de execução em segundos (1-300).                         |
| `origin`   | `String`                                  | Somente para uso interno. Reservado para instrumentação do Firecrawl. |

<div id="notes">
  ## Notas
</div>

* A API `interact` do SDK Java suporta apenas `code`. Outros SDKs expõem `prompt` como conveniência.
* Alias descontinuado: `scrapeExecute` é mapeado para `interact`, e `deleteScrapeBrowser` é mapeado para `stopInteractiveBrowser`.
