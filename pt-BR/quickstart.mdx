<div id="setup-your-development">
  ## Configure seu ambiente de desenvolvimento
</div>

Aprenda a atualizar sua documentação localmente e implantá-la para o público.

<div id="edit-and-preview">
  ### Editar e pré-visualizar
</div>

<AccordionGroup>
  <Accordion icon="github" title="Clonar sua documentação localmente">
    Durante o processo de onboarding, criamos um repositório no seu GitHub com
    o conteúdo da sua documentação. Você pode encontrar esse repositório no nosso
    [dashboard](https://dashboard.mintlify.com). Para clonar o repositório
    localmente, siga estas
    [instruções](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    no seu terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Pré-visualizar alterações">
    A pré-visualização ajuda você a confirmar que suas alterações estão como esperado. Criamos uma
    interface de linha de comando para renderizar essas alterações localmente. 1. Instale o
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) para pré-visualizar as
    alterações na documentação localmente com este comando: `npm i -g mintlify`
    2. Execute o seguinte comando na raiz da sua documentação (onde
    está o `mint.json`): `mintlify dev`
  </Accordion>
</AccordionGroup>

<div id="deploy-your-changes">
  ### Publique suas alterações
</div>

<AccordionGroup>
  <Accordion icon="message-bot" title="Instale nosso app do GitHub">
    Nosso app do GitHub publica automaticamente suas alterações no seu site de documentação, para que você
    não precise gerenciar deploys manualmente. Você pode encontrar o link de instalação no
    seu [dashboard](https://dashboard.mintlify.com). Depois que o bot for
    instalado com sucesso, deverá aparecer um check ao lado do hash do commit
    do repositório.
  </Accordion>

  <Accordion icon="rocket" title="Envie suas alterações">
    [Faça commit e push das suas alterações para o
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    para que sejam atualizadas no seu site de documentação. Se você fizer push e não vir que
    o app do GitHub publicou suas alterações com sucesso, você também pode atualizar
    manualmente sua documentação pelo nosso [dashboard](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
