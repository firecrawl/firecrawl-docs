# Firecrawl Python Agent Quickstart

This file is the canonical quickstart for external agents. It is generated from SDK source plus the v2 OpenAPI spec.

## Install

```bash
pip install firecrawl-py
```

## Authenticate

```python
import os
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])
```

## When To Use What

- `search`: start from a query and discover URLs (optionally scrape each result).
- `scrape`: you already have a URL and want content.
- `interact`: you need clicks, form fills, or post-scrape browser actions.

## Search

### Why use it

Use `search` to discover relevant URLs from a query and optionally scrape each result in one call.

### Preferred SDK method

`client.search(query, **options)`

### Example

```python
results = client.search(
    "site:example.com pricing",
    sources=["web"],
    limit=5,
)

print(results.web[0] if results.web else None)
```

### Parameters

| Parameter | Description | When to use |
| --- | --- | --- |
| `query` | Search query string. | Always. |
| `sources` | Sources to search (strings or `SourceOption` objects). | Use to limit result types. |
| `categories` | Filter categories (strings or `CategoryOption` objects). | Use to constrain results by category. |
| `limit` | Maximum number of results to return. Default: 5. | Use to cap response size. |
| `tbs` | Time-based search filter string. | Use to limit by time range. |
| `location` | Location string for localized results. | Use for geo-specific queries. |
| `ignore_invalid_urls` | Exclude invalid URLs from results. | Use when piping into other endpoints. |
| `timeout` | Request timeout in milliseconds. Default: 300000. | Use to cap search time. |
| `scrape_options` | Scrape options applied to each result (same fields as `scrape`). | Use when you want content with results. |
| `integration` | Integration identifier. | Use for tracking/analytics. |

## Scrape

### Why use it

Use `scrape` when you already know the URL and want content in specific formats.

### Preferred SDK method

`client.scrape(url, **options)`

### Example

```python
doc = client.scrape(
    "https://example.com",
    formats=["markdown", "links"],
    only_main_content=True,
    block_ads=True,
)

print(doc.markdown)
```

### Parameters

| Parameter | Description | When to use |
| --- | --- | --- |
| `url` | URL to scrape. | Always. |
| `formats` | Output formats to include (strings or format objects). | Use to control output types. |
| `headers` | Additional HTTP headers. | Use for auth, cookies, or user-agent. |
| `include_tags` | Only include content from these HTML tags. | Use to narrow extraction. |
| `exclude_tags` | Exclude content from these HTML tags. | Use to remove noise. |
| `only_main_content` | Extract main content only. | Use to drop nav/footer. |
| `timeout` | Scrape timeout in milliseconds. | Use to cap run time. |
| `wait_for` | Delay before scraping in milliseconds. | Use for JS-rendered pages. |
| `mobile` | Emulate a mobile device. | Use for mobile layouts. |
| `parsers` | Parser list or configs (for example, PDF parser settings). | Use when scraping PDFs or special formats. |
| `actions` | Browser actions to run before scraping. | Use for clicks, scrolling, or form input. |
| `location` | Location config (`country`, `languages`). | Use for geo-specific pages. |
| `skip_tls_verification` | Skip TLS certificate verification. | Use for self-signed certs. |
| `remove_base64_images` | Strip base64 images from markdown output. | Use to reduce payload size. |
| `fast_mode` | Prefer faster scraping modes. | Use when speed matters most. |
| `use_mock` | Use a mock data source (internal/testing). | Use for test fixtures. |
| `block_ads` | Enable ad and cookie blocking. | Use to reduce noise. |
| `proxy` | Proxy mode (`basic`, `stealth`, `enhanced`, `auto`). | Use for anti-bot sites. |
| `max_age` | Accept cached content younger than this age (ms). | Use to favor cache hits. |
| `store_in_cache` | Store the result in cache/index. | Use when you want reuse. |
| `profile` | Persistent browser profile (`{"name": "...", "saveChanges": True}`). | Use to keep session state. |
| `integration` | Integration identifier. | Use for tracking/analytics. |

## Interact

### Why use it

Use `interact` when the page requires clicks, form fills, or other browser actions after a scrape.

### Preferred SDK method

`client.interact(job_id, code=None, *, prompt=None, **options)`

### Example

```python
doc = client.scrape("https://example.com")
job_id = doc.metadata_typed.scrape_id
if not job_id:
    raise RuntimeError("Missing scrape_id")

run = client.interact(
    job_id,
    code="print(await page.url())",
    language="python",
    timeout=60,
)

print(run.stdout)
```

### Parameters

| Parameter | Description | When to use |
| --- | --- | --- |
| `job_id` | Scrape job ID (from `document.metadata_typed.scrape_id`). | Always. |
| `code` | Code to execute in the browser sandbox. | Use for scripted interactions. |
| `prompt` | Natural-language instruction for the browser agent. | Use when you prefer agent-driven actions. |
| `language` | Runtime language (`"python"`, `"node"`, `"bash"`). Defaults to `"node"`. | Use to match your code. |
| `timeout` | Execution timeout in seconds. | Use to limit long runs. |
| `origin` | Optional origin tag. | Use for request attribution. |

## Notes

- Parameters are snake_case in Python.
- Deprecated aliases: `scrape_execute`, `stop_interactive_browser`, `delete_scrape_browser` (use `interact` and `stop_interaction`).
- `Firecrawl` is the unified v2 client; v1 is available under `.v1` for legacy use.
