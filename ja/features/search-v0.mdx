<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## ウェブ検索と結果のスクレイピングを Firecrawl で
</div>

Firecrawl は SERP（Search Engine Results Page）API と堅牢なスクレイピング基盤を統合し、単一のエンドポイントでシームレスに検索とスクレイピングを実行できます。理由は次のとおりです。

1. **統一された検索クエリ:**
   ユーザーは SERP エンドポイント経由で検索クエリを送信します。

2. **結果の自動スクレイピング:**
   Firecrawl が検索結果を自動処理し、スクレイピング機能で各結果ページからデータを抽出します。

3. **データの提供:**
   すべての結果ページからスクレイプしたデータをクリーンなMarkdownにまとめて提供するため、すぐに利用できます。

この統合により、ユーザーは最小限の手間で効率よくウェブ検索を行い、複数ソースから網羅的なスクレイプ済みデータを取得できます。

詳細は [Search Endpoint Documentation](https://docs.firecrawl.dev/api-reference/endpoint/search) を参照してください。

<div id="search-any-query">
  ## 任意のキーワードで検索
</div>

<div id="search-endpoint">
  ### /search エンドポイント
</div>

ウェブを検索して最も関連性の高い結果を取得し、各ページをスクレイピングしてMarkdownを返します。

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // 高速な SERP API にするには false
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Markdown コンテンツ",
      "provider": "web-scraper",
      "metadata": {
        "title": "Firecrawl | LLM 向けに信頼性高くウェブをスクレイピング",
        "description": "CX とセールス向けの AI"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

</div>

```bash
pip install firecrawl-py
```


<div id="search-a-query">
  #### クエリを検索
</div>

```python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

result = app.search(query="Firecrawl とは？")
```

レスポンスは、上の curl コマンドで示したものと同様になります。


上の curl コマンドで示したものと同様のレスポンスが返ります。


<div id="with-go-sdk">
  ### Go SDK の使用

<div id="installing-go-sdk">
  #### Go SDK のインストール
</div>

```bash
go get github.com/mendableai/firecrawl-go
```


<div id="search-a-query">
  #### クエリを検索
</div>

```go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("FirecrawlApp の初期化に失敗しました: %v", err)
  }

  query := "Firecrawl とは？"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("検索に失敗しました: %v", err)
  }
  fmt.Println(searchResult)
}
```

```rust
async fn main() {
  let api_key = "YOUR_API_KEY";
  let api_url = "https://api.firecrawl.dev";
  let app = FirecrawlApp::new(api_key, api_url).expect("FirecrawlAppの初期化に失敗しました");

  let query = "Firecrawlとは？";
  let search_result = app.search(query).await;

  match search_result {
    Ok(data) => println!("検索結果: {}", data),

  }
}
```
