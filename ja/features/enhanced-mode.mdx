Firecrawl は、複雑さが異なるウェブサイトをスクレイピングできるよう、複数のプロキシタイプを提供しています。プロキシタイプは `proxy` パラメータで指定できます。


<div id="proxy-types">
  ### プロキシタイプ
</div>

Firecrawl は 3 種類のプロキシをサポートしています:

- **basic**: ほとんどのサイトをスクレイピングするためのプロキシ。高速で、たいてい問題なく動作します。
- **Enhanced**: プライバシーを維持しながら複雑なサイトをスクレイピングするためのenhanced プロキシ。速度は遅めですが、特定のサイトではより信頼性があります。
- **auto**: basic プロキシが失敗した場合、Firecrawl がenhanced プロキシで自動的に再試行します。ステルスでの再試行が成功した場合、そのスクレイプには 5 クレジットが請求されます。最初の basic が成功した場合は、通常のコストのみが請求されます。

プロキシを指定しない場合、Firecrawl は既定で auto を使用します。
