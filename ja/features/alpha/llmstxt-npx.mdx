<div id="generate-llmstxt">
  ## generate-llmstxt
</div>

Firecrawl API を使用して CLI 上で LLMs.txt ファイルを生成するシンプルな npx パッケージです。このパッケージは、指定した出力ディレクトリ（既定では「public」フォルダ）に次の2つのファイルを作成します：

- `llms.txt`: LLM 関連コンテンツの要約を含みます
- `llms-full.txt`: すべてのテキストコンテンツを含みます

<div id="usage">
  ## 使い方
</div>

このパッケージはインストールせずに npx で実行できます。Firecrawl の API キーを渡す方法は2通りあります。

<div id="1-using-command-line-arguments">
  ### 1. コマンドライン引数の使用
</div>

```bash
npx generate-llmstxt --api-key YOUR_FIRECRAWL_API_KEY
```


<div id="2-using-environment-variables">
  ### 2. 環境変数の使用
</div>

プロジェクトのルートに `.env` ファイルを作成し、API キーを追加します:

```env
FIRECRAWL_API_KEY=ここにあなたのAPIキー
```

次に、--api-key オプションなしでコマンドを実行します：

```bash
npx generate-llmstxt
```


<div id="options">
  ### オプション
</div>

- `-k, --api-key <key>`（.env に設定済みなら省略可）: Firecrawl の API キー
- `-u, --url <url>`（任意）: 解析対象の URL（デフォルト: https://example.com）
- `-m, --max-urls <number>`（任意）: 解析する URL の最大数（デフォルト: 50）
- `-o, --output-dir <path>`（任意）: 出力ディレクトリのパス（デフォルト: 'public'）

<div id="examples">
  ### 例
</div>

```bash
# 既定の出力ディレクトリでコマンドライン引数を使う
npx generate-llmstxt -k your_api_key -u https://your-website.com -m 20

# .env ファイルを使い、既定の出力ディレクトリに出力する
npx generate-llmstxt -u https://your-website.com -m 20

# 独自の出力ディレクトリを指定する
npx generate-llmstxt -k your_api_key -u https://your-website.com -o custom/path/to/output

# .env ファイルと独自の出力ディレクトリを使う
npx generate-llmstxt -u https://your-website.com -o content/llms
```


<div id="requirements">
  ## 必要条件
</div>

- Node.js 14以上
- 有効な Firecrawl API キー（コマンドラインまたは.envファイル経由）

<div id="output">
  ## 出力
</div>

このパッケージは、指定した出力ディレクトリ（既定値は「public」）に次の2つのファイルを作成します:

1. `llms.txt`: LLM関連コンテンツの要約が含まれます
2. `llms-full.txt`: すべてのテキスト本文が含まれます
