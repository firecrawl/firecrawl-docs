<div id="creating-a-custom-snippet">
  ## カスタムスニペットの作成
</div>

**前提条件**: スニペットファイルは `snippets` ディレクトリに作成する必要があります。

<Note>
  `snippets` ディレクトリ内のページはすべてスニペットとして扱われ、単独のページとしては
  レンダリングされません。スニペットを単独ページとして作成したい場合は、そのスニペットを別の
  ファイルにインポートし、コンポーネントとして呼び出してください。
</Note>

<div id="default-export">
  ### デフォルトエクスポート
</div>

1. 複数の場所で再利用したいコンテンツをスニペットファイルに追加します。必要に応じて、スニペットをインポートする際に props で値を渡して埋め込める変数を追加できます。

```mdx snippets/my-snippet.mdx
こんにちは、世界！これは複数のページで再利用したいコンテンツです。本日のキーワードは
{word}です。
```

<Warning>
  再利用したいコンテンツは、インポートを機能させるために `snippets` ディレクトリ内に置く必要があります。
</Warning>

2. スニペットを宛先ファイルにインポートします。

```mdx destination-file.mdx
---
title: タイトル
description: 説明
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## 見出し

Lorem ipsum dolor sit amet.

<MySnippet word="bananas" />
```

<div id="reusable-variables">
  ### 再利用可能な変数
</div>

1. スニペットファイルから変数をエクスポートする:

```mdx snippets/path/to/custom-variables.mdx
export const myName = 'my name';

export const myObject = { fruit: 'strawberries' };
```

2. 宛先ファイルからスニペットをインポートし、その変数を使います:

```mdx destination-file.mdx
---
title: タイトル
description: 説明
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

こんにちは、{myName} と申します。{myObject.fruit} が好きです。
```

<div id="reusable-components">
  ### 再利用可能なコンポーネント
</div>

1. スニペットファイル内で、アロー関数としてコンポーネントをエクスポートし、props を受け取るコンポーネントを作成します。

```mdx snippets/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... スニペット内容 ...</p>
  </div>
);
```

<Warning>
  矢印関数の本体内では MDX はコンパイルされません。可能な限り HTML
  構文を使うか、MDX を使う必要がある場合はデフォルトエクスポートを使用してください。
</Warning>

2. スニペットを目的のファイルにインポートし、props を渡す

```mdx destination-file.mdx
---
title: タイトル
description: 説明
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'カスタムタイトル'} />
```
