ナビゲーションメニューは、あらゆるウェブサイトにあるリンクの一覧です。

新しいページを追加するたびに `mint.json` を更新する必要がある場合がほとんどです。ページは自動的には表示されません。

<div id="navigation-syntax">
  ## ナビゲーション構文
</div>

ナビゲーション構文は再帰的で、入れ子のナビゲーショングループを作成できます。ページ名に「.mdx」を含める必要はありません。

<CodeGroup>

```json Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Nested Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

<div id="folders">
  ## フォルダ
</div>

MDXファイルをフォルダに配置し、`mint.json` のパスを更新するだけです。

たとえば、`https://yoursite.com/your-folder/your-page` にページを用意するには、`your-folder` というフォルダを作成し、その中に `your-page.mdx` というMDXファイルを入れます。

<Warning>
  他のフォルダの中に入れない限り、フォルダ名に `api` は使用できません。Mintlify は Next.js を使用しており、トップレベルの `api` フォルダは内部サーバー呼び出しのために予約されています。`api-reference` のようなフォルダ名であれば使用できます。
</Warning>

```json フォルダを使ったナビゲーション
"navigation": [
    {
        "group": "グループ名",
        "pages": ["your-folder/your-page"]
    }
]
```


<div id="hidden-pages">
  ## 非表示ページ
</div>

`mint.json` に含まれていない MDX ファイルはサイドバーには表示されませんが、検索バーから、または直接リンクでアクセスできます。
