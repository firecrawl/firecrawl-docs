<div id="inline-code">
  ### インラインコード
</div>

`word` や `phrase` をコードとして示すには、バッククォート（`）で囲みます。

```
「単語」や「語句」をコードとして示すには、バッククォート（`）で囲みます。
```


<div id="code-block">
  ### コードブロック
</div>

[フェンス付きコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)を使い、コードをバッククォート3つで囲んで、その直後にスニペットのプログラミング言語を指定するとシンタックスハイライトが有効になります。必要に応じて、プログラミング言語の後にコード名を付けることもできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("こんにちは、世界!");
    }
}
```

````md
```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
````
