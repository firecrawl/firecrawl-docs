<div id="inline-code">
  ### インラインコード
</div>

`word` や `phrase` をコードとして示すには、バッククォート（`）で囲みます。

```
「単語」や「語句」をコードとして示すには、バッククォート（`）で囲みます。
```

<div id="code-block">
  ### コードブロック
</div>

[フェンス付きコードブロック](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks)を使用するには、コードをバッククォート3つで囲み、先頭のバッククォートの直後にコードのプログラミング言語名を指定してシンタックスハイライトを有効にします。必要に応じて、プログラミング言語名の後にコードの名前を記述することもできます。

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```

class HelloWorld {
