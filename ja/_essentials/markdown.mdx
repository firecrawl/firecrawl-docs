<div id="titles">
  ## タイトル
</div>

セクションの見出しに最適です。

```md
## 見出し
```

<div id="subtitles">
  ### サブタイトル
</div>

小見出しに最適です。

```md
### 字幕
```

<Tip>
  各**タイトル**と**サブタイトル**はアンカーを生成し、右側の目次にも表示されます。
</Tip>

<div id="text-formatting">
  ## テキストの書式設定
</div>

ほとんどの Markdown の書式に対応しています。テキストの前後を `**`、`_`、`~` で囲むだけで装飾できます。

| スタイル      | 書き方               | 結果                |
| ------------- | -------------------- | ------------------- |
| 太字          | `**bold**`           | **bold**            |
| 斜体          | `_italic_`           | _italic_            |
| 取り消し線    | `~strikethrough~`    | ~strikethrough~     |

これらは組み合わせられます。たとえば、`**_bold and italic_**` と書くと、**_bold and italic_** のように表示されます。

上付き文字や下付き文字には HTML を使用します。つまり、テキストを `<sup>` または `<sub>` で囲んでください。

| 文字の種類    | 書き方                       | 結果                      |
| ------------- | ---------------------------- | ------------------------- |
| 上付き文字    | `<sup>superscript</sup>`     | <sup>superscript</sup>    |
| 下付き文字    | `<sub>subscript</sub>`       | <sub>subscript</sub>      |

<div id="linking-to-pages">
  ## ページへのリンク
</div>

テキストを `[]()` で囲むとリンクを追加できます。たとえば、`[link to google](https://google.com)` と書くと、[link to google](https://google.com) になります。

ドキュメント内のページへのリンクはルート相対にする必要があります。つまり、フォルダパス全体を含めて指定してください。たとえば、`[link to text](/writing-content/text)` はコンポーネントセクション内の「Text」ページへのリンクになります。

`[link to text](../text)` のような相対リンクは、最適化が難しいため表示が遅くなります。

<div id="singleline">
  ### シングルライン
</div>

引用ブロックを作成するには、段落の先頭に `>` を付けます。

> ドロシーは、彼女の城にある美しい部屋の数々を、彼女の後について歩いて回った。

```md
> ドロシーは、彼女に導かれて彼女の城の多くの美しい部屋を通り抜けていった。
```

<div id="multiline">
  ### 複数行
</div>

> ドロシーは、彼女の城の多くの美しい部屋を抜けて、その後をついて行った。
>
> 魔女は彼女に、鍋ややかんを洗い、床を掃き、炉に薪をくべ続けるように命じた。

```md
> ドロシーは、彼女の城の美しい部屋をいくつも抜けて、そのあとに続いた。
>
> 魔女は、鍋ややかんを洗い、床を掃き、薪をくべて火を絶やさないようにすることを彼女に命じた。
```
