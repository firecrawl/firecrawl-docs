<Info>
  **前提条件** Node.js（バージョン 18.10.0 以上）がインストールされている必要があります。
</Info>

手順 1. お使いの OS に Mintlify をインストールします:

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

手順 2. ドキュメントのディレクトリ（`mint.json` がある場所）に移動し、次のコマンドを実行します:

```bash
mintlify dev
```

ドキュメントサイトは `http://localhost:3000` で公開されています。

<div id="custom-ports">
  ### カスタムポート
</div>

Mintlify はデフォルトでポート 3000 を使用します。`--port` フラグで、Mintlify が使用するポートを変更できます。例えば、ポート 3333 で実行する場合は次のコマンドを使用します：

```bash
mintlify dev --port 3333
```

すでに使用中のポートで Mintlify を実行しようとすると、次のようなエラーが表示されます：

```md
Error: listen EADDRINUSE: アドレスは既に使用中です :::3000
```

<div id="mintlify-versions">
  ## Mintlify のバージョン
</div>

各 CLI は特定の Mintlify バージョンに紐づいています。ローカルのサイト表示が本番と異なる場合は、CLI を更新してください。

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

<div id="troubleshooting">
  ## トラブルシューティング
</div>

CLI 使用時によくある問題の解決方法を紹介します。

<AccordionGroup>
  <Accordion title="Mintlify が読み込まれない">
    Node を v18 に更新してください。`mintlify install` を実行して、もう一度お試しください。
  </Accordion>

  <Accordion title="Windows で「No such file or directory」が出る">
    `C:/Users/Username/.mintlify/` ディレクトリに移動して `mint` フォルダーを削除します。次に、この場所で Git Bash を開き、`git clone
    https://github.com/mintlify/mint.git` を実行します。

    手順 3 を繰り返してください。
  </Accordion>

  <Accordion title="不明なエラーが発生する">
    デバイスのルートディレクトリに移動し、~/.mintlify フォルダーを削除してみてください。
    その後、`mintlify dev` をもう一度実行します。
  </Accordion>
</AccordionGroup>

CLI のバージョンで何が変わったか気になりますか？[CLI の変更履歴をご覧ください。](/ja/changelog/command-line)
