<div id="using-firecrawl-with-crewai">
  ## CrewAI で Firecrawl を使う
</div>

Firecrawl は、[AI エージェントをオーケストレーションするフレームワークである CrewAI](https://www.crewai.com/) と統合されています。このページでは、フレームワークに追加された Firecrawl のツールをすべて紹介します。

<div id="installing-firecrawl-tools-inside-of-crewai">
  ### CrewAI 内での Firecrawl Tools のインストール
</div>

* [firecrawl.dev のダッシュボード](https://firecrawl.dev)から API キーを取得し、環境変数（`FIRECRAWL_API_KEY`）に設定します。
* `crewai[tools]` パッケージとあわせて [Firecrawl SDK](https://github.com/mendableai/firecrawl) をインストールします。

```
pip install firecrawl-py 'crewai[tools]'
```

  ## ツール
</div>

<div id="firecrawlcrawlwebsitetool">
  ### FirecrawlCrawlWebsiteTool
</div>

<div id="example">
  #### 例
</div>

エージェントがウェブサイトを読み込めるようにするには、次のように FirecrawlScrapeFromWebsiteTool を利用します：

```python
from crewai_tools import FirecrawlCrawlWebsiteTool

tool = FirecrawlCrawlWebsiteTool(url='firecrawl.dev')
```

<div id="arguments">
  #### 引数
</div>

- `api_key`: オプション。使用する Firecrawl の API キーを指定します。デフォルトは `FIRECRAWL_API_KEY` 環境変数の値です。
- `url`: クロールを開始するベース URL。
- `page_options`: オプション。
  - `onlyMainContent`: オプション。ヘッダー、ナビゲーション、フッターなどを除いた、ページのメインコンテンツのみを返します。
  - `includeHtml`: オプション。ページの生の HTML コンテンツを含めます。レスポンス内に `html` キーとして出力されます。
- `crawler_options`: オプション。クロール動作を制御するためのオプション。
  - `includes`: オプション。クロールに含める URL パターン。
  - `exclude`: オプション。クロールから除外する URL パターン。
  - `generateImgAltText`: オプション。LLM を使用して画像の alt テキストを生成します（有料プランが必要）。
  - `returnOnlyUrls`: オプション。`true` の場合、クロールステータスで URL のみをリストとして返します。注意: レスポンスはドキュメントのリストではなく、`data` 内に含まれる URL のリストになります。
  - `maxDepth`: オプション。クロールする最大の深さ。深さ 1 はベース URL、深さ 2 はベース URL とその直下の子ページを含み、それ以降も同様です。
  - `mode`: オプション。使用するクロールモード。Fast モードはサイトマップのないウェブサイトで 4 倍速くクロールできますが、精度が低くなる場合があり、JavaScript レンダリングが多いウェブサイトでは使用しないことを推奨します。
  - `limit`: オプション。クロールするページ数の上限。
  - `timeout`: オプション。クロール処理のタイムアウト時間（ミリ秒）。

<div id="firecrawlscrapewebsitetool">
  ### FirecrawlScrapeWebsiteツール
</div>

<div id="example">
  #### 例
</div>

エージェントがウェブサイトを読み込めるようにするには、次のように FirecrawlScrapeWebsiteTool を使用します。

```python
from crewai_tools import FirecrawlScrapeWebsiteTool

tool = FirecrawlScrapeWebsiteTool(url='firecrawl.dev')
```

<div id="arguments">
  #### 引数
</div>

- `api_key`: オプション。使用する Firecrawl API キーを指定します。デフォルトは `FIRECRAWL_API_KEY` 環境変数です。
- `url`: スクレイプする URL。
- `page_options`: オプション。
  - `onlyMainContent`: オプション。ヘッダー、ナビゲーション、フッターなどを除いたページのメインコンテンツのみを返します。
  - `includeHtml`: オプション。ページの生の HTML コンテンツを含めます。レスポンスには html キーが含まれます。
- `extractor_options`: オプション。ページコンテンツから構造化情報を LLM ベースで抽出するためのオプションです。
  - `mode`: 使用する抽出モード。現在は `llm-extraction` をサポートしています。
  - `extractionPrompt`: オプション。ページからどの情報を抽出するかを記述するプロンプト。
  - `extractionSchema`: オプション。抽出されるデータのスキーマ。
- `timeout`: オプション。リクエストのタイムアウト時間（ミリ秒単位）。

<div id="firecrawlsearchtool">
  ### Firecrawl検索ツール
</div>

<div id="example">
  #### 例
</div>

エージェントがウェブサイトを読み込めるようにするには、次のように FirecrawlSearchTool を使用します：

```python
from crewai_tools import FirecrawlSearchTool

tool = FirecrawlSearchTool(query='Firecrawl とは？')
```

<div id="arguments">
  #### 引数
</div>

- `api_key`: オプション。使用する Firecrawl の API キーを指定します。指定しない場合は `FIRECRAWL_API_KEY` 環境変数がデフォルトとして使用されます。
- `query`: 検索に使用する検索クエリ文字列。
- `page_options`: オプション。結果の形式に関するオプション。
  - `onlyMainContent`: オプション。ヘッダー、ナビゲーション、フッターなどを除いたページのメインコンテンツのみを返します。
  - `includeHtml`: オプション。ページの生の HTML コンテンツを含めます。レスポンスには html キーが出力されます。
  - `fetchPageContent`: オプション。ページ全体のコンテンツを取得します。
- `search_options`: オプション。クロール動作を制御するためのオプション。
  - `limit`: オプション。クロールするページ数の最大値。
