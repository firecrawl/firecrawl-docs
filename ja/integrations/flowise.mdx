<div id="sync-web-data-in-flowise-workflows">
  ## Flowise のワークフローで Web データを同期する
</div>

Firecrawl は [Flowise（チャットフロー・ビルダー）](https://flowiseai.com/) 内で利用できます。このページでは、Flowise 内での Firecrawl ブロックの設定と使い方を紹介します。

<img
  className="block"
  src="/images/fc_flowise_block.png"
  alt="Firecrawl Flowise ブロック"
/>

<div id="crawling-with-firecrawl-blocks">
  ### Firecrawl ブロックでのクロール
</div>

1. Firecrawl アカウントにログインして API キーを取得し、ブロックに入力します。
2. （任意）Text Splitter を接続します。
3. クロールモードを選択し、対象 URL 配下のページをクロールします。
4. クロール対象の URL を入力します。
5. 生成されたドキュメントをワークフローで利用します。

<div id="scraping-with-firecrawl-blocks">
  ### Firecrawl ブロックでのスクレイピング
</div>

1. Firecrawl アカウントにログインして API キーを取得し、ブロックに入力します。
2. （任意）Text Splitter を接続します。
3. 単一ページを取得するスクレイプモードを選択します。
4. スクレイプ対象のURLを入力します。
5. 取得したドキュメントをワークフローで使用します。
