---
title: "LLMs.txt を生成する"
openapi: "v1-openapi POST /llmstxt"
---

LLMs.txt 生成エンドポイントは、任意のウェブサイト向けに LLMs.txt と LLMs-full.txt ファイルを生成します。これらのファイルは、ウェブサイトのコンテンツを構造化し、LLM に適したフォーマットで提供することで、言語モデルによる理解・処理を容易にします。

ステータスのエンドポイントをお探しですか？[LLMs.txt Status](/ja/api-reference/endpoint/llmstxt-get) エンドポイントをご確認ください。

<div id="response-structure">
  ### レスポンス構造
</div>

レスポンスには以下が含まれます。

* **success**: リクエストの成否を示すブール値
* **id**: 生成ジョブの一意の識別子

<div id="output-format">
  ### 出力フォーマット
</div>

1. **LLMs.txt**

   * ウェブサイトの簡潔で構造化された要約
   * 主要なリンクとその説明を含む
   * 解析しやすいように Markdown で整形
   * 例:

     ```markdown
     # http://example.com llms.txt

     - [Page Title](https://example.com/page): Brief description
     - [Another Page](https://example.com/another): Another description
     ```

2. **LLMs-full.txt**（showFullText が true の場合）

   * 処理したページの全文を含む
   * 階層構造を維持
   * より詳細な情報を収録
   * 例:

     ```markdown
     # http://example.com llms-full.txt

     ## Page Title

     Full content of the page...

     ## Another Page

     More detailed content...
     ```

<div id="billing">
  ### 請求
</div>

請求は API 呼び出し数と処理した URL 数に基づきます：

* 基本料金: API 呼び出し1回につき1クレジット
* 追加料金: 処理した URL 1件につき1クレジット
* `maxUrls` パラメータで URL のコストを制御できます
