<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="tell-crawl-what-you-want">
  ### クロール内容を指示する
</div>

クロールしてほしい内容を、わかりやすい英語で記述してください:

```json
{
  "url": "https://example.com",
  "prompt": "ブログ記事とドキュメントのみをクロールし、マーケティングページは除外する"
}
```

これにより、プロンプトがクロール実行用のクローラー設定一式にマッピングされます。


<div id="improved-sitemap-control">
  ### サイトマップ制御の改善
</div>

v1 では、サイトマップの使用可否はブール値でした。v2 では、`sitemap` オプションで次を選べます:

- `"include"`（デフォルト）：サイトマップを使用しつつ、他のページも探索します。
- `"skip"`：サイトマップを完全に無視します。

<div id="new-crawling-options">
  ### 新しいクロールオプション
</div>

* `crawlEntireDomain` - 子ページだけでなくドメイン全体をクロール
* `maxDiscoveryDepth` - クロールの深さを制御（`maxDepth` を置き換え）

```json
{
  "url": "https://example.com/features",
  "crawlEntireDomain": true,
  "maxDiscoveryDepth": 2,
  "sitemap": "include"
}
```
