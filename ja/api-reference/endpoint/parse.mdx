API を介して、ローカルまたは公開アクセス可能なあらゆるドキュメントを、AIエージェント向けのクリーンなデータに変換します。

* PDF、DOCX、XLSX ファイルを Markdown または JSON に変換
* 読み順と表を保持
* ゼロデータ保持をサポート

<div id="how-to-use-document-parsing">
  ## ドキュメント解析の使い方
</div>

Firecrawl のドキュメント解析は、次の 2 つの方法で利用できます。

1. **URL ベースの解析 (`/v2/scrape`)**: 対応しているドキュメント形式を指す URL を指定します。詳細は [Scrape](/ja/api-reference/endpoint/scrape) エンドポイントを参照してください。
2. **ファイルアップロードによる解析 (`/v2/parse`)**: `multipart/form-data` を使って、ファイルのバイト列を直接アップロードします。

URL ベースの解析では、Firecrawl が拡張子またはコンテンツタイプからファイル形式を自動的に判別します。
