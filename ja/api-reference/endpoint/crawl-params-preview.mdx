  ## v2の新機能
</div>

<div id="crawl-params-preview">
  ### クロールパラメータの事前プレビュー
</div>

```json
{
  "url": "https://example.com",
  "prompt": "サイト全体をクロールし、/admin と /api は除外してください"
}   
```

<div id="response">
  ### レスポンス
</div>

```json
{
  "url": "https://example.com",
  "prompt": "ウェブサイト全体をクロールする",
  "crawlEntireDomain": true,
  "excludePaths": ["/admin/.*", "/api/.*"],
  "sitemap": "include"
}
```
