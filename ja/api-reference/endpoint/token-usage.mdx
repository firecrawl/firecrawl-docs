<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="see-where-your-tokens-go">
  ### トークンの使われ方を可視化
</div>

- FIRE-1（エージェント）のトークン使用量を追跡
- URL不要の抽出にかかるコストを確認
- 複雑なデータ抽出のトークン消費を監視
- バッチ抽出のトラッキングを強化

<div id="more-token-details">
  ### トークンの詳細
</div>

- 複雑なウェブインタラクションに必要なトークンを確認
- JavaScriptが重いサイトで発生する追加トークンを追跡
- 精度向上のためのトークン消費を監視
- 自動ページネーションにかかるコストを確認

<div id="faster-and-more-accurate">
  ### さらに高速かつ高精度に
</div>

- トークン使用量データへ迅速にアクセス
- Extract 機能のコスト内訳をより明確に
- FIRE-1 の操作をより精密にトラッキング

API は v1 と同一のまま、v2 の Extract 機能でトークン使用量の可視性が向上しました。
