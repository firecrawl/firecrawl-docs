<div id="how-it-works">
  ## 仕組み
</div>

WebサイトをAI対応のデータへ変換します。リアルタイムのWeb知識でチャットボットを強化し、最新ドキュメントでRAGシステムを構築し、ユーザーがAIアプリケーションをWebソースに接続できるようにします。

<div id="why-ai-platforms-choose-firecrawl">
  ## なぜAIプラットフォームはFirecrawlを選ぶのか
</div>

<div id="reduce-hallucinations-with-real-time-data">
  ### リアルタイムデータでハルシネーションを抑制
</div>

AIアシスタントには、古い学習データではなく最新情報が必要です。ドメイン固有の知識、技術ドキュメント、業界向けコンテンツのいずれであっても、Firecrawl はナレッジベースを最新の更新内容と同期させ、ハルシネーションを抑え、応答の正確性を高めます。

<div id="faqs">
  ## よくある質問
</div>

<AccordionGroup>
  <Accordion title="Firecrawl はどのように AI 開発プラットフォームと統合されますか？">
    Firecrawl は、AI プラットフォームに直接統合できるシンプルな API と SDK を提供します。LangChain を用いた開発、n8n のようなノーコードツール、あるいは独自フレームワークでも、Firecrawl は AI が扱いやすいクリーンで構造化されたウェブデータを提供します。
  </Accordion>

  <Accordion title="Firecrawl は AI 学習に必要なスケールに対応できますか？">
    はい。Firecrawl はエンタープライズ規模のデータ抽出向けに設計されており、AI 学習用データセットのために数百万ページを処理できます。インフラはニーズに応じて自動的にスケールします。
  </Accordion>

  <Accordion title="AI プラットフォーム向けに Firecrawl が提供するデータフォーマットは何ですか？">
    Firecrawl は、クリーンな Markdown、構造化された JSON、raw HTML、抽出画像、スクリーンショット、ニュースコンテンツなど、AI に適したフォーマットでデータを提供します。この柔軟性により、あらゆる AI プラットフォームのデータ取り込み要件に対応できます。
  </Accordion>

  <Accordion title="認証が必要なコンテンツへのアクセスはどのように処理しますか？">
    Firecrawl は、保護されたコンテンツへのアクセスに必要な認証ヘッダーやクッキーをサポートします。ログインが必要なドキュメント、ナレッジベース、会員限定サイトからデータを抽出するには、必要な認証情報を API リクエストに設定してください。
  </Accordion>

  <Accordion title="リアルタイムの AI アプリケーションに Firecrawl を利用できますか？">
    できます。API はリアルタイムのデータ抽出をサポートし、AI アプリケーションがオンデマンドで最新のウェブデータにアクセスできるようにします。最新情報を基に意思決定する AI エージェントに最適です。
  </Accordion>
</AccordionGroup>
