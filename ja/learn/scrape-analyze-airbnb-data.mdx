> 注: この例では [Firecrawl API の v0 版](/ja/v0/introduction) を使用しています。Python SDK は 0.0.20、Node SDK は 0.0.36 をインストールしてください。

> [GitHub リポジトリ (https://github.com/e2b-dev/e2b-cookbook)](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)

この例では、[Firecrawl](https://www.firecrawl.dev/) と E2B の [Code Interpreter SDK](https://github.com/e2b-dev/code-interpreter) を使って Airbnb のデータをスクレイピングし、分析する方法を示します。

分析用のコードは一切書かず、代わりに Anthropic の AI に作成させます。コードを安全に実行して結果を取得するために、E2B の Code Interpreter SDK を使用します。

<div id="prerequisites">
  ## 前提条件
</div>

* マシンに Node.js がインストールされていること
* E2B の API キー
* Firecrawl の API キー
* Anthropic の API キー

<div id="setup-run">
  ## セットアップと実行
</div>

<div id="1-clone-the-repository">
  ### 1. リポジトリをクローンする
</div>

この例の [GitHub リポジトリ](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)はこちら。

```
git clone https://github.com/e2b-dev/e2b-cookbook.git
cd e2b-cookbook/examples/scrape-and-analyze-airbnb-data-with-firecrawl
```

<div id="2-install-dependencies">
  ### 2. 依存関係のインストール
</div>

```
npm install
```

<div id="3-set-up-env">
  ### 3. `.env` をセットアップする
</div>

1. `.env.template` を `.env` にコピーする
2. [E2B APIキー](https://e2b.dev/docs/getting-started/api-key) を取得する
3. [Firecrawl APIキー](https://firecrawl.dev) を取得する
4. [Anthropic APIキー](https://anthropic.com) を取得する

<div id="4-run-the-example">
  ### 4. 例を実行する
</div>

```
npm run start
```
