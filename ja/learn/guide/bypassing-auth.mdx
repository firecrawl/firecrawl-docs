> 注意: この例は [Firecrawl API の v0 版](/ja/v0/introduction) を使用しています。Python SDK は 0.0.20、Node SDK は 0.0.36 をインストールしてください。

Firecrawl では、ヘッダーに Cookie を送信することでウェブサイトの認証を回避できます。ログインが必要なサイトからコンテンツをスクレイピングする際に役立ちます。

<div id="example">
  ### 例
</div>

認証をバイパスするには、リクエストに `headers` パラメータを含め、ブラウザのクッキーから取得した適切な値で `Cookies` ヘッダーを設定します。クッキーの取得方法が必要な場合は、次の記事を参照してください: [Grabbing cookies](https://cookie-script.com/documentation/how-to-check-cookies-on-chrome-and-firefox)

<div id="usage">
  ### 使い方
</div>

<CodeGroup>
  ```python Python
  from firecrawl import FirecrawlApp

  app = FirecrawlApp(api_key="YOUR_API_KEY")

  app.scrape_url("https://docs.firecrawl.dev", {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  })
  ```

  ```javascript JavaScript
  import { FirecrawlApp } from 'firecrawl-js';

  const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

  app.scrapeUrl('https://docs.firecrawl.dev', {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  });
  ```

  ```go Go
  import (
    "fmt"
    "log"

    "github.com/mendableai/firecrawl-go"
  )

  func main() {
    // APIキーで FirecrawlApp を初期化
    app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
    if err != nil {
      log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
    }

    params := map[string]any{
      "pageOptions": map[string]any{
        "headers": map[string]any{
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      },
    }
    scrapedData, err := app.ScrapeURL("docs.firecrawl.dev", params)
    if err != nil {
      log.Fatalf("Error occurred while scraping: %v", err)
    }
    fmt.Println(scrapedData)
  }
  ```

  ```rust Rust
  use firecrawl::FirecrawlApp;

  #[tokio::main]
  async fn main() {
      let api_key = "YOUR_API_KEY";
      let api_url = "https://api.firecrawl.dev";
      let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

      let scrape_params = json!({
          "pageOptions": {
              "headers": {
                  "Cookie": "cookie1=value1; cookie2=value2"
              }
          }
      });

      let scrape_result = app.scrape_url("https://docs.firecrawl.dev", Some(scrape_params)).await;
      match scrape_result {
          Ok(data) => println!("Scrape Result:\n{}", data),
          Err(e) => eprintln!("Scrape failed: {}", e),
      }
  }
  ```

  ```curl cURL
  curl -X POST https://api.firecrawl.dev/v0/scrape \
      -H 'Content-Type: application/json' \
      -H 'Authorization: Bearer YOUR_API_KEY' \
      -d '{
        "url": "https://example.com",
        "headers": {
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      }'
  ```
