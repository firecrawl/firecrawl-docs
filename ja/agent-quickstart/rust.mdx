<div id="firecrawl-rust-agent-quickstart">
  # Firecrawl Rust Agent クイックスタート
</div>

このファイルは、外部エージェント向けの正規のクイックスタートであり、SDK のソースコードと v2 OpenAPI 仕様から生成されています。

<div id="install">
  ## インストール
</div>

```bash
cargo add firecrawl
```

<div id="authenticate">
  ## 認証
</div>

```rust
use firecrawl::v2::Client;

let client = Client::new("fc-your-api-key")?;
```

<div id="when-to-use-what">
  ## 使い分けの目安
</div>

* `search`: クエリから始めて、まず対象を見つけたいときに使います。
* `scrape`: すでに URL があり、ページのコンテンツを取得したいときに使います。
* `interact`: ページ上でクリックやフォーム操作、またはスクレイピング後のブラウザー操作が必要なときに使います。

<div id="search">
  ## 検索
</div>

### 使う理由

`search` を使うと、クエリから関連するURLを見つけ、必要に応じて各結果をスクレイピングできます。特定のURLをスクレイピングする前に、標準の `site:` 演算子 (例: `site:docs.firecrawl.dev crawl webhooks`) を使って、クエリの対象を1つのサイトに絞れます。

<div id="preferred-sdk-method">
  ### 推奨されるSDKメソッド
</div>

`client.search(query, options)`

### 簡単な例

```rust
use firecrawl::v2::Client;

let client = Client::new("fc-your-api-key")?;
let results = client.search("site:docs.firecrawl.dev crawl webhooks", None).await?;

if let Some(web) = results.data.web {
    println!("{:?}", web.first());
}
```

<div id="complex-example">
  ### 複雑な例
</div>

```rust
use firecrawl::v2::{Client, SearchOptions, SearchSource, SearchCategory, ScrapeOptions, Format};

let client = Client::new("fc-your-api-key")?;
let options = SearchOptions {
    sources: Some(vec![SearchSource::Web, SearchSource::News]),
    categories: Some(vec![SearchCategory::Research]),
    limit: Some(10),
    tbs: Some("qdr:w".to_string()),
    location: Some("San Francisco,California,United States".to_string()),
    ignore_invalid_urls: Some(true),
    scrape_options: Some(ScrapeOptions {
        formats: Some(vec![Format::Markdown, Format::Json]),
        only_main_content: Some(true),
        ..Default::default()
    }),
    integration: Some("agent-quickstart".to_string()),
};

let results = client.search("firecrawl pricing", options).await?;
```

<div id="parameters">
  ### パラメータ
</div>

* `query`: 検索クエリ文字列。常に必須です。
* `limit`: 返す結果の最大件数。レスポンスサイズの上限を設定するために使用します。
* `sources`: 検索対象のソース。サポートされる値: `SearchSource::Web`, `SearchSource::News`, `SearchSource::Images`。
* `categories`: 結果の絞り込み条件。サポートされる値: `SearchCategory::Github`, `SearchCategory::Research`, `SearchCategory::Pdf`。
* `tbs`: 時間ベースの検索絞り込み文字列。時間範囲で絞り込むために使用します (例: `qdr:d`) 。
* `location`: ローカライズされた結果のためのロケーション文字列。地域指定のクエリに使用します。
* `ignore_invalid_urls`: 結果から無効な URL を除外します。他のエンドポイントに渡す場合に使用します。
* `timeout`: リクエストのタイムアウト (ミリ秒) 。検索時間の上限を設定するために使用します。
* `scrape_options`: 各結果に適用されるスクレイピングのオプション (`scrape` と同じフィールド) 。結果と一緒に content も取得したい場合に使用します。
* `integration`: 連携の識別子。追跡や分析に使用します。

<div id="scrape">
  ## スクレイピング
</div>

<div id="why-use-it">
  ### これを使う理由
</div>

URL がすでにわかっており、特定のフォーマットでコンテンツを取得したい場合は、`scrape` を使用します。

### 推奨SDKメソッド

`client.scrape(url, options)`

<div id="simple-example">
  ### シンプルな例
</div>

```rust
use firecrawl::v2::{Client, ScrapeOptions, Format};

let client = Client::new("fc-your-api-key")?;
let doc = client
    .scrape(
        "https://example.com",
        ScrapeOptions {
            formats: Some(vec![Format::Markdown]),
            ..Default::default()
        },
    )
    .await?;

println!("{:?}", doc.markdown);
```

<div id="complex-example">
  ### 複雑な例
</div>

```rust
use firecrawl::v2::{
    Action, ChangeTrackingMode, ChangeTrackingOptions, Client, Format, JsonOptions, LocationConfig,
    ParserConfig, ProxyType, ScrapeOptions, ScreenshotOptions, ScrollDirection, Viewport,
};

let client = Client::new("fc-your-api-key")?;
let options = ScrapeOptions {
    formats: Some(vec![Format::Markdown, Format::Json, Format::ChangeTracking, Format::Screenshot]),
    json_options: Some(JsonOptions {
        prompt: Some("Extract headings and slugs.".to_string()),
        ..Default::default()
    }),
    change_tracking_options: Some(ChangeTrackingOptions {
        modes: Some(vec![ChangeTrackingMode::GitDiff]),
        tag: Some("docs".to_string()),
        ..Default::default()
    }),
    screenshot_options: Some(ScreenshotOptions {
        full_page: Some(true),
        viewport: Some(Viewport { width: 1280, height: 720 }),
        ..Default::default()
    }),
    actions: Some(vec![
        Action::Wait {
            milliseconds: Some(1000),
            selector: None,
        },
        Action::Click {
            selector: "button#accept".to_string(),
        },
        Action::Scroll {
            direction: ScrollDirection::Down,
            selector: None,
        },
    ]),
    parsers: Some(vec![ParserConfig::Pdf {
        parser_type: "pdf".to_string(),
        max_pages: Some(5),
    }]),
    location: Some(LocationConfig {
        country: Some("US".to_string()),
        languages: Some(vec!["en-US".to_string()]),
    }),
    proxy: Some(ProxyType::Stealth),
    ..Default::default()
};

let doc = client.scrape("https://example.com/docs", options).await?;
```

<div id="parameters">
  ### パラメータ
</div>

* `url`: スクレイピング対象のURL。常に必須です。
* `formats`: 含める出力フォーマット。サポートされる値: `Format::Markdown`, `Format::Html`, `Format::RawHtml`, `Format::Links`, `Format::Images`, `Format::Screenshot`, `Format::Summary`, `Format::ChangeTracking`, `Format::Json`, `Format::Attributes`, `Format::Branding`, `Format::Audio`.
* `headers`: 追加のHTTPヘッダー。認証、Cookie、ユーザーエージェントの指定に使用します。
* `include_tags`: 指定したHTMLタグのコンテンツのみを含めます。抽出対象を絞り込む際に使用します。
* `exclude_tags`: 指定したHTMLタグのコンテンツを除外します。ノイズを取り除く際に使用します。
* `only_main_content`: メインコンテンツのみを抽出します。ナビゲーションやフッターを除外する際に使用します。
* `timeout`: スクレイピングのタイムアウト (ミリ秒) 。実行時間の上限を設定する際に使用します。
* `wait_for`: スクレイピング前の待機時間 (ミリ秒) 。JSでレンダリングされるページで使用します。
* `mobile`: モバイルデバイスをエミュレートします。モバイル向けレイアウトで使用します。
* `parsers`: パーサーのリストまたは設定。サポートされる値: `ParserConfig::Simple(String)` または `ParserConfig::Pdf { parser_type: "pdf", max_pages: Option<u32> }`.
* `actions`: スクレイピング前に実行するブラウザのアクション。サポートされるアクション: `Wait { milliseconds?, selector? }`, `Screenshot { full_page?, quality?, viewport? }`, `Click { selector }`, `Write { text }`, `Press { key }`, `Scroll { direction: Up | Down, selector? }`, `Scrape`, `ExecuteJavascript { script }`, `Pdf { format?, landscape?, scale? }`。使用可能なフォーマットは `A0`-`A6`, `Letter`, `Legal`, `Tabloid`, `Ledger` です。
* `location`: ロケーション設定 `{ country, languages }`。地域別のページで使用します。
* `skip_tls_verification`: TLS証明書の確認をスキップします。自己署名証明書で使用します。
* `remove_base64_images`: markdown出力からbase64画像を削除します。payloadサイズを削減する際に使用します。
* `block_ads`: 広告とCookieのブロックを有効にします。ノイズを減らす際に使用します。
* `proxy`: プロキシモード `ProxyType::Basic`, `ProxyType::Stealth`, `ProxyType::Enhanced`, `ProxyType::Auto`.
* `max_age`: この期間 (ms) より新しいキャッシュ済みコンテンツを許可します。キャッシュヒットを優先する際に使用します。
* `min_age`: 少なくともこの期間 (ms) 経過したキャッシュ済みコンテンツのみを返します。キャッシュのみを参照する場合に使用します。
* `store_in_cache`: 結果をキャッシュ/インデックスに保存します。再利用したい場合に使用します。
* `profile`: 永続化されたブラウザプロファイル `ProfileConfig { name, save_changes }`。セッション状態を維持する際に使用します。
* `integration`: インテグレーション識別子。追跡や分析に使用します。
* `json_options`: `Format::Json` フォーマット用のJSON extractionオプション (`schema`, `system_prompt`, `prompt`) 。
* `screenshot_options`: `Format::Screenshot` フォーマット用のスクリーンショットオプション (`full_page`, `quality`, `viewport`) 。
* `change_tracking_options`: `Format::ChangeTracking` フォーマット用のchange trackingオプション (`modes`, `schema`, `prompt`, `tag`) 。モード: `ChangeTrackingMode::GitDiff`, `ChangeTrackingMode::Json`.
* `attribute_selectors`: `Format::Attributes` フォーマット用の属性セレクター (`selector`, `attribute`) 。

<div id="interact">
  ## Interact
</div>

### なぜ使うのか

ページでスクレイピング後にクリック、フォーム入力、その他のブラウザアクションが必要な場合は、`interact` を使用します。

### 推奨SDKメソッド

`client.interact(job_id, options)`

<div id="simple-example">
  ### シンプルな例
</div>

```rust
use firecrawl::v2::{Client, ScrapeExecuteOptions};

let client = Client::new("fc-your-api-key")?;
let run = client
    .interact(
        "scrape-job-id",
        ScrapeExecuteOptions {
            code: Some("console.log(await page.url())".to_string()),
            ..Default::default()
        },
    )
    .await?;

println!("{:?}", run.stdout);
```

<div id="complex-example">
  ### 複雑な例
</div>

```rust
use firecrawl::v2::{Client, ScrapeExecuteLanguage, ScrapeExecuteOptions};

let client = Client::new("fc-your-api-key")?;
let run = client
    .interact(
        "scrape-job-id",
        ScrapeExecuteOptions {
            prompt: Some("Open the pricing page and list the plan names.".to_string()),
            language: Some(ScrapeExecuteLanguage::Node),
            timeout: Some(60),
            origin: Some("agent-docs".to_string()),
            ..Default::default()
        },
    )
    .await?;
```

<div id="parameters">
  ### パラメータ
</div>

* `job_id`: スクレイピングジョブ ID (`document.metadata.scrape_id` から取得) 。必須です。
* `code`: ブラウザサンドボックスで実行するコード。スクリプトによる操作に使用します。
* `prompt`: ブラウザエージェントへの自然言語指示。エージェント主導のアクションを使いたい場合に使用します。
* `language`: ランタイム言語 `ScrapeExecuteLanguage::Python`、`ScrapeExecuteLanguage::Node`、または `ScrapeExecuteLanguage::Bash` (デフォルトは `Node`) 。
* `timeout`: 実行タイムアウト (秒) 。長時間実行を制限するために使用します。
* `origin`: 任意のオリジンタグ。リクエスト元の識別に使用します。

<div id="notes">
  ## 注意事項
</div>

* Rust ではパラメータは snake&#95;case で、JSON は camelCase としてシリアライズされます。
* 便利なメソッド: `search_and_scrape` は、スクレイピングを有効にした状態で search を実行します。
* 非推奨のエイリアス: `scrape_execute`、`stop_interactive_browser`、`delete_scrape_browser` (`interact` と `stop_interaction` を使用してください) 。
