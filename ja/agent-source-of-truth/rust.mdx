<div id="firecrawl-rust-source-of-truth">
  # Firecrawl Rust の信頼できる唯一の情報源
</div>

このファイルは、エージェント向けの公開された正式な信頼できる唯一の情報源です。Firecrawl Rust SDK のソースコードと v2 OpenAPI 仕様から生成されています。

<div id="install">
  ## インストール
</div>

```bash
cargo add firecrawl
```

<div id="authenticate">
  ## 認証
</div>

```rust
use firecrawl::v2::Client;

let client = Client::new("fc-your-api-key")?;
// let client = Client::new_selfhosted("https://api.firecrawl.dev", Some("fc-your-api-key"))?;
```

<div id="when-to-use-what">
  ## 何をいつ使うか
</div>

* `search`: クエリから始めて、`site:` 制約を含む関連 URL を見つけます。
* `scrape`: 既知の URL からコンテンツを取得して抽出します。
* `interact`: ページでクリックやフォーム入力、またはスクレイピング後の処理が必要な場合に、ブラウザのアクションやコードを実行します。

<div id="search">
  ## 検索
</div>

<div id="why-use-it">
  ### どのような場合に使うか
</div>

クエリに基づいて候補を見つけ、Firecrawl に URL を返してほしい場合に search を使用します (必要に応じて各結果をスクレイピングすることもできます) 。たとえば、標準の `site:` 演算子を使うと、`site:docs.firecrawl.dev crawl webhooks` のように結果を特定のドメインに絞り込めます。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.search(query, options)`

<div id="simple-example">
  ### シンプルな例
</div>

```rust
use firecrawl::v2::Client;

let results = client
    .search("site:docs.firecrawl.dev webhook retries", None)
    .await?;
```

### 複雑な例

```rust
use firecrawl::v2::{
    Client, SearchOptions, SearchSource, SearchCategory, ScrapeOptions, Format, JsonOptions
};

let options = SearchOptions {
    sources: Some(vec![SearchSource::Web, SearchSource::News]),
    categories: Some(vec![SearchCategory::Research]),
    limit: Some(10),
    tbs: Some("qdr:m".to_string()),
    location: Some("San Francisco,California,United States".to_string()),
    ignore_invalid_urls: Some(true),
    timeout: Some(60000),
    scrape_options: Some(ScrapeOptions {
        formats: Some(vec![Format::Markdown, Format::Links, Format::Json]),
        json_options: Some(JsonOptions {
            prompt: Some("Extract title and key endpoints.".to_string()),
            ..Default::default()
        }),
        only_main_content: Some(true),
        ..Default::default()
    }),
    ..Default::default()
};

let results = client
    .search("site:docs.firecrawl.dev crawl webhooks", options)
    .await?;
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ                         | 型 / オプション                                           | 説明                                                |
| ----------------------------- | --------------------------------------------------- | ------------------------------------------------- |
| `query`                       | `&str`                                              | 検索クエリ。`site:domain` を使うと、結果を特定のサイトに限定できます。        |
| `options.sources`             | `Vec<SearchSource>` (`Web`, `News`, `Images`)       | 検索対象のソース。                                         |
| `options.categories`          | `Vec<SearchCategory>` (`Github`, `Research`, `Pdf`) | 結果を絞り込むカテゴリ。                                      |
| `options.limit`               | `u32`                                               | 返される結果の最大数。既定値は 5、最大値は 20 です (SDK ドキュメントによる) 。    |
| `options.tbs`                 | `String`                                            | 時間ベースの検索フィルタ (例: `qdr:d`、`qdr:w`、`sbd:1,qdr:m`) 。 |
| `options.location`            | `String`                                            | 検索結果のロケーションを指定する文字列。                              |
| `options.ignore_invalid_urls` | `bool`                                              | 他の Firecrawl エンドポイントでスクレイピングできない無効な URL を除外します。   |
| `options.timeout`             | `u32`                                               | ミリ秒単位のタイムアウト。                                     |
| `options.scrape_options`      | `ScrapeOptions`                                     | 各結果に適用されるスクレイピングのオプション (`scrape` と同じフィールド) 。      |
| `options.integration`         | `String`                                            | 内部専用。Firecrawl の計測用に予約されています。                     |

<div id="scrape">
  ## スクレイピング
</div>

### これを使う理由

URLがすでにあり、Firecrawl から構造化されたコンテンツや任意のフォーマットを取得したい場合は、スクレイピングを使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.scrape(url, options)`

### 簡単な例

```rust
use firecrawl::v2::{Client, ScrapeOptions, Format};

let doc = client
    .scrape("https://docs.firecrawl.dev", ScrapeOptions {
        formats: Some(vec![Format::Markdown]),
        ..Default::default()
    })
    .await?;
```

### 複雑な例

```rust
use firecrawl::v2::{
    Client, ScrapeOptions, Format, JsonOptions, ScreenshotOptions, ChangeTrackingOptions,
    ChangeTrackingMode, AttributeSelector, Action, ScrollDirection, ProxyType, ParserConfig
};

let options = ScrapeOptions {
    formats: Some(vec![
        Format::Markdown,
        Format::Links,
        Format::Json,
        Format::Screenshot,
        Format::ChangeTracking,
        Format::Attributes,
    ]),
    json_options: Some(JsonOptions {
        prompt: Some("Extract plan names and prices.".to_string()),
        ..Default::default()
    }),
    screenshot_options: Some(ScreenshotOptions {
        full_page: Some(true),
        quality: Some(80),
        viewport: None,
    }),
    change_tracking_options: Some(ChangeTrackingOptions {
        modes: Some(vec![ChangeTrackingMode::GitDiff]),
        tag: Some("pricing".to_string()),
        ..Default::default()
    }),
    attribute_selectors: Some(vec![AttributeSelector {
        selector: "a".to_string(),
        attribute: "href".to_string(),
    }]),
    headers: None,
    include_tags: Some(vec!["main".to_string(), "article".to_string()]),
    exclude_tags: Some(vec!["nav".to_string(), "footer".to_string()]),
    only_main_content: Some(true),
    wait_for: Some(1000),
    parsers: Some(vec![ParserConfig::Pdf {
        parser_type: "pdf".to_string(),
        max_pages: Some(5),
    }]),
    actions: Some(vec![
        Action::Click { selector: "#accept".to_string() },
        Action::Write { text: "firecrawl".to_string() },
        Action::Press { key: "Enter".to_string() },
        Action::Wait { milliseconds: Some(1500), selector: None },
        Action::Scroll { direction: ScrollDirection::Down, selector: None },
        Action::Scrape,
    ]),
    skip_tls_verification: Some(true),
    remove_base64_images: Some(true),
    fast_mode: Some(true),
    block_ads: Some(true),
    proxy: Some(ProxyType::Auto),
    max_age: Some(86400000),
    min_age: None,
    store_in_cache: Some(true),
    ..Default::default()
};

let doc = client.scrape("https://example.com/pricing", options).await?;
```

<div id="interact">
  ## Interact
</div>

### なぜこれを使うのか

ページで、スクレイピング開始後にブラウザのアクションやコードの実行 (クリック、フォーム入力、スクリプトによる手順など) が必要な場合は、interact を使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.interact(job_id, options)`

### 簡単な例

```rust
use firecrawl::v2::{Client, ScrapeOptions, Format, ScrapeExecuteOptions};

let doc = client
    .scrape("https://example.com", ScrapeOptions {
        formats: Some(vec![Format::Markdown]),
        ..Default::default()
    })
    .await?;

let job_id = doc
    .metadata
    .as_ref()
    .and_then(|m| m.scrape_id.clone())
    .expect("Missing scrape_id");

let result = client
    .interact(job_id, ScrapeExecuteOptions {
        prompt: Some("Click the pricing tab and summarize the plans.".to_string()),
        language: None,
        ..Default::default()
    })
    .await?;
```

<div id="complex-example">
  ### 複雑な使用例
</div>

```rust
use firecrawl::v2::{Client, ScrapeExecuteOptions, ScrapeExecuteLanguage};

let result = client
    .interact(
        "<scrape_job_id>",
        ScrapeExecuteOptions {
            code: Some("console.log(await page.title());".to_string()),
            language: Some(ScrapeExecuteLanguage::Node),
            timeout: Some(60),
            ..Default::default()
        },
    )
    .await?;
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ              | 型 / オプション                                          | 説明                                                  |
| ------------------ | -------------------------------------------------- | --------------------------------------------------- |
| `job_id`           | `&str`                                             | `document.metadata.scrape_id` から取得するスクレイピング job ID。 |
| `options.code`     | `String`                                           | ブラウザセッションで実行するコード。`prompt` が指定されていない場合は必須です。        |
| `options.prompt`   | `String`                                           | ブラウザ エージェントに対する自然言語の指示。`code` が指定されていない場合は必須です。     |
| `options.language` | `ScrapeExecuteLanguage` (`Python`, `Node`, `Bash`) | 実行言語。省略した場合のデフォルトは `Node` です。                       |
| `options.timeout`  | `u32`                                              | 実行タイムアウト (秒) 。                                      |
| `options.origin`   | `String`                                           | 内部専用。Firecrawl のインストルメンテーション用に予約されています。             |

<div id="notes">
  ## 注意
</div>

* 非推奨エイリアス: `scrape_execute`、`stop_interactive_browser`、`delete_scrape_browser` は、`interact` と `stop_interaction` に対応しています。
* Rust の構造体フィールドは snake&#95;case ですが、serde によって camelCase の API フィールドへシリアライズされます。
