<div id="firecrawl-python-source-of-truth">
  # Firecrawl Python の信頼できる唯一の参照元
</div>

このファイルは、エージェント向けの信頼できる唯一の公開参照元です。Firecrawl Python SDK のソースコードと v2 OpenAPI 仕様から生成されています。

<div id="install">
  ## インストール
</div>

```bash
pip install firecrawl-py
```

<div id="authenticate">
  ## 認証
</div>

```py
import os
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ.get("FIRECRAWL_API_KEY"))
# client = Firecrawl(api_key="fc-...", api_url="https://api.firecrawl.dev")
```

<div id="when-to-use-what">
  ## 使い分け
</div>

* `search`: クエリから始めて、`site:` 制約を含む関連 URL を見つけます。
* `scrape`: 既知の URL からコンテンツを取得・抽出します。
* `interact`: クリックやフォーム入力、スクレイピング後の処理が必要なページで、ブラウザのアクションやコードを実行します。

<div id="search">
  ## 検索
</div>

<div id="why-use-it">
  ### 使用する理由
</div>

クエリからURLを探索し、Firecrawlに各結果を返させたい場合 (任意でスクレイピングも可) に検索を使用します。標準の `site:` 演算子でドメインに結果を絞り込むことができます (例：`site:docs.firecrawl.dev crawl webhooks`) 。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.search(query, **options)`

<div id="simple-example">
  ### シンプルな例
</div>

```py
results = client.search("site:docs.firecrawl.dev webhook retries")
```

<div id="complex-example">
  ### 複雑な例
</div>

```py
from firecrawl.v2.types import ScrapeOptions

results = client.search(
    "site:docs.firecrawl.dev crawl webhooks",
    sources=["web", "news"],
    categories=["research"],
    limit=10,
    tbs="qdr:m",
    location="San Francisco,California,United States",
    ignore_invalid_urls=True,
    timeout=300000,
    scrape_options=ScrapeOptions(
        formats=[
            "markdown",
            "links",
            {"type": "json", "prompt": "Extract title and key endpoints."}
        ],
        only_main_content=True,
        include_tags=["main", "article"],
        exclude_tags=["nav", "footer"],
        wait_for=1000,
        actions=[
            {"type": "click", "selector": "button#accept"},
            {"type": "wait", "milliseconds": 750},
            {"type": "scrape"}
        ]
    )
)
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ                 | 型 / オプション                           | 説明                                                             |
| --------------------- | ----------------------------------- | -------------------------------------------------------------- |
| `query`               | `str`                               | 検索クエリ。`site:domain` を使用して結果を特定のサイトに絞り込めます。                     |
| `sources`             | `source名のリストまたは型付きsourceオブジェクト`     | 検索対象のソース。                                                      |
| `categories`          | `category名のリストまたは型付きcategoryオブジェクト` | 結果を絞り込むカテゴリ。                                                   |
| `limit`               | `int`                               | 返す結果の最大件数。SDKモデルのデフォルトは `5`。                                   |
| `tbs`                 | `str`                               | 時間ベースの検索フィルター (例：`qdr:d`、`qdr:w`、`sbd:1,qdr:m`) 。              |
| `location`            | `str`                               | 検索結果に使用するロケーション文字列。                                            |
| `ignore_invalid_urls` | `bool`                              | 他のFirecrawlエンドポイントでスクレイピングできない無効なURLを除外します。                    |
| `timeout`             | `int`                               | タイムアウト (ミリ秒) 。SDKモデルのデフォルトは `300000`。                          |
| `scrape_options`      | `ScrapeOptions`                     | 各結果に適用されるスクレイピングのオプション (`scrape` と同じフィールドに加え `min_age` も指定可) 。 |
| `integration`         | `str`                               | 内部専用。Firecrawlのインストルメンテーション用に予約済み。                             |

<div id="scrape">
  ## スクレイピング
</div>

<div id="why-use-it">
  ### 使用する理由
</div>

URLがすでに分かっていて、Firecrawl から構造化されたコンテンツと任意のフォーマットを返してほしい場合は、スクレイピングを使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.scrape(url, **options)`

### 簡単な例

```py
doc = client.scrape("https://docs.firecrawl.dev", formats=["markdown"])
```

<div id="complex-example">
  ### 複雑な例
</div>

```py
doc = client.scrape(
    "https://example.com/pricing",
    formats=[
        "markdown",
        "links",
        {"type": "json", "prompt": "Extract plan names and prices."},
        {"type": "screenshot", "full_page": True, "quality": 80, "viewport": {"width": 1280, "height": 720}},
        {"type": "changeTracking", "modes": ["git-diff"], "tag": "pricing"}
    ],
    headers={"User-Agent": "FirecrawlDocsBot/1.0"},
    include_tags=["main", "article"],
    exclude_tags=["nav", "footer"],
    only_main_content=True,
    wait_for=1000,
    mobile=False,
    parsers=[{"type": "pdf", "mode": "auto", "max_pages": 5}],
    actions=[
        {"type": "click", "selector": "#accept"},
        {"type": "write", "text": "firecrawl"},
        {"type": "press", "key": "Enter"},
        {"type": "wait", "milliseconds": 1500},
        {"type": "scrape"}
    ],
    location={"country": "US", "languages": ["en-US"]},
    skip_tls_verification=True,
    remove_base64_images=True,
    fast_mode=True,
    block_ads=True,
    proxy="auto",
    max_age=86400000,
    store_in_cache=True,
    profile={"name": "docs-session", "save_changes": True}
)
```

<div id="interact">
  ## Interact
</div>

### なぜ使うのか

ページで、スクレイピング開始後にブラウザのアクションやコード実行 (クリック、フォーム操作、スクリプト化された手順など) が必要な場合は、interact を使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.interact(job_id, code=None, prompt=None, language="node", timeout=None)`

<div id="simple-example">
  ### シンプルな例
</div>

```py
doc = client.scrape("https://example.com", formats=["markdown"])
job_id = doc.metadata.scrape_id if doc.metadata else None
if not job_id:
    raise ValueError("Missing scrape_id from scrape response")

result = client.interact(
    job_id,
    prompt="Click the pricing tab and summarize the plans.",
    language="node"
)
```

<div id="complex-example">
  ### 複雑な例
</div>

```py
result = client.interact(
    "<scrape_job_id>",
    code="print(await page.title())",
    language="python",
    timeout=60
)
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ      | 型 / オプション               | 説明                                             |
| ---------- | ----------------------- | ---------------------------------------------- |
| `job_id`   | `str`                   | `document.metadata.scrape_id` のスクレイピングジョブ ID。  |
| `code`     | `str`                   | ブラウザセッションで実行するコード。`prompt` が指定されていない場合は必須です。   |
| `prompt`   | `str`                   | ブラウザエージェントに対する自然言語の指示。`code` が指定されていない場合は必須です。 |
| `language` | `python, node, or bash` | 実行言語。デフォルトは `node` です。                         |
| `timeout`  | `int`                   | 実行タイムアウト (秒、1～300) 。                           |
| `origin`   | `str`                   | 内部専用。Firecrawl のインストルメンテーション用に予約されています。        |

<div id="notes">
  ## 注記
</div>

* 非推奨のエイリアス: `scrape_execute`、`stop_interactive_browser`、`delete_scrape_browser` は、`interact` と `stop_interaction` にマップされています。
* `ScrapeOptions` には `min_age` が含まれますが、`scrape` の簡易メソッドではこれを指定できません。必要な場合は、`search(..., scrape_options=...)` 経由で `ScrapeOptions` を使用してください。
* トップレベルの `Firecrawl` クライアントでは v2 メソッドを直接利用でき、従来の v1 は `client.v1` で利用できます。
