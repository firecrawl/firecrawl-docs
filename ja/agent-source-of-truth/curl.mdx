<div id="firecrawl-curl-source-of-truth">
  # Firecrawl curl の正本
</div>

このファイルは、エージェント向けの正式な公開正本です。Firecrawl v2 の OpenAPI 仕様および SDK の公開インターフェースから生成されています。



<div id="authenticate">
  ## 認証
</div>

すべてのリクエストで、`Authorization` ヘッダーに Bearer トークンを使用します。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```



<div id="when-to-use-what">
  ## 使い分け
</div>

* `search`: クエリから開始して、`site:` 制約を含む関連URLを見つけます。
* `scrape`: 既知のURLからコンテンツを取得・抽出します。
* `interact`: ページでクリックやフォーム入力、またはスクレイピング後の処理が必要な場合に、ブラウザのアクションやコードを実行します。



<div id="search">
  ## Search
</div>

<div id="why-use-it">
  ### 使用する理由
</div>

クエリからURLを探索し、Firecrawlに各結果を返させたい場合（必要に応じて各結果をスクレイピングすることも可能）に検索を使用します。標準の `site:` 演算子を使ってドメインに結果を絞り込むことができます（例：`site:docs.firecrawl.dev crawl webhooks`）。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`POST https://api.firecrawl.dev/v2/search`

<div id="simple-example">
  ### シンプルな例
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev webhook retries"
  }'
```

<div id="complex-example">
  ### 複雑な例
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev crawl webhooks",
    "sources": ["web", "news"],
    "categories": ["research"],
    "limit": 10,
    "tbs": "qdr:m",
    "location": "San Francisco,California,United States",
    "country": "US",
    "ignoreInvalidURLs": true,
    "timeout": 60000,
    "scrapeOptions": {
      "formats": [
        "markdown",
        "links",
        { "type": "json", "prompt": "Extract title and key endpoints." }
      ],
      "onlyMainContent": true,
      "includeTags": ["main", "article"],
      "excludeTags": ["nav", "footer"],
      "waitFor": 1000,
      "actions": [
        { "type": "click", "selector": "button#accept" },
        { "type": "wait", "milliseconds": 750 },
        { "type": "scrape" }
      ]
    }
  }'
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ           | 型 / オプション  | 説明                                                             |
| ------------------- | --------------- | ----------------------------------------------------------------------- | -------------------------------------------------- | ---------------- | ---------- | ------------ | ----------------------------- |
| `query`             | `string`        | 検索クエリ。`site:domain` を使用して結果を特定のサイトに絞り込めます。          |
| `sources`           | `Array<"web"    | "news"                                                                  | "images"                                           | { type: "web"    | "news"     | "images" }>` | 検索対象のソース。            |
| `categories`        | `Array<"github" | "research"                                                              | "pdf"                                              | { type: "github" | "research" | "pdf" }>`    | 結果を絞り込むカテゴリ。 |
| `limit`             | `integer`       | 返す結果の最大件数。                                                    |
| `tbs`               | `string`        | 時間ベースの検索フィルター（例：`qdr:d`、`qdr:w`、`sbd:1,qdr:m`）。 |
| `location`          | `string`        | 検索結果のロケーション文字列。                                     |
| `country`           | `string`        | 検索結果のジオターゲティング用ISOカントリーコード。                      |
| `timeout`           | `integer`       | タイムアウト（ミリ秒単位）。                                                |
| `ignoreInvalidURLs` | `boolean`       | 他のFirecrawlエンドポイントでスクレイピングできない無効なURLを除外します。      |
| `enterprise`        | `Array<"anon"   | "zdr">`                                                                 | チームで有効化されている場合のEnterprise ZDRオプション。 |
| `scrapeOptions`     | `ScrapeOptions` | 各結果に適用するスクレイピングのオプション（`scrape` と同じフィールド）。        |


<div id="scrape">
  ## スクレイピング
</div>

### 使う理由

URL がすでにあり、Firecrawl から構造化されたコンテンツと任意のフォーマットを取得したい場合は、スクレイピングを使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`POST https://api.firecrawl.dev/v2/scrape`

<div id="simple-example">
  ### シンプルな例
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://docs.firecrawl.dev",
    "formats": ["markdown"]
  }'
```

### 高度な例

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/pricing",
    "formats": [
      "markdown",
      "links",
      { "type": "json", "prompt": "Extract plan names and prices." },
      { "type": "screenshot", "fullPage": true, "quality": 80, "viewport": { "width": 1280, "height": 720 } },
      { "type": "changeTracking", "modes": ["git-diff"], "tag": "pricing" }
    ],
    "headers": { "User-Agent": "FirecrawlDocsBot/1.0" },
    "includeTags": ["main", "article"],
    "excludeTags": ["nav", "footer"],
    "onlyMainContent": true,
    "waitFor": 1000,
    "mobile": false,
    "parsers": [{ "type": "pdf", "mode": "auto", "maxPages": 5 }],
    "actions": [
      { "type": "click", "selector": "#accept" },
      { "type": "write", "text": "firecrawl" },
      { "type": "press", "key": "Enter" },
      { "type": "wait", "milliseconds": 1500 },
      { "type": "scrape" }
    ],
    "location": { "country": "US", "languages": ["en-US"] },
    "skipTlsVerification": true,
    "removeBase64Images": true,
    "blockAds": true,
    "proxy": "auto",
    "maxAge": 86400000,
    "storeInCache": true,
    "profile": { "name": "docs-session", "saveChanges": true }
  }'
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ                 | 型 / オプション          | 説明                                                                                                                  |                   |          |
| --------------------- | ------------------ | ------------------------------------------------------------------------------------------------------------------- | ----------------- | -------- |
| `url`                 | `string`           | スクレイピング対象の URL。                                                                                                     |                   |          |
| `formats`             | `Array`            | 含める出力フォーマット。`{type: "json", prompt, schema}` や `{type: "screenshot", fullPage, quality, viewport}` などの文字列またはオブジェクト。 |                   |          |
| `onlyMainContent`     | `boolean`          | メインコンテンツのみを保持します。                                                                                                   |                   |          |
| `includeTags`         | `Array<string>`    | 含めるタグ。                                                                                                              |                   |          |
| `excludeTags`         | `Array<string>`    | 除外するタグ。                                                                                                             |                   |          |
| `maxAge`              | `integer`          | この時間以内のキャッシュデータを優先します。                                                                                              |                   |          |
| `minAge`              | `integer`          | 最小経過時間の条件を満たすキャッシュデータのみを返します。                                                                                       |                   |          |
| `headers`             | `object`           | カスタムヘッダー。                                                                                                           |                   |          |
| `waitFor`             | `integer`          | スクレイピング前の待機時間 (ミリ秒) 。                                                                                               |                   |          |
| `mobile`              | `boolean`          | モバイルデバイスをエミュレートします。                                                                                                 |                   |          |
| `skipTlsVerification` | `boolean`          | TLS 証明書の確認をスキップします。                                                                                                 |                   |          |
| `timeout`             | `integer`          | タイムアウト時間 (ミリ秒) 。                                                                                                    |                   |          |
| `parsers`             | `Array`            | パーサー設定 (PDF オプションをサポート) 。                                                                                           |                   |          |
| `actions`             | `Array`            | スクレイピング前に実行するブラウザアクション。                                                                                             |                   |          |
| `location`            | `object`           | リクエストのロケーション設定。                                                                                                     |                   |          |
| `location.country`    | `string`           | ISO 国コード。                                                                                                           |                   |          |
| `location.languages`  | `Array<string>`    | 優先言語。                                                                                                               |                   |          |
| `removeBase64Images`  | `boolean`          | Markdown 出力から Base64 画像を削除します。                                                                                      |                   |          |
| `blockAds`            | `boolean`          | 広告および Cookie ポップアップのブロックを有効にします。                                                                                    |                   |          |
| `proxy`               | `&quot;basic&quot; | &quot;enhanced&quot;                                                                                                | &quot;auto&quot;` | プロキシモード。 |
| `storeInCache`        | `boolean`          | 結果をキャッシュに保存します。                                                                                                     |                   |          |
| `profile`             | `object`           | `name` と `saveChanges` を含む永続的なブラウザプロファイル。                                                                           |                   |          |
| `zeroDataRetention`   | `boolean`          | アカウントで許可されている場合にゼロデータ保持を有効にします。                                                                                     |                   |          |



<div id="interact">
  ## Interact
</div>

### なぜ使うのか

ページで、スクレイピングの開始後にブラウザのアクションやコードの実行 (クリック、フォーム操作、スクリプトによる手順) が必要な場合は、Interact を使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`POST https://api.firecrawl.dev/v2/scrape/{jobId}/interact`

### 簡単な例

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "console.log(document.title)",
    "language": "node"
  }'
```

<div id="complex-example">
  ### 複雑な例
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Click the pricing tab and summarize the plans.",
    "language": "node",
    "timeout": 60
  }'
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ      | 型 / オプション           | 説明                                             |                   |                            |
| ---------- | ------------------- | ---------------------------------------------- | ----------------- | -------------------------- |
| `jobId`    | `string`            | `data.metadata.scrapeId` のスクレイピングジョブ ID。       |                   |                            |
| `code`     | `string`            | ブラウザセッションで実行するコード。`prompt` が指定されていない場合は必須です。   |                   |                            |
| `prompt`   | `string`            | ブラウザエージェントへの自然言語による指示。`code` が指定されていない場合は必須です。 |                   |                            |
| `language` | `&quot;python&quot; | &quot;node&quot;                               | &quot;bash&quot;` | 実行言語。省略時のデフォルトは `node` です。 |
| `timeout`  | `integer`           | 実行タイムアウト (秒) 。                                 |                   |                            |
| `origin`   | `string`            | 内部使用専用です。Firecrawl の計測用に予約されています。              |                   |                            |



<div id="notes">
  ## 注記
</div>

* OpenAPI spec では現在、Interact では `code` が必須とされていますが、SDK では代わりに `prompt` も指定できます。実際には、`code` または `prompt` のいずれかを使用してください。
