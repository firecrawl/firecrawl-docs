<div id="firecrawl-java-source-of-truth">
  # Firecrawl Javaの信頼できる唯一の参照元
</div>

このファイルは、エージェント向けの公開された正規の参照元です。Firecrawl Java SDK のソースコードと v2 OpenAPI 仕様から生成されています。

<div id="install">
  ## インストール
</div>

Maven:

```xml
<dependency>
  <groupId>com.firecrawl</groupId>
  <artifactId>firecrawl-java</artifactId>
  <version>1.2.0</version>
</dependency>
```

Gradle:

```gradle
implementation("com.firecrawl:firecrawl-java:1.2.0")
```

<div id="authenticate">
  ## 認証
</div>

```java
import com.firecrawl.client.FirecrawlClient;

FirecrawlClient client = FirecrawlClient.builder()
    .apiKey(System.getenv("FIRECRAWL_API_KEY"))
    .build();
```

<div id="when-to-use-what">
  ## 何をいつ使うべきか
</div>

* `search`: クエリから始めて、`site:` 制約を含む関連URLを見つけます。
* `scrape`: 既知のURLからコンテンツを取得・抽出します。
* `interact`: ページでクリックやフォーム入力、またはスクレイピング後の処理が必要な場合に、ブラウザのアクションやコードを実行します。

<div id="search">
  ## 検索
</div>

### これを使う場面

クエリから候補を見つけ、Firecrawl に URL を返させたい場合 (必要に応じて各結果をスクレイピングすることも可能) は、search を使います。たとえば、標準の `site:` 演算子を使うと、結果を特定のドメインに絞り込めます。例: `site:docs.firecrawl.dev crawl webhooks`

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.search(query, options)`

<div id="simple-example">
  ### シンプルな例
</div>

```java
SearchData results = client.search("site:docs.firecrawl.dev webhook retries");
```

<div id="complex-example">
  ### 複雑な例
</div>

```java
import com.firecrawl.models.SearchOptions;
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;

SearchOptions options = SearchOptions.builder()
    .sources(List.of("web", "news"))
    .categories(List.of("research"))
    .limit(10)
    .tbs("qdr:m")
    .location("San Francisco,California,United States")
    .ignoreInvalidURLs(true)
    .timeout(60000)
    .scrapeOptions(
        ScrapeOptions.builder()
            .formats(List.of(
                "markdown",
                "links",
                JsonFormat.builder().prompt("Extract title and key endpoints.").build()
            ))
            .onlyMainContent(true)
            .waitFor(1000)
            .build()
    )
    .build();

SearchData results = client.search("site:docs.firecrawl.dev crawl webhooks", options);
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ                       | 型 / オプション                                                              | 説明                                                  |
| --------------------------- | ---------------------------------------------------------------------- | --------------------------------------------------- |
| `query`                     | `String`                                                               | 検索クエリ。結果を特定のサイトに限定するには `site:domain` を使用します。        |
| `options.sources`           | `List<Object>` (`"web"`, `"news"`, `"images"` or `{type: "web"}` maps) | 検索対象のソース。                                           |
| `options.categories`        | `List<Object>` (`"github"`, `"research"`, `"pdf"`)                     | 結果を絞り込むカテゴリ。                                        |
| `options.limit`             | `Integer`                                                              | 返す結果の最大数。                                           |
| `options.tbs`               | `String`                                                               | 時間ベースの検索フィルタ (例: `qdr:d`, `qdr:w`, `sbd:1,qdr:m`) 。 |
| `options.location`          | `String`                                                               | 検索結果用のロケーション文字列。                                    |
| `options.ignoreInvalidURLs` | `Boolean`                                                              | 他の Firecrawl エンドポイントでスクレイピングできない無効な URL を除外します。     |
| `options.timeout`           | `Integer`                                                              | タイムアウト (ミリ秒) 。                                      |
| `options.scrapeOptions`     | `ScrapeOptions`                                                        | 各結果に適用されるスクレイピングのオプション (`scrape` と同じフィールド) 。        |
| `options.integration`       | `String`                                                               | 内部専用。Firecrawl の計測用に予約されています。                       |

<div id="scrape">
  ## スクレイピング
</div>

<div id="why-use-it">
  ### どのような場合に使うか
</div>

URL がすでにわかっていて、Firecrawl から構造化されたコンテンツや任意のフォーマットを取得したい場合は、scrape を使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.scrape(url, options)`

<div id="simple-example">
  ### シンプルな例
</div>

```java
Document doc = client.scrape(
    "https://docs.firecrawl.dev",
    ScrapeOptions.builder()
        .formats(List.of("markdown"))
        .build()
);
```

<div id="complex-example">
  ### 複雑な例
</div>

```java
import com.firecrawl.models.ScrapeOptions;

ScrapeOptions options = ScrapeOptions.builder()
    .formats(List.of(
        "markdown",
        "links",
        Map.of("type", "json", "prompt", "Extract plan names and prices."),
        Map.of("type", "screenshot", "fullPage", true, "quality", 80, "viewport", Map.of("width", 1280, "height", 720)),
        Map.of("type", "changeTracking", "modes", List.of("git-diff"), "tag", "pricing")
    ))
    .headers(Map.of("User-Agent", "FirecrawlDocsBot/1.0"))
    .includeTags(List.of("main", "article"))
    .excludeTags(List.of("nav", "footer"))
    .onlyMainContent(true)
    .waitFor(1000)
    .mobile(false)
    .parsers(List.of(Map.of("type", "pdf", "mode", "auto", "maxPages", 5)))
    .actions(List.of(
        Map.of("type", "click", "selector", "#accept"),
        Map.of("type", "write", "text", "firecrawl"),
        Map.of("type", "press", "key", "Enter"),
        Map.of("type", "wait", "milliseconds", 1500),
        Map.of("type", "scrape")
    ))
    .location(LocationConfig.builder().country("US").languages(List.of("en-US")).build())
    .skipTlsVerification(true)
    .removeBase64Images(true)
    .blockAds(true)
    .proxy("auto")
    .maxAge(86400000L)
    .storeInCache(true)
    .build();

Document doc = client.scrape("https://example.com/pricing", options);
```

  ### パラメータ
</div>

| パラメータ                         | 型 / オプション                                                                  | 説明                                                                                                                          |
| ----------------------------- | -------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------- |
| `url`                         | `String`                                                                   | スクレイピング対象のURL。                                                                                                              |
| `options.formats`             | `List<Object>`                                                             | 含める出力フォーマット。文字列、または `{type: "json", prompt, schema}` や `{type: "screenshot", fullPage, quality, viewport}` などのフォーマットオブジェクト。 |
| `options.headers`             | `Map<String, String>`                                                      | カスタムヘッダー。                                                                                                                   |
| `options.includeTags`         | `List<String>`                                                             | 含めるタグ。                                                                                                                      |
| `options.excludeTags`         | `List<String>`                                                             | 除外するタグ。                                                                                                                     |
| `options.onlyMainContent`     | `Boolean`                                                                  | メインコンテンツのみを保持。                                                                                                              |
| `options.timeout`             | `Integer`                                                                  | タイムアウト時間 (ミリ秒) 。                                                                                                            |
| `options.waitFor`             | `Integer`                                                                  | スクレイピング開始前の待機時間 (ミリ秒) 。                                                                                                     |
| `options.mobile`              | `Boolean`                                                                  | モバイル端末をエミュレート。                                                                                                              |
| `options.parsers`             | `List<Object>`                                                             | パーサーの設定 (PDFオプションに対応) 。                                                                                                     |
| `options.actions`             | `List<Map<String, Object>>`                                                | スクレイピング前に実行するブラウザアクション。                                                                                                     |
| `options.location`            | `LocationConfig`                                                           | リクエストのロケーション設定。                                                                                                             |
| `options.skipTlsVerification` | `Boolean`                                                                  | TLS証明書の確認をスキップ。                                                                                                             |
| `options.removeBase64Images`  | `Boolean`                                                                  | Markdown出力からBase64画像を削除。                                                                                                    |
| `options.blockAds`            | `Boolean`                                                                  | 広告およびCookieポップアップのブロックを有効化。                                                                                                 |
| `options.proxy`               | `String` (`"basic"`, `"stealth"`, `"enhanced"`, `"auto"`, or a custom URL) | プロキシモード。                                                                                                                    |
| `options.maxAge`              | `Long`                                                                     | この期間内のキャッシュ済みデータを優先。                                                                                                        |
| `options.storeInCache`        | `Boolean`                                                                  | 結果をキャッシュに保存。                                                                                                                |
| `options.integration`         | `String`                                                                   | 内部専用。Firecrawlの計測用に予約済み。                                                                                                    |

<div id="interact">
  ## Interact
</div>

### 利用する理由

ページで、スクレイピング開始後にブラウザのアクションやコードの実行 (クリック、フォーム操作、スクリプトによる手順など) が必要な場合は、Interact を使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.interact(jobId, code, language, timeout, origin)`

### 簡単な例

```java
Document doc = client.scrape("https://example.com");
String jobId = doc.getMetadata() != null ? String.valueOf(doc.getMetadata().get("scrapeId")) : null;
if (jobId == null || jobId.isBlank()) {
    throw new IllegalStateException("Missing scrapeId from scrape response");
}

BrowserExecuteResponse result = client.interact(jobId, "console.log(document.title)");
```

<div id="complex-example">
  ### 複雑な例
</div>

```java
BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "console.log(await page.title());",
    "node",
    60,
    null
);
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ      | 型 / オプション                                 | 説明                                           |
| ---------- | ----------------------------------------- | -------------------------------------------- |
| `jobId`    | `String`                                  | `document.metadata.scrapeId` のスクレイピングジョブ ID。 |
| `code`     | `String`                                  | ブラウザセッションで実行するコード。                           |
| `language` | `String` (`"python"`, `"node"`, `"bash"`) | 実行言語。省略時のデフォルトは `node` です。                   |
| `timeout`  | `Integer`                                 | 実行のタイムアウト時間 (秒、1～300) 。                      |
| `origin`   | `String`                                  | 内部専用。Firecrawl の計測用に予約されています。                |

<div id="notes">
  ## 注意事項
</div>

* Java SDK の interact API は `code` のみをサポートしています。その他の SDK では、便宜上 `prompt` も利用できます。
* 非推奨のエイリアス: `scrapeExecute` は `interact` のエイリアスであり、`deleteScrapeBrowser` は `stopInteractiveBrowser` のエイリアスです。
