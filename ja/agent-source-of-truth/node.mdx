<div id="firecrawl-nodejs-source-of-truth">
  # Firecrawl Node.js の信頼できる唯一の参照元
</div>

このファイルは、エージェント向けの正規の公開信頼できる唯一の参照元です。Firecrawl JS SDK のソースコードと v2 OpenAPI 仕様から生成されています。

<div id="install">
  ## インストール
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="authenticate">
  ## 認証
</div>

```ts
import Firecrawl from "@mendable/firecrawl-js";

const client = new Firecrawl({
  apiKey: process.env.FIRECRAWL_API_KEY,
  // apiUrl: "https://api.firecrawl.dev" // セルフホスト時に任意で上書き可能
});
```

<div id="when-to-use-what">
  ## 使い分けの目安
</div>

* `search`: クエリから始めて、`site:` 制約も含めながら、関連するURLを見つけます。
* `scrape`: 既知のURLからコンテンツを取得・抽出します。
* `interact`: クリックやフォーム操作、またはスクレイピング後の処理が必要なページで、ブラウザのアクションやコードを実行します。

<div id="search">
  ## Search
</div>

### 使用する理由

クエリからURLを探索し、Firecrawlに結果を返させたい場合 (必要に応じて各結果をスクレイピングすることも可能) に使用します。標準の `site:` 演算子を使ってドメインに結果を絞り込めます。例：`site:docs.firecrawl.dev crawl webhooks`

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.search(query, options?)`

<div id="simple-example">
  ### シンプルな例
</div>

```ts
const results = await client.search("site:docs.firecrawl.dev webhook retries");
```

<div id="complex-example">
  ### 複雑な例
</div>

```ts
const results = await client.search("site:docs.firecrawl.dev crawl webhooks", {
  sources: ["web", "news"],
  categories: ["research"],
  limit: 10,
  tbs: "qdr:m",
  location: "San Francisco,California,United States",
  ignoreInvalidURLs: true,
  timeout: 60000,
  scrapeOptions: {
    formats: [
      "markdown",
      "links",
      { type: "json", prompt: "Extract title, summary, and key endpoints." }
    ],
    onlyMainContent: true,
    includeTags: ["main", "article"],
    excludeTags: ["nav", "footer"],
    waitFor: 1000,
    actions: [
      { type: "click", selector: "button#accept" },
      { type: "wait", milliseconds: 750 },
      { type: "scrape" }
    ]
  }
});
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ                       | 型 / オプション                      | 説明                                                |
| --------------------------- | ------------------------------ | ------------------------------------------------- |
| `query`                     | `string`                       | 検索クエリ。`site:domain` を指定して結果を特定のサイトに絞り込めます。        |
| `options.sources`           | `ソース名の配列、または型付きソースオブジェクトの配列`   | 検索対象のソース。                                         |
| `options.categories`        | `カテゴリ名の配列、または型付きカテゴリオブジェクトの配列` | 結果を絞り込むカテゴリ。                                      |
| `options.limit`             | `number`                       | 返す結果の最大件数。                                        |
| `options.tbs`               | `string`                       | 時間ベースの検索フィルター (例：`qdr:d`、`qdr:w`、`sbd:1,qdr:m`) 。 |
| `options.location`          | `string`                       | 検索結果に使用するロケーション文字列。                               |
| `options.ignoreInvalidURLs` | `boolean`                      | 他のFirecrawlエンドポイントでスクレイピングできない無効なURLを除外します。       |
| `options.timeout`           | `number`                       | タイムアウト (ミリ秒単位) 。                                  |
| `options.scrapeOptions`     | `ScrapeOptions`                | 各結果に適用するスクレイピングのオプション (`scrape` と同じフィールド) 。       |
| `options.integration`       | `string`                       | 内部専用。Firecrawl計測用に予約済み。                           |
| `options.origin`            | `string`                       | 内部専用。Firecrawl計測用に予約済み。                           |

<div id="scrape">
  ## スクレイピング
</div>

<div id="why-use-it">
  ### どのような場合に使うか
</div>

URLがすでにあり、Firecrawl から構造化されたコンテンツや任意のフォーマットを取得したい場合は、スクレイピングを使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.scrape(url, options?)`

<div id="simple-example">
  ### シンプルな例
</div>

```ts
const doc = await client.scrape("https://docs.firecrawl.dev", {
  formats: ["markdown"]
});
```

### 高度な例

```ts
const doc = await client.scrape("https://example.com/pricing", {
  formats: [
    "markdown",
    "links",
    { type: "json", prompt: "Extract plan names and prices." },
    { type: "screenshot", fullPage: true, quality: 80, viewport: { width: 1280, height: 720 } },
    { type: "changeTracking", modes: ["git-diff"], tag: "pricing" }
  ],
  headers: {
    "User-Agent": "FirecrawlDocsBot/1.0"
  },
  includeTags: ["main", "article"],
  excludeTags: ["nav", "footer"],
  onlyMainContent: true,
  waitFor: 1000,
  mobile: false,
  parsers: [{ type: "pdf", mode: "auto", maxPages: 5 }],
  actions: [
    { type: "click", selector: "#accept" },
    { type: "write", text: "firecrawl" },
    { type: "press", key: "Enter" },
    { type: "wait", milliseconds: 1500 },
    { type: "scrape" }
  ],
  location: { country: "US", languages: ["en-US"] },
  skipTlsVerification: true,
  removeBase64Images: true,
  fastMode: true,
  blockAds: true,
  proxy: "auto",
  maxAge: 86400000,
  storeInCache: true,
  profile: { name: "docs-session", saveChanges: true }
});
```

<div id="interact">
  ## Interact
</div>

### 使用する理由

スクレイピング開始後に、ページ上でブラウザのアクションやコードの実行 (クリック、フォーム操作、スクリプトによる手順) が必要な場合は、Interactを使用します。

<div id="preferred-sdk-method">
  ### 推奨SDKメソッド
</div>

`client.interact(jobId, args)`

<div id="simple-example">
  ### シンプルな例
</div>

```ts
const doc = await client.scrape("https://example.com", { formats: ["markdown"] });
const jobId = doc.metadata?.scrapeId;
if (!jobId) throw new Error("スクレイピングレスポンスにscrapeIdがありません");

const result = await client.interact(jobId, {
  prompt: "pricingタブをクリックして、プランを要約してください。",
  language: "node"
});
```

<div id="complex-example">
  ### 複雑な例
</div>

```ts
const result = await client.interact("<scrapeJobId>", {
  code: "console.log(await page.title());",
  language: "node",
  timeout: 60
});
```

<div id="parameters">
  ### パラメータ
</div>

| パラメータ           | 型 / オプション               | 説明                                            |
| --------------- | ----------------------- | --------------------------------------------- |
| `jobId`         | `string`                | `document.metadata.scrapeId` のスクレイピングジョブ ID。  |
| `args.code`     | `string`                | ブラウザセッション内で実行するコード。`prompt` が指定されていない場合は必須です。 |
| `args.prompt`   | `string`                | ブラウザエージェントへの自然言語での指示。`code` が指定されていない場合は必須です。 |
| `args.language` | `python, node, or bash` | 実行言語。省略時のデフォルトは `node` です。                    |
| `args.timeout`  | `number`                | 実行タイムアウト (秒) 。                                |
| `args.origin`   | `string`                | 内部専用。Firecrawl の計測用に予約されています。                 |

<div id="notes">
  ## 注意事項
</div>

* 非推奨のエイリアス: `scrapeExecute`、`stopInteractiveBrowser`、`deleteScrapeBrowser` は、`interact` と `stopInteraction` の別名です。
* 最上位の `Firecrawl` クライアントでは v2 のメソッドが直接公開されており、従来の v1 は `client.v1` で利用できます。
