<div id="better-sitemap-control">
  ### サイトマップの制御を強化
</div>

サイトマップの扱い方は3通りあります:

* `"include"` - サイトマップを利用し、さらに他のページも探索する（デフォルト）
* `"skip"` - サイトマップを完全に無視する
* `"only"` - サイトマップ内のURLのみを返す

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```


<div id="smarter-url-discovery">
  ### よりスマートなURL検出
</div>

- サイト内のすべてのページを網羅的に発見
- 結果数に上限なし（v1は500件で上限）
- さらに高速・高精度

APIはv1と同じですが、より多くのページを見つけ、複雑なサイトでもより良く動作します。
