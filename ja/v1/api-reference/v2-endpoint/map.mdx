<div id="better-sitemap-control">
  ### サイトマップの制御を強化
</div>

サイトマップの扱い方は3通りあります:

* `"include"` - サイトマップを利用し、さらに他のページも探索する（デフォルト）
* `"skip"` - サイトマップを完全に無視する
* `"only"` - サイトマップ内のURLのみを返す

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```

<div id="smarter-url-discovery">
  ### よりスマートなURLディスカバリ
</div>

- サイト内のすべてのページを検出
- 結果数に上限なし（v1では500件に制限）
- より高速かつ高精度

v1と同じAPIでありながら、より多くのページを見つけ、複雑なサイトでもよりうまく機能します。
