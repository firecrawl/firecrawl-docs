<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="better-error-messages">
  ### エラーメッセージの改良
</div>

- 自然言語プロンプトがうまく動作しない場合のわかりやすいエラー
- サイトマップや深さ制限の問題に対する具体的なメッセージ
- ドメイン全体のクロールに関する問題のためのより適切なエラー

<div id="more-error-details">
  ### エラー詳細の強化
</div>

- フォーマット処理の問題に関する詳細なエラー
- PDF とパーサーの失敗に関する具体的なメッセージ
- 位置情報ベースのクロール向けのより高度なデバッグ
- changeTracking の問題に対する明確なエラー

<div id="faster-and-easier-debugging">
  ### さらに高速で、より簡単なデバッグ
</div>

- 大規模クロールのエラー取得がさらに高速に
- エラーのグルーピングを改善
- より有用なエラーコンテキストを提供

APIはv1と同じですが、v2の機能ではエラーレポートが大幅に向上しています。
