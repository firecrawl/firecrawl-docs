<div id="whats-new-in-v2">
  ## v2 の新機能
</div>

v2 の Cancel Batch Scrape API は、v1 と同等の機能を保ちながら、信頼性がさらに向上しています。

<div id="enhanced-cancellation-process">
  ### キャンセル処理の強化
</div>

- バッチ処理におけるクリーンアップとリソース管理の改善
- 実行中のバッチ終了のハンドリングを改善
- バッチのキャンセル処理におけるエラーハンドリングを強化

API のパラメータとレスポンスフォーマットは v1 から変更されておらず、より信頼性の高いバッチキャンセルを提供しつつ、シームレスな移行を実現します。
