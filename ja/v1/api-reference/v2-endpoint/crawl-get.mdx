<div id="whats-new-in-v2">
  ## v2 の変更点
</div>

<div id="better-progress-tracking">
  ### 進捗管理の改善
</div>

- 自然言語プロンプトの挙動を可視化
- 各サイトマップモードの進捗を追跡
- ドメイン全体のクロールと深さ制限を監視

<div id="more-detailed-status">
  ### さらに詳細なステータス
</div>

- リクエストした各出力フォーマットのステータス
- PDF処理とchangeTrackingの進捗
- エラー報告の改善

<div id="faster-and-more-reliable">
  ### さらに高速で、より信頼できる
</div>

- 大規模クロールのステータス更新がより迅速に
- 進捗レポートの精度が向上
- 複雑なクロールの処理がより安定

APIはv1と同一のまま、v2の機能向けのモニタリングが強化されています。
