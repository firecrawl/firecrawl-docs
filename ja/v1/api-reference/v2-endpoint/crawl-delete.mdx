<div id="whats-new-in-v2">
  ## v2 の新機能
</div>

v2 の Cancel Crawl API は、v1 と同等の機能を保ちながら、信頼性が向上しています。

<div id="enhanced-cancellation-process">
  ### キャンセル処理の強化
</div>

- クリーンアップとリソース管理の改善
- 実行中のクロール終了のより適切な処理
- キャンセル時のエラー処理の強化

API パラメータとレスポンスフォーマットは v1 から変更はなく、より信頼性の高いクロールのキャンセルを提供しつつ、シームレスな移行を実現します。
