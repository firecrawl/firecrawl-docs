<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="all-new-formats-work-in-batches">
  ### すべての新フォーマットはバッチ処理に対応
</div>

- カスタムスクリーンショット、JSON抽出、変更トラッキングをすべてのURLで利用可能
- 新しいパーサーシステムでPDFをバッチ処理
- バッチ内のすべてのURLに同一のフォーマット設定を適用

<div id="better-batch-processing">
  ### バッチ処理の強化
</div>

- バッチ全体で国や言語を切り替えてスクレイプ
- すべてのURLでプロキシタイプ（stealth / basic / auto）を利用可能
- すべての新しいスクレイプオプションがバッチモードで利用可能

<div id="faster-and-more-reliable">
  ### さらに高速で、より信頼性の高い
</div>

- 複雑なバッチ向けのリソース管理を強化
- 進捗の追跡を改善
- エラーハンドリングを強化

APIはv1と同一で、v2の全機能がバッチ全体で機能します。
