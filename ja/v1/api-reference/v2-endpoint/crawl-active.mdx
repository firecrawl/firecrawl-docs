<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="see-what-your-crawls-are-doing">
  ### クロールの実行状況を可視化
</div>

- 自然言語プロンプトでクロールを追跡
- 各クロールのサイトマップモードを確認
- ドメイン全体のクロールと深さ設定を監視

<div id="more-details-on-active-crawls">
  ### 実行中のクロールの詳細情報
</div>

- 各クロールで使用している出力フォーマットを確認
- PDF処理や位置情報に基づくクロールを追跡
- リソース使用状況の可視化を強化

<div id="faster-and-more-reliable">
  ### より高速で、さらに信頼性向上
</div>

- 多数のクロール監視時の更新が高速化
- リソースのトラッキングが強化
- 監視の信頼性が向上

APIはv1と同一で、v2の機能の可視性が向上。
