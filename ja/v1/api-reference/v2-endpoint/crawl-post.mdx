<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="tell-the-crawler-what-you-want">
  ### クローラーに要件を伝える
</div>

クロールしてほしい内容をわかりやすい英語で記述してください。

```json
{
  "url": "https://example.com",
  "prompt": "ブログ記事とドキュメントだけをクロールし、マーケティングページはスキップする"
}
```

<div id="better-sitemap-control">
  ### より柔軟なサイトマップ制御
</div>

v1: サイトマップはオン/オフのみ。v2: 3 つのオプションがあります:

- `"include"` - サイトマップを利用しつつ、他のページも自動的に発見（デフォルト）
- `"skip"` - サイトマップを完全に無視  
- `"only"` - サイトマップ内の URL のみをクロール

<div id="new-crawling-options">
  ### 新しいクロールオプション
</div>

* `crawlEntireDomain` - 子ページだけでなくドメイン全体をクロール
* `maxDiscoveryDepth` - クロールの深さを制御（`maxDepth` の置き換え）

```json
{
  "url": "https://example.com/features",
  "crawlEntireDomain": true,
  "maxDiscoveryDepth": 2,
  "sitemap": "include"
}
```
