<div id="whats-new-in-v2">
  ## v2 の新機能
</div>

<div id="smarter-extraction">
  ### よりスマートな抽出
</div>

FIRE-1（エージェント）は手強いサイトにも対応します:

- ページ移動やページネーションを自動でクリック
- JavaScriptが多用されたサイトでも安定して動作
- データの発見と抽出により優れる

<div id="no-url-required">
  ### URLは不要
</div>

正確なURLを知らなくてもデータを抽出できます:

- 要件を記述すれば、v2が見つけます
- ウェブを検索し、最適なページから抽出します
- v1より高精度です

<div id="better-performance">
  ### パフォーマンス向上
</div>

- 処理の高速化
- より信頼性の高いエラー処理
- 精度の向上

API は v1 と同一のため、容易にアップグレードできます。
