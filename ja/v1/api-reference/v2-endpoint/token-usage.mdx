  ## v2 の最新情報
</div>

<div id="see-where-your-tokens-go">
  ### トークンの利用状況を可視化
</div>

- FIRE-1（エージェント）のトークン使用量を追跡
- URL不要の抽出にかかるコストを確認
- 複雑なデータ抽出のトークン消費を監視
- バッチ抽出のトラッキングを強化

<div id="more-token-details">
  ### さらに詳しいトークン情報
</div>

- 複雑なウェブ操作に伴うトークンを確認
- JavaScriptが多いサイトでの追加トークンを追跡
- 精度向上のためにトークンを監視
- 自動ページネーションにかかるコストを確認

<div id="faster-and-more-accurate">
  ### より高速、より高精度に
</div>

- トークン使用量データへ素早くアクセス
- Extract 機能のコスト内訳がより明確に
- FIRE-1 のオペレーションをより正確にトラッキング

API は v1 と同一のまま、v2 の Extract 機能でトークン使用量の可視性が向上しました。
