<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="better-batch-error-messages">
  ### バッチエラーのメッセージ改善
</div>

- すべてのURLで発生するフォーマット処理の問題に対して、わかりやすいエラーを表示
- PDFやパーサーの失敗に対する具体的なメッセージ
- プロキシモード別のエラー追跡を改善
- 位置情報ベースのスクレイピングで発生する問題に対して、わかりやすいエラーを表示

<div id="more-error-details">
  ### エラー詳細の強化
</div>

- changeTracking の問題向けのデバッグ情報
- ブラウザ自動化失敗時のエラー報告
- クレジットおよびリソース制限エラーの追跡を改善
- バッチ処理の不具合に対する特定的なエラー

<div id="faster-and-easier-debugging">
  ### より速く、より簡単にデバッグ
</div>

- 大規模バッチのエラー取得がより高速に
- エラーのグルーピングが向上
- さらに有用なエラーコンテキスト

API は v1 と同一で、v2 の機能をバッチで使用する際のエラー報告が大幅に改善されています。
