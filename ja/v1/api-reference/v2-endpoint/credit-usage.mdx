<div id="whats-new-in-v2">
  ## v2の新機能
</div>

<div id="see-where-your-credits-go">
  ### クレジットの使途を可視化
</div>

- 出力フォーマット別のクレジット消費を追跡
- PDFの処理・解析にかかるコストを確認
- basic／stealthのプロキシコストの内訳を表示
- ロケーション指定スクレイピングの追加コストを追跡

<div id="more-cost-details">
  ### コストの詳細をさらに充実
</div>

- changeTracking のクレジットを参照
- ブラウザ自動化機能のコストを可視化・追跡
- ソース種別ごとの検索コストの内訳
- バッチ処理コストのモニタリングを強化

<div id="faster-and-more-accurate">
  ### より高速・高精度に
</div>

- 利用状況データへ素早くアクセス
- コスト内訳をより明確に
- クレジットの追跡をより正確に

APIはv1と同一のまま、v2の機能でクレジット使用状況の可視性が向上します。
