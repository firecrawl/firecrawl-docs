<div id="whats-new-in-v2">
  ## v2 の新着情報
</div>

<div id="better-batch-progress-tracking">
  ### バッチ進捗の可視化を強化
</div>

- バッチ全体で各出力フォーマットの進捗を確認
- PDF処理とパーサー処理の進行を追跡
- 使用中の各プロキシモードを監視
- 位置情報に基づくスクレイピングの進捗を追跡

<div id="more-batch-details">
  ### さらに詳しいバッチ情報
</div>

- すべてのURLのchangeTracking進捗を確認
- ブラウザ自動化機能を監視
- バッチ問題のエラーレポートを強化
- 高度な機能のクレジット使用量を可視化

<div id="faster-and-more-reliable">
  ### より高速・より信頼性
</div>

- 大規模バッチのステータス更新がさらに高速化
- v2 機能の進捗トラッキングを強化
- ステータス報告の信頼性を向上

API は v1 と同じで、バッチにおける v2 機能のモニタリングが改善されています。
