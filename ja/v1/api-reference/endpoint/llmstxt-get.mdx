---
title: "LLMs.txt のステータス"
openapi: "/ja/api-reference/v1-openapi.json GET /llmstxt/{id}"
---

LLMs.txt の生成ジョブのステータスと結果を取得します。このエンドポイントでは、生成の完了状況を確認し、生成されたコンテンツを取得できます。

<div id="response-structure">
  ### レスポンス構造
</div>

レスポンスには次が含まれます:

- **success**: リクエストが成功したかを示すブール値
- **status**: 生成ジョブの現在ステータス:

  - `processing`: ジョブが実行中
  - `completed`: ジョブが正常に完了
  - `failed`: ジョブでエラーが発生

- **data**: 生成されたコンテンツ（ステータスが `completed` の場合）:

  - `llmstxt`: 生成された LLMs.txt の内容
  - `llmsfulltxt`: 全文の内容（showFullText が true の場合）

- **expiresAt**: 結果の有効期限を示す ISO 形式のタイムスタンプ

<div id="status-examples">
  ### ステータス例
</div>

1. **処理中のステータス**

   ```json
   {
     "success": true,
     "status": "processing",
     "data": {
       "llmstxt": "# Firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/)...",
       "llmsfulltxt": "# Firecrawl.dev llms-full.txt\n\n"
     },
     "expiresAt": "2025-03-03T23:19:18.000Z"
   }
   ```

2. **完了ステータス**
   ```json
   {
     "success": true,
     "status": "completed",
     "data": {
       "llmstxt": "# http://firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/): Transform websites into clean, LLM-ready data effortlessly.\n- [Flexible Web Scraping Pricing](https://www.firecrawl.dev/pricing): Flexible pricing plans for web scraping and data extraction.",
       "llmsfulltxt": "# http://firecrawl.dev llms-full.txt\n\n## Web Data Extraction Tool\nIntroducing /extract - Get web data with a prompt..."
     },
     "expiresAt": "2025-03-03T22:45:50.000Z"
   }
   ```

<div id="error-handling">
  ### エラー処理
</div>

生成ジョブが失敗した場合、または見つからない場合は、適切なエラーレスポンスが返されます。

```json
{
  "success": false,
  "error": "LLMs.txt の生成ジョブが見つかりません"
}
```

<div id="polling-recommendations">
  ### ポーリングの推奨事項
</div>

1. 開始時は2〜3秒間隔でポーリングする
2. 30秒経過しても処理中なら、間隔を延長する
3. ステータスが`completed`または`failed`になったらポーリングを停止する
4. レート制限を避けるため、指数バックオフを実装する
