---
title: "LLMs.txt を生成する"
openapi: "/ja/api-reference/v1-openapi.json POST /llmstxt"
---

LLMs.txt 生成エンドポイントは、任意のウェブサイトに対して LLMs.txt と LLMs-full.txt を作成します。これらのファイルはサイトのコンテンツを構造化し、LLM に扱いやすい形式で提供することで、言語モデルによる理解と処理を容易にします。

ステータス用のエンドポイントをお探しですか？[LLMs.txt Status](/ja/api-reference/endpoint/llmstxt-get) をご覧ください。

<div id="response-structure">
  ### レスポンス構造
</div>

レスポンスには以下が含まれます：

* **success**: リクエストが成功したかを示す真偽値
* **id**: 生成ジョブの一意の識別子

<div id="output-format">
  ### 出力フォーマット
</div>

1. **LLMs.txt**

   * ウェブサイトを簡潔かつ構造化して要約
   * 主要なリンクとその説明を含む
   * 解析しやすいように Markdown で整形
   * 例:

     ```markdown
     # http://example.com llms.txt

     - [Page Title](https://example.com/page): Brief description
     - [Another Page](https://example.com/another): Another description
     ```

2. **LLMs-full.txt**（showFullText が true の場合）

   * 処理したページの全文を含む
   * 階層構造を保持
   * さらに詳細な情報を含む
   * 例:

     ```markdown
     # http://example.com llms-full.txt

     ## Page Title

     Full content of the page...

     ## Another Page

     More detailed content...
     ```

<div id="billing">
  ### 請求
</div>

請求は API コール数と処理した URL 数に基づきます：

* 基本料金：API コール1回につき1クレジット
* 追加料金：処理した URL 1件につき1クレジット
* `maxUrls` パラメータで URL のコストを制御できます
