<div id="generate-llmstxt">
  ## generate-llmstxt
</div>

Firecrawl API を使用して CLI で LLMs.txt ファイルを生成するシンプルな npx パッケージです。指定した出力ディレクトリ（既定は「public」フォルダ）に次の 2 つのファイルを作成します:

- `llms.txt`: LLM 関連コンテンツの要約
- `llms-full.txt`: テキストコンテンツの全文

<div id="usage">
  ## 使用方法
</div>

このパッケージはインストール不要で NPX から実行できます。Firecrawl の API キーを指定する方法は次の2通りです。

<div id="1-using-command-line-arguments">
  ### 1. コマンドライン引数の使用
</div>

```bash
npx generate-llmstxt --api-key YOUR_FIRECRAWL_API_KEY
```


<div id="2-using-environment-variables">
  ### 2. 環境変数を使用する
</div>

プロジェクトのルートに `.env` ファイルを作成し、APIキーを追加します:

```env
FIRECRAWL_API_KEY=ここにあなたのAPIキー
```

その後、--api-key オプションを付けずにコマンドを実行します：

```bash
npx generate-llmstxt
```


<div id="options">
  ### オプション
</div>

- `-k, --api-key <key>` (.envに設定済みなら省略可): Firecrawl の API キー
- `-u, --url <url>` (任意): 解析対象の URL（デフォルト: https://example.com）
- `-m, --max-urls <number>` (任意): 解析する URL の最大数（デフォルト: 50）
- `-o, --output-dir <path>` (任意): 出力ディレクトリのパス（デフォルト: 'public'）

<div id="examples">
  ### 例
</div>

```bash
# 既定の出力ディレクトリでコマンドライン引数を使う
npx generate-llmstxt -k your_api_key -u https://your-website.com -m 20

# 既定の出力ディレクトリで .env ファイルを使う
npx generate-llmstxt -u https://your-website.com -m 20

# 出力ディレクトリを指定する（カスタム）
npx generate-llmstxt -k your_api_key -u https://your-website.com -o custom/path/to/output

# .env ファイルを使い、出力ディレクトリを指定する（カスタム）
npx generate-llmstxt -u https://your-website.com -o content/llms
```


<div id="requirements">
  ## 必要条件
</div>

- Node.js 14 以上
- 有効な Firecrawl API キー（コマンドラインまたは .env ファイルから指定）

<div id="output">
  ## 出力
</div>

このパッケージは、指定した出力ディレクトリ（デフォルトは「public」）に次の2つのファイルを作成します:

1. `llms.txt`: LLM関連コンテンツの要約を含みます
2. `llms-full.txt`: テキストコンテンツの全文を含みます
