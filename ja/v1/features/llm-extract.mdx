<div id="scrape-and-extract-structured-data-with-firecrawl">
  ## Firecrawlで構造化データをスクレイピングして抽出する
</div>

{/* <Warning>Scrape LLM Extract は今後のバージョンで廃止予定です。新しい [Extract](/features/extract) エンドポイントを使用してください。</Warning> */}

FirecrawlはAIを用いて、3つの手順でウェブページから構造化データを取得します:

1. **スキーマを設定:**
   必要なデータを、ウェブページのURLとあわせて OpenAI形式のJSONスキーマで定義します。

2. **リクエストを送信:**
   URLとスキーマを /scrape エンドポイントに送信します。詳しくはこちら:
   [Scrape Endpoint Documentation](https://docs.firecrawl.dev/api-reference/endpoint/scrape)

3. **データを取得:**
   スキーマに一致するクリーンな構造化データが返ってきます。すぐに利用可能です。

これにより、必要なフォーマットでウェブデータを迅速かつ簡単に取得できます。

<div id="json-options-object">
  ### JSON オプションオブジェクト
</div>

`jsonOptions` オブジェクトでは、次のパラメータを指定できます:

- `schema`: 抽出に使用するスキーマ。
- `systemPrompt`: 抽出に使用するシステムプロンプト。
- `prompt`: スキーマを用いない抽出に使用するプロンプト。
