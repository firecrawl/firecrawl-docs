<div id="introducing-map">
  ## /map の紹介
</div>

単一のURLからウェブサイト全体のマップを素早く取得する最も簡単な方法です。次の用途に非常に有用です:

- エンドユーザーにスクレイプ対象のリンクを選ばせたいとき
- サイト上のリンクをすばやく把握したいとき
- 特定のトピックに関連するページをスクレイプしたいとき（`search` パラメータを使用）
- サイトの特定ページだけをスクレイプしたいとき

<div id="map-with-search">
  #### 検索付きマップ
</div>

`search` パラメータを使うと、サイト内の特定のURLを検索できます。

```bash cURL
curl -X POST https://api.firecrawl.dev/v1/map \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "url": "https://firecrawl.dev",
      "search": "docs"
    }'
```

回答は、関連性の高い順に並んだ番号付きリストで返されます。

```json
{
  "status": "success",
  "links": [
    "https://docs.firecrawl.dev",
    "https://docs.firecrawl.dev/sdks/python",
    "https://docs.firecrawl.dev/learn/rag-llama3",
  ]
}
```
