<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## Firecrawlでウェブ検索し、結果をスクレイピングする
</div>

Firecrawlはsearch APIを堅牢なスクレイピング基盤と統合し、単一のエンドポイントで検索からスクレイピングまでをシームレスに提供します。理由は次のとおりです。

1. **統一された検索クエリ:**
   ユーザーはsearchエンドポイント経由で検索クエリを送信します。

2. **結果の自動スクレイピング:**
   Firecrawlが検索結果を自動処理し、各結果ページからデータを抽出します。

3. **データの提供:**
   すべての結果ページからスクレイピングしたデータはクリーンなMarkdownにまとめられ、すぐに利用できます。

この統合により、ユーザーは最小限の手間でウェブ検索を効率的に実行し、複数のソースから包括的なスクレイピング済みデータを取得できます。

詳細は[Search Endpoint Documentation](https://docs.firecrawl.dev/api-reference/endpoint/search)を参照してください。

<div id="search-any-query">
  ## 任意のクエリを検索する
</div>

<div id="search-endpoint">
  ### /search エンドポイント
</div>

ウェブを検索し、最も関連性の高い結果を取得し、各ページをスクレイピングしてMarkdownを返します。

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // 高速な SERP API にするなら false
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Markdown コンテンツ",
      "provider": "web-scraper",
      "metadata": {
        "title": "Firecrawl | LLM 向けに確実にウェブをスクレイピング",
        "description": "CX とセールス向けのAI"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="with-python-sdk">
  ### Python SDK の利用
</div>

<div id="installing-python-sdk">
  #### Python SDKのインストール
</div>

```bash
pip install firecrawl-py
```

<div id="search-a-query">
  #### クエリ検索を行う
</div>

```python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

result = app.search(query="What is firecrawl?")
```

レスポンスは、上記の curl コマンドの例と同様のものになります。

<div id="installing-javascript-sdk">

</div>

```bash
npm install @mendable/firecrawl-js
```

#### クエリで検索する

```javascript
import FirecrawlApp from '@mendable/firecrawl-js';

// APIキーを使用してFirecrawlAppを初期化
const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

// 検索を実行
const result = await app.search('What is firecrawl?');
```

レスポンスは、上記の curl コマンドで示したものと同様になります。


<div id="with-go-sdk">
  ### Go SDK を使う
</div>

<div id="installing-go-sdk">
  #### Go SDK のインストール
</div>

```bash
go get github.com/firecrawl/firecrawl-go
```

#### クエリで検索する

```go
import (
  "fmt"
  "log"

  "github.com/firecrawl/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
  }

  query := "What is firecrawl?"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("Failed to search: %v", err)
  }
  fmt.Println(searchResult)
}
```


<div id="with-rust-sdk">
  ### Rust SDK を使用する
</div>

<div id="installing-rust-sdk">
  #### Rust SDK のインストール
</div>

`Cargo.toml` に次を追加してください:

```toml
[dependencies]
firecrawl = "^0.1"
tokio = { version = "^1", features = ["full"] }
serde = { version = "^1.0", features = ["derive"] }
serde_json = "^1.0"
uuid = { version = "^1.10", features = ["v4"] }

[build-dependencies]
tokio = { version = "1", features = ["full"] }
```

#### クエリを検索

```rust
async fn main() {
  let api_key = "YOUR_API_KEY";
  let api_url = "https://api.firecrawl.dev";
  let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

  let query = "What is firecrawl?";
  let search_result = app.search(query).await;

  match search_result {
    Ok(data) => println!("Search Result: {}", data),
    Err(e) => eprintln!("Failed to search: {}", e),
  }
}
```
