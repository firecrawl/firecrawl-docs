<div id="overview">
  ## 概要
</div>

PHPのcURL拡張機能を使って、Firecrawlにスクレイピングリクエストを送信します。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl APIキー
* cURLが有効な PHP 8.1 以降

<div id="install">
  ## インストール
</div>

追加のパッケージは不要です。この例では、標準のcURL拡張機能を使用します。

<div id="configure-your-api-key">
  ## APIキーを設定する
</div>

APIキーをサーバーの環境変数に保存します。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## 最初のリクエストを送信する
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
```

<div id="complete-example">
  ## 完全なサンプル
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown', 'html'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
if ($response === false) {
    throw new RuntimeException(curl_error($ch));
}
curl_close($ch);

echo $response;
```
