<div id="overview">
  ## 概要
</div>

Goの標準ライブラリを使って、HTTP経由でFirecrawlにスクレイピングリクエストを送信します。

<div id="prerequisites">
  ## 前提条件
</div>

* FirecrawlのAPIキー
* Go 1.20+

<div id="install">
  ## インストール
</div>

追加のパッケージは不要で、この例では標準ライブラリを使用します。

<div id="configure-your-api-key">
  ## APIキーを設定する
</div>

プログラムを実行する前に、APIキーを環境変数に設定します。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## 最初のリクエストを送信する
</div>

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
    "os"
)

func main() {
    payload := map[string]any{
        "url": "https://example.com",
        "formats": []string{"markdown"},
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://api.firecrawl.dev/v2/scrape", bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("FIRECRAWL_API_KEY"))
    req.Header.Set("Content-Type", "application/json")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    var result map[string]any
    _ = json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result)
}
```

<div id="complete-example">
  ## 完全な例
</div>

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
    "os"
)

func main() {
    payload := map[string]any{
        "url": "https://example.com",
        "formats": []string{"markdown", "html"},
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://api.firecrawl.dev/v2/scrape", bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("FIRECRAWL_API_KEY"))
    req.Header.Set("Content-Type", "application/json")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    raw, _ := io.ReadAll(resp.Body)
    fmt.Println(string(raw))
}
```
