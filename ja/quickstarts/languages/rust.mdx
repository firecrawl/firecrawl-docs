<div id="overview">
  ## 概要
</div>

Firecrawl Rust SDKを使用すると、非同期Rustコードからv2 APIを呼び出せます。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl の API キー
* tokio を含む Rust ツールチェーン

<div id="install">
  ## インストール
</div>

```bash
cargo add firecrawl tokio --features tokio/full
```

<div id="configure-your-api-key">
  ## API キー を設定する
</div>

環境変数に API キー を設定します。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## 最初のリクエストを送信する
</div>

```rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    let api_key = std::env::var("FIRECRAWL_API_KEY").expect("APIキーがありません");
    let app = FirecrawlApp::new(api_key).expect("Firecrawlの初期化に失敗しました");

    let doc = app
        .scrape_url("https://example.com", None)
        .await
        .expect("スクレイピングに失敗しました");

    println!("{}", doc.markdown.unwrap_or_default());
}
```

<div id="complete-example">
  ## 完全な例
</div>

```rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    let api_key = std::env::var("FIRECRAWL_API_KEY").expect("Missing API key");
    let app = FirecrawlApp::new(api_key).expect("Failed to init Firecrawl");

    let doc = app
        .scrape_url("https://example.com", None)
        .await
        .expect("Scrape failed");

    if let Some(metadata) = doc.metadata {
        println!("Title: {}", metadata.title.unwrap_or_default());
    }
    println!("{}", doc.markdown.unwrap_or_default());
}
```
