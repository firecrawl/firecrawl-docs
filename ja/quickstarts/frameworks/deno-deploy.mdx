<div id="overview">
  ## 概要
</div>

Deno Deployのfetch APIを利用してFirecrawlを呼び出します。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl APIキー
* Deno Deployのプロジェクト

<div id="install">
  ## インストール
</div>

追加のパッケージは必要ありません。Deno には `fetch` が含まれています。

<div id="configure-environment">
  ## 環境を設定する
</div>

Deno Deploy プロジェクトの環境変数に API キーを設定します。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## サーバールートを追加
</div>

main.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

<div id="call-it-from-your-app">
  ## アプリから呼び出す
</div>

```js
const response = await fetch('https://your-app.deno.dev', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## 注意事項
</div>

* Deno Deploy では外向きの fetch リクエストしか許可されていないため、Firecrawl への呼び出しはすべてサーバー側で行ってください。
