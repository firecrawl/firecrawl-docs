<div id="overview">
  ## 概要
</div>

Laravel の HTTP クライアントを使って Firecrawl API を呼び出します。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl のAPIキー
* Laravel アプリ
* PHP 8.1+

<div id="install">
  ## インストール
</div>

追加のパッケージは不要です。Laravel には標準で HTTP クライアントが搭載されています。

<div id="configure-environment">
  ## 環境を設定する
</div>

APIキーを`.env`ファイルに追加します。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## サーバー側のルートを追加する
</div>

routes/api.php

```php
<?php

use Illuminate\Support\Facades\Http;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Route;

Route::post('/firecrawl', function (Request $request) {
    $response = Http::withToken(env('FIRECRAWL_API_KEY'))
        ->post('https://api.firecrawl.dev/v2/scrape', [
            'url' => $request->input('url'),
            'formats' => ['markdown'],
        ]);

    return $response->json();
});
```

<div id="call-it-from-your-app">
  ## アプリから呼び出す
</div>

```bash
curl -X POST http://localhost:8000/api/firecrawl               -H 'Content-Type: application/json'               -d '{"url": "https://example.com"}'
```

<div id="notes">
  ## 注意事項
</div>

* APIキー はサーバーサイドの環境変数にのみ保存してください。
