<div id="overview">
  ## 概要
</div>

Hono のルート内で Firecrawl HTTP API を直接使用することで、ランタイムを問わず動作します。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl APIキー
* Honoアプリ
* サポートされているランタイム (Node、Bun、Cloudflare、Deno)

<div id="install">
  ## インストール
</div>

```bash
npm install hono
```

<div id="configure-environment">
  ## 環境を設定する
</div>

実行環境の環境バインディングに APIキー を設定します。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## サーバールートを追加
</div>

src/index.ts

```ts
import { Hono } from 'hono';

type Bindings = { FIRECRAWL_API_KEY: string };

const app = new Hono<{ Bindings: Bindings }>();

app.post('/api/firecrawl', async (c) => {
  const { url } = await c.req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${c.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return c.json(data);
});

export default app;
```

<div id="call-it-from-your-app">
  ## アプリから呼び出す
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## 注意事項
</div>

* 使用するランタイム (Wrangler、Deno Deploy など) に応じて、環境バインディングを設定してください。
