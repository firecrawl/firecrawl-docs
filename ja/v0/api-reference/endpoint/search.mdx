title: "Search（ベータ）"
openapi: '/ja/v0/api-reference/v0-openapi.json POST /search'
---

/search エンドポイントは、検索APIにFirecrawlの機能を組み合わせ、あらゆるクエリに対して強力な検索体験を提供します。

クエリに対して自動でウェブ検索を行い、上位ページから最も関連性の高い結果をMarkdownフォーマットで返します。上位結果の各ウェブサイトを実際にスクレイプするため、常にコンテンツ全体を取得できる点がこのエンドポイントの利点です。

このエンドポイントは現在ベータ版であり、今後変更される可能性があります。
