<div id="text-formatting">
  ## 文本格式
</div>

我们支持大多数 Markdown 格式。只需在文本两侧添加 `**`、`_` 或 `~` 即可完成格式化。

| 样式     | 写法               | 结果              |
| -------- | ------------------ | ----------------- |
| 加粗     | `**bold**`         | **bold**          |
| 斜体     | `_italic_`         | _italic_          |
| 删除线   | `~strikethrough~`  | ~strikethrough~   |

这些格式可以组合使用。例如，输入 `**_bold and italic_**` 可得到 **_bold and italic_** 文本。

书写上标和下标需要使用 HTML，也就是在文本两侧添加 `<sup>` 或 `<sub>`。

| 文本类型 | 写法                       | 结果                    |
| -------- | -------------------------- | ----------------------- |
| 上标     | `<sup>superscript</sup>`   | <sup>superscript</sup>  |
| 下标     | `<sub>subscript</sub>`     | <sub>subscript</sub>    |

<div id="linking-to-pages">
  ## 链接到页面
</div>

你可以通过将文本包裹在 `[]()` 中来添加链接。比如，可以写 `[link to google](https://google.com)` 来 [link to google](https://google.com)。

文档中的页面链接需要使用根相对路径。也就是说，应当包含完整的文件夹路径。比如，`[link to text](/writing-content/text)` 会链接到我们组件部分的 “Text” 页面。

像 `[link to text](../text)` 这样的相对链接打开速度会更慢，因为我们无法对它们进行同样的优化。

<div id="singleline">
  ### 单行
</div>

要创建引用块，请在段落前添加一个 `>`。

> 多萝西跟着她穿过她城堡里许多漂亮的房间。

```md
> 多萝西跟着她穿过城堡里许多漂亮的房间。
```


<div id="multiline">
  ### 多行
</div>

> 多萝西跟着她穿过城堡里许多华丽的房间。
>
> 女巫吩咐她把锅具刷洗干净、把地扫清，还要不停地往火里添柴。

```md
> 多萝西跟着她，穿过城堡里许多华美的房间。
>
> 女巫吩咐她把锅壶洗净、打扫地板，并不停往火里添柴。
```
