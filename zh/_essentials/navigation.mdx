导航菜单是每个网站上的链接列表。

每次添加新页面时，你通常需要更新 `mint.json`。页面不会自动出现。

<div id="navigation-syntax">
  ## 导航语法
</div>

我们的导航语法是递归的，这意味着你可以创建嵌套的导航组。页面名称中无需包含 `.mdx`。

<CodeGroup>

```json Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Nested Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

<div id="folders">
  ## 文件夹
</div>

只需将你的 MDX 文件放入相应文件夹，并在 `mint.json` 中更新路径。

例如，要在 `https://yoursite.com/your-folder/your-page` 提供页面，你需要创建一个名为 `your-folder` 的文件夹，并在其中放入一个名为 `your-page.mdx` 的 MDX 文件。

<Warning>
  除非将其嵌套在其他文件夹内，否则不能使用 `api` 作为文件夹名称。Mintlify 基于 Next.js，而 Next.js 会将顶层的 `api` 文件夹保留用于内部服务器调用。像 `api-reference` 这样的文件夹名称是可接受的。
</Warning>

```json 带文件夹的导航
"navigation": [
    {
        "group": "分组名称",
        "pages": ["your-folder/your-page"]
    }
]
```


<div id="hidden-pages">
  ## 隐藏页面
</div>

未包含在 `mint.json` 中的 MDX 文件不会显示在侧边栏中，但仍可通过搜索栏访问，或直接链接到它们。
