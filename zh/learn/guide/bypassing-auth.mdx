> 注意：本示例使用的是 [Firecrawl API 的 v0 版本](/zh/v0/introduction)。你可以为 Python SDK 安装 0.0.20 版本，或为 Node SDK 安装 0.0.36 版本。

使用 Firecrawl，你可以在请求头中携带 cookies 来绕过网站的身份验证。当你需要抓取要求登录的网站内容时，这会非常有用。

<div id="example">
  ### 示例
</div>

要绕过身份验证，你需要在请求中包含 `headers` 参数，并将 `Cookies` 头设置为从浏览器中获取的相应 cookie 值。若需要帮助获取 cookies，请参考这篇文章：[Grabbing cookies](https://cookie-script.com/documentation/how-to-check-cookies-on-chrome-and-firefox)

<div id="usage">
  ### 用法
</div>

<CodeGroup>
  ```python Python
  from firecrawl import FirecrawlApp

  app = FirecrawlApp(api_key="YOUR_API_KEY")

  app.scrape_url("https://docs.firecrawl.dev", {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  })
  ```

  ```javascript JavaScript
  import { FirecrawlApp } from 'firecrawl-js';

  const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

  app.scrapeUrl('https://docs.firecrawl.dev', {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  });
  ```

  ```go Go
  import (
    "fmt"
    "log"

    "github.com/mendableai/firecrawl-go"
  )

  func main() {
    // 使用你的 API 密钥初始化 FirecrawlApp
    app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
    if err != nil {
      log.Fatalf("初始化 FirecrawlApp 失败: %v", err)
    }

    params := map[string]any{
      "pageOptions": map[string]any{
        "headers": map[string]any{
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      },
    }
    scrapedData, err := app.ScrapeURL("docs.firecrawl.dev", params)
    if err != nil {
      log.Fatalf("抓取过程中发生错误: %v", err)
    }
    fmt.Println(scrapedData)
  }
  ```

  ```rust Rust
  use firecrawl::FirecrawlApp;

  #[tokio::main]
  async fn main() {
      let api_key = "YOUR_API_KEY";
      let api_url = "https://api.firecrawl.dev";
      let app = FirecrawlApp::new(api_key, api_url).expect("初始化 FirecrawlApp 失败");

      let scrape_params = json!({
          "pageOptions": {
              "headers": {
                  "Cookie": "cookie1=value1; cookie2=value2"
              }
          }
      });

      let scrape_result = app.scrape_url("https://docs.firecrawl.dev", Some(scrape_params)).await;
      match scrape_result {
          Ok(data) => println!("抓取结果:\n{}", data),
          Err(e) => eprintln!("抓取失败: {}", e),
      }
  }
  ```

  ```curl cURL
  curl -X POST https://api.firecrawl.dev/v0/scrape \
      -H 'Content-Type: application/json' \
      -H 'Authorization: Bearer YOUR_API_KEY' \
      -d '{
        "url": "https://example.com",
        "headers": {
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      }'
  ```
