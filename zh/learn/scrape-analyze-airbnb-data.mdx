> 注意：此示例使用的是 [Firecrawl API 的 v0 版本](/zh/v0/introduction)。你可以为 Python SDK 安装 0.0.20 版本，或为 Node SDK 安装 0.0.36 版本。

> [GitHub 仓库 (https://github.com/e2b-dev/e2b-cookbook)](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)

本示例演示如何使用 [Firecrawl](https://www.firecrawl.dev/) 与 E2B 的 [Code Interpreter SDK](https://github.com/e2b-dev/code-interpreter) 抓取并分析 Airbnb 数据。

我们不会手写任何分析代码，而是让 Anthropic 的 AI 为我们生成。我们将使用 E2B 的 Code Interpreter SDK 安全运行代码并获取结果。

<img className="block" src="/images/airbnb_prices_chart.png" alt="使用 Firecrawl 和 E2B 抓取并分析 Airbnb 数据" />

<div id="prerequisites">
  ## 前置条件
</div>

* 你的设备已安装 Node.js
* 一个 E2B API 密钥
* 一个 Firecrawl API 密钥
* 一个 Anthropic API 密钥

<div id="setup-run">
  ## 安装与运行
</div>

<div id="1-clone-the-repository">
  ### 1. 克隆仓库
</div>

这是本示例的 [GitHub 仓库](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)。

```
git clone https://github.com/e2b-dev/e2b-cookbook.git
cd e2b-cookbook/examples/scrape-and-analyze-airbnb-data-with-firecrawl
```

<div id="2-install-dependencies">
  ### 2. 安装依赖项
</div>

```
npm install
```

<div id="3-set-up-env">
  ### 3. 设置 `.env`
</div>

1. 将 `.env.template` 复制为 `.env`
2. 获取 [E2B API 密钥](https://e2b.dev/docs/getting-started/api-key)
3. 获取 [Firecrawl API 密钥](https://firecrawl.dev)
4. 获取 [Anthropic API 密钥](https://anthropic.com)

<div id="4-run-the-example">
  ### 4. 运行示例
</div>

```
npm run start
```
