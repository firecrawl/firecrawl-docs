Firecrawl Enterprise 专为需要在严格的安全、合规和支持要求下大规模处理网页数据的团队打造。Enterprise 计划包含我们自助计划中的全部内容，此外还提供自定义额度、专属支持，以及下方列出的安全和访问控制功能。

<Note>
  Enterprise 功能由我们的团队负责开通。要启用这些功能或洽谈定制方案，请访问 [Enterprise page](https://www.firecrawl.dev/enterprise) 或联系 [help@firecrawl.dev](mailto:help@firecrawl.dev)。
</Note>

<div id="security-compliance">
  ## 安全与合规
</div>

* **SOC 2 Type II**：经独立审计的安全控制，涵盖数据保护和运营安全。
* **零数据保留 (ZDR)**：页面内容和提取出的数据均在内存中处理，不会在请求生命周期结束后持久化存储。适用于抓取 ([Scrape ZDR](/zh/features/scrape#zero-data-retention-zdr)) 和搜索 ([Search ZDR](/zh/features/search#zero-data-retention-zdr)) ，还包括端到端模式；在该模式下，我们的上游搜索提供商也会执行 ZDR。
* **PII 脱敏**：自动对抓取和解析输出中的个人身份信息进行脱敏处理。请参见 [PII redaction](/zh/features/scrape#pii-redaction)。
* **DPA 和定制合同**：提供数据处理协议、定制 MSA 和安全审查，以满足您的采购要求。

<div id="identity-access-management">
  ## 身份与访问管理
</div>

* **单点登录 (SSO)**：支持 SAML 和 OIDC 单点登录，让团队通过现有身份提供商 (Okta、Entra ID、Google Workspace 等) 完成身份验证。
* **SCIM 目录同步**：从你的目录中自动预配和停用用户，确保访问权限始终与组织保持同步。
* **静态 IP 允许列表**：通过专用的白名单 IP 地址路由流量，方便在你的系统中配置允许列表。
* **API 密钥 IP 限制**：将单个 API 密钥限定在特定 IP 范围内，使其仅能在获准的网络中使用。

<div id="scale-performance">
  ## 扩展与性能
</div>

* **自定义额度规模**：根据你的使用量定制额度配额，超出标准 Scale (扩展) 档位。
* **跨团队共享额度**：将多个团队归入同一组织，在团队间共享统一的额度池，同时支持按团队跟踪使用情况并生成报告。
* **支出限制**：可按 API 密钥或团队设置支出上限，帮助控制使用量和成本。
* **自定义并发**：根据你的工作负载定制并发浏览器限制，以实现最大吞吐量。请参见 [Rate Limits](/zh/rate-limits)。
* **预留并发**：将部分并发浏览器容量专门分配给特定团队或工作负载，确保高优先级任务不会因其他流量而得不到资源。
* **延长额度结转**：年度 Scale (扩展) 和 Enterprise 方案中的额度可结转一部分，而不是在月底失效。请参见 [Billing](/zh/billing)。
* **批量折扣**：针对大规模承诺提供基于用量的价格优惠。
