<div id="welcome-to-firecrawl">
  ## 欢迎使用 Firecrawl
</div>

[Firecrawl](https://firecrawl.dev?ref=github) 是一项 API 服务，输入一个 URL 即可对其进行爬取，并将内容转换为干净的 Markdown。我们会爬取所有可访问的子页面，并为每个页面提供干净的 Markdown。无需提供 sitemap。

<div id="how-to-use-it">
  ## 如何使用？
</div>

我们提供托管版的易用 API。你可以在[此处](https://firecrawl.dev/playground)找到 Playground 和文档。也支持自托管后端。

查看以下资源开始上手：

* [x] **API**：[文档](https://docs.firecrawl.dev/api-reference/introduction)
* [x] **SDKs**：[Python](https://docs.firecrawl.dev/sdks/python)、[Node](https://docs.firecrawl.dev/sdks/node)、[Go](https://docs.firecrawl.dev/sdks/go)、[Rust](https://docs.firecrawl.dev/sdks/rust)
* [x] **LLM 框架**：[LangChain（Python）](https://python.langchain.com/docs/integrations/document_loaders/firecrawl/)、[LangChain（JS）](https://js.langchain.com/docs/integrations/document_loaders/web_loaders/firecrawl)、[LlamaIndex](https://docs.llamaindex.ai/en/latest/examples/data_connectors/WebPageDemo/#using-firecrawl-reader)、[Crew.ai](https://docs.crewai.com/)、[Composio](https://composio.dev/tools/firecrawl/all)、[PraisonAI](https://docs.praison.ai/firecrawl/)、[Superinterface](https://superinterface.ai/docs/assistants/functions/firecrawl)、[Vectorize](https://docs.vectorize.io/integrations/source-connectors/firecrawl)

<div id="features">
  ### 功能
</div>

* [**Scrape**](#scraping): 抓取单个 URL，并以适用于 LLM 的 formats 返回其内容（markdown、summary、通过 [JSON 模式](#json-mode) 的结构化数据、screenshot、html）
* [**Crawl**](#crawling): 抓取网页中的所有 URL，并以适用于 LLM 的 formats 返回内容
* [**Map**](/zh/features/map): 输入网站，极速获取该站点的所有 URL
* [**Search**](/zh/features/search): 搜索网页并获取结果页面的完整内容
* [**Extract**](/zh/features/extract): 使用 AI 从单页、多页或整站提取结构化数据。

<div id="powerful-capabilities">
  ### 强大功能
</div>

* **面向 LLM 的 formats**：markdown、summary、structured data、screenshot、HTML、links、metadata
* **硬骨头场景**：代理、反机器人机制、动态内容（JS 渲染）、输出解析、编排
* **高度可定制**：排除标签、通过自定义 headers 爬取登录后页面、设置最大爬取深度等
* **媒体解析**：PDF、DOCX、图像
* **可靠性优先**：为获取所需数据而设计——无论难度多大
* **actions**：在提取数据前执行 click、scroll、input、wait 等操作

你可以在我们的[文档](https://docs.firecrawl.dev/api-reference/v2-introduction)中查阅 Firecrawl 的全部功能及其用法

<div id="open-source-vs-cloud">
  ## 开源与云端
</div>

Firecrawl 采用开源模式，遵循 [AGPL-3.0 许可证](https://github.com/mendableai/firecrawl/blob/main/LICENSE)。

为打造最佳产品，我们在开源版本之外还提供托管版 Firecrawl。云端方案使我们能够持续创新，并为所有用户提供高质量、可持续的服务。

Firecrawl Cloud 可在 [firecrawl.dev](https://firecrawl.dev) 使用，并提供一系列开源版本所不具备的功能：
