<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## 使用 Firecrawl 进行网页搜索并抓取结果
</div>

Firecrawl 将其 SERP（搜索引擎结果页）API 与强大的抓取基础设施集成，通过单一端点提供无缝的搜索与抓取功能。原因如下：

1. **统一搜索查询：**
   用户通过 SERP 端点提交搜索查询。

2. **自动化结果抓取：**
   Firecrawl 会自动处理搜索结果，并利用其抓取能力从每个结果页面提取数据。

3. **数据交付：**
   来自所有结果页面的抓取数据将被整理并以清晰的 Markdown 形式交付，开箱即用。

这种集成使用户能够高效地进行网页搜索，并以最小的工作量从多个来源获取全面的抓取数据。

更多详情，请参阅[搜索端点文档](https://docs.firecrawl.dev/api-reference/endpoint/search)。

<div id="search-any-query">
  ## 搜索任意内容
</div>

<div id="search-endpoint">
  ### /search 端点
</div>

用于进行网络搜索，获取最相关的结果，抓取每个页面并返回 markdown。

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer <YOUR_API_KEY>' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // 若需更快的 SERP API，请设为 false
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Markdown 内容",
      "provider": "web-scraper",
      "metadata": {
        "title": "Firecrawl | 为你的 LLM 稳定可靠地抓取网页",
        "description": "面向客户体验与销售的 AI"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="search-a-query">
  #### 搜索查询
</div>

```go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("初始化 FirecrawlApp 失败：%v", err)
  }

  query := "什么是 Firecrawl？"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("搜索失败：%v", err)
  }
  fmt.Println(searchResult)
}
```
