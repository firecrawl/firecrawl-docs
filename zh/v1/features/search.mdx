Firecrawl 的搜索 API 允许你进行网页搜索，并可在一次操作中可选地抓取搜索结果。

* 选择特定输出 formats（markdown、HTML、links、screenshots）
* 使用可自定义参数（location 等）进行全网搜索
* 可选以多种 formats 从搜索结果中获取内容
* 控制结果数量并设置超时

详情请参见 [Search Endpoint API Reference](https://docs.firecrawl.dev/api-reference/endpoint/search)。

常用的 `tbs` 值：

- `qdr:h` - 过去 1 小时
- `qdr:d` - 过去 24 小时
- `qdr:w` - 过去 1 周
- `qdr:m` - 过去 1 个月
- `qdr:y` - 过去 1 年

如需更精确的时间筛选，可使用自定义日期范围格式指定起止日期：

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp

# 使用你的 API 密钥初始化客户端
app = FirecrawlApp(api_key="fc-YOUR_API_KEY")

# 搜索 2024 年 12 月的结果
search_result = app.search(
    "firecrawl updates",
    limit=10,
    tbs="cdr:1,cd_min:12/1/2024,cd_max:12/31/2024"
)
```

```js JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

// 使用你的 API 密钥初始化客户端
const app = new FirecrawlApp({apiKey: "fc-YOUR_API_KEY"});

// 搜索 2024 年 12 月的结果
app.search("firecrawl updates", {
  limit: 10,
  tbs: "cdr:1,cd_min:12/1/2024,cd_max:12/31/2024"
})
.then(searchResult => {
  console.log(searchResult.data);
});
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer fc-YOUR_API_KEY" \
  -d '{
    "query": "firecrawl updates",
    "limit": 10,
    "tbs": "cdr:1,cd_min:12/1/2024,cd_max:12/31/2024"
  }'
```

</CodeGroup>

<div id="scraping-options">
  ## 抓取选项
</div>

在抓取搜索结果时，你可以指定多个输出 formats 和高级抓取选项：

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp, ScrapeOptions

# 使用你的 API 密钥初始化客户端
app = FirecrawlApp(api_key="fc-YOUR_API_KEY")

# 以多种 formats 获取搜索结果
search_result = app.search(
    "firecrawl features",
    limit=3,
    scrape_options=ScrapeOptions(formats=["markdown", "html", "links", "screenshot"])
)
```

```js JavaScript
import FirecrawlApp from '@mendable/firecrawl-js';

// 使用你的 API 密钥初始化客户端
const app = new FirecrawlApp({apiKey: "fc-YOUR_API_KEY"});

// 以多种 formats 获取搜索结果
app.search("firecrawl features", {
  limit: 3,
  scrapeOptions: {
    formats: ["markdown", "html", "links", "screenshot"]
  }
})
.then(searchResult => {
  // 以多种 formats 处理结果
  console.log(searchResult.data);
});
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v1/search \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer fc-YOUR_API_KEY" \
  -d '{
    "query": "firecrawl features",
    "limit": 3,
    "scrapeOptions": {
      "formats": ["markdown", "html", "links", "screenshot"]
    }
  }'
```

</CodeGroup>

可用 formats：

- `markdown`: 干净、格式良好的 Markdown 内容
- `html`: 处理后的 HTML 内容
- `rawHtml`: 未修改的 HTML 内容
- `links`: 页面中发现的链接列表
- `screenshot`: 页面截图
- `screenshot@fullPage`: 整页截图
- `extract`: 结构化数据提取

<div id="cost-implications">
  ## 成本影响
</div>

当搜索结果不进行抓取（未指定任何抓取选项）时，费用为每 10 条搜索结果 2 点额度。启用抓取后，除标准抓取费用外，对每条搜索结果的基础抓取不再另外收费。

但请注意以下成本因素：

- **PDF 解析**：每个 PDF 页面 1 点额度（多页 PDF 会显著增加成本）
- **隐身代理模式**：每条搜索结果 +4 点额度
- ***JSON 模式***：每条搜索结果 +4 点额度

控制成本的方法：

- 若不需要 PDF 内容，请将 `parsePDF` 设为 `false`
- 尽可能使用 `proxy: "basic"`，而非 `"stealth"`
- 使用 `limit` 参数限制搜索结果数量
