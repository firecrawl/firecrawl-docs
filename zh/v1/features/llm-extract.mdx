<div id="scrape-and-extract-structured-data-with-firecrawl">
  ## 使用 Firecrawl 抓取并提取结构化数据
</div>

{/* <Warning>Scrape LLM Extract 将在未来版本中弃用。请使用新的 [Extract](/features/extract) 端点。</Warning> */}

Firecrawl 使用 AI 通过 3 个步骤从网页获取结构化数据：

1. **设置 Schema：**
   通过定义 JSON schema（采用 OpenAI 的格式）并提供网页 URL，告知我们你需要的数据。

2. **发起请求：**
   将你的 URL 和 schema 发送到我们的 /scrape 端点。查看方法：
   [Scrape Endpoint Documentation](https://docs.firecrawl.dev/api-reference/endpoint/scrape)

3. **获取数据：**
   返回与 schema 匹配的干净、结构化数据，可立即使用。

这让你能够快速、轻松地以所需的格式获取网页数据。

<div id="json-options-object">
  ### JSON 选项对象
</div>

`jsonOptions` 对象可接受以下参数：

- `schema`：用于提取的架构。
- `systemPrompt`：用于提取的系统提示词。
- `prompt`：在未提供架构时用于提取的提示词。
