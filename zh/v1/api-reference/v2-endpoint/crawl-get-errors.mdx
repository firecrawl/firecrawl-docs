<div id="whats-new-in-v2">
  ## v2 新增内容
</div>

<div id="better-error-messages">
  ### 更清晰的错误信息
</div>

- 当自然语言提示不起作用时，给出明确的错误信息
- 针对 sitemap 和深度限制问题提供具体提示
- 针对整站爬取问题提供更友好的错误信息

<div id="more-error-details">
  ### 更多错误详情
</div>

- 提供更详细的 formats 处理错误信息
- 针对 PDF 和解析器失败给出更具体的提示
- 改进基于位置的抓取调试能力
- 为 changeTracking 问题提供更清晰的错误信息

<div id="faster-and-easier-debugging">
  ### 更快、更易用的调试
</div>

- 大规模抓取时更快获取错误
- 更智能的错误分组
- 更有用的错误上下文

API 与 v1 保持一致，但对 v2 功能的错误报告大幅提升。
