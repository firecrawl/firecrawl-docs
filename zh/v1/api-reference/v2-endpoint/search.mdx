<div id="whats-new-in-v2">
  ## v2 有哪些新变化
</div>

<div id="search-multiple-sources">
  ### 同时搜索多个来源
</div>

一次搜索网页、图片和新闻：

```json
{
  "query": "firecrawl 网页抓取",
  "sources": [
    {"type": "web", "tbs": "qdr:w", "location": "US"},
    {"type": "images"},
    {"type": "news"}
  ]
}
```

<div id="response-format-changed">
  ### 响应格式已更改
</div>

v1：平铺的结果列表。v2：按来源类型组织：

```json
{
  "success": true,
  "data": {
    "web": [/* 网页搜索结果 */],
    "images": [/* 图片搜索结果 */],
    "news": [/* 新闻搜索结果 */]
  }
}
```

<div id="new-features">
  ### 新功能
</div>

- 按时间范围筛选（“past week”、“past month”）
- 目标国家/地区可指定
- 内测期间结果上限为 500

/search 端点将网页搜索（SERP）与 Firecrawl 的抓取能力相结合，可为任意查询返回完整页面内容。

在请求中包含 `scrapeOptions`，并设置 `formats: [{"type": "markdown"}]`，即可为每个搜索结果获取完整的 Markdown 内容；否则将默认仅返回 SERP 结果（url、title、description）。你也可以使用其他 formats，例如 `{"type": "summary"}` 获取精简内容。

<div id="supported-query-operators">
  ## 支持的查询运算符
</div>

我们支持多种查询运算符，帮助你更高效地筛选搜索结果。

| 运算符 | 功能 | 示例 |
---|-|-|
| `""` | 对一段文本进行精确（非模糊）匹配 | `"Firecrawl"`
| `-` | 排除特定关键词或对其他运算符取反 | `-bad`, `-site:firecrawl.dev`
| `site:` | 仅返回来自指定网站的结果 | `site:firecrawl.dev`
| `inurl:` | 仅返回 URL 中包含某个词的结果 | `inurl:firecrawl`
| `allinurl:` | 仅返回 URL 中同时包含多个词的结果 | `allinurl:git firecrawl`
| `intitle:` | 仅返回页面标题中包含某个词的结果 | `intitle:Firecrawl`
| `allintitle:` | 仅返回页面标题中同时包含多个词的结果 | `allintitle:firecrawl playground`
| `related:` | 仅返回与特定域名相关的结果 | `related:firecrawl.dev`

<div id="location-parameter">
  ## 位置参数
</div>

使用 `location` 参数获取定向到特定地理位置的搜索结果。格式："string"。示例："Germany"、"San Francisco,California,United States"。

查看[支持的位置完整列表](https://firecrawl.dev/search_locations.json)，以了解所有可用的国家和语言。

<div id="time-based-search">
  ## 基于时间的搜索
</div>

使用 `tbs` 参数按时间范围筛选结果，支持自定义日期区间。请参阅 [Search 功能文档](https://docs.firecrawl.dev/features/search#time-based-search)以获取详细示例及支持的 formats。
