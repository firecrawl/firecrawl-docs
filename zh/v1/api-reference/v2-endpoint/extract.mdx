<div id="whats-new-in-v2">
  ## v2 新增与变更
</div>

<div id="smarter-extraction">
  ### 更智能的提取
</div>

FIRE-1（代理）能应对棘手的网站：

- 自动完成页面跳转与分页点击
- 适配重度依赖 JavaScript 的网站
- 更擅长发现和提取数据

<div id="no-url-required">
  ### 无需提供 URL
</div>

在不知道具体 URL 的情况下也能提取数据：

- 只需描述你的需求，v2 会自动找到
- 先搜索全网，再从最优页面提取
- 比 v1 更准确

<div id="better-performance">
  ### 更高性能
</div>

- 处理更快
- 错误处理更可靠
- 准确性更高

与 v1 使用相同的 API，升级很容易。
