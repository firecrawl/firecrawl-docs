<div id="whats-new-in-v2">
  ## v2 新增与变化
</div>

<div id="tell-the-crawler-what-you-want">
  ### 告诉爬虫你想要什么
</div>

用简明的英文描述需要爬取的内容：

```json
{
  "url": "https://example.com",
  "prompt": "仅抓取博客文章和文档，跳过营销页面"
}
```

<div id="better-sitemap-control">
  ### 更灵活的 Sitemap 控制
</div>

v1：sitemap 只有开/关。v2：提供三个选项：

- `"include"` - 使用 sitemap 并查找其他页面（默认）
- `"skip"` - 完全忽略 sitemap  
- `"only"` - 仅抓取 sitemap 中的 URL

<div id="new-crawling-options">
  ### 新的抓取选项
</div>

* `crawlEntireDomain` - 抓取整个域名，而不仅限于子页面
* `maxDiscoveryDepth` - 控制抓取深度（替代 `maxDepth`）

```json
{
  "url": "https://example.com/features",
  "crawlEntireDomain": true,
  "maxDiscoveryDepth": 2,
  "sitemap": "include"
}
```
