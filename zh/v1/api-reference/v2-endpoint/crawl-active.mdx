<div id="whats-new-in-v2">
  ## v2 有哪些新变化
</div>

<div id="see-what-your-crawls-are-doing">
  ### 查看爬取任务进展
</div>

- 用自然语言提示追踪爬取
- 查看每次爬取使用的站点地图模式
- 监控全站爬取与深度设置

<div id="more-details-on-active-crawls">
  ### 有关正在进行的抓取的更多详情
</div>

- 查看每次抓取所用的输出 formats
- 跟踪 PDF 处理与基于位置的抓取
- 更清晰地了解资源使用情况

<div id="faster-and-more-reliable">
  ### 更快更可靠
</div>

- 监控大量抓取任务时更新更迅速
- 资源跟踪更完善
- 监控更稳定可靠

API 与 v1 保持一致，同时对 v2 功能具备更好的可观测性。
