<div id="whats-new-in-v2">
  ## v2 有哪些新变化
</div>

v2 的取消爬取 API 在保持与 v1 相同功能的基础上，进一步提升了可靠性：

<div id="enhanced-cancellation-process">
  ### 取消流程增强
</div>

- 改进清理与资源管理
- 更完善的活动抓取终止处理
- 取消操作的错误处理更健全

API 参数和响应 formats 与 v1 保持一致，确保平滑迁移，同时提供更可靠的抓取取消能力。
