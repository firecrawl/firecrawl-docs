  ## v2 新增内容
</div>

v2 的取消批量抓取 API 在延续 v1 相同功能的同时，可靠性进一步提升：

<div id="enhanced-cancellation-process">
  ### 取消流程增强
</div>

- 改进了批处理操作的清理与资源管理
- 更可靠地处理正在运行的批次终止
- 加强了批量取消操作的错误处理

API 参数和响应 formats 与 v1 保持一致，既确保平滑迁移，又带来更可靠的批量取消功能。
