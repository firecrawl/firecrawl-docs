<div id="whats-new-in-v2">
  ## v2 新增内容
</div>

<div id="all-new-formats-work-in-batches">
  ### 全新 formats 均支持批处理
</div>

- 在所有 URL 上使用自定义截图、JSON 提取和变更跟踪
- 通过全新解析器系统批量处理 PDF
- 将相同的 format 设置应用于批次中的每个 URL

<div id="better-batch-processing">
  ### 更强的批量处理
</div>

- 为整批任务按不同国家/语言执行抓取
- 在所有 URL 上使用代理类型（basic、stealth 或 auto）
- 所有新增的抓取选项均支持批量模式

<div id="faster-and-more-reliable">
  ### 更快、更可靠
</div>

- 更佳的复杂批次资源管理
- 更完善的进度追踪
- 更健全的错误处理

与 v1 的 API 保持一致，所有 v2 功能适用于整个批次。
