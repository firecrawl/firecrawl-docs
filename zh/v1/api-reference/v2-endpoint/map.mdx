<div id="better-sitemap-control">
  ### 更灵活的 Sitemap 控制
</div>

处理 Sitemap 的三种方式：

* `"include"` - 使用 Sitemap 并同时发现其他页面（默认）
* `"skip"` - 完全忽略 Sitemap
* `"only"` - 仅返回 Sitemap 中的 URL

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```

<div id="smarter-url-discovery">
  ### 更智能的 URL 发现
</div>

- 覆盖网站上的所有页面
- 结果不限量（v1 上限为 500）
- 更快、更准确

与 v1 的 API 相同，但能发现更多页面，在复杂站点上表现更佳。
