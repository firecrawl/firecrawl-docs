<div id="whats-new-in-v2">
  ## v2 的最新功能
</div>

<div id="better-batch-error-messages">
  ### 更完善的批量错误信息
</div>

- 针对所有 URL 的 formats 处理问题提供清晰的错误提示
- 为 PDF 和解析器失败提供更具体的消息
- 对不同代理模式的错误进行更完善的跟踪
- 对基于位置的抓取问题提供清晰的错误提示

<div id="more-error-details">
  ### 更多错误详情
</div>

- 针对 changeTracking 问题的调试信息
- 浏览器自动化失败的错误报告
- 更完善的点数与资源限制错误追踪
- 针对批处理问题的具体错误

<div id="faster-and-easier-debugging">
  ### 更快、更简单的调试
</div>

- 更快速地获取大批量任务中的错误
- 更优的错误聚类/分组
- 更丰富、更有用的错误上下文

API 与 v1 保持一致，但在批处理场景的 v2 功能上提供了显著更好的错误报告。
