---
title: "LLMs.txt 状态"
openapi: "v1-openapi GET /llmstxt/{id}"
---

获取 LLMs.txt 生成任务的状态和结果。通过此端点可检查生成是否完成，并获取生成的内容。

<div id="response-structure">
  ### 响应结构
</div>

响应包含：

- **success**：布尔值，表示请求是否成功
- **status**：生成任务的当前状态：

  - `processing`：任务仍在运行
  - `completed`：任务已成功完成
  - `failed`：任务发生错误

- **data**：生成的内容（当状态为 `completed` 时）：

  - `llmstxt`：生成的 LLMs.txt 内容
  - `llmsfulltxt`：完整文本内容（如果 showFullText 为 true）

- **expiresAt**：ISO 时间戳，表示结果的失效时间

<div id="status-examples">
  ### 状态示例
</div>

1. **处理中**

   ```json
   {
     "success": true,
     "status": "processing",
     "data": {
       "llmstxt": "# Firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/)...",
       "llmsfulltxt": "# Firecrawl.dev llms-full.txt\n\n"
     },
     "expiresAt": "2025-03-03T23:19:18.000Z"
   }
   ```

2. **已完成**
   ```json
   {
     "success": true,
     "status": "completed",
     "data": {
       "llmstxt": "# http://firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/): 轻松将网站转换为干净、适配 LLM 的数据。\n- [Flexible Web Scraping Pricing](https://www.firecrawl.dev/pricing): 为网页抓取与数据提取提供灵活的定价方案。",
       "llmsfulltxt": "# http://firecrawl.dev llms-full.txt\n\n## Web Data Extraction Tool\nIntroducing /extract - 通过提示获取网页数据..."
     },
     "expiresAt": "2025-03-03T22:45:50.000Z"
   }
   ```

<div id="error-handling">
  ### 错误处理
</div>

如果生成作业失败或未找到，你会收到相应的错误响应：

```json
{
  "success": false,
  "error": "未找到 LLMs.txt 生成任务"
}
```


<div id="polling-recommendations">
  ### 轮询建议
</div>

1. 初始每隔 2–3 秒轮询一次
2. 若 30 秒后仍在处理，请加大轮询间隔
3. 当状态为 `completed` 或 `failed` 时停止轮询
4. 使用指数退避以避免触发速率限制
