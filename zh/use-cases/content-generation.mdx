<div id="how-it-works">
  ## 工作原理
</div>

Firecrawl 可从网站以多种 formats 提取洞见——包括结构化的 HTML、Markdown、JSON，以及截图。它还可按你的请求捕获图像并呈现相关的新闻内容。由此，你的 AI 生成内容既有事实依据，又能结合最新语境实现视觉增强。

<div id="what-you-can-create">
  ## 你可以创建的内容
</div>

* **销售演示**：基于潜在客户数据的定制化幻灯片
* **电子邮件营销**：规模化的个性化触达
* **营销内容**：数据驱动的博客与报告
* **社交媒体**：围绕热点话题与新闻的内容生成
* **文档**：自动更新的技术资料
* **新闻简报**：精选行业与竞品动态
* **视觉内容**：附带提取图片与截图的帖子与报告

<div id="faqs">
  ## 常见问题
</div>

<AccordionGroup>
  <Accordion title="Firecrawl 如何确保用于内容创作的数据准确性？">
    Firecrawl 直接从源网站提取数据，保留原始内容的结构和上下文。所有提取的数据都包含来源 URL 和时间戳，便于核验。
  </Accordion>

  <Accordion title="Firecrawl 能为内容生成提供哪些数据？">
    Firecrawl 可从网站提供干净的 Markdown、结构化 JSON、HTML、图像和截图。这些提取数据为你的内容生成工作流提供可靠的事实基础。
  </Accordion>

  <Accordion title="Firecrawl 能处理图像和新闻来源吗？">
    可以。Firecrawl 能提取图像、捕获截图，并抓取新闻网站的内容，帮助你创作更具视觉表现力的内容，及时掌握行业动态。
  </Accordion>

  <Accordion title="Firecrawl 能从哪些类型的网站提取数据？">
    Firecrawl 擅长从公司官网、新闻网站、博客和文档站点提取数据。具有结构化 HTML 和清晰内容层级的网站可产出更为干净的提取结果。
  </Accordion>

  <Accordion title="如何使用 Firecrawl 进行批量数据提取？">
    使用 Firecrawl 的批量抓取与 /crawl 端点，高效地从多个网站提取数据。并行处理数百个 URL，为你的内容工作流构建全面的数据集。
  </Accordion>
</AccordionGroup>
