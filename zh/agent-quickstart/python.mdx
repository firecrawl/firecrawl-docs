<div id="firecrawl-python-agent-quickstart">
  # Firecrawl Python 代理快速入门
</div>

本文件是面向外部代理的标准快速入门文档，由 SDK 源代码和 v2 OpenAPI 规范生成。

<div id="install">
  ## 安装
</div>

```bash
pip install firecrawl-py
```

<div id="authenticate">
  ## 认证
</div>

```python
import os
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])
```

<div id="when-to-use-what">
  ## 何时该用哪种方式
</div>

* `search`：当你从查询词开始并需要发现结果时使用。
* `scrape`：当你已经有 URL 并想获取页面内容时使用。
* `interact`：当页面需要点击、填写表单或抓取后的浏览器 actions 时使用。

<div id="search">
  ## 搜索
</div>

### 为什么要用它

使用 `search` 可根据查询词找到相关 URL，并可选择抓取每条结果。你也可以先用标准的 `site:` 运算符将查询限定在单个站点内 (例如 `site:docs.firecrawl.dev crawl webhooks`) ，再抓取特定 URL。

<div id="preferred-sdk-method">
  ### 推荐使用的 SDK 方法
</div>

`client.search(query, **options)`

<div id="simple-example">
  ### 简单示例
</div>

```python
results = client.search(
    "site:docs.firecrawl.dev crawl webhooks",
    limit=5,
)

print(results.web[0] if results.web else None)
```

<div id="complex-example">
  ### 复杂示例
</div>

```python
from firecrawl.v2.types import ScrapeOptions

results = client.search(
    "firecrawl pricing",
    sources=["web", "news"],
    categories=["research"],
    limit=10,
    tbs="qdr:w",
    location="San Francisco,California,United States",
    ignore_invalid_urls=True,
    scrape_options=ScrapeOptions(
        formats=[
            "markdown",
            {"type": "json", "prompt": "Extract pricing tiers and limits."},
        ],
        only_main_content=True,
        proxy="stealth",
    ),
    integration="agent-quickstart",
)
```

<div id="parameters">
  ### 参数
</div>

* `query`：搜索查询字符串。始终必填。
* `sources`：要搜索的数据来源。支持的值：`"web"`、`"news"`、`"images"` 或 `{"type": ...}` 对象。用于控制返回哪些结果数组。
* `categories`：结果筛选条件。支持的值：`"github"`、`"research"`、`"pdf"` 或 `{"type": ...}` 对象。用于按类别限制结果。
* `limit`：返回结果的最大数量。默认值：`5`。
* `tbs`：基于时间的搜索筛选字符串。用于按时间范围限制结果 (例如 `qdr:d`) 。
* `location`：用于返回本地化结果的位置字符串。用于地理位置相关的查询。
* `ignore_invalid_urls`：从结果中排除无效 URL。在将结果传递给其他端点时使用。
* `timeout`：请求超时时间 (毫秒) 。默认值：`300000`。
* `scrape_options`：应用于每个结果的抓取选项 (字段与 `scrape` 相同) 。当你希望结果中包含内容时使用。
* `integration`：集成标识符。用于跟踪/分析。

<div id="scrape">
  ## 抓取
</div>

### 为什么使用它

当你已经知道 URL，并且希望以特定格式获取内容时，请使用 `scrape`。

### 推荐的 SDK 方法

`client.scrape(url, **options)`

<div id="simple-example">
  ### 简单示例
</div>

```python
doc = client.scrape(
    "https://example.com",
    formats=["markdown"],
)

print(doc.markdown)
```

<div id="complex-example">
  ### 复杂示例
</div>

```python
doc = client.scrape(
    "https://example.com/docs",
    formats=[
        "markdown",
        {"type": "json", "prompt": "Extract headings and slugs."},
        {"type": "change_tracking", "modes": ["git-diff"], "tag": "docs"},
        {"type": "screenshot", "full_page": True, "viewport": {"width": 1280, "height": 720}},
        {"type": "attributes", "selectors": [{"selector": "a", "attribute": "href"}]},
        {"type": "query", "prompt": "What is the pricing model?"},
    ],
    only_main_content=True,
    wait_for=1500,
    actions=[
        {"type": "click", "selector": "button#accept"},
        {"type": "scroll", "direction": "down"},
    ],
    parsers=[{"type": "pdf", "mode": "auto", "max_pages": 5}],
    location={"country": "US", "languages": ["en-US"]},
    proxy="stealth",
    profile={"name": "docs-session", "save_changes": True},
    max_age=3_600_000,
)
```

<div id="parameters">
  ### 参数
</div>

* `url`: 要抓取的 URL。始终为必填项。
* `formats`: 要包含的输出 formats。接受 `ScrapeFormats` 对象 (字段为 `markdown`、`html`、`raw_html`、`summary`、`links`、`images`、`screenshot`、`change_tracking`、`json`) ，或由格式字符串/对象组成的列表。支持的格式字符串包括 `markdown`、`html`、`rawHtml`、`raw_html`、`links`、`images`、`screenshot`、`summary`、`changeTracking`、`change_tracking`、`json`、`attributes`、`branding`、`query`、`audio`。格式对象包括：`json` `{type: "json", prompt?, schema?}` (需要 `prompt` 或 `schema`) 、`changeTracking` `{type: "changeTracking" | "change_tracking", modes: ["git-diff" | "json"], schema?, prompt?, tag?}`、`screenshot` `{type: "screenshot", full_page?, quality?, viewport?}`、`attributes` `{type: "attributes", selectors: [{selector, attribute}]}`、`query` `{type: "query", prompt}`。
* `headers`: 额外的 HTTP 标头。用于 auth、cookies 或 user-agent。
* `include_tags`: 仅包含这些 HTML 标签中的内容。用于缩小提取范围。
* `exclude_tags`: 排除这些 HTML 标签中的内容。用于去除噪声。
* `only_main_content`: 仅提取主要内容。用于去掉导航栏/页脚。
* `timeout`: 抓取超时时间 (毫秒) 。用于限制运行时长。
* `wait_for`: 抓取前的延迟时间 (毫秒) 。用于处理 JS 渲染页面。
* `mobile`: 模拟移动设备。用于适配移动端布局。
* `parsers`: 解析器列表或配置。支持的 PDF 配置：`{type: "pdf", mode?: "fast" | "auto" | "ocr", max_pages?: int}`，或字符串条目 (例如 `"pdf"`) 。
* `actions`: 抓取前要执行的浏览器 actions。支持的 actions：`wait` `{type: "wait", milliseconds? | selector?}`、`screenshot` `{type: "screenshot", full_page?, quality?, viewport?}`、`click` `{type: "click", selector}`、`write` `{type: "write", text}`、`press` `{type: "press", key}`、`scroll` `{type: "scroll", direction: "up" | "down", selector?}`、`scrape` `{type: "scrape"}`、`executeJavascript` `{type: "executeJavascript", script}`、`pdf` `{type: "pdf", format?, landscape?, scale?}`，其中支持的 formats 为 `A0`-`A6`、`Letter`、`Legal`、`Tabloid`、`Ledger`。
* `location`: 位置配置 `{country, languages}`。用于特定地理位置的页面。
* `skip_tls_verification`: 跳过 TLS 证书验证。用于自签名证书。
* `remove_base64_images`: 从 markdown 输出中移除 base64 图片。用于减小 payload 大小。
* `fast_mode`: 启用快速模式。在最重视速度时使用。
* `use_mock`: 使用指定名称的 mock fixture (内部/测试) 。
* `block_ads`: 启用广告和 cookie 封禁。用于减少噪声。
* `proxy`: 代理模式 `"basic"`、`"stealth"`、`"enhanced"`、`"auto"`。
* `max_age`: 接受缓存时间小于此值 (毫秒) 的内容。用于优先命中缓存。
* `store_in_cache`: 将结果存入缓存/索引。适用于希望复用结果时。
* `profile`: 持久化浏览器配置文件 `{name, save_changes}` (也接受 `saveChanges`) 。用于保留 session 状态。
* `integration`: 集成标识符。用于跟踪/分析。

<div id="interact">
  ## 交互
</div>

<div id="why-use-it">
  ### 为什么要使用它
</div>

如果页面在抓取后还需要点击、填写表单或执行其他浏览器 actions，请使用 `interact`。

### 推荐的 SDK 方法

`client.interact(job_id, code=None, *, prompt=None, **options)`

<div id="simple-example">
  ### 简单示例
</div>

```python
run = client.interact(
    "scrape-job-id",
    code="print(await page.url())",
)

print(run.stdout)
```

<div id="complex-example">
  ### 复杂示例
</div>

```python
run = client.interact(
    "scrape-job-id",
    prompt="Open the pricing page and list the plan names.",
    language="python",
    timeout=60,
    origin="agent-docs",
)

print(run.output)
```

<div id="parameters">
  ### 参数
</div>

* `job_id`：抓取任务 ID (来自 `document.metadata_typed.scrape_id`) 。始终为必填项。
* `code`：在浏览器沙箱中执行的代码。用于脚本化交互。
* `prompt`：提供给浏览器代理的自然语言指令。适用于希望由代理执行操作时。
* `language`：运行时语言，可选 `"python"`、`"node"` 或 `"bash"` (默认值为 `"node"`) 。用于与你的代码保持一致。
* `timeout`：执行超时时间 (秒) 。用于限制运行时间过长的任务。
* `origin`：可选的来源标签。用于标注请求来源。

<div id="notes">
  ## 说明
</div>

* 在 Python 中，参数采用 snake&#95;case；SDK 会将 payload 规范化为 API 使用的 camelCase。
* 已弃用的别名：`scrape_execute`、`stop_interactive_browser`、`delete_scrape_browser` (请改用 `interact` 和 `stop_interaction`) 。
* `Firecrawl` 是统一的 v2 客户端；v1 仍可通过 `.v1` 供兼容旧版用法。
