<div id="whats-new-in-v2">
  ## v2 有哪些新内容
</div>

<div id="new-formats">
  ### 新的 formats
</div>

- `"summary"` - 获取页面内容的简洁摘要
- JSON 提取现在使用对象格式：`{ type: "json", prompt, schema }`
- 截图格式现在使用对象格式：`{ type: "screenshot", fullPage, quality, viewport }`

<div id="key-improvements">
  ### 关键改进
</div>

- **默认更快**：请求启用缓存，`maxAge` 默认为 2 天
- **更合理的默认设置**：`blockAds`、`skipTlsVerification` 和 `removeBase64Images` 默认开启
- **更强大的截图选项**：通过对象格式对截图参数进行完全控制
