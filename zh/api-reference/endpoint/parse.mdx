通过 API 将任何本地或可公开访问的文档转换为适用于 AI 代理的高质量结构化数据。

* 将 PDF、DOCX 和 XLSX 文件转换为 Markdown 或 JSON
* 保留阅读顺序和表格
* 支持 Zero Data Retention

<div id="how-to-use-document-parsing">
  ## 如何使用文档解析
</div>

Firecrawl 中的文档解析支持两种方式：

1. **基于 URL 的解析 (`/v2/scrape`)&#x20;**：提供一个指向受支持文档类型的 URL。更多详情请参阅 [Scrape](/zh/api-reference/endpoint/scrape) 端点。
2. **文件上传解析 (`/v2/parse`)&#x20;**：通过 `multipart/form-data` 直接上传文件字节。

对于基于 URL 的解析，Firecrawl 会根据文件扩展名或内容类型自动识别文件类型。
