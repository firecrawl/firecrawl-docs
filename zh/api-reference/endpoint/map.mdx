  ## v2 的新功能
</div>

<div id="better-sitemap-control">
  ### 更灵活的 Sitemap 控制
</div>

处理 Sitemap 的三种方式：

* `"include"` - 使用 Sitemap 并发现其他页面（默认）
* `"skip"` - 完全忽略 Sitemap
* `"only"` - 只返回来自 Sitemap 的 URL

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```

<div id="response-format-changed">
  ### 响应格式已更改
</div>

我们现在会在带有更多元数据的 `links` 对象数组中返回链接。

```json
{
  "url": "https://example.com",
  "links": [
    {
      "url": "https://example.com/page1",
      "title": "页面 1",
      "description": "页面 1 的说明"
    }
  ]
}
```
