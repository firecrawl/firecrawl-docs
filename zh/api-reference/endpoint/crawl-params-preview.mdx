<div id="whats-new-in-v2">
  ## v2 新变化
</div>

<div id="crawl-params-preview">
  ### 爬取参数预览
</div>

```json
{
  "url": "https://example.com",
  "prompt": "抓取整个网站，但排除 /admin 与 /api"
}   
```


<div id="response">
  ### 返回
</div>

```json
{
  "url": "https://example.com",
  "prompt": "抓取整个站点",
  "crawlEntireDomain": true,
  "excludePaths": ["/admin/.*", "/api/.*"],
  "sitemap": "include"
}
```
