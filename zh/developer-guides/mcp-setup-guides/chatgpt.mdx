<Note>
ChatGPT 中的 MCP 支持目前处于测试阶段。随着 OpenAI 持续开发，界面和可用性可能会发生变化。
</Note>

<Warning>
**可用性：** 带 MCP 连接器的开发者模式在免费套餐中不可用，需要付费 ChatGPT 订阅。不同套餐和地区的可用性和功能各不相同。请参阅 [OpenAI 关于 Developer Mode 的文档](https://help.openai.com/en/articles/12584461-developer-mode-apps-and-full-mcp-connectors-in-chatgpt-beta)，了解当前的可用性和设置说明。
</Warning>

使用 Firecrawl MCP 为 ChatGPT 添加网页抓取和搜索功能。

<div id="quick-setup">
  ## 快速开始
</div>

<div id="1-get-your-api-key">
  ### 1. 获取你的 API 密钥
</div>

在 [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys) 注册并复制你的 API 密钥。

<div id="2-enable-developer-mode">
  ### 2. 启用开发者模式
</div>

点击左下角的用户名打开 ChatGPT 设置，或者直接访问 [chatgpt.com/#settings](https://chatgpt.com/#settings)。

在设置弹窗中，滚动到页面底部并选择 **Advanced Settings（高级设置）**。将 **Developer mode（开发者模式）** 切换为开启。

<Frame>
  <img src="/images/integrations/mcp/chatgpt/enable-developer-mode.gif" alt="ChatGPT 设置中显示带有 Developer mode 开关的 Advanced Settings" />
</Frame>

<div id="3-create-the-connector">
  ### 3. 创建 Connector
</div>

启用 Developer 模式后，在同一个设置弹窗中打开 **Apps & Connectors** 选项卡。

点击页面右上角的 **Create** 按钮。

<Frame>
  <img src="/images/integrations/mcp/chatgpt/apps-connectors-create.png" alt="Apps & Connectors 选项卡中高亮显示的 Create 按钮" />
</Frame>

填写 Connector 详情：

- **Name：** `Firecrawl MCP`
- **Description：** `Web scraping, crawling, search, and content extraction`（可选）
- **MCP Server URL：** `https://mcp.firecrawl.dev/YOUR_API_KEY_HERE/v2/mcp`
- **Authentication：** `None`

将 URL 中的 `YOUR_API_KEY_HERE` 替换为你的实际 [Firecrawl API key](https://www.firecrawl.dev/app/api-keys)。

<Frame>
  <img src="/images/integrations/mcp/chatgpt/connector-form-filled.png" alt="已填写 Firecrawl MCP 详情的新 Connector 表单" />
</Frame>

勾选 **"I understand and want to continue"** 复选框，然后点击 **Create**。

<div id="4-verify-setup">
  ### 4. 验证设置
</div>

返回 ChatGPT 主界面。你应该会看到显示 **开发者模式（Developer mode）**，这表示 MCP 连接器已启用。

如果你没有看到开发者模式，请刷新页面。如果仍未出现，请再次打开设置，并在 Advanced Settings 中确认开发者模式的开关已打开。

<div id="5-access-firecrawl-tools">
  ### 5. 使用 Firecrawl 工具
</div>

要在对话中使用 Firecrawl，点击聊天输入框中的 **+** 按钮，然后选择 **More**，再选择 **Firecrawl MCP**。

<Frame>
  <img src="/images/integrations/mcp/chatgpt/select-firecrawl-mcp.png" alt="ChatGPT 聊天输入框中展开的 + 菜单，显示 More 子菜单和 Firecrawl MCP 选项" />
</Frame>

<div id="quick-demo">
  ## 快速演示
</div>

在选中 Firecrawl MCP 后，尝试使用以下提示词：

**搜索：**

```
Search for the latest React Server Components updates
```

**抓取：**

```
Scrape firecrawl.dev and tell me what it does
```

**获取文档：**

```
抓取 Vercel 的 Edge Functions 文档并进行总结
```


<div id="tool-confirmation">
  ## 工具确认
</div>

当 ChatGPT 使用 Firecrawl MCP 工具时，你会看到一个确认提示，要求你进行授权。

<Frame>
  <img src="/images/integrations/mcp/chatgpt/tool-confirmation.png" alt="ChatGPT 工具确认对话框，显示 Firecrawl MCP 请求" />
</Frame>

你可以勾选 **“Remember for this conversation”**，以避免在同一聊天会话中重复确认。这个安全措施由 OpenAI 实施，以确保 MCP 工具不会执行非预期的 actions。

确认后，ChatGPT 会执行请求并返回结果。

<Frame>
  <img src="/images/integrations/mcp/chatgpt/search-results-example.png" alt="在 ChatGPT 中显示的 Firecrawl 搜索结果示例" />
</Frame>
