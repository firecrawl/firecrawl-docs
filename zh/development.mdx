<Info>
  **前置条件** 你需要已安装 Node.js（版本 18.10.0 或更高）。
</Info>

步骤 1. 在你的操作系统上安装 Mintlify：

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

步骤 2. 前往文档所在目录（可以找到 `mint.json` 的位置），并运行以下命令：

```bash
mintlify dev
```

文档站点现已可在 `http://localhost:3000` 访问。

<div id="custom-ports">
  ### 自定义端口
</div>

Mintlify 默认使用 3000 端口。你可以使用 `--port` 参数自定义 Mintlify 的运行端口。例如，使用以下命令在 3333 端口运行：

```bash
mintlify dev --port 3333
```

如果你尝试在已占用的端口上运行 Mintlify，会看到类似如下的错误：

```md
Error：listen EADDRINUSE：地址已被占用 :::3000
```

<div id="mintlify-versions">
  ## Mintlify 版本
</div>

每个 CLI 都对应特定的 Mintlify 版本。若本地站点与生产环境显示不一致，请更新 CLI。

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

<div id="troubleshooting">
  ## 故障排查
</div>

以下是在使用 CLI 时解决常见问题的方法。

<AccordionGroup>
  <Accordion title="Mintlify 无法加载">
    请更新到 Node v18。运行 `mintlify install` 后重试。
  </Accordion>

  <Accordion title="Windows 上出现 No such file or directory">
    前往 `C:/Users/Username/.mintlify/` 目录并删除 `mint`
    文件夹。然后在此位置打开 Git Bash，运行 `git clone
    https://github.com/mintlify/mint.git`。

    重复第 3 步。
  </Accordion>

  <Accordion title="出现未知错误">
    尝试转到设备的根目录并删除 ~/.mintlify 文件夹。
    然后再次运行 `mintlify dev`。
  </Accordion>
</AccordionGroup>
