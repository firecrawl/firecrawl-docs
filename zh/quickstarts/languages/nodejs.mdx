<div id="overview">
  ## 概述
</div>

使用官方 Firecrawl Node SDK，将 URL 转换为整洁、适合 LLM 使用的内容。

<div id="prerequisites">
  ## 前置条件
</div>

* Firecrawl API 密钥
* Node.js 18+
* npm、pnpm 或 yarn

<div id="install">
  ## 安装
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-your-api-key">
  ## 配置你的 API 密钥
</div>

将 API 密钥存储在仅你的服务器可访问的环境变量中。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## 发起第一个请求
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

(async () => {
  const doc = await app.scrape('https://example.com', { formats: ['markdown'] });
  console.log(doc.markdown);
})();
```

<div id="complete-example">
  ## 完整示例
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

async function main() {
  const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
  const doc = await app.scrape('https://example.com', { formats: ['markdown', 'html'] });
  console.log(doc.metadata?.title);
  console.log(doc.markdown);
}

main().catch((error) => {
  console.error(error);
  process.exit(1);
});
```
