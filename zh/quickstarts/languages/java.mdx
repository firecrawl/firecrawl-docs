<div id="overview">
  ## 概述
</div>

使用官方 Firecrawl Java SDK 通过 Java 调用 v2 API。

<div id="prerequisites">
  ## 前置条件
</div>

* Firecrawl API 密钥
* Java 11+
* Maven 或 Gradle

<div id="install">
  ## 安装
</div>

```gradle
implementation("com.firecrawl:firecrawl-java:1.1.1")
```

<div id="configure-your-api-key">
  ## 配置 API 密钥
</div>

在运行应用前，将 API 密钥设置为环境变量。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## 发起第一个请求
</div>

```java
import com.firecrawl.client.FirecrawlClient;
import com.firecrawl.models.Document;
import com.firecrawl.models.ScrapeOptions;
import java.util.List;

FirecrawlClient client = FirecrawlClient.fromEnv();

Document doc = client.scrape(
    "https://example.com",
    ScrapeOptions.builder().formats(List.of("markdown")).build()
);

System.out.println(doc.getMarkdown());
```

<div id="complete-example">
  ## 完整示例
</div>

```java
import com.firecrawl.client.FirecrawlClient;
import com.firecrawl.models.Document;
import com.firecrawl.models.ScrapeOptions;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        FirecrawlClient client = FirecrawlClient.fromEnv();
        Document doc = client.scrape(
            "https://example.com",
            ScrapeOptions.builder().formats(List.of("markdown", "html")).build()
        );

                    if (doc.getMetadata() != null) {
                        System.out.println(doc.getMetadata().get("title"));
                    }
        System.out.println(doc.getMarkdown());
    }
}
```
