<div id="overview">
  ## 概述
</div>

使用 Bun 内置的 fetch 调用 Firecrawl 的 HTTP API。

<div id="prerequisites">
  ## 前置条件
</div>

* Firecrawl API 密钥
* Bun 1.x

<div id="install">
  ## 安装
</div>

无需额外安装任何包；Bun 原生支持 fetch。

<div id="configure-environment">
  ## 配置环境
</div>

在运行 Bun 之前，请先设置环境变量中的 API 密钥。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## 添加服务器路由
</div>

server.ts

```ts
const server = Bun.serve({
  port: 3000,
  async fetch(req) {
    const url = new URL(req.url);
    if (url.pathname != '/api/firecrawl' || req.method != 'POST') {
      return new Response('Not found', { status: 404 });
    }

    const { url: target } = await req.json();
    const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ url: target, formats: ['markdown'] }),
    });

    return new Response(await response.text(), {
      headers: { 'Content-Type': 'application/json' },
    });
  },
});

console.log(`Listening on http://localhost:${server.port}`);
```

<div id="call-it-from-your-app">
  ## 在应用中调用
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## 注意事项
</div>

* 将 API 密钥存放在服务器端，并通过 Bun 处理程序代理请求。
