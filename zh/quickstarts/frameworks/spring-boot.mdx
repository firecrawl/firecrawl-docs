<div id="overview">
  ## 概述
</div>

在 Spring Boot 的服务和控制器中使用 Firecrawl Java SDK。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl API 密钥
* Spring Boot 应用
* Java 11+

<div id="install">
  ## 安装
</div>

```gradle
implementation("com.firecrawl:firecrawl-java:1.1.1")
```

<div id="configure-environment">
  ## 配置环境
</div>

在环境变量或应用配置中设置 API 密钥。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## 添加一个服务端路由
</div>

src/main/java/com/example/firecrawl/FirecrawlService.java

```java
package com.example.firecrawl;

import com.firecrawl.client.FirecrawlClient;
import com.firecrawl.models.Document;
import com.firecrawl.models.ScrapeOptions;
import org.springframework.stereotype.Service;

import java.util.List;

@Service
public class FirecrawlService {
    private final FirecrawlClient client = FirecrawlClient.fromEnv();

    public Document scrape(String url) {
        return client.scrape(url, ScrapeOptions.builder().formats(List.of("markdown")).build());
    }
}
```

<div id="call-it-from-your-app">
  ## 在应用中调用
</div>

```java
// src/main/java/com/example/firecrawl/FirecrawlController.java
package com.example.firecrawl;

import com.firecrawl.models.Document;
import org.springframework.web.bind.annotation.*;

import java.util.Map;

@RestController
@RequestMapping("/api/firecrawl")
public class FirecrawlController {
    private final FirecrawlService service;

    public FirecrawlController(FirecrawlService service) {
        this.service = service;
    }

    @PostMapping
    public Map<String, String> scrape(@RequestBody Map<String, String> body) {
        Document doc = service.scrape(body.get("url"));
        return Map.of("markdown", doc.getMarkdown());
    }
}
```

<div id="notes">
  ## 注意事项
</div>
