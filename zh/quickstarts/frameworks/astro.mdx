<div id="overview">
  ## 概述
</div>

使用 Astro API 路由在服务器端调用 Firecrawl。

<div id="prerequisites">
  ## 前置条件
</div>

* Firecrawl API 密钥
* Astro 项目
* Node.js 18+

<div id="install">
  ## 安装
</div>

无需额外安装任何包；直接在 API 路由中使用 fetch。

<div id="configure-environment">
  ## 配置环境
</div>

将 API 密钥添加到 `.env` 文件中，并在服务端代码中读取。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## 添加服务器端路由
</div>

src/pages/api/firecrawl.ts

```ts
import type { APIRoute } from 'astro';

export const POST: APIRoute = async ({ request }) => {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${import.meta.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
};
```

<div id="call-it-from-your-app">
  ## 在应用中调用
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## 注意事项
</div>

* Firecrawl 调用请使用服务器端路由；避免在客户端发起携带 API 密钥的请求。
