<div id="overview">
  ## 概览
</div>

在 Django 视图中使用 Firecrawl Python SDK 转发抓取请求。

<div id="prerequisites">
  ## 前提条件
</div>

* Firecrawl API 密钥
* Django 应用
* Python 3.8+

<div id="install">
  ## 安装
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## 配置环境
</div>

在 Django 环境中设置 API 密钥。

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## 添加服务器端路由
</div>

app/views.py

```python
import json
import os
from django.http import JsonResponse
from django.views.decorators.csrf import csrf_exempt
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@csrf_exempt
def firecrawl_view(request):
    if request.method != "POST":
        return JsonResponse({"error": "Method not allowed"}, status=405)

    body = json.loads(request.body)
    doc = client.scrape(body["url"], formats=["markdown"])
    return JsonResponse({"markdown": doc.markdown})
```

<div id="call-it-from-your-app">
  ## 在你的应用中调用
</div>

```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## 注意事项
</div>

* 添加一个 URL 路由，将 `/api/firecrawl` 映射到 `firecrawl_view`。
