<div id="overview">
  ## 概述
</div>

在 最小 API 路由中使用 HttpClient 代理对 Firecrawl 的调用。

<div id="prerequisites">
  ## 先决条件
</div>

* Firecrawl API 密钥
* ASP.NET Core 应用
* .NET 6+

<div id="install">
  ## 安装
</div>

无需额外包；直接使用 HttpClient 和内置的 JSON 辅助工具即可。

<div id="configure-environment">
  ## 配置环境
</div>

在环境变量或应用设置中设置 `FIRECRAWL_API_KEY`。

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## 添加服务器端路由
</div>

Program.cs

```csharp
using System.Net.Http.Headers;
using System.Net.Http.Json;

var builder = WebApplication.CreateBuilder(args);
var app = builder.Build();

app.MapPost("/api/firecrawl", async (HttpRequest request) =>
{
    var payload = await request.ReadFromJsonAsync<FirecrawlRequest>();
    var apiKey = Environment.GetEnvironmentVariable("FIRECRAWL_API_KEY");

    var http = new HttpClient();
    http.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", apiKey);

    var response = await http.PostAsJsonAsync(
        "https://api.firecrawl.dev/v2/scrape",
        new { url = payload?.Url, formats = new[] { "markdown" } }
    );

    var body = await response.Content.ReadAsStringAsync();
    return Results.Content(body, "application/json");
});

app.Run();

record FirecrawlRequest(string Url);
```

<div id="call-it-from-your-app">
  ## 在应用中调用
</div>

```csharp
using System.Net.Http.Json;

var http = new HttpClient();
var response = await http.PostAsJsonAsync(
    "http://localhost:5000/api/firecrawl",
    new { url = "https://example.com" }
);

Console.WriteLine(await response.Content.ReadAsStringAsync());
```

<div id="notes">
  ## 注意事项
</div>

* 将 Firecrawl API 密钥仅保存在服务端，并通过你的 API 进行转发。
