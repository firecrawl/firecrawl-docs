<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## 使用 Firecrawl 进行网页搜索并抓取结果
</div>

Firecrawl 将其搜索 API 与强大的抓取基础设施集成，通过单一端点提供无缝的搜索与抓取功能。具体如下：

1. **统一搜索查询：**
   用户通过 search 端点提交搜索查询。

2. **自动化结果抓取：**
   Firecrawl 自动处理搜索结果，并利用其抓取能力从每个结果页面提取数据。

3. **数据交付：**
   将所有结果页面的抓取数据整理为干净的 Markdown，开箱即用。

这种集成使用户能够高效地进行网页搜索，并以最小的工作量从多个来源获取完整的抓取数据。

更多详情请参阅[Search 端点文档](https://docs.firecrawl.dev/api-reference/endpoint/search)。

<div id="search-any-query">
  ## 搜索任意内容
</div>

<div id="search-endpoint">
  ### /search 端点
</div>

用于执行网页搜索、获取最相关结果、抓取每个页面，并返回 markdown。

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // 若需更快的 SERP API，请设为 false
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Markdown 内容",
      "provider": "网页抓取器",
      "metadata": {
        "title": "Firecrawl | 为你的 LLM 可靠抓取网页",
        "description": "面向客户体验与销售的 AI"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="with-python-sdk">
  ### 使用 Python SDK
</div>

<div id="installing-python-sdk">
  #### 安装 Python SDK
</div>

```bash
pip install firecrawl-py
```

#### 搜索查询

```python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

result = app.search(query="What is firecrawl?")
```

返回结果将与上面 curl 示例中的响应类似。

```bash
go get github.com/mendableai/firecrawl-go
```

<div id="search-a-query">
  #### 执行搜索查询
</div>

```go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
  }

  query := "What is firecrawl?"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("Failed to search: %v", err)
  }
  fmt.Println(searchResult)
}
```


<div id="with-rust-sdk">
  ### 使用 Rust SDK
</div>

<div id="installing-rust-sdk">
  #### 安装 Rust SDK
</div>

将以下内容添加到你的 `Cargo.toml`：

```toml
[dependencies]
firecrawl = "^0.1"
tokio = { version = "^1", features = ["full"] }
serde = { version = "^1.0", features = ["derive"] }
serde_json = "^1.0"
uuid = { version = "^1.10", features = ["v4"] }

[build-dependencies]
tokio = { version = "1", features = ["full"] }
```

#### 执行搜索

```rust
async fn main() {
  let api_key = "YOUR_API_KEY"; // 你的 API 密钥
  let api_url = "https://api.firecrawl.dev";
  let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

  let query = "What is firecrawl?";
  let search_result = app.search(query).await;

  match search_result {
    Ok(data) => println!("Search Result: {}", data),
    Err(e) => eprintln!("Failed to search: {}", e),
  }
}
```
