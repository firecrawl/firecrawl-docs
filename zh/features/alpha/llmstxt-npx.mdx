<div id="generate-llmstxt">
  ## generate-llmstxt
</div>

一个简单的 NPX 包，可在命令行（CLI）中使用 Firecrawl API 生成 LLMs.txt 文件。该包会在你指定的输出目录中创建两个文件（默认为“public”文件夹）：

- `llms.txt`：包含与 LLM 相关内容的摘要
- `llms-full.txt`：包含完整文本内容

<div id="usage">
  ## 用法
</div>

无需安装即可通过 NPX 直接运行此软件包。你可以通过两种方式提供你的 Firecrawl API 密钥：

<div id="1-using-command-line-arguments">
  ### 1. 使用命令行参数
</div>

```bash
npx generate-llmstxt --api-key YOUR_FIRECRAWL_API_KEY
```


<div id="2-using-environment-variables">
  ### 2. 使用环境变量
</div>

在项目根目录创建一个 `.env` 文件，并添加你的 API 密钥：

```env
FIRECRAWL_API_KEY=your_api_key_here
```

然后在不使用 --api-key 选项的情况下运行该命令：

```bash
npx generate-llmstxt
```


<div id="options">
  ### 选项
</div>

- `-k, --api-key <key>`（若已在 .env 中设置则可省略）：你的 Firecrawl API 密钥
- `-u, --url <url>`（可选）：要分析的 URL（默认：https://example.com）
- `-m, --max-urls <number>`（可选）：要分析的 URL 最大数量（默认：50）
- `-o, --output-dir <path>`（可选）：输出目录路径（默认：public）

<div id="examples">
  ### 示例
</div>

```bash
# 使用命令行参数，输出到默认目录
npx generate-llmstxt -k your_api_key -u https://your-website.com -m 20

# 使用 .env 文件，输出到默认目录
npx generate-llmstxt -u https://your-website.com -m 20

# 指定自定义输出目录
npx generate-llmstxt -k your_api_key -u https://your-website.com -o custom/path/to/output

# 使用 .env 文件并指定自定义输出目录
npx generate-llmstxt -u https://your-website.com -o content/llms
```


<div id="requirements">
  ## 系统要求
</div>

- Node.js 14 或更高版本
- 有效的 Firecrawl API 密钥（通过命令行或 .env 文件提供）

<div id="output">
  ## 输出
</div>

该包会在你指定的输出目录中创建两个文件（默认路径为“public”）：

1. `llms.txt`：包含与 LLM 相关内容的摘要
2. `llms-full.txt`：包含完整文本内容
