<div id="using-firecrawl-with-crewai">
  ## 在 CrewAI 中使用 Firecrawl
</div>

Firecrawl 已与 [CrewAI（用于编排 AI 代理的框架）](https://www.crewai.com/) 集成。此页介绍集成到该框架中的所有 Firecrawl 工具。

<div id="installing-firecrawl-tools-inside-of-crewai">
  ### 在 CrewAI 中安装 Firecrawl 工具
</div>

* 前往 [firecrawl.dev 控制台](https://firecrawl.dev) 获取 API key，并将其设置为环境变量（`FIRECRAWL_API_KEY`）。
* 安装 [Firecrawl SDK](https://github.com/mendableai/firecrawl) 以及 `crewai[tools]` 包：

```
pip install firecrawl-py 'crewai[tools]'
```

<div id="firecrawlcrawlwebsitetool">
  ### FirecrawlCrawlWebsiteTool
</div>

<div id="example">
  #### 示例
</div>

按如下方式使用 FirecrawlScrapeFromWebsiteTool，使你的代理能够加载网站：

```python
from crewai_tools import FirecrawlCrawlWebsiteTool

tool = FirecrawlCrawlWebsiteTool(url='firecrawl.dev')
```


<div id="arguments">
  #### 参数
</div>

- `api_key`: 可选。指定 Firecrawl API 密钥。默认从环境变量 `FIRECRAWL_API_KEY` 读取。
- `url`: 起始爬取的基准 URL。
- `page_options`: 可选。
  - `onlyMainContent`: 可选。仅返回页面主体内容，排除页眉、导航、页脚等。
  - `includeHtml`: 可选。包含页面的原始 HTML 内容。响应中会包含一个 html 字段。
- `crawler_options`: 可选。用于控制爬取行为的选项。
  - `includes`: 可选。需要纳入爬取的 URL 模式。
  - `exclude`: 可选。需要排除的 URL 模式。
  - `generateImgAltText`: 可选。使用 LLM 为图像生成 alt 文本（需要付费方案）。
  - `returnOnlyUrls`: 可选。若为 true，仅在爬取状态中返回 URL 列表。注意：响应会在 data 中返回 URL 列表，而不是文档列表。
  - `maxDepth`: 可选。最大爬取深度。深度 1 为基准 URL，深度 2 包含基准 URL 及其直接子级，依此类推。
  - `mode`: 可选。爬取模式。Fast 模式在无站点地图的网站上速度提升 4 倍，但可能不够准确，且不应用于大量依赖 JavaScript 渲染的网站。
  - `limit`: 可选。最大爬取页面数。
  - `timeout`: 可选。爬取操作的超时时间（毫秒）。

<div id="firecrawlscrapewebsitetool">
  ### FirecrawlScrapeWebsiteTool
</div>

<div id="example">
  #### 示例
</div>

按以下方式使用 FirecrawlScrapeWebsiteTool，使你的代理能够加载网站：

```python
from crewai_tools import FirecrawlScrapeWebsiteTool

tool = FirecrawlScrapeWebsiteTool(url='firecrawl.dev')
```


<div id="arguments">
  #### 参数
</div>

- `api_key`: 可选。指定 Firecrawl API 密钥。默认使用环境变量 `FIRECRAWL_API_KEY`。
- `url`: 要抓取的 URL。
- `page_options`: 可选。
  - `onlyMainContent`: 可选。仅返回页面主体内容，排除页眉、导航、页脚等。
  - `includeHtml`: 可选。包含页面的原始 HTML 内容。响应中将包含一个 html 字段。
- `extractor_options`: 可选。基于 LLM 的结构化信息提取选项
  - `mode`: 使用的提取模式，目前支持 'llm-extraction'
  - `extractionPrompt`: 可选。用于说明需要从页面提取哪些信息的提示词
  - `extractionSchema`: 可选。要提取的数据模式（schema）
- `timeout`: 可选。请求超时时间（毫秒）

<div id="firecrawlsearchtool">
  ### Firecrawl 搜索工具
</div>

<div id="example">
  #### 示例
</div>

按如下方式使用 FirecrawlSearchTool，让你的代理能够加载网站：

```python
from crewai_tools import FirecrawlSearchTool

tool = FirecrawlSearchTool(query='什么是 Firecrawl？')
```


<div id="arguments">
  #### 参数
</div>

- `api_key`: 可选。指定 Firecrawl API 密钥。默认使用 `FIRECRAWL_API_KEY` 环境变量。
- `query`: 用于搜索的查询字符串。
- `page_options`: 可选。结果格式选项。
  - `onlyMainContent`: 可选。仅返回页面主要内容，排除页眉、导航、页脚等。
  - `includeHtml`: 可选。包含页面的原始 HTML 内容。将在响应中返回一个 html 字段。
  - `fetchPageContent`: 可选。获取页面的完整内容。
- `search_options`: 可选。用于控制爬取行为的选项。
  - `limit`: 可选。最大爬取页面数。
