<div id="sync-data-from-websites-for-dify-workflows">
  ## 为 Dify 工作流从网站同步数据
</div>

Firecrawl 可在 [Dify（LLM 工作流构建器）](https://cloud.dify.ai/) 中使用。本文介绍如何从网页抓取数据，将其解析为 Markdown，并通过其 Firecrawl 集成将其导入 Dify 知识库。

<div id="configuring-firecrawl">
  ### 配置 Firecrawl
</div>

首先，需要在设置页面的 Data Source 部分配置 Firecrawl 凭据。

<img
  className="block"
  src="/images/fc_dify_config.avif"
  alt="配置 Firecrawl 密钥"
/>

登录你的 Firecrawl 账户，获取 API Key，然后在 Dify 中输入并保存。

<img
  className="block"
  src="/images/fc_dify_savekey.png"
  alt="保存 Firecrawl 密钥"
/>

<div id="scrape-target-webpage">
  ### 抓取目标网页
</div>

现在进入有趣的部分：抓取与爬取。在知识库创建页面中，选择“Sync from website”，并输入要抓取的 URL。

<img
  className="block"
  src="/images/fc_dify_webscrape.webp"
  alt="Scraping setup"
/>

可配置选项包括：是否爬取子页面、页面爬取上限、页面抓取最大深度、排除路径、仅包含路径，以及内容提取范围。完成配置后，点击“Run”以预览解析后的页面。

<img
  className="block"
  src="/images/fc_dify_fcoptions.webp"
  alt="Set Firecrawl configuration"
/>

<div id="review-import-results">
  ### 查看导入结果
</div>

将网页解析后的文本导入后，它会被存储到知识库文档中。查看导入结果，然后点击 Add URL 继续导入新的网页。

<img
  className="block"
  src="/images/fc_dify_results.webp"
  alt="查看 Firecrawl 抓取结果"
/>
