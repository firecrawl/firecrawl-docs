<div id="sync-web-data-in-flowise-workflows">
  ## 在 Flowise 工作流中同步网页数据
</div>

Firecrawl 可以在 [Flowise（Chatflow 构建器）](https://flowiseai.com/) 中使用。本文介绍如何在 Flowise 中配置并使用 Firecrawl 模块。

<img
  className="block"
  src="/images/fc_flowise_block.png"
  alt="Firecrawl Flowise 模块"
/>

<div id="crawling-with-firecrawl-blocks">
  ### 使用 Firecrawl 块进行爬取
</div>

1. 登录你的 Firecrawl 账户获取 API Key，并在该块中填写。
2. （可选）连接 Text Splitter。
3. 选择爬取模式，以抓取目标 URL 下的页面。
4. 输入要爬取的目标 URL。
5. 在你的工作流中使用生成的文档。

<div id="scraping-with-firecrawl-blocks">
  ### 使用 Firecrawl 块进行抓取
</div>

1. 登录您的 Firecrawl 账户并获取 API 密钥，然后在块中输入该密钥。
2. （可选）连接文本分割器。
3. 选择抓取模式以抓取单个页面。
4. 输入要抓取的目标 URL。
5. 在您的工作流中使用生成的文档。
