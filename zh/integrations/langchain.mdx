> 注意：该集成仍在使用 [Firecrawl API 的 v0 版本](/zh/v0/introduction)。你可以安装 Python SDK 的 0.0.20 版本，或 Node SDK 的 0.0.36 版本。

<div id="installation">
  ## 安装
</div>

```bash
pip install firecrawl-py==0.0.20
```

<div id="usage">
  ## 用法
</div>

你需要获取自己的 API 密钥。请参见 https://firecrawl.dev

```python
from langchain_community.document_loaders import FireCrawlLoader

loader = FireCrawlLoader(
    api_key="YOUR_API_KEY", url="https://firecrawl.dev", mode="crawl"
)

docs = loader.load()
```

<div id="modes">
  ### 模式
</div>

Scrape：抓取单个 URL，并返回 Markdown。
Crawl：抓取该 URL 及所有可访问的子页面，并为每个页面返回 Markdown。

```python
loader = FireCrawlLoader(
    api_key="YOUR_API_KEY",
    url="https://firecrawl.dev",
    mode="scrape",
)

data = loader.load()
```

<div id="crawler-options">
  ### 爬虫选项
</div>

你也可以向 loader 传入参数（params）。这些参数将作为一个选项字典传递给爬虫。更多信息请参阅 Firecrawl API 文档。

<div id="langchain-js">
  ## Langchain JS
</div>

要在 Langchain JS 中使用它，可以通过 npm 安装：

```bash
npm install @mendableai/firecrawl-js
```

然后，你可以这样使用：

```typescript
import { FireCrawlLoader } from "langchain/document_loaders/web/firecrawl";

const loader = new FireCrawlLoader({
  url: "https://firecrawl.dev", // 需要抓取的 URL
  apiKey: process.env.FIRECRAWL_API_KEY, // 可选，默认读取环境变量中的 `FIRECRAWL_API_KEY`。
  mode: "scrape", // 爬取模式。"scrape" 用于单个 URL，"crawl" 用于抓取所有可访问的子页面
  params: {
    // 参见 Firecrawl API 文档的可选参数
    // API 文档：https://docs.firecrawl.dev
  },
});

const docs = await loader.load();
```
