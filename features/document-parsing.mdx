Document parsing extracts structured content from files like PDFs, Excel spreadsheets, and Word documents. Pass a URL that points to a supported file, and Firecrawl automatically detects the format, parses the content, and returns clean markdown.

- Automatic format detection from URL extension or content-type header
- Outputs clean, structured markdown for all supported formats
- Works with `/scrape`, `/crawl`, and `/batch/scrape`

## Supported formats

| Format | Extensions | Notes |
|--------|-----------|-------|
| PDF | `.pdf` | Text extraction with optional OCR for scanned documents |
| Excel | `.xlsx`, `.xls` | Each worksheet becomes an HTML table under an H2 heading |
| Word | `.docx`, `.doc`, `.odt`, `.rtf` | Preserves headings, paragraphs, lists, tables, and basic formatting |

## Basic usage

Pass a URL pointing to a supported document. Firecrawl detects the file type and parses it automatically.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

result = firecrawl.scrape("https://example.com/report.pdf")

print(result.markdown)
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const result = await firecrawl.scrape('https://example.com/report.pdf');

console.log(result.markdown);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/report.pdf"
  }'
```

</CodeGroup>

## PDF parsing modes

Use the `parsers` option to control how PDFs are processed. This is useful when working with scanned documents or image-heavy pages where default text extraction may not be sufficient.

| Mode | Description |
|------|-------------|
| `auto` | Attempts fast text-based extraction first, falls back to OCR if needed. This is the default. |
| `fast` | Text-based parsing only (embedded text). Fastest option, but will not extract text from scanned or image-heavy pages. |
| `ocr` | Forces OCR parsing on every page. Use for scanned documents or when `auto` misclassifies a page. |

<CodeGroup>

```python Python
# Force OCR for a scanned document
result = firecrawl.scrape(
    "https://example.com/scanned-report.pdf",
    parsers=[{"type": "pdf", "mode": "ocr", "maxPages": 20}]
)

print(result.markdown)
```

```js Node
// Force OCR for a scanned document
const result = await firecrawl.scrape('https://example.com/scanned-report.pdf', {
  parsers: [{ type: "pdf", mode: "ocr", maxPages: 20 }]
});

console.log(result.markdown);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/scanned-report.pdf",
    "parsers": [{ "type": "pdf", "mode": "ocr", "maxPages": 20 }]
  }'
```

</CodeGroup>

To use the default `auto` mode, pass the parser without a `mode`:

<CodeGroup>

```python Python
result = firecrawl.scrape(
    "https://example.com/report.pdf",
    parsers=[{"type": "pdf"}]
)
```

```js Node
const result = await firecrawl.scrape('https://example.com/report.pdf', {
  parsers: [{ type: "pdf" }]
});
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/report.pdf",
    "parsers": [{ "type": "pdf" }]
  }'
```

</CodeGroup>

## Output format

All supported document types are converted to clean, structured markdown. For example, an Excel file with multiple sheets produces:

```markdown
## Sheet1

| Name   | Value |
|--------|-------|
| Item 1 | 100   |
| Item 2 | 200   |

## Sheet2

| Date       | Description   |
|------------|---------------|
| 2023-01-01 | First quarter |
```

## Configuration reference

The full set of options available when passing a `parsers` object for PDF parsing:

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `type` | `string` | (required) | Must be `"pdf"` |
| `mode` | `string` | `"auto"` | Parsing strategy: `"auto"`, `"fast"`, or `"ocr"` |
| `maxPages` | `number` | (none) | Maximum number of pages to parse |
