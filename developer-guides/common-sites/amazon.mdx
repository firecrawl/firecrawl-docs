Extract structured product information, pricing, reviews, and search results from Amazon using Firecrawl's scrape, search, crawl, and batch scrape endpoints.

- **Scrape** a single product page for markdown or structured JSON
- **Search** for products across Amazon using web search
- **Map** category pages to discover product URLs
- **Crawl** multiple pages from categories or search results
- **Batch scrape** a list of known product URLs simultaneously

## Setup

Install the Firecrawl SDK for your language:

<CodeGroup>

```bash Python
pip install firecrawl
```

```bash Node
npm install @mendable/firecrawl-js
```

</CodeGroup>

## Scrape a product page

Scrape a single Amazon product page and return its content as markdown.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

result = firecrawl.scrape(
    "https://www.amazon.com/dp/B0DZZWMB2L",
    formats=["markdown"],
    only_main_content=True
)

print(result.markdown)
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const result = await firecrawl.scrape('https://www.amazon.com/dp/B0DZZWMB2L', {
  formats: ['markdown'],
  onlyMainContent: true
});

console.log(result.markdown);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.amazon.com/dp/B0DZZWMB2L",
    "formats": ["markdown"],
    "onlyMainContent": true
  }'
```

</CodeGroup>

You can also request other formats such as `html` or `links` by adding them to the `formats` array.

## Extract structured data with JSON mode

Use JSON extraction to pull specific product fields into a structured schema. Define the fields you want (title, price, rating, availability, features) and Firecrawl returns validated JSON.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

result = firecrawl.scrape(
    "https://www.amazon.com/dp/B0DZZWMB2L",
    formats=[{
        "type": "json",
        "schema": {
            "type": "object",
            "properties": {
                "title": { "type": "string" },
                "price": { "type": "string" },
                "rating": { "type": "number" },
                "availability": { "type": "string" },
                "features": {
                    "type": "array",
                    "items": { "type": "string" }
                }
            }
        }
    }]
)

print(result.json)
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';
import { z } from 'zod';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const ProductSchema = z.object({
  title: z.string(),
  price: z.string(),
  rating: z.number(),
  availability: z.string(),
  features: z.array(z.string())
});

const result = await firecrawl.scrape('https://www.amazon.com/dp/B0DZZWMB2L', {
  formats: [{
    type: 'json',
    schema: z.toJSONSchema(ProductSchema)
  }]
});

const jsonData = typeof result.json === 'string'
  ? JSON.parse(result.json)
  : result.json;
const validated = ProductSchema.parse(jsonData);

console.log(validated);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.amazon.com/dp/B0DZZWMB2L",
    "formats": [{
      "type": "json",
      "schema": {
        "type": "object",
        "properties": {
          "title": { "type": "string" },
          "price": { "type": "string" },
          "rating": { "type": "number" },
          "availability": { "type": "string" },
          "features": {
            "type": "array",
            "items": { "type": "string" }
          }
        }
      }
    }]
  }'
```

</CodeGroup>

## Search for products

Use Firecrawl's search endpoint to find Amazon products by keyword. The `site:amazon.com` operator restricts results to Amazon.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

results = firecrawl.search(
    "gaming laptop site:amazon.com",
    limit=10,
    scrape_options={
        "formats": ["markdown"]
    }
)

for item in results:
    print(item.url, item.markdown[:200])
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const results = await firecrawl.search('gaming laptop site:amazon.com', {
  limit: 10,
  sources: [{ type: 'web' }],
  scrapeOptions: {
    formats: ['markdown']
  }
});

console.log(results);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "gaming laptop site:amazon.com",
    "limit": 10,
    "scrapeOptions": {
      "formats": ["markdown"]
    }
  }'
```

</CodeGroup>

You can also search for `news` or `images` by adding additional source types.

## Map a category page

Use the map endpoint to discover all linked URLs on an Amazon category or best sellers page. Map returns URLs only, without page content.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

result = firecrawl.map(
    "https://www.amazon.com/Best-Sellers-Electronics/zgbs/electronics"
)

print(result.links)
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const result = await firecrawl.map(
  'https://www.amazon.com/Best-Sellers-Electronics/zgbs/electronics'
);

console.log(result.links);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/map" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.amazon.com/Best-Sellers-Electronics/zgbs/electronics"
  }'
```

</CodeGroup>

## Crawl multiple pages

Crawl Amazon search results or category pages to scrape content from multiple pages in one operation.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

result = firecrawl.crawl(
    "https://www.amazon.com/s?k=mechanical+keyboards",
    limit=10,
    scrape_options={
        "formats": ["markdown"]
    }
)

for page in result.data:
    print(page.metadata.url)
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const result = await firecrawl.crawl('https://www.amazon.com/s?k=mechanical+keyboards', {
  limit: 10,
  scrapeOptions: {
    formats: ['markdown']
  }
});

console.log(result.data);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/crawl" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://www.amazon.com/s?k=mechanical+keyboards",
    "limit": 10,
    "scrapeOptions": {
      "formats": ["markdown"]
    }
  }'
```

</CodeGroup>

## Batch scrape product URLs

Scrape multiple known Amazon product URLs simultaneously. Batch scrape is ideal when you already have a list of product pages to extract.

<CodeGroup>

```python Python
from firecrawl import Firecrawl

firecrawl = Firecrawl(api_key="fc-YOUR-API-KEY")

result = firecrawl.batch_scrape(
    [
        "https://www.amazon.com/dp/B0DZZWMB2L",
        "https://www.amazon.com/dp/B0FP47DNFQ",
        "https://www.amazon.com/dp/B0FL4RMGSH"
    ],
    formats=["markdown"]
)

print(result.status, result.completed, result.total)
```

```js Node
import Firecrawl from '@mendable/firecrawl-js';

const firecrawl = new Firecrawl({ apiKey: "fc-YOUR-API-KEY" });

const result = await firecrawl.batchScrape([
  'https://www.amazon.com/dp/B0DZZWMB2L',
  'https://www.amazon.com/dp/B0FP47DNFQ',
  'https://www.amazon.com/dp/B0FL4RMGSH'
], {
  options: {
    formats: ['markdown']
  },
  pollInterval: 2,
  timeout: 120
});

console.log(result.status, result.completed, result.total);
```

```bash cURL
curl -s -X POST "https://api.firecrawl.dev/v2/batch/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "urls": [
      "https://www.amazon.com/dp/B0DZZWMB2L",
      "https://www.amazon.com/dp/B0FP47DNFQ",
      "https://www.amazon.com/dp/B0FL4RMGSH"
    ],
    "formats": ["markdown"]
  }'
```

</CodeGroup>
