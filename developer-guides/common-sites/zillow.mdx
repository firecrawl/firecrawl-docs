## Setup

```bash
npm install @mendable/firecrawl-js zod
```

## Overview

When scraping Zillow, you'll typically want to:
- Extract property listings and details
- Get Zestimate values and price history
- Monitor market trends and inventory
- Analyze neighborhood demographics
- Track rental prices and availability

## Scrape with JSON Mode

Extract structured property data using Zod schemas.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';
import { z } from 'zod';

// Define Zod schema
const PropertySchema = z.object({
    address: z.string(),
    price: z.string(),
    bedrooms: z.number(),
    bathrooms: z.number()
});

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

const result = await firecrawl.scrape('https://www.zillow.com/seattle-wa/', {
    formats: [{
        type: 'json',
        schema: z.toJSONSchema(PropertySchema)
    }],
});

// Parse and validate with Zod
const jsonData = typeof result.json === 'string' ? JSON.parse(result.json) : result.json;
const validated = PropertySchema.parse(jsonData);

console.log('âœ… Validated property data:');
console.log(validated);
```

## Search

Find properties on Zillow.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

const searchResult = await firecrawl.search('seattle homes for sale site:zillow.com', {
    limit: 10,
    sources: [{ type: 'web' }], // { type: 'news' }, { type: 'images' }
    scrapeOptions: {
        formats: ['markdown']
    }
});

console.log(searchResult);
```

## Scrape

Scrape a single Zillow property listing.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

const result = await firecrawl.scrape('https://www.zillow.com/seattle-wa/', {
    formats: ['markdown'], // i.e. html, links, etc.
    onlyMainContent: true
});

console.log(result);
```

## Map

Discover all available URLs in a Zillow neighborhood or area. Note: Map returns URLs only, without content.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

const mapResult = await firecrawl.map('https://www.zillow.com/seattle-wa/capitol-hill/');

console.log(mapResult.links);
// Returns array of URLs without content
```

## Crawl

Crawl multiple property pages from a Zillow search or area.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

const crawlResult = await firecrawl.crawl('https://www.zillow.com/seattle-wa/', {
    limit: 10,
    scrapeOptions: {
        formats: ['markdown']
    }
});

console.log(crawlResult.data);
```

## Batch Scrape

Scrape multiple Zillow property URLs simultaneously.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

// Wait for completion
const job = await firecrawl.batchScrape([
    'https://www.zillow.com/seattle-wa/',
    'https://www.zillow.com/seattle-wa-98116/west-seattle_att/',
    'https://www.zillow.com/seattle-wa/houses/'],
    {
        options: {
            formats: ['markdown']
        },
        pollInterval: 2,
        timeout: 120
    }
);


console.log(job.status, job.completed, job.total);

console.log(job);
```

## Batch Scrape with JSON Mode

Extract structured data from multiple properties at once.

```typescript
import FirecrawlApp from '@mendable/firecrawl-js';
import { z } from 'zod';

// Define Zod schema
const PropertySchema = z.object({
    address: z.string(),
    price: z.string(),
    bedrooms: z.number(),
    bathrooms: z.number()
});

const firecrawl = new FirecrawlApp({ apiKey: process.env.FIRECRAWL_API_KEY });

// Wait for completion
const job = await firecrawl.batchScrape([
    'https://www.zillow.com/seattle-wa/',
    'https://www.zillow.com/seattle-wa-98116/west-seattle_att/'],
    {
        options: {
            formats: [{
                type: 'json',
                schema: z.toJSONSchema(PropertySchema)
            }]
        },
        pollInterval: 2,
        timeout: 120
    }
);

console.log(job.status, job.completed, job.total);

console.log(job);
```
