# Firecrawl Docs

Documentation for Firecrawl.

## Guidelines

1. **Do not modify localized files** - When making changes, do not touch localized/translated files. This includes:
   - Files with locale suffixes (e.g., `*.de.mdx`, `*.es.json`)
   - Files in locale-specific directories (e.g., `/[locale]/`, `/en/`, `/de/`, `/es/`)
   - Translation/dictionary JSON files
   - Any i18n-related content files

   Only modify the base/source files. Translations are managed separately.
