# Firecrawl Python Source of Truth

This file is the canonical public source of truth for agents. It is generated from the Firecrawl Python SDK source and the v2 OpenAPI spec.

## Install

```bash
pip install firecrawl-py
```

## Authenticate

```py
import os
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ.get("FIRECRAWL_API_KEY"))
# client = Firecrawl(api_key="fc-...", api_url="https://api.firecrawl.dev")
```

## When To Use What

- `search`: start from a query to discover relevant URLs, including `site:` constraints.
- `scrape`: fetch and extract content from a known URL.
- `interact`: run browser actions or code when a page needs clicks, forms, or post-scrape steps.

## Search

### Why use it

Use search when you need discovery from a query and want Firecrawl to return URLs (and optionally scrape each result). You can constrain results to a domain with the standard `site:` operator, for example `site:docs.firecrawl.dev crawl webhooks`.

### Preferred SDK method

`client.search(query, **options)`

### Simple Example

```py
results = client.search("site:docs.firecrawl.dev webhook retries")
```

### Complex Example

```py
from firecrawl.v2.types import ScrapeOptions

results = client.search(
    "site:docs.firecrawl.dev crawl webhooks",
    sources=["web", "news"],
    categories=["research"],
    limit=10,
    tbs="qdr:m",
    location="San Francisco,California,United States",
    ignore_invalid_urls=True,
    timeout=300000,
    scrape_options=ScrapeOptions(
        formats=[
            "markdown",
            "links",
            {"type": "json", "prompt": "Extract title and key endpoints."}
        ],
        only_main_content=True,
        include_tags=["main", "article"],
        exclude_tags=["nav", "footer"],
        wait_for=1000,
        actions=[
            {"type": "click", "selector": "button#accept"},
            {"type": "wait", "milliseconds": 750},
            {"type": "scrape"}
        ]
    )
)
```

### Parameters

| Parameter | Type / Options | Description |
| --- | --- | --- |
| `query` | `str` | Search query. Use `site:domain` to restrict results to a site. |
| `sources` | `list of source names or typed source objects` | Sources to search. |
| `categories` | `list of category names or typed category objects` | Categories to filter results. |
| `limit` | `int` | Maximum number of results to return. Default is `5` in the SDK model. |
| `tbs` | `str` | Time-based search filter (for example `qdr:d`, `qdr:w`, `sbd:1,qdr:m`). |
| `location` | `str` | Location string for search results. |
| `ignore_invalid_urls` | `bool` | Exclude invalid URLs that other Firecrawl endpoints cannot scrape. |
| `timeout` | `int` | Timeout in milliseconds. Default is `300000` in the SDK model. |
| `scrape_options` | `ScrapeOptions` | Scrape options applied to each result (same fields as `scrape`, plus `min_age`). |
| `integration` | `str` | Internal only. Reserved for Firecrawl instrumentation. |

## Scrape

### Why use it

Use scrape when you already have a URL and want Firecrawl to return structured content and optional formats.

### Preferred SDK method

`client.scrape(url, **options)`

### Simple Example

```py
doc = client.scrape("https://docs.firecrawl.dev", formats=["markdown"])
```

### Complex Example

```py
doc = client.scrape(
    "https://example.com/pricing",
    formats=[
        "markdown",
        "links",
        {"type": "json", "prompt": "Extract plan names and prices."},
        {"type": "screenshot", "full_page": True, "quality": 80, "viewport": {"width": 1280, "height": 720}},
        {"type": "changeTracking", "modes": ["git-diff"], "tag": "pricing"}
    ],
    headers={"User-Agent": "FirecrawlDocsBot/1.0"},
    include_tags=["main", "article"],
    exclude_tags=["nav", "footer"],
    only_main_content=True,
    wait_for=1000,
    mobile=False,
    parsers=[{"type": "pdf", "mode": "auto", "max_pages": 5}],
    actions=[
        {"type": "click", "selector": "#accept"},
        {"type": "write", "text": "firecrawl"},
        {"type": "press", "key": "Enter"},
        {"type": "wait", "milliseconds": 1500},
        {"type": "scrape"}
    ],
    location={"country": "US", "languages": ["en-US"]},
    skip_tls_verification=True,
    remove_base64_images=True,
    fast_mode=True,
    block_ads=True,
    proxy="auto",
    max_age=86400000,
    store_in_cache=True,
    profile={"name": "docs-session", "save_changes": True}
)
```

### Parameters

| Parameter | Type / Options | Description |
| --- | --- | --- |
| `url` | `str` | URL to scrape. |
| `formats` | `List[FormatOption]` or `ScrapeFormats` | Output formats to include. Accepts strings or objects for advanced formats, or `ScrapeFormats` with boolean toggles. |
| `formats[].type` | `format name such as markdown, html, rawHtml, links, images, screenshot, summary, changeTracking, json, attributes, branding, query, audio, raw_html, or change_tracking` | Format selector for object entries (snake_case aliases accepted). |
| `formats[].prompt` | `str` | Prompt for `json` or `query` formats. |
| `formats[].schema` | `dict` | JSON schema for `json` or for change tracking with `json` mode. |
| `formats[].modes` | `list of change-tracking modes such as git-diff or json` | Change tracking modes. |
| `formats[].tag` | `str` | Change tracking tag to isolate history. |
| `formats[].full_page` | `bool` | Full-page screenshot when `type: "screenshot"`. |
| `formats[].quality` | `int` | Screenshot quality (1-100). |
| `formats[].viewport` | `{ "width": int, "height": int }` | Screenshot viewport size. |
| `formats[].selectors` | `List[{"selector": str, "attribute": str}]` | Attribute extraction selectors for `type: "attributes"`. |
| `headers` | `Dict[str, str]` | Custom headers. |
| `include_tags` | `List[str]` | Tags to include. |
| `exclude_tags` | `List[str]` | Tags to exclude. |
| `only_main_content` | `bool` | Keep only the main content. |
| `timeout` | `int` | Timeout in milliseconds. |
| `wait_for` | `int` | Wait time in milliseconds before scraping. |
| `mobile` | `bool` | Emulate a mobile device. |
| `parsers` | `list of parser names or PDF parser option objects` | Parser configuration (PDF options supported). |
| `actions` | `List[Action]` | Browser actions to run before scraping. |
| `actions[].type` | `action name such as wait, screenshot, click, write, press, scroll, scrape, executeJavascript, or pdf` | Action type. |
| `actions[].milliseconds` | `int` | Wait duration when `type: "wait"`. |
| `actions[].selector` | `str` | Selector for `wait`, `click`, or `scroll`. |
| `actions[].text` | `str` | Text for `write`. |
| `actions[].key` | `str` | Key to press for `press`. |
| `actions[].direction` | `up or down` | Scroll direction. |
| `actions[].script` | `str` | JavaScript for `executeJavascript`. |
| `actions[].format` | `PDF page size such as A0, A1, A2, A3, A4, A5, A6, Letter, Legal, Tabloid, or Ledger` | PDF page size for `pdf`. |
| `actions[].landscape` | `bool` | PDF landscape mode. |
| `actions[].scale` | `float` | PDF scale factor. |
| `actions[].full_page` | `bool` | Screenshot action full page. |
| `actions[].quality` | `int` | Screenshot action quality. |
| `actions[].viewport` | `{ "width": int, "height": int }` | Screenshot action viewport. |
| `location` | `{ "country": str, "languages": List[str] }` | Location settings for the request. |
| `skip_tls_verification` | `bool` | Skip TLS certificate verification. |
| `remove_base64_images` | `bool` | Strip base64 images from markdown output. |
| `fast_mode` | `bool` | Faster scraping with reduced fidelity. |
| `use_mock` | `str` | Internal only. Reserved for SDK testing. |
| `block_ads` | `bool` | Enable ad and cookie popup blocking. |
| `proxy` | `basic, stealth, enhanced, or auto` | Proxy mode. |
| `max_age` | `int` | Prefer cached data up to this age. |
| `store_in_cache` | `bool` | Store results in cache. |
| `profile` | `dict` | Persistent browser profile, for example `{ "name": "profile", "save_changes": True }`. |
| `integration` | `str` | Internal only. Reserved for Firecrawl instrumentation. |

## Interact

### Why use it

Use interact when a page requires browser actions or code execution after a scrape starts (clicks, forms, or scripted steps).

### Preferred SDK method

`client.interact(job_id, code=None, prompt=None, language="node", timeout=None)`

### Simple Example

```py
doc = client.scrape("https://example.com", formats=["markdown"])
job_id = doc.metadata.scrape_id if doc.metadata else None
if not job_id:
    raise ValueError("Missing scrape_id from scrape response")

result = client.interact(
    job_id,
    prompt="Click the pricing tab and summarize the plans.",
    language="node"
)
```

### Complex Example

```py
result = client.interact(
    "<scrape_job_id>",
    code="print(await page.title())",
    language="python",
    timeout=60
)
```

### Parameters

| Parameter | Type / Options | Description |
| --- | --- | --- |
| `job_id` | `str` | Scrape job ID from `document.metadata.scrape_id`. |
| `code` | `str` | Code to execute in the browser session. Required if `prompt` is not provided. |
| `prompt` | `str` | Natural language instruction for the browser agent. Required if `code` is not provided. |
| `language` | `python, node, or bash` | Execution language. Defaults to `node`. |
| `timeout` | `int` | Execution timeout in seconds (1-300). |
| `origin` | `str` | Internal only. Reserved for Firecrawl instrumentation. |

## Notes

- Deprecated aliases: `scrape_execute`, `stop_interactive_browser`, and `delete_scrape_browser` map to `interact` and `stop_interaction`.
- `ScrapeOptions` includes `min_age`, but the `scrape` convenience method does not expose it. Use `ScrapeOptions` via `search(..., scrape_options=...)` if you need it.
- The top-level `Firecrawl` client exposes v2 methods directly; legacy v1 is available under `client.v1`.
