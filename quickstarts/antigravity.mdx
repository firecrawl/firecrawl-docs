Add Firecrawl's hosted Search, Scrape, and Parse tools to [Google Antigravity](https://antigravity.google/) via MCP. You can try the hosted server without an account or API key.

## Quick Setup: Hosted Keyless MCP

Open Antigravity settings (`Cmd/Ctrl + ,`), search for **MCP Servers**, and add:

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/v2/mcp"
    }
  }
}
```

Reload the window (`Cmd/Ctrl + Shift + P` → `Reload Window`). Firecrawl Search, Scrape, and Parse are now available to the agent.

## Quick Demo

In an Antigravity agent chat:

```
Search the web for "Vercel AI SDK v5 release notes" and summarize.
```

```
Scrape https://docs.firecrawl.dev/ai-onboarding and list every linked guide.
```

## Authenticated Local Server: Full Access

Use the local MCP server with a Firecrawl API key when you need account-gated tools such as Crawl, Map, Extract, or Interact. Sign up at [firecrawl.dev/app](https://www.firecrawl.dev/app), copy your API key, and add:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Replace `fc-YOUR-API-KEY` with your Firecrawl API key, then reload the window.

## Troubleshooting

- **Server shows as "failed"** — check the MCP output panel for stderr.
- **Missing `npx`** — install Node.js 18+ or use the hosted keyless URL above.
- **Tools not invoked** — explicitly mention Firecrawl (for example, "Use Firecrawl to scrape…") in your first prompt.
