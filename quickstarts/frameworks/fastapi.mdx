## Overview
Use the Firecrawl Python SDK inside a FastAPI route handler.

## Prerequisites
- A Firecrawl API key
- FastAPI app
- Python 3.8+

## Install
```bash
pip install firecrawl-py
```

## Configure environment
Set the API key in your server environment.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

## Add a server route
app/main.py

```python
import os
from fastapi import FastAPI
from firecrawl import Firecrawl

app = FastAPI()
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def scrape(body: dict):
    doc = client.scrape(body["url"], formats=["markdown"])
    return {"markdown": doc.markdown}
```

## Call it from your app
```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

## Notes
- Keep the Firecrawl client initialized once to reuse HTTP connections.
