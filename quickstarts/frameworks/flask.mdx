## Overview
Use the Firecrawl Python SDK inside a Flask server endpoint.

## Prerequisites
- A Firecrawl API key
- Flask app
- Python 3.8+

## Install
```bash
pip install firecrawl-py
```

## Configure environment
Set your API key in the environment before starting Flask.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

## Add a server route
app.py

```python
import os
from flask import Flask, request, jsonify
from firecrawl import Firecrawl

app = Flask(__name__)
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def firecrawl():
    url = request.json["url"]
    doc = client.scrape(url, formats=["markdown"])
    return jsonify(markdown=doc.markdown)
```

## Call it from your app
```python
import requests

response = requests.post(
    "http://localhost:5000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

## Notes
- Keep the Firecrawl client initialized once and reuse it across requests.
