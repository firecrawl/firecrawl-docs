## Overview
Use the Firecrawl Node SDK inside an Express server route.

## Prerequisites
- A Firecrawl API key
- Express app
- Node.js 18+

## Install
```bash
npm install @mendable/firecrawl-js
```

## Configure environment
Store the API key in `.env` or your process environment.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
server.js

```js
import express from 'express';
import Firecrawl from '@mendable/firecrawl-js';

const app = express();
app.use(express.json());

const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

app.post('/api/firecrawl', async (req, res) => {
  const { url } = req.body;
  const doc = await client.scrape(url, { formats: ['markdown'] });
  res.json({ markdown: doc.markdown });
});

app.listen(3000, () => console.log('Server listening on :3000'));
```

## Call it from your app
```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

## Notes
- Keep Firecrawl calls on the server; never expose API keys in client bundles.
