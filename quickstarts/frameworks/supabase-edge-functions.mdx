## Overview
Use a Deno-based Edge Function to call the Firecrawl HTTP API.

## Prerequisites
- A Firecrawl API key
- Supabase project
- Supabase CLI

## Install
No extra packages required; Supabase Edge Functions run on Deno with fetch.

## Configure environment
Set the API key as a Supabase secret.

```bash
supabase secrets set FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
supabase/functions/firecrawl/index.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

## Call it from your app
```js
const response = await fetch('https://project.supabase.co/functions/v1/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

## Notes
- Edge Functions use Deno, so avoid Node-only packages.
