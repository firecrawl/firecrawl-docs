## Overview
Use Deno Deploy's fetch API to call Firecrawl.

## Prerequisites
- A Firecrawl API key
- Deno Deploy project

## Install
No extra packages required; Deno includes fetch.

## Configure environment
Set the API key in the Deno Deploy project environment.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
main.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

## Call it from your app
```js
const response = await fetch('https://your-app.deno.dev', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

## Notes
- Deno Deploy only allows outbound fetch requests; keep all Firecrawl calls server-side.
