## Overview
Wire Firecrawl into a NestJS service and expose it via a controller.

## Prerequisites
- A Firecrawl API key
- NestJS app
- Node.js 18+

## Install
```bash
npm install @mendable/firecrawl-js
```

## Configure environment
Expose the API key as a server-only environment variable.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
src/firecrawl/firecrawl.service.ts

```ts
import { Injectable } from '@nestjs/common';
import Firecrawl from '@mendable/firecrawl-js';

@Injectable()
export class FirecrawlService {
  private readonly client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

  async scrape(url: string) {
    return this.client.scrape(url, { formats: ['markdown'] });
  }
}
```

## Call it from your app
```ts
// src/firecrawl/firecrawl.controller.ts
import { Body, Controller, Post } from '@nestjs/common';
import { FirecrawlService } from './firecrawl.service';

@Controller('api/firecrawl')
export class FirecrawlController {
  constructor(private readonly firecrawl: FirecrawlService) {}

  @Post()
  async create(@Body('url') url: string) {
    const doc = await this.firecrawl.scrape(url);
    return { markdown: doc.markdown };
  }
}
```

## Notes
- Register `FirecrawlService` and `FirecrawlController` in a module.
- Keep the API key out of client-side code.
