## Overview
Create an action route in Remix to call Firecrawl on the server.

## Prerequisites
- A Firecrawl API key
- Remix app
- Node.js 18+

## Install
```bash
npm install @mendable/firecrawl-js
```

## Configure environment
Store the API key in `.env` or your deployment environment.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
app/routes/api.firecrawl.ts

```ts
import { json } from '@remix-run/node';
import Firecrawl from '@mendable/firecrawl-js';

export const action = async ({ request }) => {
  const { url } = await request.json();
  const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

  const doc = await client.scrape(url, { formats: ['markdown'] });
  return json({ markdown: doc.markdown });
};
```

## Call it from your app
```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

## Notes
- Use Remix actions or loaders to keep Firecrawl calls server-side.
