## Overview
Use the Firecrawl Python SDK in a Django view to proxy scrape requests.

## Prerequisites
- A Firecrawl API key
- Django app
- Python 3.8+

## Install
```bash
pip install firecrawl-py
```

## Configure environment
Set the API key in your Django environment.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

## Add a server route
app/views.py

```python
import json
import os
from django.http import JsonResponse
from django.views.decorators.csrf import csrf_exempt
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@csrf_exempt
def firecrawl_view(request):
    if request.method != "POST":
        return JsonResponse({"error": "Method not allowed"}, status=405)

    body = json.loads(request.body)
    doc = client.scrape(body["url"], formats=["markdown"])
    return JsonResponse({"markdown": doc.markdown})
```

## Call it from your app
```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

## Notes
- Add a URL route that maps `/api/firecrawl` to `firecrawl_view`.
