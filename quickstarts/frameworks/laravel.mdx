## Overview
Use Laravel's HTTP client to call the Firecrawl API.

## Prerequisites
- A Firecrawl API key
- Laravel app
- PHP 8.1+

## Install
No extra packages required; Laravel ships with an HTTP client.

## Configure environment
Add the API key to your `.env` file.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
routes/api.php

```php
<?php

use Illuminate\Support\Facades\Http;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Route;

Route::post('/firecrawl', function (Request $request) {
    $response = Http::withToken(env('FIRECRAWL_API_KEY'))
        ->post('https://api.firecrawl.dev/v2/scrape', [
            'url' => $request->input('url'),
            'formats' => ['markdown'],
        ]);

    return $response->json();
});
```

## Call it from your app
```bash
curl -X POST http://localhost:8000/api/firecrawl               -H 'Content-Type: application/json'               -d '{"url": "https://example.com"}'
```

## Notes
- Keep the API key in server-side environment variables only.
