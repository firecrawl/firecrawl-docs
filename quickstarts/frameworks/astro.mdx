## Overview
Use an Astro API route to call Firecrawl on the server.

## Prerequisites
- A Firecrawl API key
- Astro project
- Node.js 18+

## Install
No extra packages required; use fetch in the API route.

## Configure environment
Add the API key to `.env` and read it from server-side code.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

## Add a server route
src/pages/api/firecrawl.ts

```ts
import type { APIRoute } from 'astro';

export const POST: APIRoute = async ({ request }) => {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${import.meta.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
};
```

## Call it from your app
```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

## Notes
- Use server routes for Firecrawl calls; avoid client-side requests with the API key.
