Add Firecrawl's hosted Search, Scrape, and Parse tools to [Sourcegraph Amp](https://ampcode.com) via MCP. You can try the hosted server without an account or API key.

## Quick Setup: Hosted Keyless MCP

Open Amp settings and add this MCP server:

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/v2/mcp"
    }
  }
}
```

Reload the Amp window. Firecrawl Search, Scrape, and Parse are now available to the agent.

## Authenticated Local Server: Full Access

Use the local MCP server with a Firecrawl API key when you need account-gated tools such as Crawl, Map, Extract, or Interact. Sign up at [firecrawl.dev/app](https://www.firecrawl.dev/app), copy your API key, and add:

```json
{
  "amp.mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Replace `fc-YOUR-API-KEY` with your Firecrawl API key, then reload Amp.

## Troubleshooting

- **Server fails to start** — check Amp's MCP log view for stderr output.
- **Missing `npx`** — install Node.js 18+ or use the hosted keyless URL above.
