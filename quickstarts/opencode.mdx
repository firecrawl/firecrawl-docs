Add Firecrawl's keyless Search, Scrape, and Parse tools to [OpenCode](https://opencode.ai) via MCP. Configure credentials later when you need the full tool set.

## Quick Setup

### 1. Add Firecrawl to OpenCode

OpenCode reads config from `~/.config/opencode/config.json` (global) or `./opencode.json` in your project. Add:

```json
{
  "$schema": "https://opencode.ai/config.json",
  "mcp": {
    "firecrawl": {
      "type": "remote",
      "url": "https://mcp.firecrawl.dev/v2/mcp",
      "enabled": true
    }
  }
}
```

This keyless connection exposes Search, Scrape, and Parse. For full access, configure `Authorization: Bearer fc-YOUR_API_KEY` as a remote MCP header, or use the local server with `FIRECRAWL_API_KEY` in its environment.

### 2. Start OpenCode

```bash
opencode
```

OpenCode loads MCP servers on startup. Confirm Firecrawl is attached:

```
/mcp
```

## Troubleshooting

- **Server not attached** — run `opencode doctor` to inspect MCP load errors.
- **Permission denied on `npx`** — install Node.js 18+ and ensure your shell picks up the install (`which npx`).
