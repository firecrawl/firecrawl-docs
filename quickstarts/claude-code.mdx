Add web scraping and search capabilities to Claude Code with Firecrawl MCP.

## Quick Setup

### 1. Try the hosted server

```bash
claude mcp add --transport http firecrawl https://mcp.firecrawl.dev/v2/mcp
```

This keyless connection exposes Search, Scrape, and Parse. If you need the full tool set, add your API key as a header:

```bash
claude mcp add --transport http firecrawl https://mcp.firecrawl.dev/v2/mcp --header "Authorization: Bearer fc-YOUR_API_KEY"
```

For local stdio, run `claude mcp add firecrawl -e FIRECRAWL_API_KEY=fc-YOUR_API_KEY -- npx -y firecrawl-mcp`.

Done! You can now search and scrape the web from Claude Code.
