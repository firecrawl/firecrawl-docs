## Overview
Use the official Firecrawl Node SDK to turn a URL into clean, LLM-ready content.

## Prerequisites
- A Firecrawl API key
- Node.js 18+
- npm, pnpm, or yarn

## Install
```bash
npm install @mendable/firecrawl-js
```

## Configure your API key
Store your API key in an environment variable that only your server can access.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

## Make your first request
```js
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

(async () => {
  const doc = await app.scrape('https://example.com', { formats: ['markdown'] });
  console.log(doc.markdown);
})();
```

## Complete example
```js
import Firecrawl from '@mendable/firecrawl-js';

async function main() {
  const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
  const doc = await app.scrape('https://example.com', { formats: ['markdown', 'html'] });
  console.log(doc.metadata?.title);
  console.log(doc.markdown);
}

main().catch((error) => {
  console.error(error);
  process.exit(1);
});
```
