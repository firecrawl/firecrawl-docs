Add Firecrawl's hosted Search, Scrape, and Parse tools to [Google Gemini CLI](https://github.com/google-gemini/gemini-cli) via MCP. You can try the hosted server without an account or API key.

## Quick Setup: Hosted Keyless MCP

Gemini CLI reads MCP config from `~/.gemini/settings.json` (global) or `.gemini/settings.json` in your project. Add:

```json
{
  "mcpServers": {
    "firecrawl": {
      "url": "https://mcp.firecrawl.dev/v2/mcp"
    }
  }
}
```

Launch Gemini CLI:

```bash
gemini
```

Confirm the server is loaded:

```
/mcp list
```

You should see Firecrawl Search, Scrape, and Parse.

## Authenticated Local Server: Full Access

Use the local MCP server with a Firecrawl API key when you need account-gated tools such as Crawl, Map, Extract, or Interact. Sign up at [firecrawl.dev/app](https://www.firecrawl.dev/app), copy your API key, and replace the hosted entry with:

```json
{
  "mcpServers": {
    "firecrawl": {
      "command": "npx",
      "args": ["-y", "firecrawl-mcp"],
      "env": {
        "FIRECRAWL_API_KEY": "fc-YOUR-API-KEY"
      }
    }
  }
}
```

Restart Gemini CLI after changing `settings.json`.

## Troubleshooting

- **Tools don't show up** — restart Gemini CLI after editing `settings.json`; MCP servers are loaded at startup.
- **`spawn npx ENOENT`** — install Node.js 18+ or use the hosted keyless URL above.
- **Rate-limited** — configure the authenticated local server or review plans at [firecrawl.dev/pricing](https://www.firecrawl.dev/pricing).
