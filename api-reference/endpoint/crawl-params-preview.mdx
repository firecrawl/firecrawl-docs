## What's New in v2

### Crawl Params Preview

```json
{
  "url": "https://example.com",
  "prompt": "Crawl the entire website but exclude /admin and /api"
}   
```

### Response

```json
{
  "url": "https://example.com",
  "prompt": "Crawl the entire website",
  "crawlEntireDomain": true,
  "excludePaths": ["/admin/.*", "/api/.*"],
  "sitemap": "include"
}
```
