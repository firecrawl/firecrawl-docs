---
title: "Agent Completed"
description: "Fired when an agent job finishes successfully"
---

<Note>
  Configure webhooks by passing a `webhook` object in your [POST /agent](/api-reference/endpoint/agent) request body.
</Note>

## Headers

Your endpoint receives these headers with every webhook delivery:

<ParamField header="Content-Type" type="string" required>
  Always `application/json`.
</ParamField>

<ParamField header="X-Firecrawl-Signature" type="string">
  HMAC-SHA256 signature of the raw request body, formatted as `sha256=<hex>`. Present when an HMAC secret is configured in your [account settings](https://www.firecrawl.dev/app/settings?tab=advanced). See [Webhook Security](/webhooks/security) for verification details.
</ParamField>

Any custom headers you specified in the `webhook.headers` configuration are also included.

## Payload

<ParamField body="success" type="boolean" required>
  Always `true` for this event.
</ParamField>

<ParamField body="type" type="string" required>
  Always `agent.completed`.
</ParamField>

<ParamField body="id" type="string" required>
  The agent job ID, matching the `id` returned by [POST /agent](/api-reference/endpoint/agent).
</ParamField>

<ParamField body="webhookId" type="string" required>
  Unique identifier for this webhook delivery. Use for deduplication — the same `webhookId` is sent if a delivery is retried.
</ParamField>

<ParamField body="data" type="array" required>
  Array with a single object containing the final result.

  <Expandable title="Result fields">
    <ParamField body="data[].creditsUsed" type="number">
      Total credits consumed by the agent job.
    </ParamField>

    <ParamField body="data[].data" type="object">
      The extracted data matching the schema you provided in the agent request. If no schema was provided, this contains the agent's unstructured output.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="metadata" type="object">
  The custom metadata object from your webhook configuration, if provided.
</ParamField>
