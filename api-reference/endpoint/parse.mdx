Convert any local or publicly accessible document into clean data for AI agents via API.

- Turn PDF, DOCX, and XLSX files into Markdown or JSON
- Preserves reading order and tables
- Zero Data Retention support

## How to Use Document Parsing

Document parsing in Firecrawl works in two ways:

1. **URL-based parsing (`/v2/scrape`)**: provide a URL that points to a supported document type. Check out the [Scrape](/api-reference/endpoint/scrape) endpoint for more details.
2. **File upload parsing (`/v2/parse`)**: upload file bytes directly with `multipart/form-data`.

For URL-based parsing, Firecrawl detects file type from extension or content type automatically.
