## Overview

Composio exposes Firecrawl as a toolkit so your agents can search and scrape the web without custom API wiring.

## Quick start

1. Open the [Firecrawl toolkit in Composio](https://docs.composio.dev/toolkits/firecrawl)
2. Authenticate with your Firecrawl API key
3. Call Firecrawl tools from your Composio-powered agents or workflows

## What you can do

| Capability | Use case |
|------------|----------|
| **Scrape** | Fetch page content for agent context |
| **Crawl** | Gather multi-page site data |
| **Search** | Discover sources before extraction |
