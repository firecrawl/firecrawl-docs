## Overview

Praison AI can call Firecrawl as a tool so agents scrape and gather web content during runs.

## Quick start

1. Follow the [Praison Firecrawl guide](https://docs.praison.ai/firecrawl/)
2. Configure your Firecrawl API key in Praison
3. Enable the Firecrawl tool for your agents

## What you can do

| Capability | Use case |
|------------|----------|
| **Scrape** | Fetch page content into agent context |
| **Crawl** | Collect multi-page research sets |
| **Search** | Find sources before scraping |
