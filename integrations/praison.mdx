## Overview

Praison AI can call Firecrawl as a tool so agents scrape web pages and use the cleaned content during runs.

## Quick start

1. Follow the [Praison Firecrawl guide](https://docs.praison.ai/firecrawl/) or the [`FirecrawlTool` reference](https://docs.praison.ai/docs/tools/external/firecrawl)
2. Set `FIRECRAWL_API_KEY` for cloud Firecrawl, or point at a local instance such as `http://localhost:3002`
3. Pass a Firecrawl tool to your agent (for example `tools=[FirecrawlTool()]` or a custom scrape function)

## What you can do

| Capability | Use case |
|------------|----------|
| **Scrape** | Fetch cleaned page content into agent context |
| **Crawl** | Collect pages from a site (`FirecrawlTool.crawl`, see Praison external docs) |
