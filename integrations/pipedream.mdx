## Overview

Use the Firecrawl Pipedream app to scrape pages, crawl sites, search the web, map URLs, and extract structured data as actions in your workflows.

## Quick start

1. Open the [Firecrawl app on Pipedream](https://pipedream.com/apps/firecrawl/)
2. Connect your Firecrawl API key
3. Add Firecrawl actions to a new or existing workflow

## What you can do

| Capability | Use case |
|------------|----------|
| **Scrape** | Pull page content when a webhook or schedule fires |
| **Crawl** | Start a multi page crawl job (poll with Get Crawl Data) |
| **Search** | Discover sources before downstream steps |
| **Map** | List URLs on a site, optionally filtered by a search term |
| **Extract** | Pull structured data from one or more URLs |
