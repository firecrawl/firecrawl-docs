## Overview

Use the Firecrawl Pipedream app to scrape pages, crawl sites, and search the web as steps in event-driven workflows.

## Quick start

1. Open the [Firecrawl app on Pipedream](https://pipedream.com/apps/firecrawl/)
2. Connect your Firecrawl API key
3. Add Firecrawl actions to a new or existing workflow

## What you can do

| Capability | Use case |
|------------|----------|
| **Scrape** | Pull content when a webhook or schedule fires |
| **Crawl** | Process multi-page sites in a pipeline |
| **Search** | Discover URLs before downstream steps |
