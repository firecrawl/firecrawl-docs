## Overview

Lovable apps can use Firecrawl to scrape and crawl websites into clean, structured content in chat while you build and in published apps. Firecrawl is available as an app and chat connector.

## Quick start

1. Open **Connectors** in Lovable and select **Firecrawl**
2. Add a connection with **Managed by Lovable**, or bring your own Firecrawl API key
3. Link the connection to your project, then ask Lovable to scrape or crawl with Firecrawl

## What you can build

| Capability | Use case |
|------------|----------|
| **Scrape** | Turn URLs into clean markdown or structured JSON |
| **Crawl** | Gather pages across a site, including subpages |
| **Extract** | Pull fields like product details, tables, and metadata |
