```python Python
from firecrawl import Firecrawl

app = Firecrawl(api_key="fc-YOUR-API-KEY")

# 1. Scrape the page
result = app.scrape("https://example.com", formats=["markdown"])
scrape_id = result.metadata["scrapeId"]
print(result.markdown)

# 2. Interact — ask the AI to do something on the page
response = app.interact(scrape_id, prompt="What is the main heading on this page?")
print(response.output)  # "The main heading is 'Example Domain'."

# 3. Stop the session
app.stop_interaction(scrape_id)
```
