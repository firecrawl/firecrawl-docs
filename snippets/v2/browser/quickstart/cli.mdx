```bash CLI (Python)
# 1. Launch a session
firecrawl browser launch

# 2. Execute Playwright Python code
firecrawl browser execute '
await page.goto("https://news.ycombinator.com")
title = await page.title()
print(title)
'

# 3. Close
firecrawl browser close
```
