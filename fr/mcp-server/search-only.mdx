Le [Firecrawl MCP Server](/fr/mcp-server) à l’adresse `https://mcp.firecrawl.dev/v2/mcp` est le point de terminaison par défaut et fournit l’ensemble complet d’outils Firecrawl.

<div id="endpoint">
  ## Point de terminaison
</div>

```text
https://mcp.firecrawl.dev/v2/mcp-search
```

Chaque requête nécessite une authentification : connectez-vous avec OAuth directement via le point de terminaison, ou transmettez une Firecrawl API key dans l’en-tête `Authorization`.

```json
{
  "mcpServers": {
    "firecrawl-search": {
      "url": "https://mcp.firecrawl.dev/v2/mcp-search",
      "headers": {
        "Authorization": "Bearer fc-VOTRE_CLÉ_API"
      }
    }
  }
}
```

<div id="available-tools">
  ## Outils disponibles
</div>

Le point de terminaison dédié à la recherche expose exactement ces six outils en lecture seule :

| Outil                               | Objectif                                                                    |
| ----------------------------------- | --------------------------------------------------------------------------- |
| `firecrawl_search`                  | Rechercher sur le web et renvoyer des résultats classés                     |
| `firecrawl_research_search_papers`  | Rechercher des articles scientifiques                                       |
| `firecrawl_research_inspect_paper`  | Obtenir les métadonnées d’un article                                        |
| `firecrawl_research_related_papers` | Trouver des articles associés via des liens de citation                     |
| `firecrawl_research_read_paper`     | Lire des passages du texte intégral d’un article                            |
| `firecrawl_research_search_github`  | Rechercher dans l’historique des dépôts publics et dans les fichiers README |

<div id="differences-from-the-full-mcp-endpoint">
  ## Différences par rapport au point de terminaison MCP complet
</div>

* L’ensemble d’outils est fixe et en lecture seule.
* Les outils scrape, map, crawl, extract, agent, interact, parse, monitor et feedback ne sont pas disponibles.
* `firecrawl_search` n’accepte pas `scrapeOptions` et ne permet pas de demander la récupération du contenu des pages.
