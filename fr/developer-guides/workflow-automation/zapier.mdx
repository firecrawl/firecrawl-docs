<div id="firecrawl-actions">
  ## Actions Firecrawl
</div>

| Action | Cas d’usage |
|--------|-------------|
| **Scrape URL** | Capture rapide de données sur une page unique |
| **Crawl Website** | Scraping complet du site sur plusieurs pages |
| **Extract Structured Data** | Extraction basée sur l’IA avec schémas personnalisés |
| **Search Web** | Automatisation de la recherche avec recherche + scraping |
| **Map Website** | Analyse SEO et cartographie de la structure du site |

</div>

<CardGroup cols={2}>
  <Card title="Pour commencer" icon="play">
    1. Inscrivez-vous sur [firecrawl.dev](https://firecrawl.dev)
    2. Récupérez votre clé API
    3. Créez un Zap dans Zapier
    4. Connectez Firecrawl avec votre clé API
    5. Choisissez votre flux de travail et activez-le
  </Card>

  <Card title="Bonnes pratiques" icon="lightbulb">
    - Utilisez « /Scrape URL » pour une seule page (plus rapide)
    - Planifiez intelligemment (toutes les heures/quotidien/hebdomadaire)
    - Testez d’abord dans le playground Firecrawl
    - Ajoutez une gestion des erreurs pour les extractions échouées
    - Utilisez des filtres pour éviter les exécutions inutiles
  </Card>
</CardGroup>
