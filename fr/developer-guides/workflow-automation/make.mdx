    Recherchez « Firecrawl » et sélectionnez l’action souhaitée

  <Step title="Connectez-vous avec la clé API">
    Ajoutez votre clé API Firecrawl pour vous authentifier
  </Step>

  <Step title="Configurez et testez">
    Configurez les paramètres de votre workflow et exécutez un test
  </Step>
</Steps>

<div id="firecrawl-actions-overview">
  ## Aperçu des actions Firecrawl
</div>

| Module | Cas d’usage | Idéal pour |
|--------|-------------|------------|
| **Scrape a Website** | Extraction de données sur une page | Capture rapide de données |
| **Crawl a Website** | Collecte de contenu multi‑pages | Scraping complet d’un site |
| **Extract a Website** | Extraction structurée par IA | Besoins de données complexes |
| **Search a Website** | SERP + contenu intégral | Automatisation de la recherche |
| **Map a Website** | Découverte d’URL | Analyse SEO |

<div id="best-practices">
  ## Bonnes pratiques
</div>

<CardGroup cols={2}>
  <Card title="Performance" icon="gauge-high">
    - Utilisez **Scrape** pour une seule page (le plus rapide)
    - Utilisez **Crawl** avec des limites pour les sites volumineux
    - Planifiez adéquatement pour éviter le throttling
    - Ajoutez des modules de gestion des erreurs
  </Card>

  <Card title="Optimisation des coûts" icon="piggy-bank">
    - Planifiez de manière stratégique (toutes les heures/quotidien/hebdomadaire)
    - Utilisez des filtres pour éviter les exécutions inutiles
    - Définissez des limites de crawl pour maîtriser l’usage de l’API
    - Testez d’abord dans le playground Firecrawl
  </Card>
</CardGroup>

<div id="make-vs-zapier-vs-n8n">
  ## Make vs Zapier vs n8n
</div>

| Fonctionnalité | Make | Zapier | n8n |
|---------|------|--------|-----|
| **Configuration** | Éditeur visuel, cloud | No-code, cloud | Auto-hébergé ou cloud |
| **Tarification** | Au nombre d’opérations | À la tâche | Forfait mensuel |
| **Intégrations** | 3 000+ applications | 8 000+ applications | 400+ intégrations |
| **Complexité** | Workflows avancés | Workflows simples | Workflows complexes |
| **Idéal pour** | Automatisation visuelle, complexité intermédiaire | Automatisation rapide | Contrôle pour développeurs |
| **Courbe d’apprentissage** | Modérée | Facile | Modérée à avancée |

<Tip>
**Astuce pro :** Make excelle dans la conception de workflows visuels et d’automatisations complexes. Idéal pour les équipes qui ont besoin de plus de contrôle que Zapier tout en préférant la construction visuelle à l’approche orientée code de n8n.
</Tip>
