</Warning>

Ajoutez des fonctionnalités de scraping et de recherche web à ChatGPT avec Firecrawl MCP.

<div id="quick-setup">
  ## Configuration rapide
</div>

<div id="1-get-your-api-key">
  ### 1. Obtenir votre clé API
</div>

Inscrivez-vous sur [firecrawl.dev/app/api-keys](https://www.firecrawl.dev/app/api-keys) et copiez votre clé API.

<div id="2-enable-developer-mode">
  ### 2. Activer le mode développeur
</div>

Ouvrez les paramètres de ChatGPT en cliquant sur votre nom d'utilisateur dans le coin inférieur gauche, ou accédez directement à [chatgpt.com/#settings](https://chatgpt.com/#settings).

Dans la fenêtre des paramètres, faites défiler jusqu'en bas et sélectionnez **Advanced Settings**. Activez **Developer mode** en basculant l’interrupteur sur ON.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/enable-developer-mode.gif" alt="Paramètres de ChatGPT affichant Advanced Settings avec l’interrupteur Developer mode" />
</Frame>

<div id="3-create-the-connector">
  ### 3. Créer le connecteur
</div>

Avec le mode Developer activé, allez dans l'onglet **Apps & Connectors** dans la même fenêtre de paramètres.

Cliquez sur le bouton **Create** dans le coin supérieur droit.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/apps-connectors-create.png" alt="Onglet Apps & Connectors avec le bouton Create mis en évidence" />
</Frame>

Renseignez les détails du connecteur :

- **Name :** `Firecrawl MCP`
- **Description :** `Web scraping, crawling, search, and content extraction` (facultatif)
- **MCP Server URL :** `https://mcp.firecrawl.dev/YOUR_API_KEY_HERE/v2/mcp`
- **Authentication :** `None`

Remplacez `YOUR_API_KEY_HERE` dans l’URL par votre [clé API Firecrawl](https://www.firecrawl.dev/app/api-keys).

<Frame>
  <img src="/images/integrations/mcp/chatgpt/connector-form-filled.png" alt="Nouveau formulaire de connecteur rempli avec les détails de Firecrawl MCP" />
</Frame>

Cochez la case **"I understand and want to continue"**, puis cliquez sur **Create**.

<div id="4-verify-setup">
  ### 4. Vérifier la configuration
</div>

Retournez à l'interface principale de ChatGPT. Vous devriez voir le **Mode développeur** affiché, ce qui indique que les connecteurs MCP sont actifs.

Si vous ne voyez pas le Mode développeur, actualisez la page. S'il n'apparaît toujours pas, rouvrez les paramètres et vérifiez que le Mode développeur est bien activé dans les paramètres avancés.

<div id="5-access-firecrawl-tools">
  ### 5. Accéder aux outils Firecrawl
</div>

Pour utiliser Firecrawl dans une conversation, cliquez sur le bouton **+** dans la zone de saisie du chat, puis sélectionnez **More**, puis **Firecrawl MCP**.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/select-firecrawl-mcp.png" alt="Zone de saisie de conversation de ChatGPT affichant le menu + développé avec le sous-menu More et l’option Firecrawl MCP" />
</Frame>

<div id="quick-demo">
  ## Démo rapide
</div>

Avec Firecrawl MCP sélectionné, essayez les prompts suivants :

**Recherche :**

```
Search for the latest React Server Components updates
```

**Scraping :**

```
Scrape firecrawl.dev and tell me what it does
```

**Obtenir la documentation :**

```
Extrais la documentation Vercel sur les edge functions et fais-en un résumé
```


<div id="tool-confirmation">
  ## Confirmation de l’outil
</div>

Lorsque ChatGPT utilise les outils MCP de Firecrawl, une invite de confirmation s’affiche pour demander votre approbation.

<Frame>
  <img src="/images/integrations/mcp/chatgpt/tool-confirmation.png" alt="Boîte de dialogue de confirmation d’outil de ChatGPT affichant une requête Firecrawl MCP" />
</Frame>

Vous pouvez cocher **« Se souvenir pour cette conversation »** pour éviter les confirmations répétées au cours de la même session de chat. Cette mesure de sécurité est mise en place par OpenAI pour garantir que les outils MCP n’exécutent pas d’actions non intentionnelles.

Une fois la demande confirmée, ChatGPT exécutera la requête et renverra les résultats.
