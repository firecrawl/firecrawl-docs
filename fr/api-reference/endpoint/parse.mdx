Convertissez tout document local ou accessible publiquement en données structurées pour les agents IA via l&#39;API.

* Convertissez les fichiers PDF, DOCX et XLSX en Markdown ou en JSON
* Préserve l&#39;ordre de lecture et les tableaux
* Prise en charge de Zero Data Retention

<div id="how-to-use-document-parsing">
  ## Comment utiliser l’analyse de documents
</div>

L’analyse de documents dans Firecrawl fonctionne de deux manières :

1. **Analyse à partir d’une URL (`/v2/scrape`)** : fournissez une URL pointant vers un type de document pris en charge. Consultez le point de terminaison [Scrape](/fr/api-reference/endpoint/scrape) pour plus de détails.
2. **Analyse par envoi de fichier (`/v2/parse`)** : envoyez directement les octets du fichier avec `multipart/form-data`.

Pour l’analyse à partir d’une URL, Firecrawl détecte automatiquement le type de fichier à partir de l’extension ou du type de contenu.
