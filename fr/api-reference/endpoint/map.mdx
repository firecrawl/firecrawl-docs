<div id="better-sitemap-control">
  ### Meilleur contrôle des sitemaps
</div>

Trois façons de gérer les sitemaps :

* `"include"` - Utiliser le sitemap et trouver d’autres pages (par défaut)
* `"skip"` - Ignorer complètement le sitemap
* `"only"` - Renvoyer uniquement les URL du sitemap

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```

<div id="response-format-changed">
  ### Format de réponse modifié
</div>

Nous renvoyons désormais les liens dans le tableau d’objets `links` avec des métadonnées enrichies.

```json
{
  "url": "https://example.com",
  "links": [
    {
      "url": "https://example.com/page1",
      "title": "Page 1",
      "description": "Description de la Page 1"
    }
  ]
}
```
