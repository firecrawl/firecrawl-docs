---
title: "Consulter ou lire un article"
openapi: "/fr/api-reference/v2-openapi.json GET /search/research/papers/{id}"
---

Consultez les métadonnées d’un article à partir de son identifiant, ou ajoutez un paramètre `query` pour lire les passages de texte intégral les plus pertinents pour une question.

Les identifiants acceptés incluent les valeurs canoniques `paperId` ainsi que les valeurs `primaryId` propres à la source (par ex. `arxiv:1706.03762`).

Pour un aperçu du workflow, consultez le [guide de l’index Research](/fr/features/research).
