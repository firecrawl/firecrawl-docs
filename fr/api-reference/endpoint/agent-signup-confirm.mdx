---
title: "Confirmer l’inscription de l’agent"
description: "Vérifier et associer une clé API provisionnée par un agent au compte du sponsor."
---

Confirme une inscription d’agent en attente. Si le sponsor possède déjà un compte Firecrawl, la clé y est rattachée. Sinon, le compte sandbox est converti en compte complet sous l’e-mail du sponsor. Aucune authentification requise — le jeton de vérification fait foi.

<div id="headers">
  ## En-têtes
</div>

| En-tête        | Valeur             |
| -------------- | ------------------ |
| `Content-Type` | `application/json` |

<div id="request-body">
  ## Corps de la requête
</div>

| Paramètre            | Type   | Obligatoire | Description                                                                       |
| -------------------- | ------ | ----------- | --------------------------------------------------------------------------------- |
| `agent_signup_token` | string | **Oui**     | Le jeton de vérification figurant dans le lien de confirmation envoyé par e-mail. |

<div id="response">
  ## Response
</div>

| Field     | Type    | Description                               |
| --------- | ------- | ----------------------------------------- |
| `success` | boolean | Indique si la confirmation a réussi       |
| `message` | string  | Message de résultat lisible par un humain |

<div id="example-request">
  ### Exemple de requête
</div>

```bash
curl -X POST "https://api.firecrawl.dev/v2/agent-signup/confirm" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_signup_token": "a1b2c3d4..."
  }'
```


<div id="example-response">
  ### Exemple de réponse
</div>

```json
{
  "success": true,
  "message": "Agent key confirmed and linked to your account."
}
```

<div id="what-happens-on-confirm">
  ## Ce qui se passe lors de la confirmation
</div>

- **Compte existant** : la clé API de l’agent est déplacée vers l’équipe du sponsor. Le compte sandboxé est désactivé.
- **Aucun compte** : l’e-mail du compte sandboxé est mis à jour avec l’e-mail du sponsor, et devient ainsi son vrai compte.
- Dans les deux cas, le plafond de 50 crédits est supprimé et la clé hérite de l’offre complète du sponsor.

<div id="error-responses">
  ## Réponses d&#39;erreur
</div>

| État | Erreur | Quand |
|--------|-------|------|
| `400` | Requête invalide | `agent_signup_token` manquant |
| `403` | Le délai de vérification est dépassé | La fenêtre de confirmation de 5 jours a expiré |
| `403` | Cette inscription de l’agent a été bloquée | Le sponsor a déjà bloqué cette clé |
| `404` | Jeton de vérification invalide ou expiré | Jeton introuvable ou déjà utilisé |
