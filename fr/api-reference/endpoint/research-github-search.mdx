---
title: "Recherche dans l’historique GitHub"
openapi: "/fr/api-reference/v2-openapi.json GET /search/research/github"
---

Recherchez dans l’historique des issues et des PR GitHub, des discussions et des fichiers README des dépôts. Les résultats incluent les métadonnées du dépôt, les URL, des extraits et le contenu Markdown correspondant lorsqu’il est disponible.

Pour une vue d’ensemble du workflow, consultez le [guide de l’index de recherche](/fr/features/research).
