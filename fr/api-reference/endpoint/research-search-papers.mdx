---
title: "Rechercher des articles scientifiques"
openapi: "/fr/api-reference/v2-openapi.json GET /search/research/papers"
---

Recherchez des articles scientifiques par sujet, méthode, benchmark, auteur ou catégorie. Les résultats incluent le `paperId` canonique, le `primaryId` préféré, les identifiants source, le titre, le résumé, le score et, éventuellement, des signaux de classement.

Pour une vue d’ensemble du workflow, consultez le [guide de l’index de recherche](/fr/features/research).
