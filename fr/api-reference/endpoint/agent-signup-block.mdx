---
title: "Bloquer l’inscription Agent"
description: "Bloquer et désactiver une API key provisionnée par un agent."
---

Bloque une inscription d’agent en attente, en désactivant immédiatement l’API key et en bannissant le account en sandbox. Utilisez cette option si l’agent a créé une clé sans autorisation. Aucune authentification n’est requise — le jeton de vérification fait office de preuve.

<div id="headers">
  ## En-têtes
</div>

| En-tête        | Valeur             |
| -------------- | ------------------ |
| `Content-Type` | `application/json` |

<div id="request-body">
  ## Corps de la requête
</div>

| Parameter            | Type   | Obligatoire | Description                                                                    |
| -------------------- | ------ | ----------- | ------------------------------------------------------------------------------ |
| `agent_signup_token` | string | **Oui**     | Le jeton de vérification présent dans le lien de confirmation reçu par e-mail. |

<div id="response">
  ## Réponse
</div>

| Champ | Type | Description |
|-------|------|-------------|
| `success` | boolean | Indique si le bloc a réussi |
| `message` | string | Message de résultat compréhensible par un humain |

<div id="example-request">
  ### Exemple de requête
</div>

```bash
curl -X POST "https://api.firecrawl.dev/v2/agent-signup/block" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_signup_token": "a1b2c3d4..."
  }'
```


<div id="example-response">
  ### Exemple de réponse
</div>

```json
{
  "success": true,
  "message": "Agent key has been blocked and disabled."
}
```

<div id="what-happens-on-block">
  ## Ce qui se passe lors d’un blocage
</div>

- La clé API de l’agent est **supprimée**.
- L’équipe en sandbox est **bannie**.
- L’enregistrement du sponsor est marqué comme `"blocked"`, ce qui empêche de futures inscriptions d’agent pour cet e-mail.
- Toute requête API supplémentaire utilisant la clé bloquée renverra `403`. 

<div id="error-responses">
  ## Réponses d’erreur
</div>

| État  | Erreur                                 | Quand                                        |
| ----- | -------------------------------------- | -------------------------------------------- |
| `400` | Requête invalide                       | `agent_signup_token` manquant                |
| `404` | Jeton de vérification invalide         | Jeton introuvable                            |
| `409` | Cette clé d’agent a déjà été confirmée | Impossible de bloquer une clé déjà confirmée |
