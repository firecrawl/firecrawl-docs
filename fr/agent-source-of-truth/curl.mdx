<div id="firecrawl-curl-source-of-truth">
  # Firecrawl curl source de référence
</div>

Ce fichier constitue la référence publique faisant autorité pour les agents. Il est généré à partir de la spécification OpenAPI v2 de Firecrawl et des interfaces du SDK.

<div id="authenticate">
  ## Authentification
</div>

Toutes les requêtes utilisent un jeton Bearer dans l’en-tête `Authorization`.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="when-to-use-what">
  ## Quand utiliser quoi
</div>

* `search` : partir d’une requête pour découvrir des URL pertinentes, y compris avec des contraintes `site:`.
* `scrape` : récupérer et extraire le contenu d’une URL connue.
* `interact` : exécuter des actions dans le navigateur ou du code lorsqu’une page nécessite des clics, des formulaires ou des étapes post-scraping.

<div id="search">
  ## Search
</div>

<div id="why-use-it">
  ### Pourquoi l&#39;utiliser
</div>

Utilisez la recherche lorsque vous avez besoin d&#39;explorer des résultats à partir d&#39;une requête et souhaitez que Firecrawl renvoie des URLs (et scrape optionnellement chaque résultat). Vous pouvez restreindre les résultats à un domaine avec l&#39;opérateur standard `site:`, par exemple `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Méthode recommandée du SDK
</div>

`POST https://api.firecrawl.dev/v2/search`

<div id="simple-example">
  ### Exemple simple
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev webhook retries"
  }'
```

<div id="complex-example">
  ### Exemple complexe
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev crawl webhooks",
    "sources": ["web", "news"],
    "categories": ["research"],
    "limit": 10,
    "tbs": "qdr:m",
    "location": "San Francisco,California,United States",
    "country": "US",
    "ignoreInvalidURLs": true,
    "timeout": 60000,
    "scrapeOptions": {
      "formats": [
        "markdown",
        "links",
        { "type": "json", "prompt": "Extract title and key endpoints." }
      ],
      "onlyMainContent": true,
      "includeTags": ["main", "article"],
      "excludeTags": ["nav", "footer"],
      "waitFor": 1000,
      "actions": [
        { "type": "click", "selector": "button#accept" },
        { "type": "wait", "milliseconds": 750 },
        { "type": "scrape" }
      ]
    }
  }'
```

<div id="parameters">
  ### Paramètres
</div>

| Paramètre           | Type / Options                                             | Description                                                                                       |
| ------------------- | ---------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| `query`             | `string`                                                   | Requête de recherche. Utilisez `site:domain` pour restreindre les résultats à un site.            |
| `sources`           | `tableau de noms de source ou d’objets source typés`       | Sources à interroger.                                                                             |
| `categories`        | `tableau de noms de catégorie ou d’objets catégorie typés` | Catégories pour filtrer les résultats.                                                            |
| `limit`             | `integer`                                                  | Nombre maximum de résultats à renvoyer.                                                           |
| `tbs`               | `string`                                                   | Filtre de recherche temporel (par exemple `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).                       |
| `location`          | `string`                                                   | Chaîne de localisation pour les résultats de recherche.                                           |
| `country`           | `string`                                                   | Code pays ISO pour le ciblage géographique des résultats de recherche.                            |
| `timeout`           | `integer`                                                  | Timeout en millisecondes.                                                                         |
| `ignoreInvalidURLs` | `boolean`                                                  | Exclure les URLs invalides que les autres points de terminaison Firecrawl ne peuvent pas scraper. |
| `enterprise`        | `tableau de modes Enterprise tels que anon ou zdr`         | Options Enterprise ZDR lorsqu&#39;elles sont activées pour votre équipe.                          |
| `scrapeOptions`     | `ScrapeOptions`                                            | Options de scrape appliquées à chaque résultat (mêmes champs que `scrape`).                       |

<div id="scrape">
  ## Scrape
</div>

### Pourquoi l’utiliser

Utilisez scrape lorsque vous avez déjà une URL et souhaitez que Firecrawl renvoie du contenu structuré et des formats optionnels.

<div id="preferred-sdk-method">
  ### Méthode recommandée du SDK
</div>

`POST https://api.firecrawl.dev/v2/scrape`

<div id="simple-example">
  ### Exemple simple
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://docs.firecrawl.dev",
    "formats": ["markdown"]
  }'
```

<div id="complex-example">
  ### Exemple complexe
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/pricing",
    "formats": [
      "markdown",
      "links",
      { "type": "json", "prompt": "Extract plan names and prices." },
      { "type": "screenshot", "fullPage": true, "quality": 80, "viewport": { "width": 1280, "height": 720 } },
      { "type": "changeTracking", "modes": ["git-diff"], "tag": "pricing" }
    ],
    "headers": { "User-Agent": "FirecrawlDocsBot/1.0" },
    "includeTags": ["main", "article"],
    "excludeTags": ["nav", "footer"],
    "onlyMainContent": true,
    "waitFor": 1000,
    "mobile": false,
    "parsers": [{ "type": "pdf", "mode": "auto", "maxPages": 5 }],
    "actions": [
      { "type": "click", "selector": "#accept" },
      { "type": "write", "text": "firecrawl" },
      { "type": "press", "key": "Enter" },
      { "type": "wait", "milliseconds": 1500 },
      { "type": "scrape" }
    ],
    "location": { "country": "US", "languages": ["en-US"] },
    "skipTlsVerification": true,
    "removeBase64Images": true,
    "blockAds": true,
    "proxy": "auto",
    "maxAge": 86400000,
    "storeInCache": true,
    "profile": { "name": "docs-session", "saveChanges": true }
  }'
```

<div id="parameters">
  ### Paramètres
</div>

| Paramètre             | Type / Options           | Description                                                                                                                                      |
| --------------------- | ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `url`                 | `string`                 | URL à scraper.                                                                                                                                   |
| `formats`             | `Array`                  | Formats de sortie à inclure. Chaînes ou objets tels que `{type: "json", prompt, schema}` ou `{type: "screenshot", fullPage, quality, viewport}`. |
| `onlyMainContent`     | `boolean`                | Conserver uniquement le contenu principal.                                                                                                       |
| `includeTags`         | `Array<string>`          | Balises à inclure.                                                                                                                               |
| `excludeTags`         | `Array<string>`          | Balises à exclure.                                                                                                                               |
| `maxAge`              | `integer`                | Privilégier les données en cache jusqu&#39;à cet âge.                                                                                            |
| `minAge`              | `integer`                | Retourner les données en cache uniquement si elles ont au moins cet âge minimum.                                                                 |
| `headers`             | `object`                 | En-têtes personnalisés.                                                                                                                          |
| `waitFor`             | `integer`                | Temps d&#39;attente en millisecondes avant le scraping.                                                                                          |
| `mobile`              | `boolean`                | Émuler un appareil mobile.                                                                                                                       |
| `skipTlsVerification` | `boolean`                | Ignorer la vérification du certificat TLS.                                                                                                       |
| `timeout`             | `integer`                | Délai d&#39;expiration en millisecondes.                                                                                                         |
| `parsers`             | `Array`                  | Configuration des parseurs (options PDF prises en charge).                                                                                       |
| `actions`             | `Array`                  | Actions du navigateur à exécuter avant le scraping.                                                                                              |
| `location`            | `object`                 | Paramètres de localisation de la requête.                                                                                                        |
| `location.country`    | `string`                 | Code pays ISO.                                                                                                                                   |
| `location.languages`  | `Array<string>`          | Langues préférées.                                                                                                                               |
| `removeBase64Images`  | `boolean`                | Supprimer les images en base64 de la sortie markdown.                                                                                            |
| `blockAds`            | `boolean`                | Activer le blocage des publicités et des fenêtres contextuelles de cookies.                                                                      |
| `proxy`               | `basic, avancé, ou auto` | Mode proxy.                                                                                                                                      |
| `storeInCache`        | `boolean`                | Stocker les résultats dans le cache.                                                                                                             |
| `profile`             | `object`                 | Profil de navigateur persistant avec `name` et `saveChanges`.                                                                                    |
| `zeroDataRetention`   | `boolean`                | Activer l&#39;option zero data retention si votre compte l&#39;autorise.                                                                         |

<div id="interact">
  ## Interact
</div>

### Pourquoi l’utiliser

Utilisez Interact lorsqu’une page nécessite des actions dans le navigateur ou l’exécution de code après le lancement d’un scrape (clics, formulaires ou étapes scriptées).

<div id="preferred-sdk-method">
  ### Méthode recommandée du SDK
</div>

`POST https://api.firecrawl.dev/v2/scrape/{jobId}/interact`

<div id="simple-example">
  ### Exemple simple
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "console.log(document.title)",
    "language": "node"
  }'
```

<div id="complex-example">
  ### Exemple complexe
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Click the pricing tab and summarize the plans.",
    "language": "node",
    "timeout": 60
  }'
```

<div id="parameters">
  ### Paramètres
</div>

| Paramètre  | Type / options         | Description                                                                                        |
| ---------- | ---------------------- | -------------------------------------------------------------------------------------------------- |
| `jobId`    | `string`               | ID de tâche de scraping issu de `data.metadata.scrapeId`.                                          |
| `code`     | `string`               | Code à exécuter dans la session de navigateur. Obligatoire si `prompt` n’est pas fourni.           |
| `prompt`   | `string`               | Instruction en langage naturel pour l’agent de navigateur. Obligatoire si `code` n’est pas fourni. |
| `language` | `python, node ou bash` | Langage d’exécution. `node` est utilisé par défaut s’il est omis.                                  |
| `timeout`  | `integer`              | Timeout d’exécution en secondes.                                                                   |
| `origin`   | `string`               | Usage interne uniquement. Réservé à l’instrumentation de Firecrawl.                                |

<div id="notes">
  ## Remarques
</div>

* La spécification OpenAPI indique actuellement que `code` est requis pour Interact, mais les SDK acceptent aussi `prompt` comme alternative. En pratique, utilisez soit `code`, soit `prompt`.
