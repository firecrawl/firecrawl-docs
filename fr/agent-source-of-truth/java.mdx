<div id="firecrawl-java-source-of-truth">
  # Source de référence de Firecrawl Java
</div>

Ce fichier est la référence publique officielle pour les agents. Il est généré à partir du code source du SDK Java de Firecrawl et de la spécification OpenAPI v2.

<div id="install">
  ## Installation
</div>

Maven :

```xml
<dependency>
  <groupId>com.firecrawl</groupId>
  <artifactId>firecrawl-java</artifactId>
  <version>1.2.0</version>
</dependency>
```

Gradle :

```gradle
implementation("com.firecrawl:firecrawl-java:1.2.0")
```

<div id="authenticate">
  ## Authentification
</div>

```java
import com.firecrawl.client.FirecrawlClient;

FirecrawlClient client = FirecrawlClient.builder()
    .apiKey(System.getenv("FIRECRAWL_API_KEY"))
    .build();
```

<div id="when-to-use-what">
  ## Quand utiliser lequel
</div>

* `search` : partir d’une requête pour trouver des URL pertinentes, y compris avec des contraintes `site:`.
* `scrape` : récupérer et extraire le contenu d’une URL connue.
* `interact` : exécuter des actions dans le navigateur ou du code lorsqu’une page nécessite des clics, des formulaires ou des étapes après le scraping.

<div id="search">
  ## Recherche
</div>

### Pourquoi l’utiliser

Utilisez la recherche lorsque vous avez besoin de trouver des résultats à partir d’une requête et que vous souhaitez que Firecrawl renvoie des URL (et, éventuellement, scrape chaque résultat). Vous pouvez limiter les résultats à un domaine avec l’opérateur standard `site:`, par exemple `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Méthode recommandée du SDK
</div>

`client.search(query, options)`

<div id="simple-example">
  ### Exemple simple
</div>

```java
SearchData results = client.search("site:docs.firecrawl.dev webhook retries");
```

<div id="complex-example">
  ### Exemple complexe
</div>

```java
import com.firecrawl.models.SearchOptions;
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;

SearchOptions options = SearchOptions.builder()
    .sources(List.of("web", "news"))
    .categories(List.of("research"))
    .limit(10)
    .tbs("qdr:m")
    .location("San Francisco,California,United States")
    .ignoreInvalidURLs(true)
    .timeout(60000)
    .scrapeOptions(
        ScrapeOptions.builder()
            .formats(List.of(
                "markdown",
                "links",
                JsonFormat.builder().prompt("Extract title and key endpoints.").build()
            ))
            .onlyMainContent(true)
            .waitFor(1000)
            .build()
    )
    .build();

SearchData results = client.search("site:docs.firecrawl.dev crawl webhooks", options);
```

<div id="parameters">
  ### Paramètres
</div>

| Paramètre                   | Type / Options                                                               | Description                                                                                       |
| --------------------------- | ---------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------- |
| `query`                     | `String`                                                                     | Requête de recherche. Utilisez `site:domain` pour limiter les résultats à un site.                |
| `options.sources`           | `List<Object>` (`"web"`, `"news"`, `"images"` ou des objets `{type: "web"}`) | Sources à interroger.                                                                             |
| `options.categories`        | `List<Object>` (`"github"`, `"research"`, `"pdf"`)                           | Catégories permettant de filtrer les résultats.                                                   |
| `options.limit`             | `Integer`                                                                    | Nombre maximal de résultats à renvoyer.                                                           |
| `options.tbs`               | `String`                                                                     | Filtre de recherche basé sur le temps (par exemple `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).              |
| `options.location`          | `String`                                                                     | Paramètre de localisation pour les résultats de recherche.                                        |
| `options.ignoreInvalidURLs` | `Boolean`                                                                    | Exclut les URL non valides que les autres points de terminaison Firecrawl ne peuvent pas scraper. |
| `options.timeout`           | `Integer`                                                                    | Délai d&#39;expiration en millisecondes.                                                          |
| `options.scrapeOptions`     | `ScrapeOptions`                                                              | Options de scrape appliquées à chaque résultat (mêmes champs que pour `scrape`).                  |
| `options.integration`       | `String`                                                                     | Interne uniquement. Réservé à l&#39;instrumentation interne de Firecrawl.                         |

<div id="scrape">
  ## Scrape
</div>

<div id="why-use-it">
  ### Pourquoi l&#39;utiliser
</div>

Utilisez scrape si vous avez déjà une URL et souhaitez que Firecrawl renvoie du contenu structuré ainsi que des formats en option.

<div id="preferred-sdk-method">
  ### Méthode recommandée du SDK
</div>

`client.scrape(url, options)`

<div id="simple-example">
  ### Exemple simple
</div>

```java
Document doc = client.scrape(
    "https://docs.firecrawl.dev",
    ScrapeOptions.builder()
        .formats(List.of("markdown"))
        .build()
);
```

<div id="complex-example">
  ### Exemple complexe
</div>

```java
import com.firecrawl.models.ScrapeOptions;

ScrapeOptions options = ScrapeOptions.builder()
    .formats(List.of(
        "markdown",
        "links",
        Map.of("type", "json", "prompt", "Extract plan names and prices."),
        Map.of("type", "screenshot", "fullPage", true, "quality", 80, "viewport", Map.of("width", 1280, "height", 720)),
        Map.of("type", "changeTracking", "modes", List.of("git-diff"), "tag", "pricing")
    ))
    .headers(Map.of("User-Agent", "FirecrawlDocsBot/1.0"))
    .includeTags(List.of("main", "article"))
    .excludeTags(List.of("nav", "footer"))
    .onlyMainContent(true)
    .waitFor(1000)
    .mobile(false)
    .parsers(List.of(Map.of("type", "pdf", "mode", "auto", "maxPages", 5)))
    .actions(List.of(
        Map.of("type", "click", "selector", "#accept"),
        Map.of("type", "write", "text", "firecrawl"),
        Map.of("type", "press", "key", "Enter"),
        Map.of("type", "wait", "milliseconds", 1500),
        Map.of("type", "scrape")
    ))
    .location(LocationConfig.builder().country("US").languages(List.of("en-US")).build())
    .skipTlsVerification(true)
    .removeBase64Images(true)
    .blockAds(true)
    .proxy("auto")
    .maxAge(86400000L)
    .storeInCache(true)
    .build();

Document doc = client.scrape("https://example.com/pricing", options);
```

<div id="parameters">
  ### Paramètres
</div>

| Paramètre                     | Type / Options                                                                     | Description                                                                                                                                                |
| ----------------------------- | ---------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `url`                         | `String`                                                                           | URL à scraper.                                                                                                                                             |
| `options.formats`             | `List<Object>`                                                                     | Formats de sortie à inclure. Chaînes ou objets de format tels que `{type: "json", prompt, schema}` ou `{type: "screenshot", fullPage, quality, viewport}`. |
| `options.headers`             | `Map<String, String>`                                                              | En-têtes personnalisés.                                                                                                                                    |
| `options.includeTags`         | `List<String>`                                                                     | Balises à inclure.                                                                                                                                         |
| `options.excludeTags`         | `List<String>`                                                                     | Balises à exclure.                                                                                                                                         |
| `options.onlyMainContent`     | `Boolean`                                                                          | Conserver uniquement le contenu principal.                                                                                                                 |
| `options.timeout`             | `Integer`                                                                          | Délai d’expiration en millisecondes.                                                                                                                       |
| `options.waitFor`             | `Integer`                                                                          | Temps d’attente en millisecondes avant le scraping.                                                                                                        |
| `options.mobile`              | `Boolean`                                                                          | Émuler un appareil mobile.                                                                                                                                 |
| `options.parsers`             | `List<Object>`                                                                     | Configuration de l’analyseur (options PDF prises en charge).                                                                                               |
| `options.actions`             | `List<Map<String, Object>>`                                                        | Actions du navigateur à exécuter avant le scraping.                                                                                                        |
| `options.location`            | `LocationConfig`                                                                   | Paramètres de localisation pour la requête.                                                                                                                |
| `options.skipTlsVerification` | `Boolean`                                                                          | Ignorer la vérification du certificat TLS.                                                                                                                 |
| `options.removeBase64Images`  | `Boolean`                                                                          | Supprimer les images en base64 de la sortie markdown.                                                                                                      |
| `options.blockAds`            | `Boolean`                                                                          | Activer le blocage des publicités et des fenêtres contextuelles de cookies.                                                                                |
| `options.proxy`               | `String` (`"basic"`, `"stealth"`, `"enhanced"`, `"auto"` ou une URL personnalisée) | Mode proxy.                                                                                                                                                |
| `options.maxAge`              | `Long`                                                                             | Privilégier les données en cache jusqu’à cet âge.                                                                                                          |
| `options.storeInCache`        | `Boolean`                                                                          | Stocker les résultats dans le cache.                                                                                                                       |
| `options.integration`         | `String`                                                                           | Interne uniquement. Réservé à l’instrumentation Firecrawl.                                                                                                 |

<div id="interact">
  ## Interact
</div>

### Pourquoi l’utiliser

Utilisez interact lorsqu’une page nécessite des actions du navigateur ou l’exécution de code une fois le scrape lancé (clics, formulaires ou étapes scriptées).

<div id="preferred-sdk-method">
  ### Méthode recommandée du SDK
</div>

`client.interact(jobId, code, language, timeout, origin)`

<div id="simple-example">
  ### Exemple simple
</div>

```java
Document doc = client.scrape("https://example.com");
String jobId = doc.getMetadata() != null ? String.valueOf(doc.getMetadata().get("scrapeId")) : null;
if (jobId == null || jobId.isBlank()) {
    throw new IllegalStateException("Missing scrapeId from scrape response");
}

BrowserExecuteResponse result = client.interact(jobId, "console.log(document.title)");
```

<div id="complex-example">
  ### Exemple complexe
</div>

```java
BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "console.log(await page.title());",
    "node",
    60,
    null
);
```

<div id="parameters">
  ### Paramètres
</div>

| Paramètre  | Type / Options                            | Description                                                                       |
| ---------- | ----------------------------------------- | --------------------------------------------------------------------------------- |
| `jobId`    | `String`                                  | ID de la tâche de scraping issue de `document.metadata.scrapeId`.                 |
| `code`     | `String`                                  | Code à exécuter dans la session de navigateur.                                    |
| `language` | `String` (`"python"`, `"node"`, `"bash"`) | Langage d&#39;exécution. Par défaut, `node` est utilisé si ce paramètre est omis. |
| `timeout`  | `Integer`                                 | Délai d&#39;expiration de l&#39;exécution, en secondes (1-300).                   |
| `origin`   | `String`                                  | Usage interne uniquement. Réservé à l&#39;instrumentation de Firecrawl.           |

<div id="notes">
  ## Remarques
</div>

* L’API interact du SDK Java ne prend en charge que `code`. Les autres SDK proposent `prompt` par commodité.
* Alias déprécié : `scrapeExecute` correspond à `interact`, et `deleteScrapeBrowser` à `stopInteractiveBrowser`.
