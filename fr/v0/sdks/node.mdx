> Remarque : ceci utilise la [version v0 de l’API Firecrawl](/fr/v0/introduction), en cours de dépréciation. Nous vous recommandons de passer à la [v1](/fr/sdks/node).

<div id="installation">
  ## Installation
</div>

Pour installer le SDK Node Firecrawl, vous pouvez utiliser npm :

```bash
npm install @mendable/firecrawl-js@0.0.36
```

<div id="usage">
  ## Utilisation
</div>

1. Obtenez une clé d’API sur [firecrawl.dev](https://firecrawl.dev)
2. Définissez la clé d’API comme variable d’environnement nommée `FIRECRAWL_API_KEY`, ou transmettez-la en paramètre à la classe `FirecrawlApp`.

Voici un exemple d’utilisation du SDK avec gestion des erreurs :

```js
import FirecrawlApp from '@mendable/firecrawl-js';

// Initialisez FirecrawlApp avec votre clé d'API
const app = new FirecrawlApp({ apiKey: "YOUR_API_KEY" });

// Extraire une seule URL
const url = 'https://docs.firecrawl.dev';
const scrapedData = await app.scrapeUrl(url);

// Explorer un site web
const crawlUrl = 'https://docs.firecrawl.dev';
const params = {
  crawlerOptions: {
    excludes: ['blog/'],
    includes: [], // laisser vide pour inclure toutes les pages
    limit: 1000,
  },
  pageOptions: {
      onlyMainContent: true
  }
};

const crawlResult = await app.crawlUrl(crawlUrl, params);
```

<div id="scraping-a-url">
  ### Extraire une URL
</div>

Pour extraire le contenu d’une URL unique avec gestion des erreurs, utilisez la méthode `scrapeUrl`. Elle prend l’URL en paramètre et renvoie les données extraites sous forme de dictionnaire.

```js
const url = 'https://example.com';
const scrapedData = await app.scrapeUrl(url);
```

<div id="crawling-a-website">
  ### Explorer un site web
</div>

Pour explorer un site web avec gestion des erreurs, utilisez la méthode `crawlUrl`. Elle accepte l’URL de départ et des paramètres optionnels. L’argument `params` vous permet de définir des options supplémentaires pour la tâche d’exploration, comme le nombre maximal de pages à explorer, les domaines autorisés et le format de sortie.

```js
const crawlUrl = 'https://example.com';

const params = {
  crawlerOptions: {
    excludes: ['blog/'],
    includes: [], // laisser vide pour inclure toutes les pages
    limit: 1000,
  },
  pageOptions: {
    onlyMainContent: true
  }
};

const waitUntilDone = true;
const pollInterval = 5;

const crawlResult = await app.crawlUrl(
  crawlUrl,
  params,
  waitUntilDone,
  pollInterval
);
```

<div id="checking-crawl-status">
  ### Vérifier l’état d’un crawl
</div>

Pour vérifier l’état d’un job de crawl avec gestion des erreurs, utilisez la méthode `checkCrawlStatus`. Elle prend l’ID du job en paramètre et renvoie l’état actuel du crawl.

```js
const status = await app.checkCrawlStatus(jobId);
```

<div id="extracting-structured-data-from-a-url">
  ### Extraire des données structurées à partir d’une URL
</div>

Avec l’extraction via LLM, vous pouvez facilement extraire des données structurées depuis n’importe quelle URL. Nous prenons en charge les schémas Zod pour vous simplifier la vie. Voici comment l’utiliser :

```js
import FirecrawlApp from "@mendable/firecrawl-js";
import { z } from "zod";

const app = new FirecrawlApp({
  apiKey: "fc-YOUR_API_KEY",
});

// Définir le schéma d’extraction des contenus
const schema = z.object({
  top: z
    .array(
      z.object({
        title: z.string(),
        points: z.number(),
        by: z.string(),
        commentsURL: z.string(),
      })
    )
    .length(5)
    .describe("Les 5 meilleurs articles sur Hacker News"),
});

const scrapeResult = await app.scrapeUrl("https://firecrawl.dev", {
  extractorOptions: { extractionSchema: schema },
});

console.log(scrapeResult.data["llm_extraction"]);
```

<div id="search-for-a-query">
  ### Rechercher une requête
</div>

Avec la méthode `search`, vous pouvez lancer une recherche dans un moteur et obtenir les principaux résultats, ainsi que le contenu de la page pour chacun. La méthode prend la requête en paramètre et renvoie les résultats de recherche.

```js
const query = 'Qu’est-ce que Mendable ?';
const searchResults = await app.search(query, {
  pageOptions: {
    fetchPageContent: true // Récupérer le contenu de la page pour chaque résultat de recherche
  }
});
```
