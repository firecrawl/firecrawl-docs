<div id="scraping-with-firecrawl">
  ## Scraping avec Firecrawl
</div>

Firecrawl convertit les pages web en Markdown, idéal pour les applications LLM. Voici pourquoi :

1. **Complexités gérées :**
   Gère les proxys, la mise en cache, les limites de débit et le contenu bloqué par JavaScript pour un scraping fluide.

2. **Contenu dynamique :**
   Récupère des données depuis des sites rendus en JavaScript, des PDF, des images, etc.

3. **Conversion en Markdown ou en données structurées :**
   Convertit les données collectées en Markdown propre ou en sortie structurée, parfait pour le traitement par des LLM ou toute autre tâche.

Pour plus de détails, consultez la [référence de l’API du point de terminaison /scrape](https://docs.firecrawl.dev/api-reference/endpoint/scrape).

<div id="usage">
  ### Utilisation
</div>

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

content = app.scrape_url("https://mendable.ai")
```

```JavaScript JavaScript
import { FirecrawlApp } from 'firecrawl-js';

const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

const content = await app.scrapeUrl('https://mendable.ai');
```

```go Go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
    log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
  }

  content, err := app.ScrapeURL("mendable.ai", nil)
  if err != nil {
    log.Fatalf("Failed to scrape URL: %v", err)
  }
  fmt.Println(content)
}
```

```rust Rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    // Initialiser FirecrawlApp avec la clé API
    let api_key = "YOUR_API_KEY";
    let api_url = "https://api.firecrawl.dev";
    let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

    // Récupérer le contenu de l’URL
    let scrape_result = app.scrape_url("https://mendable.ai", None).await;

    // Afficher le résultat
  match scrape_result {
    Ok(data) => println!("Résultat du scraping:\n{}", data["markdown"]),
    Err(e) => eprintln!("Échec du scraping : {}", e),
  }
}
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v0/scrape \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "url": "https://mendable.ai"
    }'
```
