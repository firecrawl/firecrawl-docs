---
title: "Obtenir l’état d’un crawl"
openapi: '/fr/v0/api-reference/v0-openapi.json GET /crawl/status/{jobId}'
---

Ce point de terminaison renvoie l’état d’un job de crawl. Si le job n’est pas terminé, la réponse inclut du contenu dans `partial_data`. Une fois le job terminé, le contenu est disponible dans `data`.

**Nous vous recommandons de suivre vos jobs de crawl, car les résultats d’état peuvent expirer au bout de 24 heures.**
