Le point de terminaison /search combine une API de recherche avec la puissance de Firecrawl pour offrir une expérience de recherche performante pour toute requête.

Il recherche automatiquement le Web en fonction de la requête et renvoie les résultats les plus pertinents des pages principales au format Markdown. L’avantage de ce point de terminaison est qu’il scrute réellement chaque site Web figurant parmi les premiers résultats, afin que vous obteniez toujours le contenu complet.

Ce point de terminaison est actuellement en bêta et peut évoluer.
