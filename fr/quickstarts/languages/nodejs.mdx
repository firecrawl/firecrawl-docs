<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le SDK Node officiel de Firecrawl pour convertir une URL en contenu propre, adapté aux LLM.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Node.js 18+
* npm, pnpm ou yarn

<div id="install">
  ## Installation
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-your-api-key">
  ## Configurez votre clé API
</div>

Stockez votre clé API dans une variable d’environnement à laquelle seul votre serveur peut accéder.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Effectuez votre première requête
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

(async () => {
  const doc = await app.scrape('https://example.com', { formats: ['markdown'] });
  console.log(doc.markdown);
})();
```

<div id="complete-example">
  ## Exemple complet
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

async function main() {
  const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
  const doc = await app.scrape('https://example.com', { formats: ['markdown', 'html'] });
  console.log(doc.metadata?.title);
  console.log(doc.markdown);
}

main().catch((error) => {
  console.error(error);
  process.exit(1);
});
```
