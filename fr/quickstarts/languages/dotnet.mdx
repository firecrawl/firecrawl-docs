<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez HttpClient pour envoyer une requête de scraping à Firecrawl depuis .NET.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* .NET 6+

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; cet exemple utilise HttpClient.

<div id="configure-your-api-key">
  ## Configurez votre clé API
</div>

Définissez votre clé API comme variable d’environnement pour le processus.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Envoyez votre première requête
</div>

```csharp
using System.Net.Http.Headers;
using System.Net.Http.Json;

var apiKey = Environment.GetEnvironmentVariable("FIRECRAWL_API_KEY");
var http = new HttpClient();
http.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", apiKey);

var payload = new { url = "https://example.com", formats = new[] { "markdown" } };
var response = await http.PostAsJsonAsync("https://api.firecrawl.dev/v2/scrape", payload);
var body = await response.Content.ReadAsStringAsync();

Console.WriteLine(body);
```

<div id="complete-example">
  ## Exemple complet
</div>

```csharp
using System.Net.Http.Headers;
using System.Net.Http.Json;

var apiKey = Environment.GetEnvironmentVariable("FIRECRAWL_API_KEY");
var http = new HttpClient();
http.DefaultRequestHeaders.Authorization = new AuthenticationHeaderValue("Bearer", apiKey);

var payload = new { url = "https://example.com", formats = new[] { "markdown", "html" } };
var response = await http.PostAsJsonAsync("https://api.firecrawl.dev/v2/scrape", payload);
response.EnsureSuccessStatusCode();

var body = await response.Content.ReadAsStringAsync();
Console.WriteLine(body);
```
