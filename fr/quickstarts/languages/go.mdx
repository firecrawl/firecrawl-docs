<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez la bibliothèque standard de Go pour envoyer une requête de scraping à Firecrawl via HTTP.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Go 1.20+

<div id="install">
  ## Installation
</div>

Aucun paquet supplémentaire n’est nécessaire ; cet exemple utilise la bibliothèque standard.

<div id="configure-your-api-key">
  ## Configurez votre clé API
</div>

Définissez votre clé API en tant que variable d’environnement avant d’exécuter votre programme.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Effectuez votre première requête
</div>

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
    "os"
)

func main() {
    payload := map[string]any{
        "url": "https://example.com",
        "formats": []string{"markdown"},
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://api.firecrawl.dev/v2/scrape", bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("FIRECRAWL_API_KEY"))
    req.Header.Set("Content-Type", "application/json")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    var result map[string]any
    _ = json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result)
}
```

<div id="complete-example">
  ## Exemple complet
</div>

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
    "os"
)

func main() {
    payload := map[string]any{
        "url": "https://example.com",
        "formats": []string{"markdown", "html"},
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://api.firecrawl.dev/v2/scrape", bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("FIRECRAWL_API_KEY"))
    req.Header.Set("Content-Type", "application/json")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    raw, _ := io.ReadAll(resp.Body)
    fmt.Println(string(raw))
}
```
