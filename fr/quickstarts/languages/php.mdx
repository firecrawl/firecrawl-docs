<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez l’extension cURL de PHP pour envoyer une requête de scraping à Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* PHP 8.1+ avec cURL activé

<div id="install">
  ## Installation
</div>

Aucun paquet supplémentaire n’est nécessaire ; cet exemple utilise l’extension cURL intégrée.

<div id="configure-your-api-key">
  ## Configurez votre clé API
</div>

Enregistrez votre clé API dans une variable d’environnement sur le serveur.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Faites votre première requête
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
```

<div id="complete-example">
  ## Exemple complet
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown', 'html'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
if ($response === false) {
    throw new RuntimeException(curl_error($ch));
}
curl_close($ch);

echo $response;
```
