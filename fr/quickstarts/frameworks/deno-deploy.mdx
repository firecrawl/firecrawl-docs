<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez l’API fetch de Deno Deploy pour appeler Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Un projet Deno Deploy

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; Deno inclut fetch.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez la clé API dans l’environnement du projet Deno Deploy.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

main.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('https://your-app.deno.dev', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Deno Deploy n’autorise que les requêtes sortantes via fetch ; gardez tous les appels à Firecrawl côté serveur.
