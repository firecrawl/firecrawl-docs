<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le client HTTP de Laravel pour appeler l’API Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application Laravel
* PHP 8.1+

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; Laravel inclut un client HTTP.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Ajoutez la clé API à votre fichier `.env`.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

routes/api.php

```php
<?php

use Illuminate\Support\Facades\Http;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Route;

Route::post('/firecrawl', function (Request $request) {
    $response = Http::withToken(env('FIRECRAWL_API_KEY'))
        ->post('https://api.firecrawl.dev/v2/scrape', [
            'url' => $request->input('url'),
            'formats' => ['markdown'],
        ]);

    return $response->json();
});
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```bash
curl -X POST http://localhost:8000/api/firecrawl               -H 'Content-Type: application/json'               -d '{"url": "https://example.com"}'
```

<div id="notes">
  ## Remarques
</div>

* Conservez la clé d’API uniquement dans des variables d’environnement côté serveur.
