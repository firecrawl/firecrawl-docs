<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez un gestionnaire Lambda pour relayer en toute sécurité les requêtes Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* AWS Lambda (environnement d’exécution Node.js 18)

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; utilisez fetch avec Node.js 18.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez `FIRECRAWL_API_KEY` dans les variables d’environnement de votre fonction Lambda.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

index.mjs

```js
export const handler = async (event) => {
  const { url } = JSON.parse(event.body || '{}');

  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return {
    statusCode: 200,
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data),
  };
};
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('https://your-api-id.execute-api.region.amazonaws.com/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

  ## Notes
