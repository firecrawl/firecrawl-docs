<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le `fetch` intégré de Bun pour appeler l’API HTTP de Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Bun 1.x

<div id="install">
  ## Installation
</div>

Aucun paquet supplémentaire n’est nécessaire ; Bun inclut fetch par défaut.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez votre clé API dans l’environnement avant d’exécuter Bun.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

server.ts

```ts
const server = Bun.serve({
  port: 3000,
  async fetch(req) {
    const url = new URL(req.url);
    if (url.pathname != '/api/firecrawl' || req.method != 'POST') {
      return new Response('Not found', { status: 404 });
    }

    const { url: target } = await req.json();
    const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ url: target, formats: ['markdown'] }),
    });

    return new Response(await response.text(), {
      headers: { 'Content-Type': 'application/json' },
    });
  },
});

console.log(`Listening on http://localhost:${server.port}`);
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Conservez la clé API sur le serveur et faites transiter les requêtes via votre handler Bun.
