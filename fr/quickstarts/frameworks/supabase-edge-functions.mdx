<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez une fonction Edge basée sur Deno pour appeler l’API HTTP de Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Clé API Firecrawl
* Projet Supabase
* CLI Supabase

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; les fonctions Edge de Supabase s’exécutent sur Deno avec fetch.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez la clé API en tant que secret Supabase.

```bash
supabase secrets set FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

supabase/functions/firecrawl/index.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('https://project.supabase.co/functions/v1/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Les fonctions Edge utilisent Deno, évitez donc les packages réservés à Node.
