<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez directement l’API HTTP de Firecrawl dans une route Hono pour qu’elle fonctionne sur tous les environnements d’exécution.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application Hono
* Un environnement d’exécution pris en charge (Node, Bun, Cloudflare, Deno)

<div id="install">
  ## Installation
</div>

```bash
npm install hono
```

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Exposez la clé API via les liaisons d’environnement de votre environnement d’exécution.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

src/index.ts

```ts
import { Hono } from 'hono';

type Bindings = { FIRECRAWL_API_KEY: string };

const app = new Hono<{ Bindings: Bindings }>();

app.post('/api/firecrawl', async (c) => {
  const { url } = await c.req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${c.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return c.json(data);
});

export default app;
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Configurez les liaisons d’environnement pour l’environnement d’exécution de votre choix (Wrangler, Deno Deploy, etc.).
