<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le SDK Python de Firecrawl dans le gestionnaire d’une route FastAPI.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé d’API Firecrawl
* Une application FastAPI
* Python 3.8+

<div id="install">
  ## Installation
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configurez l’environnement
</div>

Définissez la clé API dans l’environnement de votre serveur.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

app/main.py

```python
import os
from fastapi import FastAPI
from firecrawl import Firecrawl

app = FastAPI()
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def scrape(body: dict):
    doc = client.scrape(body["url"], formats=["markdown"])
    return {"markdown": doc.markdown}
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Remarques
</div>

* N’initialisez le client Firecrawl qu’une seule fois afin de réutiliser les connexions HTTP.
