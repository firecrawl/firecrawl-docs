<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le SDK Node de Firecrawl dans une route de serveur Fastify.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application Fastify
* Node.js 18+

<div id="install">
  ## Installer
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Ajoutez la clé API à l’environnement de votre serveur.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route au serveur
</div>

server.js

```js
import Fastify from 'fastify';
import Firecrawl from '@mendable/firecrawl-js';

const app = Fastify();
const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

app.post('/api/firecrawl', async (request) => {
  const { url } = request.body as { url: string };
  const doc = await client.scrape(url, { formats: ['markdown'] });
  return { markdown: doc.markdown };
});

app.listen({ port: 3000 });
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

<div id="notes">
  ## Remarques
</div>

* Conservez les requêtes Firecrawl côté serveur afin de protéger votre clé API.
