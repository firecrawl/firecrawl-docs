<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le module Net::HTTP de Ruby pour appeler l’API Firecrawl depuis un contrôleur.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application Rails
* Ruby 3.0+

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; utilisez la bibliothèque standard de Ruby.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez la clé API dans votre environnement Rails.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

app/controllers/firecrawl&#95;controller.rb

```rb
require 'net/http'
require 'uri'
require 'json'

class FirecrawlController < ApplicationController
  protect_from_forgery with: :null_session

  def create
    uri = URI('https://api.firecrawl.dev/v2/scrape')
    request = Net::HTTP::Post.new(uri)
    request['Authorization'] = "Bearer #{ENV.fetch('FIRECRAWL_API_KEY')}"
    request['Content-Type'] = 'application/json'
    request.body = { url: params[:url], formats: ['markdown'] }.to_json

    response = Net::HTTP.start(uri.hostname, uri.port, use_ssl: true) do |http|
      http.request(request)
    end

    render json: JSON.parse(response.body)
  end
end
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```rb
require 'net/http'
require 'json'

uri = URI('http://localhost:3000/api/firecrawl')
response = Net::HTTP.post(uri, { url: 'https://example.com' }.to_json, 'Content-Type' => 'application/json')

puts JSON.parse(response.body)
```

<div id="notes">
  ## Remarques
</div>

* Ajoutez `post '/api/firecrawl', to: 'firecrawl#create'` à `config/routes.rb`.
