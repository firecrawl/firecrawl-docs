<div id="overview">
  ## Vue d’ensemble
</div>

Créez une route action dans Remix pour appeler Firecrawl côté serveur.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application Remix
* Node.js 18+

<div id="install">
  ## Installation
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Stockez la clé API dans `.env` ou dans votre environnement de déploiement.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

app/routes/api.firecrawl.ts

```ts
import { json } from '@remix-run/node';
import Firecrawl from '@mendable/firecrawl-js';

export const action = async ({ request }) => {
  const { url } = await request.json();
  const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

  const doc = await client.scrape(url, { formats: ['markdown'] });
  return json({ markdown: doc.markdown });
};
```

<div id="call-it-from-your-app">
  ## Utilisez-le depuis votre application
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

<div id="notes">
  ## Remarques
</div>

* Utilisez les actions ou les loaders de Remix pour garder les appels à Firecrawl du côté serveur.
