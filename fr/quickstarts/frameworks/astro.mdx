<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez une route API Astro pour appeler Firecrawl côté serveur.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Un projet Astro
* Node.js 18+

<div id="install">
  ## Installation
</div>

Aucun paquet supplémentaire n’est nécessaire ; utilisez fetch dans la route d’API.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Ajoutez la clé API dans `.env`, puis récupérez-la dans le code côté serveur.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route de serveur
</div>

src/pages/api/firecrawl.ts

```ts
import type { APIRoute } from 'astro';

export const POST: APIRoute = async ({ request }) => {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${import.meta.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
};
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Utilisez des routes serveur pour les appels à Firecrawl ; évitez les requêtes côté client avec la clé API.
