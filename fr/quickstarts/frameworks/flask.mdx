<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez le SDK Python de Firecrawl dans un point de terminaison d’un serveur Flask.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application Flask
* Python 3.8+

<div id="install">
  ## Installation
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez votre clé API dans l’environnement avant de démarrer Flask.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Ajouter une route au serveur
</div>

app.py

```python
import os
from flask import Flask, request, jsonify
from firecrawl import Firecrawl

app = Flask(__name__)
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def firecrawl():
    url = request.json["url"]
    doc = client.scrape(url, formats=["markdown"])
    return jsonify(markdown=doc.markdown)
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```python
import requests

response = requests.post(
    "http://localhost:5000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Notes
</div>

* Initialisez le client Firecrawl une seule fois et réutilisez-le pour toutes les requêtes.
