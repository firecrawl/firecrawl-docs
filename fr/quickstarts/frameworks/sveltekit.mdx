<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez un point de terminaison SvelteKit `+server.ts` pour appeler Firecrawl sur le serveur.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Une application SvelteKit
* Node.js 18+

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; utilisez `fetch` depuis le point de terminaison serveur.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Enregistrez la clé API dans `.env` et chargez-la depuis `$env/static/private`.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route serveur
</div>

src/routes/api/firecrawl/+server.ts

```ts
import { json } from '@sveltejs/kit';
import { FIRECRAWL_API_KEY } from '$env/static/private';

export async function POST({ request }) {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return json(data);
}
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Conservez la clé API dans des variables d’environnement privées ; évitez de l’exposer via `$env/static/public`.
