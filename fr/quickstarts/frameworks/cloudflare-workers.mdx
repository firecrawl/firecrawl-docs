<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez `fetch` dans un Worker pour appeler l’API HTTP de Firecrawl.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Un projet Cloudflare Workers
* Wrangler

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; les Workers incluent fetch par défaut.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez la clé API en tant que secret Worker avec Wrangler.

```bash
wrangler secret put FIRECRAWL_API_KEY
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

src/index.ts

```ts
export interface Env {
  FIRECRAWL_API_KEY: string;
}

export default {
  async fetch(request: Request, env: Env): Promise<Response> {
    if (request.method !== 'POST') {
      return new Response('Method not allowed', { status: 405 });
    }

    const { url } = await request.json();
    const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${env.FIRECRAWL_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ url, formats: ['markdown'] }),
    });

    return new Response(await response.text(), {
      headers: { 'Content-Type': 'application/json' },
    });
  },
};
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('https://your-worker.example.workers.dev', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Remarques
</div>

* Les Workers s’exécutent à la périphérie du réseau, donc faites toujours passer les requêtes Firecrawl par le Worker.
