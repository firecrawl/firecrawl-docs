<div id="overview">
  ## Vue d’ensemble
</div>

Utilisez une fonction Vercel comme proxy pour les appels à Firecrawl sans exposer votre clé API.

<div id="prerequisites">
  ## Prérequis
</div>

* Une clé API Firecrawl
* Un projet Vercel
* Node.js 18+

<div id="install">
  ## Installation
</div>

Aucun package supplémentaire n’est nécessaire ; utilisez fetch dans la fonction.

<div id="configure-environment">
  ## Configurer l’environnement
</div>

Définissez `FIRECRAWL_API_KEY` dans `.env.local` ainsi que dans les paramètres de votre projet Vercel.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Ajouter une route côté serveur
</div>

api/firecrawl.ts

```ts
import type { VercelRequest, VercelResponse } from '@vercel/node';

export default async function handler(req: VercelRequest, res: VercelResponse) {
  if (req.method !== 'POST') {
    res.status(405).send('Method not allowed');
    return;
  }

  const { url } = req.body;
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  res.status(200).json(data);
}
```

<div id="call-it-from-your-app">
  ## Appelez-le depuis votre application
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

  ## Notes
</div>

* Configurez les variables d’environnement dans Vercel pour préserver la confidentialité de la clé API.
