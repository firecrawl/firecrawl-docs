<div id="inline-code">
  ### Code en ligne
</div>

Pour indiquer qu’un `mot` ou une `expression` est du code, encadrez-le de backticks (`).

```
Pour indiquer qu’un mot ou une expression est du code, entourez-les d’accents graves (`).
```


<div id="code-block">
  ### Bloc de code
</div>

Utilisez des [blocs de code délimités](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) en encadrant le code de trois accents graves, puis indiquez la langue de programmation de votre extrait juste après les accents d’ouverture pour activer la coloration syntaxique. Vous pouvez également indiquer le nom de votre code après la langue de programmation, de façon optionnelle.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Bonjour, le monde !");
    }
}
```
