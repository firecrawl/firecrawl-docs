<div id="creating-a-custom-snippet">
  ## Créer un extrait personnalisé
</div>

**Condition préalable** : vous devez créer votre fichier d’extrait dans le répertoire `snippets`.

<Note>
  Toute page située dans le répertoire `snippets` sera traitée comme un extrait et ne
  sera pas rendue en tant que page autonome. Si vous souhaitez créer une page
  autonome à partir de l’extrait, importez l’extrait dans un autre fichier et
  utilisez-le comme composant.
</Note>

<div id="default-export">
  ### Export par défaut
</div>

1. Ajoutez du contenu au fichier de snippet que vous souhaitez réutiliser à plusieurs
   emplacements. Vous pouvez aussi définir des variables à renseigner via des props
   lors de l’import du snippet.

```mdx extraits/mon-extrait.mdx
Bonjour, monde ! Voici mon contenu que je souhaite réutiliser sur plusieurs pages. Mon mot‑clé du
jour est {word}.
```

<Warning>
  Le contenu que vous souhaitez réutiliser doit se trouver dans le répertoire `snippets`
  pour que l’import fonctionne.
</Warning>

2. Importez l’extrait dans votre fichier de destination.

```mdx fichier-de-destination.mdx
---
title: Mon titre
description: Ma description
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## En-tête

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

<div id="reusable-variables">
  ### Variables réutilisables
</div>

1. Exportez une variable depuis votre fichier de snippet :

```mdx extraits/chemin/vers/variables-personnalisées.mdx
export const myName = 'mon nom';

export const myObject = { fruit: 'fraises' };
```

2. Importez l’extrait depuis votre fichier cible et utilisez la variable :

```mdx fichier-de-destination.mdx
---
title: Mon titre
description: Ma description
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Bonjour, je m’appelle {myName} et j’aime {myObject.fruit}.
```

<div id="reusable-components">
  ### Composants réutilisables
</div>

1. Dans votre fichier de snippet, créez un composant qui reçoit des props en exportant
   votre composant sous forme de fonction fléchée.

```mdx extraits/custom-component.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... extrait de contenu ...</p>
  </div>
);
```

<Warning>
  MDX ne se compile pas dans le corps d&#39;une fonction fléchée. Préférez la syntaxe HTML
  lorsque c&#39;est possible, ou exportez par défaut si vous devez utiliser du MDX.
</Warning>

2. Importez le snippet dans votre fichier de destination et transmettez les props

```mdx fichier-de-destination.mdx
---
title: Mon titre
description: Ma description
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Titre personnalisé'} />
```
