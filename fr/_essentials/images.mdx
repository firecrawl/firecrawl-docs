<div id="image">
  ## Image
</div>

<div id="using-markdown">
  ### Utiliser Markdown
</div>

La [syntaxe Markdown](https://www.markdownguide.org/basic-syntax/#images) permet d’ajouter des images avec le code suivant

```md
![Titre](/path/image.jpg)
```

Notez que la taille du fichier image doit être inférieure à 5 Mo. Sinon, nous recommandons d’héberger l’image sur un service comme [Cloudinary](https://cloudinary.com/) ou [S3](https://aws.amazon.com/s3/). Vous pouvez ensuite utiliser cette URL et l’intégrer.

<div id="using-embeds">
  ### Utiliser les embeds
</div>

Pour bénéficier d’une personnalisation plus fine des images, vous pouvez également utiliser les [embeds](/fr/writing-content/embed) pour ajouter des images

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

Charge une autre page HTML au sein du document. Le plus souvent utilisé pour intégrer des vidéos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
