> Remarque : cet exemple utilise la [version v0 de l’API Firecrawl](/fr/v0/introduction). Vous pouvez installer la version 0.0.20 du SDK Python ou la 0.0.36 du SDK Node.

> [Dépôt GitHub (https://github.com/e2b-dev/e2b-cookbook)](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)

Cet exemple montre comment extraire des données Airbnb et les analyser avec [Firecrawl](https://www.firecrawl.dev/) et le [Code Interpreter SDK](https://github.com/e2b-dev/code-interpreter) d’E2B.

Nous n’écrirons aucun code d’analyse ; nous demanderons plutôt à l’IA d’Anthropic de l’écrire pour nous. Nous utiliserons le Code Interpreter SDK d’E2B pour exécuter le code en toute sécurité et obtenir les résultats.

<div id="prerequisites">
  ## Prérequis
</div>

* Node.js installé sur votre machine
* Une clé API E2B
* Une clé API Firecrawl
* Une clé API Anthropic

<div id="setup-run">
  ## Configuration et exécution
</div>

<div id="1-clone-the-repository">
  ### 1. Cloner le dépôt
</div>

Voici le [dépôt GitHub](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl) pour cet exemple.

```
git clone https://github.com/e2b-dev/e2b-cookbook.git
cd e2b-cookbook/examples/scrape-and-analyze-airbnb-data-with-firecrawl
```

<div id="2-install-dependencies">
  ### 2. Installez les dépendances
</div>

```
npm install
```

<div id="3-set-up-env">
  ### 3. Configurer `.env`
</div>

1. Copier `.env.template` en `.env`
2. Récupérer la [clé API E2B](https://e2b.dev/docs/getting-started/api-key)
3. Récupérer la [clé API Firecrawl](https://firecrawl.dev)
4. Récupérer la [clé API Anthropic](https://anthropic.com)

<div id="4-run-the-example">
  ### 4. Exécuter l’exemple
</div>

```
npm run start
```
