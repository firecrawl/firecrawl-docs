> Remarque : cet exemple utilise la [version v0 de l’API Firecrawl](/fr/v0/introduction). Vous pouvez installer la version 0.0.20 du SDK Python ou la 0.0.36 du SDK Node.

Avec Firecrawl, vous pouvez envoyer des cookies dans les en-têtes pour contourner l’authentification sur un site web. Cela peut être utile lorsque vous devez scraper du contenu sur un site nécessitant une connexion.

<div id="example">
  ### Exemple
</div>

Pour contourner l’authentification, incluez le paramètre `headers` dans votre requête et définissez l’en-tête `Cookie` avec les valeurs appropriées, récupérées depuis les cookies de votre navigateur. Si vous avez besoin d’aide pour récupérer des cookies, consultez cet article : [Grabbing cookies](https://cookie-script.com/documentation/how-to-check-cookies-on-chrome-and-firefox)

<div id="usage">
  ### Utilisation
</div>

<CodeGroup>
  ```python Python
  from firecrawl import FirecrawlApp

  app = FirecrawlApp(api_key="YOUR_API_KEY")

  app.scrape_url("https://docs.firecrawl.dev", {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  })
  ```

  ```javascript JavaScript
  import { FirecrawlApp } from 'firecrawl-js';

  const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

  app.scrapeUrl('https://docs.firecrawl.dev', {
    pageOptions:{
      headers:{
        "Cookie": "cookie1=value1; cookie2=value2"
      }
    }
  });
  ```

  ```go Go
  import (
    "fmt"
    "log"

    "github.com/mendableai/firecrawl-go"
  )

  func main() {
    // Initialiser FirecrawlApp avec votre clé API
    app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
    if err != nil {
      log.Fatalf("Échec de l'initialisation de FirecrawlApp : %v", err)
    }

    params := map[string]any{
      "pageOptions": map[string]any{
        "headers": map[string]any{
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      },
    }
    scrapedData, err := app.ScrapeURL("docs.firecrawl.dev", params)
    if err != nil {
      log.Fatalf("Erreur lors du scraping : %v", err)
    }
    fmt.Println(scrapedData)
  }
  ```

  ```rust Rust
  use firecrawl::FirecrawlApp;

  #[tokio::main]
  async fn main() {
      let api_key = "YOUR_API_KEY";
      let api_url = "https://api.firecrawl.dev";
      let app = FirecrawlApp::new(api_key, api_url).expect("Échec de l'initialisation de FirecrawlApp");

      let scrape_params = json!({
          "pageOptions": {
              "headers": {
                  "Cookie": "cookie1=value1; cookie2=value2"
              }
          }
      });

      let scrape_result = app.scrape_url("https://docs.firecrawl.dev", Some(scrape_params)).await;
      match scrape_result {
          Ok(data) => println!("Résultat du scraping:\n{}", data),
          Err(e) => eprintln!("Échec du scraping : {}", e),
      }
  }
  ```

  ```curl cURL
  curl -X POST https://api.firecrawl.dev/v0/scrape \

      -d '{
        "url": "https://example.com",
        "headers": {
          "Cookie": "cookie1=value1; cookie2=value2"
        }
      }'
  ```
