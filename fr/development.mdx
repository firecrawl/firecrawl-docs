<Info>
  **Prérequis** Vous devez avoir installé Node.js (version 18.10.0 ou
  supérieure).
</Info>

Étape 1. Installez Mintlify sur votre système :

<CodeGroup>
  ```bash npm
  npm i -g mintlify
  ```

  ```bash yarn
  yarn global add mintlify
  ```
</CodeGroup>

Étape 2. Rendez-vous à l’emplacement où se trouvent les documents (là où se trouve `mint.json`) et exécutez la commande suivante :

```bash
mintlify dev
```

Le site de documentation est maintenant accessible à l’adresse `http://localhost:3000`.

<div id="custom-ports">
  ### Ports personnalisés
</div>

Par défaut, Mintlify utilise le port 3000. Vous pouvez utiliser l’option `--port` pour définir le port sur lequel Mintlify s’exécute. Par exemple, utilisez cette commande pour l’exécuter sur le port 3333 :

```bash
mintlify dev --port 3333
```

Vous verrez une erreur de ce type si vous tentez d’exécuter Mintlify sur un port déjà utilisé :

```md
Erreur : listen EADDRINUSE : adresse déjà utilisée :::3000
```

<div id="mintlify-versions">
  ## Versions de Mintlify
</div>

Chaque CLI est associée à une version spécifique de Mintlify. Mettez à jour la CLI si votre site en local diffère de l’environnement de production.

<CodeGroup>
  ```bash npm
  npm i -g mintlify@latest
  ```

  ```bash yarn
  yarn global upgrade mintlify
  ```
</CodeGroup>

<div id="troubleshooting">
  ## Dépannage
</div>

Voici comment résoudre quelques problèmes courants lors de l&#39;utilisation de la CLI.

<AccordionGroup>
  <Accordion title="Mintlify ne se charge pas">
    Mettez à jour Node vers la v18. Exécutez `mintlify install`, puis réessayez.
  </Accordion>

  <Accordion title="Aucun fichier ou dossier de ce type sur Windows">
    Accédez au répertoire `C:/Users/Username/.mintlify/` et supprimez le dossier
    `mint`. Ensuite, ouvrez Git Bash à cet emplacement et exécutez `git clone
    https://github.com/mintlify/mint.git`.

    Répétez l&#39;étape 3.
  </Accordion>

  <Accordion title="Erreur inconnue">
    Essayez d&#39;accéder à la racine de votre appareil et de supprimer le dossier ~/.mintlify.
    Ensuite, exécutez à nouveau `mintlify dev`.
  </Accordion>
</AccordionGroup>

Vous voulez savoir ce qui a changé dans une version de la CLI ? [Consultez le changelog de la CLI.](/fr/changelog/command-line)
