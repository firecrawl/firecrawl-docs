<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## Rechercher sur le web et extraire les résultats avec Firecrawl
</div>

Firecrawl intègre son API SERP (Search Engine Results Page) à son infrastructure de scraping robuste pour offrir, via un seul point de terminaison, une expérience fluide de recherche et d’extraction. Voici comment :

1. **Requête de recherche unifiée :**
   Les utilisateurs envoient une requête de recherche via le point de terminaison SERP.

2. **Scraping automatisé des résultats :**
   Firecrawl traite automatiquement les résultats de recherche et utilise ses capacités de scraping pour extraire les données de chaque page de résultats.

3. **Livraison des données :**
   Les données extraites de toutes les pages de résultats sont compilées et livrées dans un markdown propre — prêtes à l’emploi.

Cette intégration permet d’effectuer efficacement des recherches web et d’obtenir, avec un minimum d’effort, des données complètes extraites de multiples sources.

Pour plus de détails, consultez la [documentation du point de terminaison /search](https://docs.firecrawl.dev/api-reference/endpoint/search).

<div id="search-any-query">
  ## Rechercher n’importe quelle requête
</div>

<div id="search-endpoint">
  ### point de terminaison /search
</div>

Permet d’effectuer une recherche sur le web, d’obtenir les résultats les plus pertinents, d’extraire chaque page et de renvoyer le markdown.

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer VOTRE_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // false pour une API SERP plus rapide
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Contenu Markdown",
      "provider": "web-scraper",
      "metadata": {
        "title": "Firecrawl | Récupérez le web de manière fiable pour vos LLM",
        "description": "IA pour l’expérience client et les ventes"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="search-a-query">
  #### Rechercher une requête
</div>

```go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("Échec de l’initialisation de FirecrawlApp : %v", err)
  }

  query := "Qu’est-ce que Firecrawl ?"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("Échec de la recherche : %v", err)
  }
  fmt.Println(searchResult)
}
```
