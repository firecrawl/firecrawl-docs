<div id="generate-llmstxt">
  ## generate-llmstxt
</div>

Un simple package NPX qui génère des fichiers llms.txt via la CLI en utilisant l’API Firecrawl. Ce package crée deux fichiers dans le répertoire de sortie spécifié (par défaut, le dossier « public ») :

- `llms.txt` : contient un résumé du contenu lié aux LLM
- `llms-full.txt` : contient le texte intégral

<div id="usage">
  ## Utilisation
</div>

Vous pouvez exécuter ce package avec NPX sans l’installer. Il existe deux manières de fournir votre clé d’API Firecrawl :

<div id="1-using-command-line-arguments">
  ### 1. Utiliser des arguments en ligne de commande
</div>

```bash
npx generate-llmstxt --api-key VOTRE_CLEF_API_FIRECRAWL
```


<div id="2-using-environment-variables">
  ### 2. Utiliser des variables d’environnement
</div>

Créez un fichier `.env` à la racine de votre projet et ajoutez votre clé d’API :

```env
FIRECRAWL_API_KEY=votre_cle_api_ici
```

Puis exécutez la commande sans l’option --api-key :

```bash
npx generate-llmstxt
```


<div id="options">
  ### Options
</div>

- `-k, --api-key <key>` (facultatif si défini dans .env) : votre clé API Firecrawl
- `-u, --url <url>` (facultatif) : URL à analyser (par défaut : https://example.com)
- `-m, --max-urls <number>` (facultatif) : nombre maximal d’URL à analyser (par défaut : 50)
- `-o, --output-dir <path>` (facultatif) : chemin du répertoire de sortie (par défaut : « public »)

<div id="examples">
  ### Exemples
</div>

```bash
# Utiliser un argument de ligne de commande avec le répertoire de sortie par défaut
npx generate-llmstxt -k your_api_key -u https://your-website.com -m 20

# Utiliser un fichier .env avec le répertoire de sortie par défaut
npx generate-llmstxt -u https://your-website.com -m 20

# Indiquer un répertoire de sortie personnalisé
npx generate-llmstxt -k your_api_key -u https://your-website.com -o custom/path/to/output

# Utiliser un fichier .env avec un répertoire de sortie personnalisé
npx generate-llmstxt -u https://your-website.com -o content/llms
```


<div id="requirements">
  ## Prérequis
</div>

- Node.js 14 ou ultérieur
- Une clé API Firecrawl valide (via la ligne de commande ou un fichier .env)

<div id="output">
  ## Sortie
</div>

Le package crée deux fichiers dans le répertoire de sortie indiqué (par défaut « public ») :

1. `llms.txt` : contient un résumé du contenu lié aux LLM
2. `llms-full.txt` : contient le texte intégral
