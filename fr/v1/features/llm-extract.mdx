<div id="scrape-and-extract-structured-data-with-firecrawl">
  ## Extraire des données structurées avec Firecrawl
</div>

{/* <Warning>Scrape LLM Extract sera supprimé dans de futures versions. Veuillez utiliser le nouveau point de terminaison [Extract](/features/extract).</Warning> */}

Firecrawl utilise l’IA pour obtenir des données structurées à partir de pages web en 3 étapes :

1. **Définir le schéma :**
   Indiquez les données souhaitées en définissant un schéma JSON (au format OpenAI) ainsi que l’URL de la page.

2. **Envoyer la requête :**
   Transmettez votre URL et votre schéma à notre point de terminaison /scrape. Voir comment faire :
   [Documentation du point de terminaison /scrape](https://docs.firecrawl.dev/api-reference/endpoint/scrape)

3. **Récupérer vos données :**
   Recevez des données propres et structurées correspondant à votre schéma, prêtes à l’emploi.

Ainsi, l’obtention de données web au format dont vous avez besoin devient rapide et simple.

<div id="json-options-object">
  ### Objet d’options JSON
</div>

L’objet `jsonOptions` accepte les paramètres suivants :

- `schema` : Le schéma à utiliser pour l’extraction.
- `systemPrompt` : L’invite système à utiliser pour l’extraction.
- `prompt` : L’invite à utiliser pour l’extraction sans schéma.
