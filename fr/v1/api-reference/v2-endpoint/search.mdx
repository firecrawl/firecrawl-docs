<div id="whats-new-in-v2">
  ## Nouveautés de la v2
</div>

<div id="search-multiple-sources">
  ### Rechercher dans plusieurs sources
</div>

Recherchez simultanément le web, les images et l’actualité :

```json
{
  "query": "firecrawl web scraping",
  "sources": [
    {"type": "web", "tbs": "qdr:w", "location": "US"},
    {"type": "images"},
    {"type": "news"}
  ]
}
```


<div id="response-format-changed">
  ### Format de réponse modifié
</div>

v1 : liste à plat des résultats. v2 : organisé par type de source :

```json
{
  "success": true,
  "data": {
    "web": [/* résultats Web */],
    "images": [/* résultats image */],
    "news": [/* résultats d’actualités */]
  }
}
```


<div id="new-features">
  ### Nouvelles fonctionnalités
</div>

- Filtrer par plage de dates (« past week », « past month »)
- Cibler des pays/régions spécifiques
- Résultats plafonnés à 500 pendant l’alpha

Le point de terminaison de recherche combine la recherche web (SERP) et les capacités de scraping de Firecrawl pour renvoyer le contenu complet des pages pour toute requête.

Incluez `scrapeOptions` avec `formats: [{"type": "markdown"}]` pour obtenir le contenu Markdown complet pour chaque résultat de recherche ; sinon, vous recevrez par défaut les résultats du SERP (url, title, description). Vous pouvez aussi utiliser d’autres formats comme `{"type": "summary"}` pour un contenu condensé.

<div id="supported-query-operators">
  ## Opérateurs de requête pris en charge
</div>

Nous prenons en charge une variété d’opérateurs de requête qui permettent d’affiner vos recherches.

| Opérateur | Fonctionnalité | Exemples |
---|-|-|
| `""` | Correspondance exacte d’une chaîne de texte (non floue) | `"Firecrawl"`
| `-` | Exclut certains mots-clés ou annule d’autres opérateurs | `-bad`, `-site:firecrawl.dev`
| `site:` | Renvoie uniquement des résultats provenant d’un site web spécifié | `site:firecrawl.dev`
| `inurl:` | Renvoie uniquement des résultats qui incluent un mot dans l’URL | `inurl:firecrawl`
| `allinurl:` | Renvoie uniquement des résultats qui incluent plusieurs mots dans l’URL | `allinurl:git firecrawl`
| `intitle:` | Renvoie uniquement des résultats qui incluent un mot dans le titre de la page | `intitle:Firecrawl`
| `allintitle:` | Renvoie uniquement des résultats qui incluent plusieurs mots dans le titre de la page | `allintitle:firecrawl playground`
| `related:` | Renvoie uniquement des résultats liés à un domaine spécifique | `related:firecrawl.dev`

<div id="location-parameter">
  ## Paramètre de localisation
</div>

Utilisez le paramètre `location` pour obtenir des résultats de recherche géociblés. Format : `"string"`. Exemples : `"Germany"`, `"San Francisco,California,United States"`.

Consultez la [liste complète des emplacements pris en charge](https://firecrawl.dev/search_locations.json) pour connaître tous les pays et langues disponibles.

<div id="time-based-search">
  ## Recherche par période
</div>

Utilisez le paramètre `tbs` pour filtrer les résultats selon des périodes, y compris des plages de dates personnalisées. Consultez la [documentation de la recherche](https://docs.firecrawl.dev/features/search#time-based-search) pour des exemples détaillés et les formats pris en charge.
