<div id="whats-new-in-v2">
  ## Nouveautés de la v2
</div>

<div id="better-error-messages">
  ### Des messages d’erreur améliorés
</div>

- Erreurs claires lorsque les invites en langage naturel échouent
- Messages précis pour les problèmes de sitemap et de limite de profondeur
- Messages plus pertinents pour les problèmes d’exploration à l’échelle du domaine

<div id="more-error-details">
  ### Plus de détails sur les erreurs
</div>

- Erreurs détaillées pour les problèmes de traitement des formats
- Messages spécifiques pour les échecs liés aux PDF et aux parsers
- Meilleures informations de débogage pour l'exploration basée sur la localisation
- Messages clairs pour les problèmes de suiviDesModifications

<div id="faster-and-easier-debugging">
  ### Débogage plus rapide et plus simple
</div>

- Récupération des erreurs accélérée pour les crawls de grande taille
- Regroupement des erreurs amélioré
- Contexte d’erreur plus pertinent

Même API que v1, bien meilleur rapport d’erreurs pour les fonctionnalités de v2.
