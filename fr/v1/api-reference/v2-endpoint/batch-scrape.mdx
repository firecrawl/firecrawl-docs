<div id="whats-new-in-v2">
  ## Nouveautés de la v2
</div>

<div id="all-new-formats-work-in-batches">
  ### Tous les nouveaux formats fonctionnent par lots
</div>

- Utilisez des captures d’écran personnalisées, l’extraction JSON et le suivi des modifications sur toutes les URL
- Traitez les PDF par lots avec le nouveau système de parsing
- Appliquez les mêmes paramètres de format à chaque URL du lot

<div id="better-batch-processing">
  ### Traitement par lots amélioré
</div>

- Extraire depuis différents pays/langues pour des lots complets
- Utiliser les modes de proxy stealth, basic ou auto sur toutes les URL
- Toutes les nouvelles options d’extraction fonctionnent en mode batch

<div id="faster-and-more-reliable">
  ### Plus rapide et plus fiable
</div>

- Meilleure gestion des ressources pour les lots complexes
- Suivi de l’avancement amélioré
- Gestion des erreurs renforcée

Même API que v1, toutes les fonctionnalités de v2 s’appliquent à l’ensemble du lot.
