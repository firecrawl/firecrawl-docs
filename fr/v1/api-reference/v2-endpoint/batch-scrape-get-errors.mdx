<div id="whats-new-in-v2">
  ## Nouveautés de la v2
</div>

<div id="better-batch-error-messages">
  ### Meilleurs messages d’erreur pour le traitement par lots
</div>

- Messages clairs pour les problèmes de traitement des formats sur toutes les URL
- Messages spécifiques pour les échecs liés aux PDF et aux parseurs
- Meilleur suivi des erreurs selon les différents modes de proxy
- Messages clairs pour les problèmes de scraping liés à la localisation

<div id="more-error-details">
  ### Détails supplémentaires sur les erreurs
</div>

- Informations de diagnostic pour les problèmes de suivi des modifications
- Signalement des erreurs en cas d’échec de l’automatisation du navigateur
- Meilleur suivi des erreurs liées aux crédits et aux limites de ressources
- Erreurs spécifiques pour les problèmes de traitement par lots

<div id="faster-and-easier-debugging">
  ### Débogage plus rapide et plus simple
</div>

- Récupération d’erreurs accélérée pour les gros lots
- Regroupement des erreurs amélioré
- Contexte d’erreur plus pertinent

Même API qu’en v1, mais un bien meilleur reporting des erreurs pour les fonctionnalités de la v2 en traitement par lots.
