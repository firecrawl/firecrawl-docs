<div id="better-sitemap-control">
  ### Meilleur contrôle des sitemaps
</div>

Trois façons de gérer les sitemaps :

* `"include"` - Utiliser le sitemap + découvrir d’autres pages (par défaut)
* `"skip"` - Ignorer complètement le sitemap
* `"only"` - Ne retourner que les URL du sitemap

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```


<div id="smarter-url-discovery">
  ### Découverte d’URL plus intelligente
</div>

- Repère toutes les pages d’un site web
- Pas de limite de résultats (v1 plafonné à 500)
- Plus rapide et plus précise

Même API qu’en v1, mais repère davantage de pages et gère mieux les sites complexes.
