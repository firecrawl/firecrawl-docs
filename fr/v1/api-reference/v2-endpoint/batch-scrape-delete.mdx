<div id="whats-new-in-v2">
  ## Nouveautés de la v2
</div>

L’API Cancel Batch Scrape en v2 conserve les mêmes fonctionnalités que la v1, avec une fiabilité accrue :

<div id="enhanced-cancellation-process">
  ### Processus d’annulation amélioré
</div>

- Nettoyage et gestion des ressources améliorés pour les opérations par lots
- Meilleure gestion de l’arrêt des lots actifs
- Gestion des erreurs renforcée pour les opérations d’annulation de lots

Les paramètres de l’API et le format de réponse restent inchangés par rapport à v1, garantissant une migration fluide tout en offrant une annulation de lots plus fiable.
