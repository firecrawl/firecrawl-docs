<div id="whats-new-in-v2">
  ## Nouveautés de la v2
</div>

<div id="tell-the-crawler-what-you-want">
  ### Indiquez au crawler ce que vous voulez
</div>

Décrivez en anglais simple ce qu’il doit explorer :

```json
{
  "url": "https://example.com",
  "prompt": "Ne parcourir que les articles de blog et la documentation, ignorer les pages marketing"
}
```

<div id="better-sitemap-control">
  ### Meilleur contrôle des sitemaps
</div>

v1 : sitemaps activés/désactivés. v2 : trois options :

- `"include"` - Utiliser le sitemap et découvrir d'autres pages (par défaut)
- `"skip"` - Ignorer complètement le sitemap  
- `"only"` - Explorer uniquement les URL du sitemap

<div id="new-crawling-options">
  ### Nouvelles options de crawl
</div>

* `crawlEntireDomain` - Explorer tout le domaine, pas seulement les pages enfant
* `maxDiscoveryDepth` - Contrôle de la profondeur de crawl (remplace `maxDepth`)

```json
{
  "url": "https://example.com/features",
  "crawlEntireDomain": true,
  "maxDiscoveryDepth": 2,
  "sitemap": "include"
}
```
