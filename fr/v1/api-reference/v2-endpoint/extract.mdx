  ## Nouveautés de la v2
</div>

<div id="smarter-extraction">
  ### Extraction plus intelligente
</div>

L’agent FIRE-1 gère les sites web complexes :

- Clique automatiquement dans les pages et la pagination
- Fonctionne sur les sites très dépendants de JavaScript
- Repère et extrait les données plus efficacement

<div id="no-url-required">
  ### Aucune URL requise
</div>

Extrayez des données sans connaître l’URL exacte :

- Décrivez ce que vous cherchez, v2 le trouve
- Effectue une recherche web et extrait les meilleures pages
- Plus précis que v1

<div id="better-performance">
  ### Meilleures performances
</div>

- Traitement plus rapide
- Gestion des erreurs plus fiable
- Précision accrue

Même API que la v1, mise à niveau facile.
