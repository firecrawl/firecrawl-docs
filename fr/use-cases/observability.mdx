<div id="how-it-works">
  ## Fonctionnement
</div>

Exploitez les capacités d’extraction de Firecrawl pour bâtir des systèmes d’observabilité pour vos sites web. Extrayez le contenu des pages, analysez les évolutions dans le temps, validez les déploiements et mettez en place des workflows de supervision qui garantissent le bon fonctionnement de vos sites.

<div id="what-you-can-monitor">
  ## Ce que vous pouvez surveiller
</div>

* **Disponibilité** : Uptime, temps de réponse, taux d’erreurs
* **Contenu** : Modifications de texte, mises à jour d’images, décalages de mise en page
* **Performance** : Temps de chargement des pages, tailles des ressources, Core Web Vitals
* **Sécurité** : Certificats SSL, en-têtes de sécurité, erreurs de configuration
* **SEO** : Balises meta, données structurées, validité du sitemap

<div id="faqs">
  ## FAQs
</div>

<AccordionGroup>
  <Accordion title="Comment Firecrawl aide-t-il à la supervision de sites web ?">
    Firecrawl extrait à la demande le contenu et la structure des sites. Créez des systèmes de supervision qui appellent l’API de Firecrawl pour vérifier des pages, comparer les données extraites à des références et déclencher vos propres alertes en cas de changement.
  </Accordion>

  <Accordion title="Puis-je superviser des applications très riches en JavaScript ?">
    Oui ! Firecrawl exécute entièrement le JavaScript, ce qui le rend idéal pour superviser les SPA modernes, les applications React et le contenu dynamique. Nous capturons la page telle que les utilisateurs la voient, pas seulement le HTML brut.
  </Accordion>

  <Accordion title="À quelle vitesse puis-je détecter des problèmes sur un site ?">
    Firecrawl extrait les données en temps réel à chaque appel. Concevez votre système de supervision pour vérifier les sites à la fréquence requise — de minute en minute pour les pages critiques à une fois par jour pour les contrôles de routine.
  </Accordion>

  <Accordion title="Puis-je valider des éléments spécifiques d’une page ?">
    Oui. Utilisez l’API d’extraction pour récupérer des éléments précis comme les prix, les niveaux de stock ou du contenu critique. Intégrez une logique de validation dans votre système de supervision pour vérifier que les informations importantes sont présentes et correctes.
  </Accordion>

  <Accordion title="Comment puis-je intégrer Firecrawl avec des systèmes d’alerte ?">
    Firecrawl fournit des webhooks que vous pouvez utiliser pour intégrer vos outils d’alerte. Envoyez les données extraites vers PagerDuty, Slack, l’e-mail ou toute plateforme de supervision via des connecteurs qui traitent les réponses de Firecrawl.
  </Accordion>
</AccordionGroup>
