<div id="sync-web-data-in-flowise-workflows">
  ## Synchroniser des données web dans des workflows Flowise
</div>

Firecrawl peut être utilisé dans [Flowise, le constructeur de Chatflows](https://flowiseai.com/). Cette page présente la configuration et l’utilisation d’un bloc Firecrawl dans Flowise.

<img
  className="block"
  src="/images/fc_flowise_block.png"
  alt="Bloc Firecrawl pour Flowise"
/>

<div id="crawling-with-firecrawl-blocks">
  ### Exploration avec les blocs Firecrawl
</div>

1. Connectez-vous à votre compte Firecrawl, récupérez votre clé API, puis saisissez-la dans le bloc.
2. (Facultatif) Connectez Text Splitter.
3. Sélectionnez le mode d’exploration pour explorer les pages situées sous l’URL cible.
4. Saisissez l’URL cible à explorer.
5. Utilisez les documents obtenus dans vos workflows.

<div id="scraping-with-firecrawl-blocks">
  ### Extraction avec les blocs Firecrawl
</div>

1. Connectez-vous à votre compte Firecrawl, récupérez votre clé API, puis saisissez-la dans le bloc.
2. (Facultatif) Connectez Text Splitter.
3. Sélectionnez le mode d’extraction pour récupérer une seule page.
4. Saisissez l’URL cible à extraire.
5. Utilisez les documents générés dans vos workflows.
