<div id="usage">
  ## Utilisation
</div>

Avec Camel AI et Firecrawl, vous pouvez rapidement créer des systèmes multi-agents qui exploitent des données provenant du web.

<div id="using-firecrawl-to-gather-an-entire-website">
  ### Utiliser Firecrawl pour collecter l’intégralité d’un site web
</div>

```python
mock_app = MockFirecrawlApp.return_value
firecrawl = Firecrawl(
    api_key='FC_API_KEY', api_url='https://api.test.com'
)
url = 'https://example.com'
response = [{'markdown': 'Contenu Markdown'}]
mock_app.crawl_url.return_value = respons
result = firecrawl.markdown_crawl(url)
```


<div id="using-firecrawl-to-gather-a-single-page">
  ### Utiliser Firecrawl pour récupérer une page unique
</div>

```python
mock_app = MockFirecrawlApp.return_value
firecrawl = Firecrawl(
    api_key='test_api_key', api_url='https://api.test.com'
)
url = 'https://example.com'
response = 'Contenu récupéré'
mock_app.scrape_url.return_value = response

result = firecrawl.scrape(url)
```
