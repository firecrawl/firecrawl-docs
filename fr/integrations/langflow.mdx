<div id="sync-web-data-in-langflow-workflows">
  ## Synchroniser des données web dans des workflows Langflow
</div>

Firecrawl peut être utilisé dans [Langflow, l’outil de création de workflows IA](https://www.langflow.org/). Cette page explique comment configurer et utiliser un bloc Firecrawl dans Langflow.

<img
  className="block"
  src="/images/fc_langflow_block.png"
  alt="Bloc Firecrawl dans Langflow"
/>

<div id="scraping-with-firecrawl-blocks">
  ### Scraping avec les blocs Firecrawl
</div>

1. Connectez-vous à votre compte Firecrawl, récupérez votre clé API, puis saisissez-la dans le bloc ou passez-la depuis une autre partie du workflow.
2. (Facultatif) Connectez Text Splitter.
3. Sélectionnez le mode de scraping pour récupérer une seule page.
4. Indiquez l’URL cible à scraper ou passez-la depuis une autre partie du workflow.
5. Configurez les options de page et les options d’extraction selon le site et les données recherchées. Vous pouvez aussi les passer depuis une autre partie du workflow.
6. Utilisez les données dans vos workflows.
