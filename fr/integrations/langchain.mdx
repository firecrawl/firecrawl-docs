> Remarque : cette intégration utilise encore la [version v0 de l’API Firecrawl](/fr/v0/introduction). Vous pouvez installer la version 0.0.20 pour le SDK Python ou la 0.0.36 pour le SDK Node.

<div id="installation">
  ## Installation
</div>

```bash
pip install firecrawl-py==0.0.20
```

<div id="usage">
  ## Utilisation
</div>

Vous devez obtenir votre propre clé API. Consultez https://firecrawl.dev

```python
from langchain_community.document_loaders import FireCrawlLoader

loader = FireCrawlLoader(
    api_key="VOTRE_CLE_API", url="https://firecrawl.dev", mode="crawl"
)

docs = loader.load()
```

<div id="modes">
  ### Modes
</div>

Scrape: Scrape une seule URL et renvoie le markdown.
Crawl: Crawl l’URL et toutes les sous-pages accessibles, puis renvoie le markdown pour chacune.

```python
loader = FireCrawlLoader(
    api_key="VOTRE_API_KEY",
    url="https://firecrawl.dev",
    mode="scrape",
)

data = loader.load()
```

<div id="crawler-options">
  ### Options du crawler
</div>

Vous pouvez également transmettre des paramètres au loader. Il s’agit d’un dictionnaire d’options à passer au crawler. Consultez la documentation de l’API Firecrawl pour plus d’informations.

<div id="langchain-js">
  ## LangChain JS
</div>

Pour l’utiliser avec LangChain JS, vous pouvez l’installer via npm :

```bash
npm install @firecrawl/firecrawl-js
```

Ensuite, vous pouvez l’utiliser ainsi :

```typescript
import { FireCrawlLoader } from "langchain/document_loaders/web/firecrawl";

const loader = new FireCrawlLoader({
  url: "https://firecrawl.dev", // L’URL à extraire
  apiKey: process.env.FIRECRAWL_API_KEY, // Optionnel ; par défaut, utilise `FIRECRAWL_API_KEY` dans votre environnement.
  mode: "scrape", // Mode d’exécution du crawler. Utilisez "scrape" pour une seule URL ou "crawl" pour toutes les sous-pages accessibles
  params: {
    // Paramètres optionnels selon la documentation de l’API Firecrawl
    // Pour la documentation de l’API, rendez-vous sur https://docs.firecrawl.dev
  },
});

const docs = await loader.load();
```
