<div id="sync-data-from-websites-for-dify-workflows">
  ## Synchroniser des données depuis des sites web pour les workflows Dify
</div>

Firecrawl peut être utilisé dans [Dify, le concepteur de workflows LLM](https://cloud.dify.ai/). Cette page explique comment extraire des données d’une page web, les convertir en Markdown, puis les importer dans la base de connaissances de Dify via leur intégration Firecrawl.

<div id="configuring-firecrawl">
  ### Configuration de Firecrawl
</div>

Commencez par configurer les identifiants Firecrawl dans la section Source de données de la page Paramètres.

<img
  className="block"
  src="/images/fc_dify_config.avif"
  alt="Configurer la clé Firecrawl"
/>

Connectez-vous à votre compte Firecrawl, récupérez votre clé d’API, puis saisissez-la et enregistrez-la dans Dify.

<img
  className="block"
  src="/images/fc_dify_savekey.png"
  alt="Enregistrer la clé Firecrawl"
/>

<div id="scrape-target-webpage">
  ### Scraper la page web cible
</div>

Passons à la partie intéressante : le scraping et le crawling. Sur la page de création de la base de connaissances, sélectionnez Sync from website et saisissez l’URL à scraper.

<img
  className="block"
  src="/images/fc_dify_webscrape.webp"
  alt="Configuration du scraping"
/>

Les options de configuration incluent : Activer le crawl des sous-pages, Limite de pages à crawler, Profondeur maximale de scraping par page, Chemins exclus, Chemins à inclure uniquement, et Portée de l’extraction de contenu. Une fois la configuration terminée, cliquez sur Run pour prévisualiser les pages analysées.

<img
  className="block"
  src="/images/fc_dify_fcoptions.webp"
  alt="Définir la configuration Firecrawl"
/>

<div id="review-import-results">
  ### Examiner les résultats de l’importation
</div>

Après avoir importé le texte analysé depuis la page web, il est enregistré dans les documents de la base de connaissances. Consultez les résultats de l’importation et cliquez sur Add URL pour continuer à importer de nouvelles pages web.

<img
  className="block"
  src="/images/fc_dify_results.webp"
  alt="Voir les résultats du scraping Firecrawl"
/>
