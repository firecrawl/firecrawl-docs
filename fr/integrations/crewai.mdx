<div id="using-firecrawl-with-crewai">
  ## Utilisation de Firecrawl avec CrewAI
</div>

Firecrawl est intégré à [CrewAI, le framework d’orchestration d’agents IA](https://www.crewai.com/). Cette page présente l’ensemble des outils Firecrawl ajoutés au framework.

<div id="installing-firecrawl-tools-inside-of-crewai">
  ### Installer les outils Firecrawl dans CrewAI
</div>

* Récupérez une clé API depuis votre [tableau de bord firecrawl.dev](https://firecrawl.dev) et définissez-la dans vos variables d’environnement (`FIRECRAWL_API_KEY`).
* Installez le [SDK Firecrawl](https://github.com/mendableai/firecrawl) ainsi que le package `crewai[tools]` :

```
pip install firecrawl-py 'crewai[tools]'
```

<div id="firecrawlcrawlwebsitetool">
  ### OutilFirecrawlCrawlWebsite
</div>

<div id="example">
  #### Exemple
</div>

Utilisez FirecrawlScrapeFromWebsiteTool comme suit pour permettre à votre agent de charger des sites web :

```python
from crewai_tools import FirecrawlCrawlWebsiteTool

tool = FirecrawlCrawlWebsiteTool(url='firecrawl.dev')
```

<div id="arguments">
  #### Arguments
</div>

- `api_key` : Facultatif. Spécifie la clé d’API Firecrawl. La valeur par défaut est la variable d’environnement `FIRECRAWL_API_KEY`.
- `url` : L’URL de base à partir de laquelle démarrer le crawl.
- `page_options` : Facultatif.
  - `onlyMainContent` : Facultatif. Retourne uniquement le contenu principal de la page en excluant les en-têtes, barres de navigation, pieds de page, etc.
  - `includeHtml` : Facultatif. Inclut le contenu HTML brut de la page. Génère une clé `html` dans la réponse.
- `crawler_options` : Facultatif. Options pour contrôler le comportement du crawler.
  - `includes` : Facultatif. Modèles d’URL à inclure dans le crawl.
  - `exclude` : Facultatif. Modèles d’URL à exclure du crawl.
  - `generateImgAltText` : Facultatif. Génère le texte alternatif des images à l’aide de LLMs (nécessite un plan payant).
  - `returnOnlyUrls` : Facultatif. Si `true`, renvoie uniquement les URL sous forme de liste dans le statut du crawl. Remarque : la réponse sera une liste d’URL dans `data`, et non une liste de documents.
  - `maxDepth` : Facultatif. Profondeur maximale du crawl. La profondeur 1 correspond à l’URL de base, la profondeur 2 inclut l’URL de base et ses enfants directs, et ainsi de suite.
  - `mode` : Facultatif. Mode de crawling à utiliser. Le mode `fast` crawle les sites 4× plus vite lorsqu’ils n’ont pas de sitemap, mais peut être moins précis et ne doit pas être utilisé sur les sites fortement rendus en JavaScript.
  - `limit` : Facultatif. Nombre maximal de pages à crawler.
  - `timeout` : Facultatif. Délai d’expiration en millisecondes pour l’opération de crawling.

<div id="firecrawlscrapewebsitetool">
  ### OutilFirecrawlScrapeWebsite
</div>

<div id="example">
  #### Exemple
</div>

Utilisez l’outil FirecrawlScrapeWebsiteTool comme suit pour permettre à votre agent de charger des sites web :

```python
from crewai_tools import FirecrawlScrapeWebsiteTool

tool = FirecrawlScrapeWebsiteTool(url='firecrawl.dev')
```

<div id="arguments">
  #### Arguments
</div>

- `api_key` : Optionnel. Spécifie la clé API Firecrawl. La valeur par défaut est la variable d’environnement `FIRECRAWL_API_KEY`.
- `url` : L’URL à scraper.
- `page_options` : Optionnel.
  - `onlyMainContent` : Optionnel. Retourne uniquement le contenu principal de la page, en excluant les en-têtes, barres de navigation, pieds de page, etc.
  - `includeHtml` : Optionnel. Inclut le contenu HTML brut de la page. Renverra une clé `html` dans la réponse.
- `extractor_options` : Optionnel. Options pour l’extraction basée sur un LLM d’informations structurées à partir du contenu de la page.
  - `mode` : Le mode d’extraction à utiliser, prend actuellement en charge `llm-extraction`.
  - `extractionPrompt` : Optionnel. Un prompt décrivant quelles informations doivent être extraites de la page.
  - `extractionSchema` : Optionnel. Le schéma des données à extraire.
- `timeout` : Optionnel. Délai d’expiration en millisecondes pour la requête.

<div id="firecrawlsearchtool">
  ### Outil de recherche Firecrawl
</div>

<div id="example">
  #### Exemple
</div>

Utilisez FirecrawlSearchTool comme suit pour permettre à votre agent de charger des sites web :

```python
from crewai_tools import FirecrawlSearchTool

tool = FirecrawlSearchTool(query='Qu’est-ce que Firecrawl ?')
```

<div id="arguments">
  #### Arguments
</div>

- `api_key`: Facultatif. Spécifie la clé API Firecrawl. La valeur par défaut est la variable d'environnement `FIRECRAWL_API_KEY`.
- `query`: La chaîne de requête à utiliser pour la recherche.
- `page_options`: Facultatif. Options pour le formatage des résultats.
  - `onlyMainContent`: Facultatif. Renvoie uniquement le contenu principal de la page, en excluant les en-têtes, barres de navigation, pieds de page, etc.
  - `includeHtml`: Facultatif. Inclut le contenu HTML brut de la page. Produira une clé html dans la réponse.
  - `fetchPageContent`: Facultatif. Récupère le contenu complet de la page.
- `search_options`: Facultatif. Options pour contrôler le comportement d'exploration.
  - `limit`: Facultatif. Nombre maximal de pages à explorer.
