## What's New in v2

### All New Formats Work in Batches
- Use custom screenshots, JSON extraction, change tracking across all URLs
- Process PDFs in batches with new parser system
- Apply same format settings to every URL in batch

### Better Batch Processing
- Scrape from different countries/languages for entire batches
- Use stealth, basic, or auto proxy modes across all URLs
- All new scraping options work in batch mode

### Faster and More Reliable
- Better resource management for complex batches
- Improved progress tracking
- Better error handling

Same API as v1, all v2 features work across entire batch.
