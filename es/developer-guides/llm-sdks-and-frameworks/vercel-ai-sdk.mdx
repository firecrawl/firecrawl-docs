Herramientas de Firecrawl para Vercel AI SDK v5. Web scraping, búsqueda, rastreo y extracción de datos para aplicaciones de IA.

<div id="install">
  ## Instalación
</div>

```bash
npm install firecrawl-aisdk ai @ai-sdk/openai
```

Configura las variables de entorno:

```bash
FIRECRAWL_API_KEY=fc-your-key
OPENAI_API_KEY=sk-your-key
```

<Note>
  Estos ejemplos usan OpenAI, pero las herramientas de Firecrawl funcionan con cualquier proveedor del SDK de IA de Vercel, incluidos Anthropic, Google, Mistral y más. Consulta la lista completa de [proveedores compatibles](https://ai-sdk.dev/providers/ai-sdk-providers).
</Note>

<div id="quick-start">
  ## Inicio rápido
</div>

```typescript
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { scrapeTool } from 'firecrawl-aisdk';

const { text } = await generateText({
  model: openai('gpt-5-mini'),
  prompt: 'Extrae datos de https://firecrawl.dev y resume lo que hace',
  tools: { scrape: scrapeTool },
});
```


<div id="available-tools">
  ## Herramientas disponibles
</div>

```typescript
import {
  scrapeTool,         // Extrae una URL individual
  searchTool,         // Busca en la web
  mapTool,            // Descubre URLs en un sitio
  crawlTool,          // Rastrea múltiples páginas
  batchScrapeTool,    // Extrae múltiples URLs
  extractTool,        // Extrae datos estructurados
  pollTool,           // Consulta trabajos asíncronos
  statusTool,         // Verifica el estado del trabajo
  cancelTool,         // Cancela trabajos
} from 'firecrawl-aisdk';
```

<div id="crawl">
  ### Crawl
</div>

Operación asíncrona: incluye `pollTool` para comprobar el estado del trabajo.

```typescript
import { generateText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { crawlTool, pollTool } from 'firecrawl-aisdk';

const { text } = await generateText({
  model: openai('gpt-5-mini'),
  prompt: 'Rastrear https://docs.firecrawl.dev (límite 3 páginas) y resumir',
  tools: { crawl: crawlTool, poll: pollTool },
});

console.log(text);
```

<div id="stream">
  ### Streaming
</div>

```typescript
import { streamText } from 'ai';
import { openai } from '@ai-sdk/openai';
import { scrapeTool } from 'firecrawl-aisdk';

const result = streamText({
  model: openai('gpt-5-mini'),
  prompt: 'Extrae https://firecrawl.dev y explica qué hace',
  tools: { scrape: scrapeTool },
});

for await (const chunk of result.textStream) {
  process.stdout.write(chunk);
}
```
