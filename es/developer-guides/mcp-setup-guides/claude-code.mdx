Añade funciones de scraping y búsqueda web a Claude Code con Firecrawl MCP.

<div id="quick-setup">
  ## Configuración rápida
</div>

<div id="1-get-your-api-key">
  ### 1. Obtén tu clave de API
</div>

Crea una cuenta en [firecrawl.dev/app](https://firecrawl.dev/app) y copia tu clave de API.

<div id="2-add-firecrawl-mcp-server">
  ### 2. Añade el servidor MCP de Firecrawl
</div>

```bash
claude mcp add firecrawl -e FIRECRAWL_API_KEY=tu-clave-de-api -- npx -y firecrawl-mcp
```

Reemplaza `your-api-key` por tu clave real de la API de Firecrawl.

¡Listo! Ahora puedes buscar y extraer datos de la web desde Claude Code.


<div id="quick-demo">
  ## Demostración rápida
</div>

Prueba esto en Claude Code:

**Buscar en la web:**

```
Busca las últimas funciones de Next.js 15
```

**Extraer una página:**

```
Extrae datos de firecrawl.dev y dime qué hace
```

**Obtener la documentación:**

```
Encuentra y extrae la documentación de la API de Stripe sobre intenciones de pago
```

Claude usará automáticamente las herramientas de búsqueda y rastreo de Firecrawl para obtener la información.
