<div id="dify-integration-overview">
  ## Descripción general de la integración con Dify
</div>

Dify es una plataforma de código abierto para desarrollar aplicaciones con LLM. El complemento oficial de Firecrawl permite rastrear y extraer datos de la web directamente en tus flujos de trabajo de IA.

<CardGroup cols={2}>
  <Card title="Aplicaciones de Chatflow y Workflow" icon="diagram-project">
    Crea canalizaciones visuales con nodos de Firecrawl para extraer datos
  </Card>

  <Card title="Aplicaciones de Agentes" icon="robot">
    Da a los agentes de IA la capacidad de extraer datos web en tiempo real y bajo demanda
  </Card>
</CardGroup>

<div id="firecrawl-tools-in-dify">
  ## Herramientas de Firecrawl en Dify
</div>

<AccordionGroup>
  <Accordion title="Scrape" icon="file-code">

Convierte cualquier URL en datos limpios y estructurados. Transforma HTML en bruto en información accionable.

**Casos de uso:** Extrae datos de productos, extrae contenido de artículos, obtén datos estructurados con el modo JSON.

  </Accordion>

  <Accordion title="Crawl" icon="spider">

Realiza rastreos recursivos de sitios web y subdominios para recopilar contenido extenso.

**Casos de uso:** Extracción de contenido de todo el sitio, rastreo de documentación, recopilación de datos multipágina.

  </Accordion>

  <Accordion title="Map" icon="sitemap">

Genera un mapa completo de todas las URL presentes en un sitio web.

**Casos de uso:** Análisis de la estructura del sitio, auditoría SEO, descubrimiento de URL para scraping por lotes.

  </Accordion>

  <Accordion title="Crawl Job" icon="list-check">

Obtén resultados de scraping por ID de tarea o cancela tareas en curso.

**Casos de uso:** Supervisa rastreos de larga duración, gestiona flujos de trabajo de scraping asíncrono, cancela operaciones cuando sea necesario.

  </Accordion>
</AccordionGroup>

<div id="getting-started">
  ## Primeros pasos
</div>

<Steps>
  <Step title="Instalar el plugin de Firecrawl">
    Accede al [Dify Plugin Marketplace](https://marketplace.dify.ai/plugins/langgenius/firecrawl) e instala la herramienta Firecrawl
  </Step>

  <Step title="Obtener la clave de API de Firecrawl">
    Visita [Firecrawl API Keys](https://www.firecrawl.dev/app/api-keys) y crea una nueva clave de API
  </Step>

  <Step title="Autorizar en Dify">
    Ve a **Plugins > Firecrawl > To Authorize** e introduce tu clave de API
  </Step>

  <Step title="Agregar a tu flujo de trabajo">
    Arrastra las herramientas de Firecrawl a tu Chatflow, Workflow o aplicación Agent
  </Step>

  <Step title="Configurar y probar">
    Configura los parámetros y prueba tu flujo de trabajo
  </Step>
</Steps>

<div id="usage-patterns">
  ## Patrones de uso
</div>

<Tabs>
  <Tab title="Chatflow Apps">
**Integración visual de pipelines**

1. Agrega el nodo de Firecrawl a tu pipeline
2. Selecciona la acción (Map, Crawl, Scrape)
3. Define las variables de entrada
4. Ejecuta el pipeline secuencialmente

**Flujo de ejemplo:**
```
Entrada del usuario → Firecrawl (Scrape) → Procesamiento con LLM → Respuesta
```
  </Tab>

  <Tab title="Workflow Apps">
**Procesamiento de datos automatizado**

Crea flujos de trabajo de varios pasos con:
- Scraping programado
- Transformación de datos
- Almacenamiento en base de datos
- Notificaciones

**Flujo de ejemplo:**
```
Disparador programado → Firecrawl (Crawl) → Procesamiento de datos → Almacenamiento
```
  </Tab>

  <Tab title="Agent Apps">
**Acceso web con IA**

Da a los agentes capacidades de web scraping en tiempo real:

1. Agrega la herramienta de Firecrawl al agente
2. El agente decide de forma autónoma cuándo hacer scraping
3. El LLM analiza el contenido extraído
4. El agente ofrece respuestas fundamentadas

**Caso de uso:** Agentes de soporte al cliente que consultan documentación actualizada
  </Tab>
</Tabs>

<div id="firecrawl-actions">
  ## Acciones de Firecrawl
</div>

| Herramienta | Descripción | Ideal para |
|------|-------------|----------|
| **Scrape** | Extracción de datos de una sola página | Captura rápida de contenido |
