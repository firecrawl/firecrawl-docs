<div id="firecrawl-actions">
  ## Acciones de Firecrawl
</div>

| Acción | Caso de uso |
|--------|-------------|
| **Scrape URL** | Captura rápida de datos de una sola página |
| **Crawl Website** | Rastreo completo del sitio con múltiples páginas |
| **Extract Structured Data** | Extracción con IA mediante esquemas personalizados |
| **Search Web** | Automatización de investigación con búsqueda + scraping |
| **Map Website** | Análisis SEO y mapeo de la estructura del sitio |

    - Usa filtros para evitar ejecuciones innecesarias
