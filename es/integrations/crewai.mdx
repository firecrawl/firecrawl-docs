<div id="using-firecrawl-with-crewai">
  ## Uso de Firecrawl con CrewAI
</div>

Firecrawl está integrado con [CrewAI, el framework para orquestar agentes de IA](https://www.crewai.com/). Esta página presenta todas las herramientas de Firecrawl incorporadas al framework.

<div id="installing-firecrawl-tools-inside-of-crewai">
  ### Instalar Firecrawl Tools dentro de CrewAI
</div>

* Obtén una clave de API desde tu [panel de firecrawl.dev](https://firecrawl.dev) y configúrala como variable de entorno (`FIRECRAWL_API_KEY`).
* Instala el [SDK de Firecrawl](https://github.com/mendableai/firecrawl) junto con el paquete `crewai[tools]`:

```
pip install firecrawl-py 'crewai[tools]'
```

<div id="firecrawlcrawlwebsitetool">
  ### Herramienta FirecrawlCrawlWebsite
</div>

<div id="example">
  #### Ejemplo
</div>

Utiliza FirecrawlScrapeFromWebsiteTool de la siguiente manera para que tu agente cargue sitios web:

```python
from crewai_tools import FirecrawlCrawlWebsiteTool

tool = FirecrawlCrawlWebsiteTool(url='firecrawl.dev')
```


<div id="arguments">
  #### Argumentos
</div>

- `api_key`: Opcional. Especifica la clave de API de Firecrawl. De forma predeterminada, se toma de la variable de entorno `FIRECRAWL_API_KEY`.
- `url`: La URL base desde la que comenzar el rastreo.
- `page_options`: Opcional.
  - `onlyMainContent`: Opcional. Devuelve solo el contenido principal de la página, excluyendo encabezados, navegación, pies de página, etc.
  - `includeHtml`: Opcional. Incluye el HTML sin procesar de la página. Agregará una clave html en la respuesta.
- `crawler_options`: Opcional. Opciones para controlar el comportamiento del rastreo.
  - `includes`: Opcional. Patrones de URL a incluir en el rastreo.
  - `exclude`: Opcional. Patrones de URL a excluir del rastreo.
  - `generateImgAltText`: Opcional. Genera texto alternativo para las imágenes usando LLMs (requiere un plan de pago).
  - `returnOnlyUrls`: Opcional. Si es true, devuelve únicamente las URLs como una lista en el estado del rastreo. Nota: la respuesta será una lista de URLs dentro de data, no una lista de documentos.
  - `maxDepth`: Opcional. Profundidad máxima del rastreo. La profundidad 1 es la URL base, la profundidad 2 incluye la URL base y sus hijos directos, y así sucesivamente.
  - `mode`: Opcional. Modo de rastreo a usar. El modo Fast rastrea 4 veces más rápido en sitios sin sitemap, pero puede ser menos preciso y no debería usarse en sitios con un renderizado intensivo de JavaScript.
  - `limit`: Opcional. Número máximo de páginas a rastrear.
  - `timeout`: Opcional. Tiempo de espera en milisegundos para la operación de rastreo.

<div id="firecrawlscrapewebsitetool">
  ### Herramienta FirecrawlScrapeWebsite
</div>

<div id="example">
  #### Ejemplo
</div>

Usa FirecrawlScrapeWebsiteTool de la siguiente manera para permitir que tu agente cargue sitios web:

```python
from crewai_tools import FirecrawlScrapeWebsiteTool

tool = FirecrawlScrapeWebsiteTool(url='firecrawl.dev')
```


<div id="arguments">
  #### Argumentos
</div>

- `api_key`: Opcional. Especifica la clave de API de Firecrawl. El valor predeterminado es la variable de entorno `FIRECRAWL_API_KEY`.
- `url`: La URL que se va a extraer.
- `page_options`: Opcional.
  - `onlyMainContent`: Opcional. Devuelve solo el contenido principal de la página, excluyendo encabezados, navegación, pies de página, etc.
  - `includeHtml`: Opcional. Incluye el HTML en bruto de la página. Generará una clave html en la respuesta.
- `extractor_options`: Opcional. Opciones para la extracción basada en LLM de información estructurada del contenido de la página.
  - `mode`: El modo de extracción a usar; actualmente admite 'llm-extraction'.
  - `extractionPrompt`: Opcional. Un prompt que describe qué información extraer de la página.
  - `extractionSchema`: Opcional. El esquema de los datos que se van a extraer.
- `timeout`: Opcional. Tiempo de espera en milisegundos para la solicitud.

<div id="firecrawlsearchtool">
  ### FirecrawlSearchTool
</div>

<div id="example">
  #### Ejemplo
</div>

Utiliza FirecrawlSearchTool de la siguiente manera para que tu agente pueda cargar sitios web:

```python
from crewai_tools import FirecrawlSearchTool

tool = FirecrawlSearchTool(query='¿Qué es Firecrawl?')
```


<div id="arguments">
  #### Argumentos
</div>

- `api_key`: Opcional. Especifica la clave de API de Firecrawl. De forma predeterminada, se usa la variable de entorno `FIRECRAWL_API_KEY`.
- `query`: La cadena de consulta que se utilizará para la búsqueda.
- `page_options`: Opcional. Opciones para el formato de los resultados.
  - `onlyMainContent`: Opcional. Devuelve solo el contenido principal de la página, excluyendo encabezados, barras de navegación, pies de página, etc.
  - `includeHtml`: Opcional. Incluye el HTML sin procesar de la página. Devolverá una clave html en la respuesta.
  - `fetchPageContent`: Opcional. Obtiene el contenido completo de la página.
- `search_options`: Opcional. Opciones para controlar el comportamiento del rastreo.
  - `limit`: Opcional. Número máximo de páginas a rastrear.
