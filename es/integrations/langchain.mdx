> Nota: esta integración aún utiliza la [versión v0 de la API de Firecrawl](/es/v0/introduction). Puedes instalar la versión 0.0.20 del SDK de Python o la 0.0.36 del SDK de Node.

<div id="installation">
  ## Instalación
</div>

```bash
pip install firecrawl-py==0.0.20
```

<div id="usage">
  ## Uso
</div>

Necesitas tu propia clave de API. Consulta https://firecrawl.dev

```python
from langchain_community.document_loaders import FireCrawlLoader

loader = FireCrawlLoader(
    api_key="TU_CLAVE_API", url="https://firecrawl.dev", mode="crawl"
)

docs = loader.load()
```

<div id="modes">
  ### Modos
</div>

Scrape: Extrae una única URL y devuelve el markdown.
Crawl: Rastrea la URL y todas las subpáginas accesibles y devuelve el markdown de cada una.

```python
loader = FireCrawlLoader(
    api_key="TU_API_KEY",
    url="https://firecrawl.dev",
    mode="scrape",
)

data = loader.load()
```

<div id="crawler-options">
  ### Opciones del crawler
</div>

También puedes pasar parámetros al loader. Es un diccionario de opciones que se envían al crawler. Consulta la documentación de la API de Firecrawl para obtener más información.

<div id="langchain-js">
  ## Langchain JS
</div>

Para usarlo en Langchain JS, puedes instalarlo con npm:

```bash
npm install @mendableai/firecrawl-js
```

Luego puedes usarlo así:

```typescript
import { FireCrawlLoader } from "langchain/document_loaders/web/firecrawl";

const loader = new FireCrawlLoader({
  url: "https://firecrawl.dev", // La URL que se va a extraer
  apiKey: process.env.FIRECRAWL_API_KEY, // Opcional; por defecto usa `FIRECRAWL_API_KEY` en tu entorno.
  mode: "scrape", // El modo en el que se ejecuta el rastreador. Puede ser "scrape" para URLs individuales o "crawl" para todas las subpáginas accesibles
  params: {
    // parámetros opcionales según la documentación de la API de Firecrawl
    // Para la documentación de la API, visita https://docs.firecrawl.dev
  },
});

const docs = await loader.load();
```
