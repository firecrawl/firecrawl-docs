<div id="sync-data-from-websites-for-dify-workflows">
  ## Sincroniza datos desde sitios web para flujos de trabajo de Dify
</div>

Firecrawl puede usarse dentro de [Dify, el constructor de flujos de trabajo con LLM](https://cloud.dify.ai/). Esta página explica cómo extraer datos de una página web, convertirlos a Markdown e importarlos en la base de conocimiento de Dify usando su integración con Firecrawl.

<div id="configuring-firecrawl">
  ### Configurar Firecrawl
</div>

Primero, configura las credenciales de Firecrawl en la sección Data Source de la página de Settings.

<img
  className="block"
  src="/images/fc_dify_config.avif"
  alt="Configurar la clave de Firecrawl"
/>

Inicia sesión en tu cuenta de Firecrawl, obtén tu API Key y luego introdúcela y guárdala en Dify.

<img
  className="block"
  src="/images/fc_dify_savekey.png"
  alt="Guardar la clave de Firecrawl"
/>

<div id="scrape-target-webpage">
  ### Extraer la página de destino
</div>

Ahora viene la parte divertida: scraping y rastreo. En la página de creación de la base de conocimientos, selecciona Sync from website e introduce la URL que deseas extraer.

<img
  className="block"
  src="/images/fc_dify_webscrape.webp"
  alt="Configuración de scraping"
/>

Las opciones de configuración incluyen: rastrear subpáginas o no, límite de rastreo de páginas, profundidad máxima de scraping, rutas excluidas, rutas incluidas únicamente y alcance de extracción de contenido. Cuando termines la configuración, haz clic en Run para previsualizar las páginas analizadas.

<img
  className="block"
  src="/images/fc_dify_fcoptions.webp"
  alt="Configurar Firecrawl"
/>

<div id="review-import-results">
  ### Revisa los resultados de la importación
</div>

Después de importar el texto procesado desde la página web, este se guarda en los documentos de la base de conocimientos. Revisa los resultados de la importación y haz clic en Add URL para seguir importando nuevas páginas web.

<img
  className="block"
  src="/images/fc_dify_results.webp"
  alt="Ver los resultados del scraping de Firecrawl"
/>
