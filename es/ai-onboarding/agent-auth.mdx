**Predeterminado: ruta A.** Si no puedes generar un WorkOS ID-JAG —o no estás seguro— usa el onboarding con intervención humana mediante [`SKILL.md`](https://www.firecrawl.dev/agent-onboarding/SKILL.md) (consulta [Obtener credenciales](/es/ai-onboarding#get-credentials)).

Esta página cubre **solo la ruta B**: el registro de WorkOS ID-JAG para Firecrawl. No es la especificación de ejecución; sigue el documento canónico [`auth.md`](https://www.firecrawl.dev/auth.md) para cada paso, requisito de claims y código de error.
