<div id="creating-a-custom-snippet">
  ## Crear un snippet personalizado
</div>

**Requisito previo**: Debes crear el archivo del snippet en el directorio `snippets`.

<Note>
  Cualquier página en el directorio `snippets` se tratará como un snippet y no
  se renderizará como una página independiente. Si quieres crear una página
  independiente a partir del snippet, importa el snippet en otro archivo y úsalo como
  componente.
</Note>

<div id="default-export">
  ### Exportación predeterminada
</div>

1. Agrega contenido al archivo del snippet que quieras reutilizar en múltiples
   ubicaciones. Opcionalmente, puedes agregar variables que se puedan completar mediante props
   cuando importes el snippet.

```mdx fragmentos/mi-fragmento.mdx
¡Hola, mundo! Este es el contenido que quiero reutilizar en varias páginas. Mi palabra clave del
día es {word}.
```

<Warning>
  El contenido que deseas reutilizar debe estar dentro del directorio `snippets`
  para que la importación funcione.
</Warning>

2. Importa el snippet en tu archivo de destino.

```mdx archivo-de-destino.mdx
---
title: Mi título
description: Mi descripción
---

import MySnippet from '/snippets/path/to/my-snippet.mdx';

## Encabezado

Lorem impsum dolor sit amet.

<MySnippet word="bananas" />
```

<div id="reusable-variables">
  ### Variables reutilizables
</div>

1. Exporta una variable desde tu archivo de snippet:

```mdx fragmentos/ruta/a/variables-personalizadas.mdx
export const myName = 'mi nombre';

export const myObject = { fruit: 'fresas' };
```

2. Importa el snippet desde tu archivo de destino y utiliza la variable:

```mdx archivo-destino.mdx
---
title: Mi título
description: Mi descripción
---

import { myName, myObject } from '/snippets/path/to/custom-variables.mdx';

Hola, me llamo {myName} y me gusta la {myObject.fruit}.
```

<div id="reusable-components">
  ### Componentes reutilizables
</div>

1. Dentro de tu archivo de snippet, crea un componente que reciba props exportando
   el componente como una función flecha.

```mdx snippets/componente-personalizado.mdx
export const MyComponent = ({ title }) => (
  <div>
    <h1>{title}</h1>
    <p>... contenido del snippet ...</p>
  </div>
);
```

<Warning>
  MDX no se compila dentro del cuerpo de una función flecha. Mantén la sintaxis
  HTML cuando sea posible o usa una exportación predeterminada si necesitas usar MDX.
</Warning>

2. Importa el snippet en tu archivo de destino y pásale las props

```mdx archivo-destino.mdx
---
title: Mi título
description: Mi descripción
---

import { MyComponent } from '/snippets/custom-component.mdx';

Lorem ipsum dolor sit amet.

<MyComponent title={'Título personalizado'} />
```
