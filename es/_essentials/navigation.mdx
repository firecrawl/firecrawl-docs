El menú de navegación es la lista de enlaces presente en cada sitio web.

Es probable que tengas que actualizar `mint.json` cada vez que agregues una nueva página. Las páginas no se muestran automáticamente.

<div id="navigation-syntax">
  ## Sintaxis de navegación
</div>

Nuestra sintaxis de navegación es recursiva, lo que significa que puedes crear grupos de navegación anidados. No es necesario incluir `.mdx` en los nombres de las páginas.

<CodeGroup>

```json Regular Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": ["quickstart"]
    }
]
```

```json Nested Navigation
"navigation": [
    {
        "group": "Getting Started",
        "pages": [
            "quickstart",
            {
                "group": "Nested Reference Pages",
                "pages": ["nested-reference-page"]
            }
        ]
    }
]
```

</CodeGroup>

<div id="folders">
  ## Carpetas
</div>

Simplemente coloca tus archivos MDX en carpetas y actualiza las rutas en `mint.json`.

Por ejemplo, para tener una página en `https://yoursite.com/your-folder/your-page`, crearías una carpeta llamada `your-folder` que contenga un archivo MDX llamado `your-page.mdx`.

<Warning>
  No puedes usar `api` como nombre de carpeta a menos que la anides dentro de otra. Mintlify usa Next.js, que reserva la carpeta de primer nivel `api` para llamadas internas del servidor. Un nombre de carpeta como `api-reference` sería aceptado.
</Warning>

```json Navegación con carpeta
"navigation": [
    {
        "group": "Nombre del grupo",
        "pages": ["your-folder/your-page"]
    }
]
```

<div id="hidden-pages">
  ## Páginas ocultas
</div>

Los archivos MDX que no estén incluidos en `mint.json` no aparecerán en la barra lateral, pero se pueden acceder a ellos desde la barra de búsqueda o enlazando directamente.
