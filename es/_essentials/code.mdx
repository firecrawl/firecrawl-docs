<div id="inline-code">
  ### Código en línea
</div>

Para indicar que una `palabra` o `frase` es código, enciérrala entre comillas invertidas (`).

```
Para indicar una `palabra` o `frase` como código, enciérrala entre acentos graves (`).
```

<div id="code-block">
  ### Bloque de código
</div>

Usa [bloques de código delimitados](https://www.markdownguide.org/extended-syntax/#fenced-code-blocks) encerrando el código entre tres comillas invertidas (`) y, después de las comillas iniciales, indica el lenguaje de programación de tu fragmento para obtener resaltado de sintaxis. Opcionalmente, también puedes escribir el nombre de tu fragmento de código después del lenguaje de programación.

```java HelloWorld.java
class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
```
