  ## Imagen
</div>

<div id="using-markdown">
  ### Uso de Markdown
</div>

La [sintaxis de Markdown](https://www.markdownguide.org/basic-syntax/#images) te permite añadir imágenes con el siguiente código

```md
![Título](/path/image.jpg)
```

Ten en cuenta que el tamaño del archivo de imagen debe ser inferior a 5 MB. De lo contrario, recomendamos alojarla en un servicio como [Cloudinary](https://cloudinary.com/) o [S3](https://aws.amazon.com/s3/). Luego puedes usar esa URL e insertarla.

<div id="using-embeds">
  ### Uso de embeds
</div>

Para tener más flexibilidad con las imágenes, también puedes usar [embeds](/es/writing-content/embed) para añadir imágenes

```html
<img height="200" src="/path/image.jpg" />
```

<div id="iframes">
  ### iFrames
</div>

Carga otra página HTML dentro del documento. Se usa más comúnmente para incrustar vídeos.

```html
<iframe src="https://www.youtube.com/embed/4KzFe50RQkQ"> </iframe>
```
