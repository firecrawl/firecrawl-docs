<div id="setup-your-development">
  ## Configura tu entorno de desarrollo
</div>

Aprende a actualizar tu documentación localmente y a publicarla.

<div id="edit-and-preview">
  ### Edita y vista previa
</div>

<AccordionGroup>
  <Accordion icon="github" title="Clona tu documentación localmente">
    Durante el proceso de incorporación, creamos un repositorio en tu GitHub con
    el contenido de tu documentación. Puedes encontrar este repositorio en nuestro
    [panel](https://dashboard.mintlify.com). Para clonar el repositorio
    localmente, sigue estas
    [instrucciones](https://docs.github.com/en/repositories/creating-and-managing-repositories/cloning-a-repository)
    en tu terminal.
  </Accordion>

  <Accordion icon="rectangle-terminal" title="Vista previa de los cambios">
    La vista previa te ayuda a asegurarte de que tus cambios se vean como esperas. Creamos una
    interfaz de línea de comandos para renderizar estos cambios localmente. 1. Instala la
    [Mintlify CLI](https://www.npmjs.com/package/mintlify) para obtener una vista previa de los
    cambios de la documentación localmente con este comando: `npm i -g mintlify`
    2. Ejecuta el siguiente comando en la raíz de tu documentación (donde está
    `mint.json`): `mintlify dev`
  </Accordion>
</AccordionGroup>

<div id="deploy-your-changes">
  ### Despliega tus cambios
</div>

<AccordionGroup>
  <Accordion icon="message-bot" title="Instala nuestra app de GitHub">
    Nuestra app de GitHub despliega automáticamente tus cambios en tu sitio de documentación, por lo que
    no necesitas gestionar los despliegues tú mismo. Puedes encontrar el enlace para instalarla en
    tu [panel](https://dashboard.mintlify.com). Una vez que el bot se haya
    instalado correctamente, debería aparecer una marca de verificación junto al hash del commit
    del repositorio.
  </Accordion>

  <Accordion icon="rocket" title="Sube (push) tus cambios">
    [Haz commit y realiza push de tus cambios a
    Git](https://docs.github.com/en/get-started/using-git/pushing-commits-to-a-remote-repository#about-git-push)
    para que se reflejen en tu sitio de documentación. Si haces push y no ves que
    la app de GitHub haya desplegado correctamente tus cambios, también puedes actualizar
    manualmente tu documentación desde nuestro [panel](https://dashboard.mintlify.com).
  </Accordion>
</AccordionGroup>
