<div id="scraping-with-firecrawl">
  ## Extracción con Firecrawl
</div>

Firecrawl convierte páginas web en markdown, ideal para aplicaciones con LLM. Aquí tienes por qué:

1. **Complejidades gestionadas:**
   Gestiona proxies, caché, Límites de tasa y contenido bloqueado por JavaScript para una extracción fluida.

2. **Contenido dinámico:**
   Recopila datos de sitios web renderizados con JavaScript, PDF, imágenes, etc.

3. **Conversión a markdown o datos estructurados:**
   Convierte los datos recopilados a markdown limpio o a salida estructurada, perfecto para el procesamiento con LLM u otras tareas.

Para más detalles, consulta la [referencia del punto de conexión /scrape](https://docs.firecrawl.dev/api-reference/endpoint/scrape).

<div id="usage">
  ### Uso
</div>

<CodeGroup>

```python Python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

content = app.scrape_url("https://mendable.ai")
```

```JavaScript JavaScript
import { FirecrawlApp } from 'firecrawl-js';

const app = new FirecrawlApp({ apiKey: 'YOUR_API_KEY' });

const content = await app.scrapeUrl('https://mendable.ai');
```

```go Go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
    log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
  }

  content, err := app.ScrapeURL("mendable.ai", nil)
  if err != nil {
    log.Fatalf("Failed to scrape URL: %v", err)
  }
  fmt.Println(content)
}
```

```rust Rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    // Inicializa FirecrawlApp con la clave de API
    let api_key = "YOUR_API_KEY";
    let api_url = "https://api.firecrawl.dev";
    let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

    // Extrae la URL
    let scrape_result = app.scrape_url("https://mendable.ai", None).await;

    // Imprime el resultado de la extracción
  match scrape_result {
    Ok(data) => println!("Resultado de la extracción:\n{}", data["markdown"]),
    Err(e) => eprintln!("La extracción falló: {}", e),
  }
}
```

```bash cURL
curl -X POST https://api.firecrawl.dev/v0/scrape \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "url": "https://mendable.ai"
    }'
```
