---
title: "Search (Beta)"
openapi: 'v0-openapi POST /search'
---

El punto de conexión /search combina una API de búsqueda con el poder de Firecrawl para ofrecer una experiencia de búsqueda potente para cualquier consulta.

Busca automáticamente en la web la consulta y devuelve los resultados más relevantes de las páginas principales en formato markdown. La ventaja de este punto de conexión es que realmente rastrea y extrae cada sitio web de los resultados principales, por lo que siempre obtienes el contenido completo.

Este punto de conexión está actualmente en beta y puede cambiar.
