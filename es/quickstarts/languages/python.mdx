<div id="overview">
  ## Descripción general
</div>

Usa el SDK oficial de Firecrawl para Python para recuperar una URL y devolver una salida estructurada.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de la API de Firecrawl
* Python 3.8+
* pip

<div id="install">
  ## Instalación
</div>

```bash
pip install firecrawl-py
```

<div id="configure-your-api-key">
  ## Configura tu clave de API
</div>

Guarda tu clave de API en una variable de entorno del servidor.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Haz tu primera petición
</div>

```python
import os
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

doc = client.scrape("https://example.com", formats=["markdown"])
print(doc.markdown)
```

<div id="complete-example">
  ## Ejemplo completo
</div>

```python
import os
from firecrawl import Firecrawl

def main() -> None:
    client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])
    doc = client.scrape("https://example.com", formats=["markdown", "html"])
    print(doc.metadata.title if doc.metadata else "")
    print(doc.markdown)

if __name__ == "__main__":
    main()
```
