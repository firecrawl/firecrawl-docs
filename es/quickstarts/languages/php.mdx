<div id="overview">
  ## Descripción general
</div>

Usa la extensión cURL de PHP para enviar una solicitud de scraping a Firecrawl.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* PHP 8.1+ con cURL activado

<div id="install">
  ## Instalación
</div>

No se requieren paquetes adicionales; este ejemplo usa la extensión cURL incluida.

<div id="configure-your-api-key">
  ## Configura tu clave de API
</div>

Guarda tu clave de API en una variable de entorno del servidor.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Realiza tu primera solicitud
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
curl_close($ch);

echo $response;
```

<div id="complete-example">
  ## Ejemplo completo
</div>

```php
<?php

$apiKey = getenv('FIRECRAWL_API_KEY');
$payload = json_encode([
    'url' => 'https://example.com',
    'formats' => ['markdown', 'html'],
]);

$ch = curl_init('https://api.firecrawl.dev/v2/scrape');
curl_setopt_array($ch, [
    CURLOPT_RETURNTRANSFER => true,
    CURLOPT_POST => true,
    CURLOPT_HTTPHEADER => [
        'Authorization: Bearer ' . $apiKey,
        'Content-Type: application/json',
    ],
    CURLOPT_POSTFIELDS => $payload,
]);

$response = curl_exec($ch);
if ($response === false) {
    throw new RuntimeException(curl_error($ch));
}
curl_close($ch);

echo $response;
```
