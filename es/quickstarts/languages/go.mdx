<div id="overview">
  ## Descripción general
</div>

Usa la biblioteca estándar de Go para realizar una solicitud de scraping a Firecrawl mediante HTTP.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Go 1.20+

<div id="install">
  ## Instalación
</div>

No se requieren paquetes adicionales; este ejemplo usa la biblioteca estándar.

<div id="configure-your-api-key">
  ## Configura tu clave de API
</div>

Define tu clave de API como una variable de entorno antes de ejecutar tu programa.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Realiza tu primera solicitud
</div>

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
    "os"
)

func main() {
    payload := map[string]any{
        "url": "https://example.com",
        "formats": []string{"markdown"},
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://api.firecrawl.dev/v2/scrape", bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("FIRECRAWL_API_KEY"))
    req.Header.Set("Content-Type", "application/json")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    var result map[string]any
    _ = json.NewDecoder(resp.Body).Decode(&result)
    fmt.Println(result)
}
```

<div id="complete-example">
  ## Ejemplo completo
</div>

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "io"
    "net/http"
    "os"
)

func main() {
    payload := map[string]any{
        "url": "https://example.com",
        "formats": []string{"markdown", "html"},
    }

    body, _ := json.Marshal(payload)
    req, _ := http.NewRequest("POST", "https://api.firecrawl.dev/v2/scrape", bytes.NewReader(body))
    req.Header.Set("Authorization", "Bearer "+os.Getenv("FIRECRAWL_API_KEY"))
    req.Header.Set("Content-Type", "application/json")

    resp, err := http.DefaultClient.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    raw, _ := io.ReadAll(resp.Body)
    fmt.Println(string(raw))
}
```
