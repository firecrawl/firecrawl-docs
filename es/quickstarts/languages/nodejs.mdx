<div id="overview">
  ## Descripción general
</div>

Usa el SDK oficial de Firecrawl para Node.js para convertir una URL en contenido limpio y listo para LLM.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Node.js 18+
* npm, pnpm o yarn

<div id="install">
  ## Instalación
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-your-api-key">
  ## Configura tu clave de API
</div>

Guarda tu clave de API en una variable de entorno a la que solo tu servidor pueda acceder.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Realiza tu primera solicitud
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

(async () => {
  const doc = await app.scrape('https://example.com', { formats: ['markdown'] });
  console.log(doc.markdown);
})();
```

<div id="complete-example">
  ## Ejemplo completo
</div>

```js
import Firecrawl from '@mendable/firecrawl-js';

async function main() {
  const app = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });
  const doc = await app.scrape('https://example.com', { formats: ['markdown', 'html'] });
  console.log(doc.metadata?.title);
  console.log(doc.markdown);
}

main().catch((error) => {
  console.error(error);
  process.exit(1);
});
```
