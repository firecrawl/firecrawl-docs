<div id="overview">
  ## Descripción general
</div>

Usa el SDK de Firecrawl para Rust para llamar a la API v2 desde código Rust asíncrono.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Toolchain de Rust con tokio

<div id="install">
  ## Instalación
</div>

```bash
cargo add firecrawl tokio --features tokio/full
```

<div id="configure-your-api-key">
  ## Configura tu clave de API
</div>

Define tu clave de API como una variable de entorno.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="make-your-first-request">
  ## Realiza tu primera solicitud
</div>

```rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    let api_key = std::env::var("FIRECRAWL_API_KEY").expect("Falta la clave de API");
    let app = FirecrawlApp::new(api_key).expect("Error al inicializar Firecrawl");

    let doc = app
        .scrape_url("https://example.com", None)
        .await
        .expect("Error en el scraping");

    println!("{}", doc.markdown.unwrap_or_default());
}
```

<div id="complete-example">
  ## Ejemplo completo
</div>

```rust
use firecrawl::FirecrawlApp;

#[tokio::main]
async fn main() {
    let api_key = std::env::var("FIRECRAWL_API_KEY").expect("Missing clave de API");
    let app = FirecrawlApp::new(api_key).expect("Failed to init Firecrawl");

    let doc = app
        .scrape_url("https://example.com", None)
        .await
        .expect("scraping failed");

    if let Some(metadata) = doc.metadata {
        println!("Title: {}", metadata.title.unwrap_or_default());
    }
    println!("{}", doc.markdown.unwrap_or_default());
}
```
