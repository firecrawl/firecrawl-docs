<div id="overview">
  ## Descripción general
</div>

Usa el SDK de Python de Firecrawl en un endpoint de un servidor Flask.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación Flask
* Python 3.8+

<div id="install">
  ## Instalación
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configurar el entorno
</div>

Configura tu clave de API en el entorno antes de iniciar Flask.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Añade una ruta en el servidor
</div>

app.py

```python
import os
from flask import Flask, request, jsonify
from firecrawl import Firecrawl

app = Flask(__name__)
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def firecrawl():
    url = request.json["url"]
    doc = client.scrape(url, formats=["markdown"])
    return jsonify(markdown=doc.markdown)
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```python
import requests

response = requests.post(
    "http://localhost:5000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Notas
</div>

* Mantén el cliente de Firecrawl inicializado una sola vez y reutilízalo entre solicitudes.
