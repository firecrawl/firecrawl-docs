<div id="overview">
  ## Descripción general
</div>

Crea una ruta de acción en Remix para invocar Firecrawl desde el servidor.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación Remix
* Node.js 18+

<div id="install">
  ## Instalación
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-environment">
  ## Configura el entorno
</div>

Guarda la clave de API en `.env` o en tu entorno de despliegue.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta de servidor
</div>

app/routes/api.firecrawl.ts

```ts
import { json } from '@remix-run/node';
import Firecrawl from '@mendable/firecrawl-js';

export const action = async ({ request }) => {
  const { url } = await request.json();
  const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

  const doc = await client.scrape(url, { formats: ['markdown'] });
  return json({ markdown: doc.markdown });
};
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

<div id="notes">
  ## Notas
</div>

* Usa las actions o loaders de Remix para mantener las llamadas a Firecrawl en el servidor.
