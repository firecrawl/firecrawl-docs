<div id="overview">
  ## Descripción general
</div>

Usa una Edge Function basada en Deno para llamar a la API HTTP de Firecrawl.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de la API de Firecrawl
* Un proyecto de Supabase
* La CLI de Supabase

<div id="install">
  ## Instalación
</div>

No se requieren paquetes adicionales; las Edge Functions de Supabase se ejecutan en Deno con fetch.

<div id="configure-environment">
  ## Configurar el entorno
</div>

Configura la clave de API como un secreto en Supabase.

```bash
supabase secrets set FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta del servidor
</div>

supabase/functions/firecrawl/index.ts

```ts
Deno.serve(async (req) => {
  const { url } = await req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${Deno.env.get('FIRECRAWL_API_KEY')}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
});
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```js
const response = await fetch('https://project.supabase.co/functions/v1/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Las Edge Functions usan Deno, así que evita los paquetes solo para Node.
