<div id="overview">
  ## Descripción general
</div>

Usa el cliente HTTP de Laravel para llamar a la API de Firecrawl.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación Laravel
* PHP 8.1+

<div id="install">
  ## Instalación
</div>

No se requieren paquetes adicionales; Laravel incluye un cliente HTTP.

<div id="configure-environment">
  ## Configurar el entorno
</div>

Añade la clave de API a tu archivo `.env`.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Agrega una ruta al servidor
</div>

routes/api.php

```php
<?php

use Illuminate\Support\Facades\Http;
use Illuminate\Http\Request;
use Illuminate\Support\Facades\Route;

Route::post('/firecrawl', function (Request $request) {
    $response = Http::withToken(env('FIRECRAWL_API_KEY'))
        ->post('https://api.firecrawl.dev/v2/scrape', [
            'url' => $request->input('url'),
            'formats' => ['markdown'],
        ]);

    return $response->json();
});
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```bash
curl -X POST http://localhost:8000/api/firecrawl               -H 'Content-Type: application/json'               -d '{"url": "https://example.com"}'
```

<div id="notes">
  ## Notas
</div>

* Mantén la clave de API solo en variables de entorno del servidor.
