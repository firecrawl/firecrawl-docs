<div id="overview">
  ## Descripción general
</div>

Usa una ruta de la API de Astro para llamar a Firecrawl desde el servidor.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Un proyecto de Astro
* Node.js 18+

<div id="install">
  ## Instalación
</div>

No se requieren paquetes adicionales; usa `fetch` en la ruta de la API.

<div id="configure-environment">
  ## Configura el entorno
</div>

Añade la clave de API a `.env` y léela desde el código del servidor.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta del servidor
</div>

src/pages/api/firecrawl.ts

```ts
import type { APIRoute } from 'astro';

export const POST: APIRoute = async ({ request }) => {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${import.meta.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  return new Response(await response.text(), {
    headers: { 'Content-Type': 'application/json' },
  });
};
```

<div id="call-it-from-your-app">
  ## Úsalo desde tu aplicación
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Usa rutas del servidor para las llamadas a Firecrawl; evita las solicitudes desde el cliente con la clave de API.
