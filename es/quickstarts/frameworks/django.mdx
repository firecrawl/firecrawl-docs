<div id="overview">
  ## Descripción general
</div>

Usa el SDK de Python de Firecrawl en una vista de Django para servir de proxy a las solicitudes de scraping.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación de Django
* Python 3.8+

<div id="install">
  ## Instalación
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configura el entorno
</div>

Establece la clave de API en tu entorno de Django.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Añade una ruta en el servidor
</div>

app/views.py

```python
import json
import os
from django.http import JsonResponse
from django.views.decorators.csrf import csrf_exempt
from firecrawl import Firecrawl

client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@csrf_exempt
def firecrawl_view(request):
    if request.method != "POST":
        return JsonResponse({"error": "Method not allowed"}, status=405)

    body = json.loads(request.body)
    doc = client.scrape(body["url"], formats=["markdown"])
    return JsonResponse({"markdown": doc.markdown})
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Notas
</div>

* Añade una ruta URL que asocie `/api/firecrawl` con `firecrawl_view`.
