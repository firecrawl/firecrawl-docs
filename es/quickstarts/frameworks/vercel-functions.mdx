<div id="overview">
  ## Resumen
</div>

Usa una función de Vercel para que actúe como proxy de las llamadas a Firecrawl sin exponer tu clave de API.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Un proyecto de Vercel
* Node.js 18+

<div id="install">
  ## Instalación
</div>

No se requieren paquetes adicionales; usa fetch dentro de la función.

<div id="configure-environment">
  ## Configura el entorno
</div>

Define `FIRECRAWL_API_KEY` en `.env.local` y en la configuración de tu proyecto en Vercel.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Agrega una ruta del servidor
</div>

api/firecrawl.ts

```ts
import type { VercelRequest, VercelResponse } from '@vercel/node';

export default async function handler(req: VercelRequest, res: VercelResponse) {
  if (req.method !== 'POST') {
    res.status(405).send('Method not allowed');
    return;
  }

  const { url } = req.body;
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  res.status(200).json(data);
}
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```js
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Configura variables de entorno en Vercel para mantener privada la clave de API.
