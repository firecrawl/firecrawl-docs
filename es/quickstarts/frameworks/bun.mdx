<div id="overview">
  ## Resumen
</div>

Usa el `fetch` integrado de Bun para llamar a la API HTTP de Firecrawl.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Bun 1.x

<div id="install">
  ## Instalar
</div>

No se requieren paquetes adicionales; Bun incluye fetch de forma nativa.

<div id="configure-environment">
  ## Configura el entorno
</div>

Configura la clave de API en tu entorno antes de ejecutar Bun.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta al servidor
</div>

server.ts

```ts
const server = Bun.serve({
  port: 3000,
  async fetch(req) {
    const url = new URL(req.url);
    if (url.pathname != '/api/firecrawl' || req.method != 'POST') {
      return new Response('Not found', { status: 404 });
    }

    const { url: target } = await req.json();
    const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
      method: 'POST',
      headers: {
        Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
        'Content-Type': 'application/json',
      },
      body: JSON.stringify({ url: target, formats: ['markdown'] }),
    });

    return new Response(await response.text(), {
      headers: { 'Content-Type': 'application/json' },
    });
  },
});

console.log(`Listening on http://localhost:${server.port}`);
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Mantén la clave de API en el servidor y redirige las solicitudes a través de tu handler de Bun.
