<div id="overview">
  ## Descripción general
</div>

Usa un endpoint `+server.ts` de SvelteKit para llamar a Firecrawl desde el servidor.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación SvelteKit
* Node.js 18+

<div id="install">
  ## Instalación
</div>

No se necesitan paquetes adicionales; usa `fetch` desde el endpoint del servidor.

<div id="configure-environment">
  ## Configurar el entorno
</div>

Guarda la clave de API en `.env` y cárgala desde `$env/static/private`.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta de servidor
</div>

src/routes/api/firecrawl/+server.ts

```ts
import { json } from '@sveltejs/kit';
import { FIRECRAWL_API_KEY } from '$env/static/private';

export async function POST({ request }) {
  const { url } = await request.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return json(data);
}
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Mantén la clave de API en variables de entorno privadas; evita exponerla a través de `$env/static/public`.
