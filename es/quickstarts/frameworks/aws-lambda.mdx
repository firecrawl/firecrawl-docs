<div id="overview">
  ## Resumen
</div>

Usa un manejador de Lambda para servir de proxy seguro para las solicitudes de Firecrawl.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de la API de Firecrawl
* AWS Lambda (entorno de ejecución de Node.js 18)

<div id="install">
  ## Instalación
</div>

No se necesitan paquetes adicionales; usa fetch en Node.js 18.

<div id="configure-environment">
  ## Configura el entorno
</div>

Define `FIRECRAWL_API_KEY` en las variables de entorno de tu Lambda.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta de servidor
</div>

index.mjs

```js
export const handler = async (event) => {
  const { url } = JSON.parse(event.body || '{}');

  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${process.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return {
    statusCode: 200,
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify(data),
  };
};
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```js
const response = await fetch('https://your-api-id.execute-api.region.amazonaws.com/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Configura API Gateway (o las URL de funciones de Lambda) para exponer el endpoint de tu Lambda.
