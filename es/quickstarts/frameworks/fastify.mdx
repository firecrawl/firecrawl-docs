<div id="overview">
  ## Descripción general
</div>

Usa el SDK de Firecrawl para Node en una ruta del servidor Fastify.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación Fastify
* Node.js 18+

<div id="install">
  ## Instalación
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="configure-environment">
  ## Configura el entorno
</div>

Añade la clave de API al entorno del servidor.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta del servidor
</div>

server.js

```js
import Fastify from 'fastify';
import Firecrawl from '@mendable/firecrawl-js';

const app = Fastify();
const client = new Firecrawl({ apiKey: process.env.FIRECRAWL_API_KEY });

app.post('/api/firecrawl', async (request) => {
  const { url } = request.body as { url: string };
  const doc = await client.scrape(url, { formats: ['markdown'] });
  return { markdown: doc.markdown };
});

app.listen({ port: 3000 });
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```js
const response = await fetch('http://localhost:3000/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.markdown);
```

<div id="notes">
  ## Notas
</div>

* Mantén las solicitudes de Firecrawl en el servidor para proteger tu clave de API.
