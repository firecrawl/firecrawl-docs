<div id="overview">
  ## Descripción general
</div>

Usa el SDK de Python de Firecrawl en un controlador de rutas de FastAPI.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación FastAPI
* Python 3.8+

<div id="install">
  ## Instalación
</div>

```bash
pip install firecrawl-py
```

<div id="configure-environment">
  ## Configurar el entorno
</div>

Define la clave de API en el entorno de tu servidor.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```

<div id="add-a-server-route">
  ## Agrega una ruta al servidor
</div>

app/main.py

```python
import os
from fastapi import FastAPI
from firecrawl import Firecrawl

app = FastAPI()
client = Firecrawl(api_key=os.environ["FIRECRAWL_API_KEY"])

@app.post("/api/firecrawl")
def scrape(body: dict):
    doc = client.scrape(body["url"], formats=["markdown"])
    return {"markdown": doc.markdown}
```

<div id="call-it-from-your-app">
  ## Llámalo desde tu aplicación
</div>

```python
import requests

response = requests.post(
    "http://localhost:8000/api/firecrawl",
    json={"url": "https://example.com"},
)

print(response.json().get("markdown"))
```

<div id="notes">
  ## Notas
</div>

* Inicializa el cliente de Firecrawl una sola vez para reutilizar las conexiones HTTP.
