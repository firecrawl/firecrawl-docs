<div id="overview">
  ## Descripción general
</div>

Usa la API HTTP de Firecrawl directamente en una ruta de Hono para que funcione en distintos entornos de ejecución.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Una clave de API de Firecrawl
* Una aplicación de Hono
* Un entorno de ejecución compatible (Node, Bun, Cloudflare, Deno)

<div id="install">
  ## Instalar
</div>

```bash
npm install hono
```

<div id="configure-environment">
  ## Configurar el entorno
</div>

Expón la clave de API mediante la vinculación de entorno de tu entorno de ejecución.

```bash
FIRECRAWL_API_KEY=fc-your-api-key
```

<div id="add-a-server-route">
  ## Añade una ruta en el servidor
</div>

src/index.ts

```ts
import { Hono } from 'hono';

type Bindings = { FIRECRAWL_API_KEY: string };

const app = new Hono<{ Bindings: Bindings }>();

app.post('/api/firecrawl', async (c) => {
  const { url } = await c.req.json();
  const response = await fetch('https://api.firecrawl.dev/v2/scrape', {
    method: 'POST',
    headers: {
      Authorization: `Bearer ${c.env.FIRECRAWL_API_KEY}`,
      'Content-Type': 'application/json',
    },
    body: JSON.stringify({ url, formats: ['markdown'] }),
  });

  const data = await response.json();
  return c.json(data);
});

export default app;
```

<div id="call-it-from-your-app">
  ## Invócalo desde tu aplicación
</div>

```ts
const response = await fetch('/api/firecrawl', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ url: 'https://example.com' }),
});

const data = await response.json();
console.log(data.data?.markdown ?? data);
```

<div id="notes">
  ## Notas
</div>

* Configura las vinculaciones de entorno para el runtime que elijas (Wrangler, Deno Deploy, etc.).
