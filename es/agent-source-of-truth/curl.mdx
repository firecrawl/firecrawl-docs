<div id="firecrawl-curl-source-of-truth">
  # Fuente canónica de curl para Firecrawl
</div>

Este archivo es la fuente pública canónica para los agentes. Se genera a partir de la especificación OpenAPI v2 de Firecrawl y de las interfaces del SDK.



<div id="authenticate">
  ## Autenticación
</div>

Todas las solicitudes usan un token Bearer en el encabezado `Authorization`.

```bash
export FIRECRAWL_API_KEY="fc-your-api-key"
```



<div id="when-to-use-what">
  ## Cuándo usar cada opción
</div>

* `search`: empieza con una consulta para descubrir URL relevantes, incluidas las restricciones `site:`.
* `scrape`: obtiene y extrae contenido de una URL conocida.
* `interact`: ejecuta acciones del navegador o código cuando una página requiere clics, formularios o pasos posteriores al scraping.



<div id="search">
  ## Search
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa search cuando necesites descubrir resultados a partir de una consulta y quieras que Firecrawl devuelva URLs (y, opcionalmente, haga scraping de cada resultado). Puedes restringir los resultados a un dominio con el operador estándar `site:`, por ejemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`POST https://api.firecrawl.dev/v2/search`

<div id="simple-example">
  ### Ejemplo simple
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev webhook retries"
  }'
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/search" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "query": "site:docs.firecrawl.dev crawl webhooks",
    "sources": ["web", "news"],
    "categories": ["research"],
    "limit": 10,
    "tbs": "qdr:m",
    "location": "San Francisco,California,United States",
    "country": "US",
    "ignoreInvalidURLs": true,
    "timeout": 60000,
    "scrapeOptions": {
      "formats": [
        "markdown",
        "links",
        { "type": "json", "prompt": "Extract title and key endpoints." }
      ],
      "onlyMainContent": true,
      "includeTags": ["main", "article"],
      "excludeTags": ["nav", "footer"],
      "waitFor": 1000,
      "actions": [
        { "type": "click", "selector": "button#accept" },
        { "type": "wait", "milliseconds": 750 },
        { "type": "scrape" }
      ]
    }
  }'
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro           | Tipo / Opciones  | Descripción                                                             |
| ------------------- | --------------- | ----------------------------------------------------------------------- | -------------------------------------------------- | ---------------- | ---------- | ------------ | ----------------------------- |
| `query`             | `string`        | Consulta de búsqueda. Usa `site:domain` para restringir los resultados a un sitio. |
| `sources`           | `Array<"web"    | "news"                                                                  | "images"                                           | { type: "web"    | "news"     | "images" }>` | Fuentes donde buscar.         |
| `categories`        | `Array<"github" | "research"                                                              | "pdf"                                              | { type: "github" | "research" | "pdf" }>`    | Categorías para filtrar los resultados. |
| `limit`             | `integer`       | Número máximo de resultados a devolver.                                 |
| `tbs`               | `string`        | Filtro de búsqueda por tiempo (por ejemplo `qdr:d`, `qdr:w`, `sbd:1,qdr:m`). |
| `location`          | `string`        | Ubicación para los resultados de búsqueda.                              |
| `country`           | `string`        | Código de país ISO para geolocalizar los resultados de búsqueda.        |
| `timeout`           | `integer`       | Timeout en milisegundos.                                                |
| `ignoreInvalidURLs` | `boolean`       | Excluye URLs inválidas que otros endpoints de Firecrawl no pueden hacer scraping. |
| `enterprise`        | `Array<"anon"   | "zdr">`                                                                 | Opciones Enterprise ZDR cuando están habilitadas para tu equipo. |
| `scrapeOptions`     | `ScrapeOptions` | Opciones de scraping aplicadas a cada resultado (mismos campos que `scrape`). |


<div id="scrape">
  ## Scraping
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa scrape cuando ya tengas una URL y quieras que Firecrawl devuelva contenido estructurado y formatos opcionales.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`POST https://api.firecrawl.dev/v2/scrape`

<div id="simple-example">
  ### Ejemplo simple
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://docs.firecrawl.dev",
    "formats": ["markdown"]
  }'
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "url": "https://example.com/pricing",
    "formats": [
      "markdown",
      "links",
      { "type": "json", "prompt": "Extract plan names and prices." },
      { "type": "screenshot", "fullPage": true, "quality": 80, "viewport": { "width": 1280, "height": 720 } },
      { "type": "changeTracking", "modes": ["git-diff"], "tag": "pricing" }
    ],
    "headers": { "User-Agent": "FirecrawlDocsBot/1.0" },
    "includeTags": ["main", "article"],
    "excludeTags": ["nav", "footer"],
    "onlyMainContent": true,
    "waitFor": 1000,
    "mobile": false,
    "parsers": [{ "type": "pdf", "mode": "auto", "maxPages": 5 }],
    "actions": [
      { "type": "click", "selector": "#accept" },
      { "type": "write", "text": "firecrawl" },
      { "type": "press", "key": "Enter" },
      { "type": "wait", "milliseconds": 1500 },
      { "type": "scrape" }
    ],
    "location": { "country": "US", "languages": ["en-US"] },
    "skipTlsVerification": true,
    "removeBase64Images": true,
    "blockAds": true,
    "proxy": "auto",
    "maxAge": 86400000,
    "storeInCache": true,
    "profile": { "name": "docs-session", "saveChanges": true }
  }'
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro             | Tipo / Opciones    | Descripción                                                                                                                                         |                   |                |
| --------------------- | ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------- | ----------------- | -------------- |
| `url`                 | `string`           | URL para hacer scraping.                                                                                                                            |                   |                |
| `formats`             | `Array`            | Formatos de salida que se incluirán. Cadenas u objetos como `{type: "json", prompt, schema}` o `{type: "screenshot", fullPage, quality, viewport}`. |                   |                |
| `onlyMainContent`     | `boolean`          | Conserva solo el contenido principal.                                                                                                               |                   |                |
| `includeTags`         | `Array<string>`    | Etiquetas que se incluirán.                                                                                                                         |                   |                |
| `excludeTags`         | `Array<string>`    | Etiquetas que se excluirán.                                                                                                                         |                   |                |
| `maxAge`              | `integer`          | Prioriza los datos en caché hasta esta antigüedad.                                                                                                  |                   |                |
| `minAge`              | `integer`          | Solo devuelve datos en caché si alcanzan la antigüedad mínima.                                                                                      |                   |                |
| `headers`             | `object`           | Encabezados personalizados.                                                                                                                         |                   |                |
| `waitFor`             | `integer`          | Tiempo de espera en milisegundos antes de hacer scraping.                                                                                           |                   |                |
| `mobile`              | `boolean`          | Emula un dispositivo móvil.                                                                                                                         |                   |                |
| `skipTlsVerification` | `boolean`          | Omite la verificación del certificado TLS.                                                                                                          |                   |                |
| `timeout`             | `integer`          | Tiempo de espera en milisegundos.                                                                                                                   |                   |                |
| `parsers`             | `Array`            | Configuración del parser (se admiten opciones de PDF).                                                                                              |                   |                |
| `actions`             | `Array`            | Acciones del navegador que se ejecutarán antes del scraping.                                                                                        |                   |                |
| `location`            | `object`           | Configuración de ubicación para la solicitud.                                                                                                       |                   |                |
| `location.country`    | `string`           | Código ISO del país.                                                                                                                                |                   |                |
| `location.languages`  | `Array<string>`    | Idiomas preferidos.                                                                                                                                 |                   |                |
| `removeBase64Images`  | `boolean`          | Elimina las imágenes en base64 de la salida en markdown.                                                                                            |                   |                |
| `blockAds`            | `boolean`          | Activa el bloqueo de anuncios y ventanas emergentes de cookies.                                                                                     |                   |                |
| `proxy`               | `&quot;basic&quot; | &quot;enhanced&quot;                                                                                                                                | &quot;auto&quot;` | Modo de proxy. |
| `storeInCache`        | `boolean`          | Almacena los resultados en caché.                                                                                                                   |                   |                |
| `profile`             | `object`           | Perfil persistente del navegador con `name` y `saveChanges`.                                                                                        |                   |                |
| `zeroDataRetention`   | `boolean`          | Habilita la retención de datos cero cuando tu cuenta lo permita.                                                                                    |                   |                |



<div id="interact">
  ## Interact
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa Interact cuando una página requiera acciones del navegador o la ejecución de código una vez iniciado el scraping (clics, formularios o pasos con scripts).

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`POST https://api.firecrawl.dev/v2/scrape/{jobId}/interact`

<div id="simple-example">
  ### Ejemplo simple
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "code": "console.log(document.title)",
    "language": "node"
  }'
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```bash
curl -sS "https://api.firecrawl.dev/v2/scrape/<jobId>/interact" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "prompt": "Click the pricing tab and summarize the plans.",
    "language": "node",
    "timeout": 60
  }'
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro  | Tipo / Opciones     | Descripción                                                                                            |                   |                                                                        |
| ---------- | ------------------- | ------------------------------------------------------------------------------------------------------ | ----------------- | ---------------------------------------------------------------------- |
| `jobId`    | `string`            | ID del trabajo de scraping de `data.metadata.scrapeId`.                                                |                   |                                                                        |
| `code`     | `string`            | Código que se ejecutará en la sesión del navegador. Obligatorio si no se proporciona `prompt`.         |                   |                                                                        |
| `prompt`   | `string`            | Instrucción en lenguaje natural para el agente del navegador. Obligatorio si no se proporciona `code`. |                   |                                                                        |
| `language` | `&quot;python&quot; | &quot;node&quot;                                                                                       | &quot;bash&quot;` | Lenguaje de ejecución. Usa `node` de forma predeterminada si se omite. |
| `timeout`  | `integer`           | Tiempo de espera de la ejecución en segundos.                                                          |                   |                                                                        |
| `origin`   | `string`            | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                                 |                   |                                                                        |



<div id="notes">
  ## Notas
</div>

* La especificación de OpenAPI marca actualmente `code` como obligatorio para interact, pero los SDK admiten `prompt` como alternativa. En la práctica, usa `code` o `prompt`.
