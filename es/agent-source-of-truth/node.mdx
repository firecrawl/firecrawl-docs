<div id="firecrawl-nodejs-source-of-truth">
  # Fuente canónica de Firecrawl Node.js
</div>

Este archivo es la fuente pública oficial para los agentes. Se genera a partir del código fuente del SDK de Firecrawl JS y de la especificación OpenAPI v2.

<div id="install">
  ## Instalación
</div>

```bash
npm install @mendable/firecrawl-js
```

<div id="authenticate">
  ## Autenticación
</div>

```ts
import Firecrawl from "@mendable/firecrawl-js";

const client = new Firecrawl({
  apiKey: process.env.FIRECRAWL_API_KEY,
  // apiUrl: "https://api.firecrawl.dev" // anulación opcional para instancias autogestionadas
});
```

<div id="when-to-use-what">
  ## Cuándo usar cada opción
</div>

* `search`: empieza con una consulta para encontrar URL relevantes, incluidas las restricciones con `site:`.
* `scrape`: obtiene y extrae contenido de una URL conocida.
* `interact`: ejecuta acciones del navegador o código cuando una página requiere clics, formularios o pasos posteriores al scraping.

<div id="search">
  ## Search
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa search cuando necesites descubrir resultados a partir de una consulta y quieras que Firecrawl devuelva URLs (y, opcionalmente, haga scraping de cada resultado). Puedes restringir los resultados a un dominio con el operador estándar `site:`, por ejemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.search(query, options?)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```ts
const results = await client.search("site:docs.firecrawl.dev webhook retries");
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```ts
const results = await client.search("site:docs.firecrawl.dev crawl webhooks", {
  sources: ["web", "news"],
  categories: ["research"],
  limit: 10,
  tbs: "qdr:m",
  location: "San Francisco,California,United States",
  ignoreInvalidURLs: true,
  timeout: 60000,
  scrapeOptions: {
    formats: [
      "markdown",
      "links",
      { type: "json", prompt: "Extract title, summary, and key endpoints." }
    ],
    onlyMainContent: true,
    includeTags: ["main", "article"],
    excludeTags: ["nav", "footer"],
    waitFor: 1000,
    actions: [
      { type: "click", selector: "button#accept" },
      { type: "wait", milliseconds: 750 },
      { type: "scrape" }
    ]
  }
});
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro                   | Tipo / Opciones                                     | Descripción                                                                              |
| --------------------------- | --------------------------------------------------- | ---------------------------------------------------------------------------------------- |
| `query`                     | `string`                                            | Consulta de búsqueda. Usa `site:dominio` para restringir los resultados a un sitio.      |
| `options.sources`           | `array of source names or typed source objects`     | Fuentes en las que buscar.                                                               |
| `options.categories`        | `array of category names or typed category objects` | Categorías para filtrar los resultados.                                                  |
| `options.limit`             | `number`                                            | Número máximo de resultados a retornar.                                                  |
| `options.tbs`               | `string`                                            | Filtro de búsqueda por tiempo (por ejemplo `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).             |
| `options.location`          | `string`                                            | Cadena de ubicación para los resultados de búsqueda.                                     |
| `options.ignoreInvalidURLs` | `boolean`                                           | Excluye URLs inválidas que otros endpoints de Firecrawl no pueden procesar con scraping. |
| `options.timeout`           | `number`                                            | Timeout en milisegundos.                                                                 |
| `options.scrapeOptions`     | `ScrapeOptions`                                     | Opciones de scraping aplicadas a cada resultado (mismos campos que `scrape`).            |
| `options.integration`       | `string`                                            | Solo uso interno. Reservado para la instrumentación de Firecrawl.                        |
| `options.origin`            | `string`                                            | Solo uso interno. Reservado para la instrumentación de Firecrawl.                        |

<div id="scrape">
  ## Scraping
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa scraping cuando ya tengas una URL y quieras que Firecrawl devuelva contenido estructurado y formatos opcionales.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.scrape(url, options?)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```ts
const doc = await client.scrape("https://docs.firecrawl.dev", {
  formats: ["markdown"]
});
```

### Ejemplo avanzado

```ts
const doc = await client.scrape("https://example.com/pricing", {
  formats: [
    "markdown",
    "links",
    { type: "json", prompt: "Extract plan names and prices." },
    { type: "screenshot", fullPage: true, quality: 80, viewport: { width: 1280, height: 720 } },
    { type: "changeTracking", modes: ["git-diff"], tag: "pricing" }
  ],
  headers: {
    "User-Agent": "FirecrawlDocsBot/1.0"
  },
  includeTags: ["main", "article"],
  excludeTags: ["nav", "footer"],
  onlyMainContent: true,
  waitFor: 1000,
  mobile: false,
  parsers: [{ type: "pdf", mode: "auto", maxPages: 5 }],
  actions: [
    { type: "click", selector: "#accept" },
    { type: "write", text: "firecrawl" },
    { type: "press", key: "Enter" },
    { type: "wait", milliseconds: 1500 },
    { type: "scrape" }
  ],
  location: { country: "US", languages: ["en-US"] },
  skipTlsVerification: true,
  removeBase64Images: true,
  fastMode: true,
  blockAds: true,
  proxy: "auto",
  maxAge: 86400000,
  storeInCache: true,
  profile: { name: "docs-session", saveChanges: true }
});
```

<div id="interact">
  ## Interact
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa interact cuando una página requiere acciones del navegador o la ejecución de código después de que se inicia un scraping (clics, formularios o pasos con scripts).

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.interact(jobId, args)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```ts
const doc = await client.scrape("https://example.com", { formats: ["markdown"] });
const jobId = doc.metadata?.scrapeId;
if (!jobId) throw new Error("Falta scrapeId en la respuesta del scraping");

const result = await client.interact(jobId, {
  prompt: "Haz clic en la pestaña de precios y resume los planes.",
  language: "node"
});
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```ts
const result = await client.interact("<scrapeJobId>", {
  code: "console.log(await page.title());",
  language: "node",
  timeout: 60
});
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro       | Tipo / Opciones       | Descripción                                                                                            |
| --------------- | --------------------- | ------------------------------------------------------------------------------------------------------ |
| `jobId`         | `string`              | ID del trabajo de scraping de `document.metadata.scrapeId`.                                            |
| `args.code`     | `string`              | Código que se ejecutará en la sesión del navegador. Obligatorio si no se proporciona `prompt`.         |
| `args.prompt`   | `string`              | Instrucción en lenguaje natural para el agente del navegador. Obligatorio si no se proporciona `code`. |
| `args.language` | `python, node o bash` | Lenguaje de ejecución. El valor predeterminado es `node` si se omite.                                  |
| `args.timeout`  | `number`              | Tiempo de espera de la ejecución en segundos.                                                          |
| `args.origin`   | `string`              | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                                 |

<div id="notes">
  ## Notas
</div>

* Alias obsoletos: `scrapeExecute`, `stopInteractiveBrowser` y `deleteScrapeBrowser` corresponden a `interact` y `stopInteraction`.
* El cliente `Firecrawl` de nivel superior expone directamente los métodos de v2; la versión heredada de v1 está disponible en `client.v1`.
