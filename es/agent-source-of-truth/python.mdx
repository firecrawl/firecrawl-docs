<div id="search">
  ## Cuándo usar cada opción
</div>

* `search`: empieza con una consulta para descubrir URL relevantes, incluidas las restricciones `site:`.
* `scrape`: obtiene y extrae contenido de una URL conocida.
* `interact`: ejecuta acciones del navegador o código cuando una página requiere clics, formularios o pasos posteriores al scraping.



## Search

<div id="preferred-sdk-method">
  ### Por qué usarlo
</div>

Usa search cuando necesites descubrir resultados a partir de una consulta y quieras que Firecrawl devuelva URLs (y, opcionalmente, realice scraping de cada resultado). Puedes limitar los resultados a un dominio con el operador estándar `site:`, por ejemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="simple-example">
  ### Método recomendado del SDK
</div>

`client.search(query, **options)`

<div id="complex-example">
  ### Ejemplo simple
</div>

```py
results = client.search("site:docs.firecrawl.dev webhook retries")
```

<div id="parameters">
  ### Ejemplo complejo
</div>

```py
from firecrawl.v2.types import ScrapeOptions

results = client.search(
    "site:docs.firecrawl.dev crawl webhooks",
    sources=["web", "news"],
    categories=["research"],
    limit=10,
    tbs="qdr:m",
    location="San Francisco,California,United States",
    ignore_invalid_urls=True,
    timeout=300000,
    scrape_options=ScrapeOptions(
        formats=[
            "markdown",
            "links",
            {"type": "json", "prompt": "Extract title and key endpoints."}
        ],
        only_main_content=True,
        include_tags=["main", "article"],
        exclude_tags=["nav", "footer"],
        wait_for=1000,
        actions=[
            {"type": "click", "selector": "button#accept"},
            {"type": "wait", "milliseconds": 750},
            {"type": "scrape"}
        ]
    )
)
```

### Parámetros

| Parámetro             | Tipo / Opciones | Descripción                                                                                        |
| --------------------- | --------------- | -------------------------------------------------------------------------------- | -------- | ----------------- | ---------- | ----------- | ----------------------------- |
| `query`               | `str`           | Consulta de búsqueda. Usa `site:domain` para restringir los resultados a un sitio.                 |
| `sources`             | `List["web"     | "news"                                                                           | "images" | {"type": "web"    | "news"     | "images"}]` | Fuentes en las que buscar.    |
| `categories`          | `List["github"  | "research"                                                                       | "pdf"    | {"type": "github" | "research" | "pdf"}]`    | Categorías para filtrar los resultados. |
| `limit`               | `int`           | Número máximo de resultados a devolver. El valor predeterminado es `5` en el modelo del SDK.       |
| `tbs`                 | `str`           | Filtro de búsqueda por tiempo (por ejemplo, `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).                     |
| `location`            | `str`           | Cadena de ubicación para los resultados de búsqueda.                                               |
| `ignore_invalid_urls` | `bool`          | Excluye URLs no válidas que otros endpoints de Firecrawl no pueden procesar con scraping.          |
| `timeout`             | `int`           | Tiempo de espera en milisegundos. El valor predeterminado es `300000` en el modelo del SDK.        |
| `scrape_options`      | `ScrapeOptions` | Opciones de scraping aplicadas a cada resultado (los mismos campos que `scrape`, más `min_age`).   |
| `integration`         | `str`           | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                             |


## Scraping

<div id="preferred-sdk-method">
  ### Por qué usarlo
</div>

Usa el scraping cuando ya tengas una URL y quieras que Firecrawl devuelva contenido estructurado y formatos opcionales.

<div id="simple-example">
  ### Método recomendado del SDK
</div>

`client.scrape(url, **options)`

<div id="complex-example">
  ### Ejemplo simple
</div>

```py
doc = client.scrape("https://docs.firecrawl.dev", formats=["markdown"])
```

<div id="parameters">
  ### Ejemplo complejo
</div>

```py
doc = client.scrape(
    "https://example.com/pricing",
    formats=[
        "markdown",
        "links",
        {"type": "json", "prompt": "Extract plan names and prices."},
        {"type": "screenshot", "full_page": True, "quality": 80, "viewport": {"width": 1280, "height": 720}},
        {"type": "changeTracking", "modes": ["git-diff"], "tag": "pricing"}
    ],
    headers={"User-Agent": "FirecrawlDocsBot/1.0"},
    include_tags=["main", "article"],
    exclude_tags=["nav", "footer"],
    only_main_content=True,
    wait_for=1000,
    mobile=False,
    parsers=[{"type": "pdf", "mode": "auto", "max_pages": 5}],
    actions=[
        {"type": "click", "selector": "#accept"},
        {"type": "write", "text": "firecrawl"},
        {"type": "press", "key": "Enter"},
        {"type": "wait", "milliseconds": 1500},
        {"type": "scrape"}
    ],
    location={"country": "US", "languages": ["en-US"]},
    skip_tls_verification=True,
    remove_base64_images=True,
    fast_mode=True,
    block_ads=True,
    proxy="auto",
    max_age=86400000,
    store_in_cache=True,
    profile={"name": "docs-session", "save_changes": True}
)
```

### Parámetros



| Parameter                | Type / Options                               | Description                                                                                                          |
| ------------------------ | -------------------------------------------- | -------------------------------------------------------------------------------------------------------------------- | ---------------------- | -------------------------- | --------------------------------------------- | ------------ | --------- | ------------------- | ------- | ------------ | ---------- | ------------------------ | ------- | ---------- | ------------------ | ----------------------------------------------------------------- |
| `url`                    | `str`                                        | URL a scrapear.                                                                                                       |
| `formats`                | `List[FormatOption]` or `ScrapeFormats`      | Formatos de salida a incluir. Acepta cadenas u objetos para formatos avanzados, o `ScrapeFormats` con toggles booleanos. |
| `formats[].type`         | `"markdown"                                  | "html"                                                                                                               | "rawHtml"              | "links"                    | "images"                                      | "screenshot" | "summary" | "changeTracking"    | "json"  | "attributes" | "branding" | "query"                  | "audio" | "raw_html" | "change_tracking"` | Selector de formato para entradas de objeto (se aceptan alias en snake&#95;case). |
| `formats[].prompt`       | `str`                                        | Prompt para los formatos `json` o `query`.                                                                                |
| `formats[].schema`       | `dict`                                       | JSON schema para `json` o para seguimiento de cambios en modo `json`.                                                      |
| `formats[].modes`        | `List["git-diff"                             | "json"]`                                                                                                             | Modos de seguimiento de cambios. |
| `formats[].tag`          | `str`                                        | Etiqueta de seguimiento de cambios para aislar el historial.                                                                              |
| `formats[].full_page`    | `bool`                                       | Screenshot de página completa cuando `type: "screenshot"`.                                                                      |
| `formats[].quality`      | `int`                                        | Calidad del screenshot (1-100).                                                                                          |
| `formats[].viewport`     | `{ "width": int, "height": int }`            | Tamaño del viewport del screenshot.                                                                                            |
| `formats[].selectors`    | `List[{"selector": str, "attribute": str}]`  | Selectores de extracción de atributos para `type: "attributes"`.                                                             |
| `headers`                | `Dict[str, str]`                             | Headers personalizados.                                                                                                      |
| `include_tags`           | `List[str]`                                  | Etiquetas a incluir.                                                                                                     |
| `exclude_tags`           | `List[str]`                                  | Etiquetas a excluir.                                                                                                     |
| `only_main_content`      | `bool`                                       | Conservar solo el contenido principal.                                                                                          |
| `timeout`                | `int`                                        | Timeout en milisegundos.                                                                                             |
| `wait_for`               | `int`                                        | Tiempo de espera en milisegundos antes del scraping.                                                                           |
| `mobile`                 | `bool`                                       | Emular un dispositivo móvil.                                                                                             |
| `parsers`                | `List[str                                    | {"type": "pdf", "mode": "fast"                                                                                       | "auto"                 | "ocr", "max_pages": int}]` | Configuración del parser (se admiten opciones de PDF). |
| `actions`                | `List[Action]`                               | Acciones del browser a ejecutar antes del scraping.                                                                              |
| `actions[].type`         | `"wait"                                      | "screenshot"                                                                                                         | "click"                | "write"                    | "press"                                       | "scroll"     | "scrape"  | "executeJavascript" | "pdf"`  | Tipo de acción. |
| `actions[].milliseconds` | `int`                                        | Duración de la espera cuando `type: "wait"`.                                                                                   |
| `actions[].selector`     | `str`                                        | Selector para `wait`, `click` o `scroll`.                                                                           |
| `actions[].text`         | `str`                                        | Texto para `write`.                                                                                                    |
| `actions[].key`          | `str`                                        | Tecla a presionar para `press`.                                                                                            |
| `actions[].direction`    | `"up"                                        | "down"`                                                                                                              | Dirección del scroll.      |
| `actions[].script`       | `str`                                        | JavaScript para `executeJavascript`.                                                                                  |
| `actions[].format`       | `"A0"                                        | "A1"                                                                                                                 | "A2"                   | "A3"                       | "A4"                                          | "A5"         | "A6"      | "Letter"            | "Legal" | "Tabloid"    | "Ledger"`  | Tamaño de página PDF para `pdf`. |
| `actions[].landscape`    | `bool`                                       | Modo horizontal (landscape) del PDF.                                                                                                  |
| `actions[].scale`        | `float`                                      | Factor de escala del PDF.                                                                                                    |
| `actions[].full_page`    | `bool`                                       | Screenshot de página completa para la acción.                                                                                         |
| `actions[].quality`      | `int`                                        | Calidad del screenshot para la acción.                                                                                           |
| `actions[].viewport`     | `{ "width": int, "height": int }`            | Viewport del screenshot para la acción.                                                                                          |
| `location`               | `{ "country": str, "languages": List[str] }` | Configuración de ubicación para la solicitud.                                                                                   |
| `skip_tls_verification`  | `bool`                                       | Omitir la verificación del certificado TLS.                                                                                   |
| `remove_base64_images`   | `bool`                                       | Eliminar imágenes en base64 de la salida en markdown.                                                                            |
| `fast_mode`              | `bool`                                       | Scraping más rápido con fidelidad reducida.                                                                               |
| `use_mock`               | `str`                                        | Solo uso interno. Reservado para pruebas del SDK.                                                                             |
| `block_ads`              | `bool`                                       | Habilita el bloqueo de anuncios y popups de cookies.                                                                                 |
| `proxy`                  | `"basic"                                     | "stealth"                                                                                                            | "enhanced"             | "auto"`                    | Modo de proxy.                                   |
| `max_age`                | `int`                                        | Usa datos en caché con una antigüedad máxima de este valor.                                                                                   |
| `store_in_cache`         | `bool`                                       | Almacena los resultados en caché.                                                                                              |
| `profile`                | `dict`                                       | Perfil de browser persistente, por ejemplo `{ "name": "profile", "save_changes": True }`.                               |
| `integration`            | `str`                                        | Solo uso interno. Reservado para la instrumentación de Firecrawl.                                                               |





## Interact

<div id="preferred-sdk-method">
  ### Por qué usarlo
</div>

Usa interact cuando una página requiera acciones del navegador o ejecutar código una vez iniciado el scraping (clics, formularios o pasos automatizados con scripts).

<div id="simple-example">
  ### Método recomendado del SDK
</div>

`client.interact(job_id, code=None, prompt=None, language="node", timeout=None)`

<div id="complex-example">
  ### Ejemplo simple
</div>

```py
doc = client.scrape("https://example.com", formats=["markdown"])
job_id = doc.metadata.scrape_id if doc.metadata else None
if not job_id:
    raise ValueError("Missing scrape_id from scrape response")

result = client.interact(
    job_id,
    prompt="Click the pricing tab and summarize the plans.",
    language="node"
)
```

<div id="parameters">
  ### Ejemplo complejo
</div>

```py
result = client.interact(
    "<scrape_job_id>",
    code="print(await page.title())",
    language="python",
    timeout=60
)
```

### Parámetros

| Parameter  | Type / Options      | Description                                                                                            |                   |                                                           |
| ---------- | ------------------- | ------------------------------------------------------------------------------------------------------ | ----------------- | --------------------------------------------------------- |
| `job_id`   | `str`               | ID del trabajo de scraping de `document.metadata.scrape_id`.                                           |                   |                                                           |
| `code`     | `str`               | Código que se ejecutará en la sesión del navegador. Obligatorio si no se proporciona `prompt`.         |                   |                                                           |
| `prompt`   | `str`               | Instrucción en lenguaje natural para el agente del navegador. Obligatorio si no se proporciona `code`. |                   |                                                           |
| `language` | `&quot;python&quot; | &quot;node&quot;                                                                                       | &quot;bash&quot;` | Lenguaje de ejecución. El valor predeterminado es `node`. |
| `timeout`  | `int`               | Tiempo máximo de ejecución en segundos (1-300).                                                        |                   |                                                           |
| `origin`   | `str`               | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                                 |                   |                                                           |



<div id="source-of-truth">
  ## Notas
</div>

* Alias obsoletos: `scrape_execute`, `stop_interactive_browser` y `delete_scrape_browser` apuntan a `interact` y `stop_interaction`.
* `ScrapeOptions` incluye `min_age`, pero el método auxiliar `scrape` no lo expone. Usa `ScrapeOptions` a través de `search(..., scrape_options=...)` si lo necesitas.
* El cliente principal de `Firecrawl` expone directamente los métodos de v2; la versión heredada v1 está disponible en `client.v1`.
