<div id="firecrawl-rust-source-of-truth">
  # Fuente oficial de referencia de Firecrawl Rust
</div>

Este archivo es la fuente pública canónica para los agentes. Se genera a partir del código fuente del SDK de Rust de Firecrawl y de la especificación OpenAPI v2.

<div id="install">
  ## Instalación
</div>

```bash
cargo add firecrawl
```

<div id="authenticate">
  ## Autenticación
</div>

```rust
use firecrawl::v2::Client;

let client = Client::new("fc-your-api-key")?;
// let client = Client::new_selfhosted("https://api.firecrawl.dev", Some("fc-your-api-key"))?;
```

<div id="when-to-use-what">
  ## Cuándo usar cada opción
</div>

* `search`: parte de una consulta para encontrar URL relevantes, incluidas las restricciones `site:`.
* `scrape`: obtiene y extrae contenido de una URL conocida.
* `interact`: ejecuta acciones en el navegador o código cuando una página requiere clics, formularios o pasos posteriores al scraping.

<div id="search">
  ## Búsqueda
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa search cuando necesites encontrar resultados a partir de una consulta y quieras que Firecrawl devuelva URL (y, opcionalmente, haga scraping de cada resultado). Puedes limitar los resultados a un dominio con el operador estándar `site:`, por ejemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.search(query, options)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```rust
use firecrawl::v2::Client;

let results = client
    .search("site:docs.firecrawl.dev webhook retries", None)
    .await?;
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```rust
use firecrawl::v2::{
    Client, SearchOptions, SearchSource, SearchCategory, ScrapeOptions, Format, JsonOptions
};

let options = SearchOptions {
    sources: Some(vec![SearchSource::Web, SearchSource::News]),
    categories: Some(vec![SearchCategory::Research]),
    limit: Some(10),
    tbs: Some("qdr:m".to_string()),
    location: Some("San Francisco,California,United States".to_string()),
    ignore_invalid_urls: Some(true),
    timeout: Some(60000),
    scrape_options: Some(ScrapeOptions {
        formats: Some(vec![Format::Markdown, Format::Links, Format::Json]),
        json_options: Some(JsonOptions {
            prompt: Some("Extract title and key endpoints.".to_string()),
            ..Default::default()
        }),
        only_main_content: Some(true),
        ..Default::default()
    }),
    ..Default::default()
};

let results = client
    .search("site:docs.firecrawl.dev crawl webhooks", options)
    .await?;
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro                     | Tipo / Opciones                                     | Descripción                                                                                                    |
| ----------------------------- | --------------------------------------------------- | -------------------------------------------------------------------------------------------------------------- |
| `query`                       | `&str`                                              | Consulta de búsqueda. Usa `site:domain` para limitar los resultados a un sitio.                                |
| `options.sources`             | `Vec<SearchSource>` (`Web`, `News`, `Images`)       | Fuentes en las que buscar.                                                                                     |
| `options.categories`          | `Vec<SearchCategory>` (`Github`, `Research`, `Pdf`) | Categorías para filtrar los resultados.                                                                        |
| `options.limit`               | `u32`                                               | Número máximo de resultados que se devolverán. Predeterminado: 5; máximo: 20 (según la documentación del SDK). |
| `options.tbs`                 | `String`                                            | Filtro de búsqueda basado en tiempo (por ejemplo, `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).                            |
| `options.location`            | `String`                                            | Cadena de texto de ubicación para los resultados de búsqueda.                                                  |
| `options.ignore_invalid_urls` | `bool`                                              | Excluye las URL no válidas que otros endpoints de Firecrawl no pueden hacer scraping.                          |
| `options.timeout`             | `u32`                                               | Timeout en milisegundos.                                                                                       |
| `options.scrape_options`      | `ScrapeOptions`                                     | Opciones de scraping aplicadas a cada resultado (los mismos campos que `scrape`).                              |
| `options.integration`         | `String`                                            | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                                         |

<div id="scrape">
  ## Scraping
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa scraping si ya tienes una URL y quieres que Firecrawl devuelva contenido estructurado y formatos opcionales.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.scrape(url, options)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```rust
use firecrawl::v2::{Client, ScrapeOptions, Format};

let doc = client
    .scrape("https://docs.firecrawl.dev", ScrapeOptions {
        formats: Some(vec![Format::Markdown]),
        ..Default::default()
    })
    .await?;
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```rust
use firecrawl::v2::{
    Client, ScrapeOptions, Format, JsonOptions, ScreenshotOptions, ChangeTrackingOptions,
    ChangeTrackingMode, AttributeSelector, Action, ScrollDirection, ProxyType, ParserConfig
};

let options = ScrapeOptions {
    formats: Some(vec![
        Format::Markdown,
        Format::Links,
        Format::Json,
        Format::Screenshot,
        Format::ChangeTracking,
        Format::Attributes,
    ]),
    json_options: Some(JsonOptions {
        prompt: Some("Extract plan names and prices.".to_string()),
        ..Default::default()
    }),
    screenshot_options: Some(ScreenshotOptions {
        full_page: Some(true),
        quality: Some(80),
        viewport: None,
    }),
    change_tracking_options: Some(ChangeTrackingOptions {
        modes: Some(vec![ChangeTrackingMode::GitDiff]),
        tag: Some("pricing".to_string()),
        ..Default::default()
    }),
    attribute_selectors: Some(vec![AttributeSelector {
        selector: "a".to_string(),
        attribute: "href".to_string(),
    }]),
    headers: None,
    include_tags: Some(vec!["main".to_string(), "article".to_string()]),
    exclude_tags: Some(vec!["nav".to_string(), "footer".to_string()]),
    only_main_content: Some(true),
    wait_for: Some(1000),
    parsers: Some(vec![ParserConfig::Pdf {
        parser_type: "pdf".to_string(),
        max_pages: Some(5),
    }]),
    actions: Some(vec![
        Action::Click { selector: "#accept".to_string() },
        Action::Write { text: "firecrawl".to_string() },
        Action::Press { key: "Enter".to_string() },
        Action::Wait { milliseconds: Some(1500), selector: None },
        Action::Scroll { direction: ScrollDirection::Down, selector: None },
        Action::Scrape,
    ]),
    skip_tls_verification: Some(true),
    remove_base64_images: Some(true),
    fast_mode: Some(true),
    block_ads: Some(true),
    proxy: Some(ProxyType::Auto),
    max_age: Some(86400000),
    min_age: None,
    store_in_cache: Some(true),
    ..Default::default()
};

let doc = client.scrape("https://example.com/pricing", options).await?;
```

<div id="interact">
  ## Interact
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa interact cuando una página requiere acciones del navegador o la ejecución de código después de iniciar un scraping (clics, formularios o pasos con scripts).

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.interact(job_id, options)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```rust
use firecrawl::v2::{Client, ScrapeOptions, Format, ScrapeExecuteOptions};

let doc = client
    .scrape("https://example.com", ScrapeOptions {
        formats: Some(vec![Format::Markdown]),
        ..Default::default()
    })
    .await?;

let job_id = doc
    .metadata
    .as_ref()
    .and_then(|m| m.scrape_id.clone())
    .expect("Missing scrape_id");

let result = client
    .interact(job_id, ScrapeExecuteOptions {
        prompt: Some("Click the pricing tab and summarize the plans.".to_string()),
        language: None,
        ..Default::default()
    })
    .await?;
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```rust
use firecrawl::v2::{Client, ScrapeExecuteOptions, ScrapeExecuteLanguage};

let result = client
    .interact(
        "<scrape_job_id>",
        ScrapeExecuteOptions {
            code: Some("console.log(await page.title());".to_string()),
            language: Some(ScrapeExecuteLanguage::Node),
            timeout: Some(60),
            ..Default::default()
        },
    )
    .await?;
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro          | Tipo / Opciones                                    | Descripción                                                                                            |
| ------------------ | -------------------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| `job_id`           | `&str`                                             | ID del trabajo de scraping de `document.metadata.scrape_id`.                                           |
| `options.code`     | `String`                                           | Código que se ejecutará en la sesión del navegador. Obligatorio si no se proporciona `prompt`.         |
| `options.prompt`   | `String`                                           | Instrucción en lenguaje natural para el agente del navegador. Obligatorio si no se proporciona `code`. |
| `options.language` | `ScrapeExecuteLanguage` (`Python`, `Node`, `Bash`) | Lenguaje de ejecución. El valor predeterminado es `Node` si se omite.                                  |
| `options.timeout`  | `u32`                                              | Tiempo límite de ejecución en segundos.                                                                |
| `options.origin`   | `String`                                           | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                                 |

<div id="notes">
  ## Notas
</div>

* Alias en desuso: `scrape_execute`, `stop_interactive_browser` y `delete_scrape_browser` corresponden a `interact` y `stop_interaction`.
* Los campos de las estructuras de Rust usan snake&#95;case, pero se serializan como campos camelCase de la API mediante serde.
