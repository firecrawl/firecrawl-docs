<div id="firecrawl-java-source-of-truth">
  # Fuente canónica de Firecrawl para Java
</div>

Este archivo es la fuente pública oficial canónica para agentes. Se genera a partir del código fuente del SDK de Java de Firecrawl y de la especificación OpenAPI v2.

<div id="install">
  ## Instalación
</div>

Maven:

```xml
<dependency>
  <groupId>com.firecrawl</groupId>
  <artifactId>firecrawl-java</artifactId>
  <version>1.2.0</version>
</dependency>
```

Gradle:

```gradle
implementation("com.firecrawl:firecrawl-java:1.2.0")
```

<div id="authenticate">
  ## Autenticación
</div>

```java
import com.firecrawl.client.FirecrawlClient;

FirecrawlClient client = FirecrawlClient.builder()
    .apiKey(System.getenv("FIRECRAWL_API_KEY"))
    .build();
```

<div id="when-to-use-what">
  ## Cuándo usar cada opción
</div>

* `search`: empieza con una consulta para descubrir URLs relevantes, incluidas las restricciones `site:`.
* `scrape`: obtiene y extrae contenido de una URL conocida.
* `interact`: ejecuta acciones en el navegador o código cuando una página requiere clics, formularios o pasos posteriores al scraping.

<div id="search">
  ## Búsqueda
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa search cuando necesites descubrir URLs a partir de una consulta y quieras que Firecrawl las devuelva (y, opcionalmente, haga scraping de cada resultado). Puedes limitar los resultados a un dominio con el operador estándar `site:`, por ejemplo `site:docs.firecrawl.dev crawl webhooks`.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.search(query, options)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```java
SearchData results = client.search("site:docs.firecrawl.dev webhook retries");
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```java
import com.firecrawl.models.SearchOptions;
import com.firecrawl.models.ScrapeOptions;
import com.firecrawl.models.JsonFormat;

SearchOptions options = SearchOptions.builder()
    .sources(List.of("web", "news"))
    .categories(List.of("research"))
    .limit(10)
    .tbs("qdr:m")
    .location("San Francisco,California,United States")
    .ignoreInvalidURLs(true)
    .timeout(60000)
    .scrapeOptions(
        ScrapeOptions.builder()
            .formats(List.of(
                "markdown",
                "links",
                JsonFormat.builder().prompt("Extract title and key endpoints.").build()
            ))
            .onlyMainContent(true)
            .waitFor(1000)
            .build()
    )
    .build();

SearchData results = client.search("site:docs.firecrawl.dev crawl webhooks", options);
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro                   | Tipo / Opciones                                                          | Descripción                                                                                  |
| --------------------------- | ------------------------------------------------------------------------ | -------------------------------------------------------------------------------------------- |
| `query`                     | `String`                                                                 | Consulta de búsqueda. Usa `site:domain` para restringir los resultados a un sitio.           |
| `options.sources`           | `List<Object>` (`"web"`, `"news"`, `"images"` o objetos `{type: "web"}`) | Fuentes en las que buscar.                                                                   |
| `options.categories`        | `List<Object>` (`"github"`, `"research"`, `"pdf"`)                       | Categorías para filtrar resultados.                                                          |
| `options.limit`             | `Integer`                                                                | Número máximo de resultados devueltos.                                                       |
| `options.tbs`               | `String`                                                                 | Filtro de búsqueda basado en tiempo (por ejemplo, `qdr:d`, `qdr:w`, `sbd:1,qdr:m`).          |
| `options.location`          | `String`                                                                 | Ubicación para los resultados de búsqueda.                                                   |
| `options.ignoreInvalidURLs` | `Boolean`                                                                | Excluye las URL no válidas de las que otros endpoints de Firecrawl no pueden hacer scraping. |
| `options.timeout`           | `Integer`                                                                | Tiempo de espera en milisegundos.                                                            |
| `options.scrapeOptions`     | `ScrapeOptions`                                                          | Opciones de scraping aplicadas a cada resultado (los mismos campos que en `scrape`).         |
| `options.integration`       | `String`                                                                 | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                       |

<div id="scrape">
  ## Scraping
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa `scrape` si ya tienes una URL y quieres que Firecrawl devuelva contenido estructurado y formatos opcionales.

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.scrape(url, options)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```java
Document doc = client.scrape(
    "https://docs.firecrawl.dev",
    ScrapeOptions.builder()
        .formats(List.of("markdown"))
        .build()
);
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```java
import com.firecrawl.models.ScrapeOptions;

ScrapeOptions options = ScrapeOptions.builder()
    .formats(List.of(
        "markdown",
        "links",
        Map.of("type", "json", "prompt", "Extract plan names and prices."),
        Map.of("type", "screenshot", "fullPage", true, "quality", 80, "viewport", Map.of("width", 1280, "height", 720)),
        Map.of("type", "changeTracking", "modes", List.of("git-diff"), "tag", "pricing")
    ))
    .headers(Map.of("User-Agent", "FirecrawlDocsBot/1.0"))
    .includeTags(List.of("main", "article"))
    .excludeTags(List.of("nav", "footer"))
    .onlyMainContent(true)
    .waitFor(1000)
    .mobile(false)
    .parsers(List.of(Map.of("type", "pdf", "mode", "auto", "maxPages", 5)))
    .actions(List.of(
        Map.of("type", "click", "selector", "#accept"),
        Map.of("type", "write", "text", "firecrawl"),
        Map.of("type", "press", "key", "Enter"),
        Map.of("type", "wait", "milliseconds", 1500),
        Map.of("type", "scrape")
    ))
    .location(LocationConfig.builder().country("US").languages(List.of("en-US")).build())
    .skipTlsVerification(true)
    .removeBase64Images(true)
    .blockAds(true)
    .proxy("auto")
    .maxAge(86400000L)
    .storeInCache(true)
    .build();

Document doc = client.scrape("https://example.com/pricing", options);
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro                     | Tipo / Opciones                                                                   | Descripción                                                                                                                                                        |
| ----------------------------- | --------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `url`                         | `String`                                                                          | URL que se va a scrapear.                                                                                                                                          |
| `options.formats`             | `List<Object>`                                                                    | Formatos de salida que se deben incluir. Cadenas u objetos de formato como `{type: "json", prompt, schema}` o `{type: "screenshot", fullPage, quality, viewport}`. |
| `options.headers`             | `Map<String, String>`                                                             | Encabezados personalizados.                                                                                                                                        |
| `options.includeTags`         | `List<String>`                                                                    | Etiquetas que se deben incluir.                                                                                                                                    |
| `options.excludeTags`         | `List<String>`                                                                    | Etiquetas que se deben excluir.                                                                                                                                    |
| `options.onlyMainContent`     | `Boolean`                                                                         | Mantener solo el contenido principal.                                                                                                                              |
| `options.timeout`             | `Integer`                                                                         | Tiempo de espera en milisegundos.                                                                                                                                  |
| `options.waitFor`             | `Integer`                                                                         | Tiempo de espera en milisegundos antes del scraping.                                                                                                               |
| `options.mobile`              | `Boolean`                                                                         | Emular un dispositivo móvil.                                                                                                                                       |
| `options.parsers`             | `List<Object>`                                                                    | Configuración del analizador (compatible con opciones de PDF).                                                                                                     |
| `options.actions`             | `List<Map<String, Object>>`                                                       | Acciones del navegador que se ejecutarán antes del scraping.                                                                                                       |
| `options.location`            | `LocationConfig`                                                                  | Configuración de ubicación para la solicitud.                                                                                                                      |
| `options.skipTlsVerification` | `Boolean`                                                                         | Omitir la verificación del certificado TLS.                                                                                                                        |
| `options.removeBase64Images`  | `Boolean`                                                                         | Eliminar las imágenes en base64 de la salida en markdown.                                                                                                          |
| `options.blockAds`            | `Boolean`                                                                         | Activar el bloqueo de anuncios y ventanas emergentes de cookies.                                                                                                   |
| `options.proxy`               | `String` (`"basic"`, `"stealth"`, `"enhanced"`, `"auto"` o una URL personalizada) | Modo de proxy.                                                                                                                                                     |
| `options.maxAge`              | `Long`                                                                            | Priorizar datos en caché hasta esta antigüedad.                                                                                                                    |
| `options.storeInCache`        | `Boolean`                                                                         | Almacenar los resultados en caché.                                                                                                                                 |
| `options.integration`         | `String`                                                                          | Solo para uso interno. Reservado para la instrumentación de Firecrawl.                                                                                             |

<div id="interact">
  ## Interact
</div>

<div id="why-use-it">
  ### Por qué usarlo
</div>

Usa interact cuando una página requiera acciones del navegador o la ejecución de código una vez iniciado el scraping (clics, formularios o pasos automatizados mediante scripts).

<div id="preferred-sdk-method">
  ### Método recomendado del SDK
</div>

`client.interact(jobId, code, language, timeout, origin)`

<div id="simple-example">
  ### Ejemplo simple
</div>

```java
Document doc = client.scrape("https://example.com");
String jobId = doc.getMetadata() != null ? String.valueOf(doc.getMetadata().get("scrapeId")) : null;
if (jobId == null || jobId.isBlank()) {
    throw new IllegalStateException("Falta scrapeId en la respuesta del scraping");
}

BrowserExecuteResponse result = client.interact(jobId, "console.log(document.title)");
```

<div id="complex-example">
  ### Ejemplo complejo
</div>

```java
BrowserExecuteResponse result = client.interact(
    "<scrapeJobId>",
    "console.log(await page.title());",
    "node",
    60,
    null
);
```

<div id="parameters">
  ### Parámetros
</div>

| Parámetro  | Tipo / Opciones                           | Descripción                                                            |
| ---------- | ----------------------------------------- | ---------------------------------------------------------------------- |
| `jobId`    | `String`                                  | ID del trabajo de scraping en `document.metadata.scrapeId`.            |
| `code`     | `String`                                  | Código que se ejecutará en la sesión del navegador.                    |
| `language` | `String` (`"python"`, `"node"`, `"bash"`) | Lenguaje de ejecución. Si se omite, el valor predeterminado es `node`. |
| `timeout`  | `Integer`                                 | Tiempo de espera de la ejecución en segundos (1-300).                  |
| `origin`   | `String`                                  | Solo para uso interno. Reservado para la instrumentación de Firecrawl. |

<div id="notes">
  ## Notas
</div>

* La API `interact` del SDK de Java solo admite `code`. Otros SDK exponen `prompt` por comodidad.
* Alias obsoleto: `scrapeExecute` apunta a `interact` y `deleteScrapeBrowser` apunta a `stopInteractiveBrowser`.
