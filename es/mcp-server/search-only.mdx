El [Firecrawl MCP Server](/es/mcp-server) en `https://mcp.firecrawl.dev/v2/mcp` es el endpoint predeterminado y ofrece el conjunto completo de herramientas de Firecrawl.

<div id="endpoint">
  ## Endpoint
</div>

```text
https://mcp.firecrawl.dev/v2/mcp-search
```

Toda solicitud requiere autenticación: conéctate con OAuth usando el endpoint directamente, o incluye una Firecrawl API key en la cabecera `Authorization`.

```json
{
  "mcpServers": {
    "firecrawl-search": {
      "url": "https://mcp.firecrawl.dev/v2/mcp-search",
      "headers": {
        "Authorization": "Bearer fc-YOUR_API_KEY"
      }
    }
  }
}
```

<div id="available-tools">
  ## Herramientas disponibles
</div>

El endpoint de solo búsqueda expone exactamente estas seis herramientas de solo lectura:

| Herramienta                         | Propósito                                                            |
| ----------------------------------- | -------------------------------------------------------------------- |
| `firecrawl_search`                  | buscar en la web y devolver resultados ordenados                     |
| `firecrawl_research_search_papers`  | Buscar artículos de investigación                                    |
| `firecrawl_research_inspect_paper`  | Obtener metadatos de un artículo                                     |
| `firecrawl_research_related_papers` | Encontrar artículos relacionados a través de enlaces de citación     |
| `firecrawl_research_read_paper`     | Leer pasajes de texto completo de un artículo                        |
| `firecrawl_research_search_github`  | Buscar en el historial de repositorios públicos y en archivos README |

<div id="differences-from-the-full-mcp-endpoint">
  ## Diferencias con el endpoint completo de MCP
</div>

* El conjunto de herramientas es fijo y de solo lectura.
* Las herramientas scrape, map, crawl, extract, agent, interact, parse, monitor y feedback no están disponibles.
* `firecrawl_search` no acepta `scrapeOptions` y no puede solicitar la recuperación del contenido de la página.
