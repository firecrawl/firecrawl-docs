---
title: "Generar LLMs.txt"
openapi: "v1-openapi POST /llmstxt"
---

El punto de conexión para generar LLMs.txt crea los archivos LLMs.txt y LLMs-full.txt para cualquier sitio web. Estos archivos ofrecen un formato estructurado y optimizado para LLMs del contenido del sitio, lo que facilita que los modelos de lenguaje comprendan y procesen la información.

¿Buscas el punto de conexión de estado? Consulta [Estado de LLMs.txt](/es/api-reference/endpoint/llmstxt-get).

<div id="response-structure">
  ### Estructura de la respuesta
</div>

La respuesta incluye:

* **success**: Booleano que indica si la solicitud se realizó correctamente
* **id**: Identificador único del trabajo de generación

<div id="output-format">
  ### Formato de salida
</div>

1. **LLMs.txt**

   * Resumen conciso y estructurado del sitio web
   * Incluye enlaces clave y descripciones
   * Formateado en Markdown para facilitar el parseo
   * Ejemplo:

     ```markdown
     # http://example.com llms.txt

     - [Título de la página](https://example.com/page): Descripción breve
     - [Otra página](https://example.com/another): Otra descripción
     ```

2. **LLMs-full.txt** (cuando showFullText es true)

   * Contiene el contenido completo de las páginas procesadas
   * Mantiene la estructura jerárquica
   * Incluye información más detallada
   * Ejemplo:

     ```markdown
     # http://example.com llms-full.txt

     ## Título de la página

     Contenido completo de la página...

     ## Otra página

     Contenido más detallado...
     ```

<div id="billing">
  ### Facturación
</div>

La facturación se basa en llamadas a la API y en las URL procesadas:

* Costo base: 1 crédito por llamada a la API
* Adicional: 1 crédito por URL procesada
* Controla los costos por URL con el parámetro `maxUrls`
