---
title: "Bloquear el registro del agente"
description: "Bloquea y desactiva una clave de API aprovisionada por un agente."
---

Bloquea un registro pendiente de un agente y desactiva de inmediato la clave de API, además de bloquear la cuenta en entorno aislado. Úsalo si el agente creó una clave sin autorización. No se requiere autenticación: el token de verificación sirve como prueba.

<div id="headers">
  ## Encabezados
</div>

| Encabezado     | Valor              |
| -------------- | ------------------ |
| `Content-Type` | `application/json` |

<div id="request-body">
  ## Cuerpo de la solicitud
</div>

| Parámetro            | Tipo   | Obligatorio | Descripción                                                                 |
| -------------------- | ------ | ----------- | --------------------------------------------------------------------------- |
| `agent_signup_token` | string | **Sí**      | El token de verificación del enlace de confirmación del correo electrónico. |

<div id="response">
  ## respuesta
</div>

| Field | Type | descripción |
|-------|------|-------------|
| `success` | boolean | Indica si el bloque se ejecutó correctamente |
| `message` | string | Mensaje de resultado legible para humanos |

<div id="example-request">
  ### Ejemplo de solicitud
</div>

```bash
curl -X POST "https://api.firecrawl.dev/v2/agent-signup/block" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_signup_token": "a1b2c3d4..."
  }'
```


<div id="example-response">
  ### Respuesta de ejemplo
</div>

```json
{
  "success": true,
  "message": "Agent key has been blocked and disabled."
}
```


<div id="what-happens-on-block">
  ## Qué ocurre al bloquear
</div>

* La API key del agent se **elimina**.
* El equipo en sandbox queda **vetado**.
* El registro del patrocinador se marca como `"blocked"`, lo que impide nuevos registros de agent para ese email.
* Cualquier solicitud posterior a la API que use la API key bloqueada devolverá `403`.

<div id="error-responses">
  ## Respuestas de error
</div>

| Estado | Error                              | Cuándo                                            |
| ------ | ---------------------------------- | ------------------------------------------------- |
| `400`  | Solicitud no válida                | Falta `agent_signup_token`                        |
| `404`  | Token de verificación no válido    | Token no encontrado                               |
| `409`  | Esta clave de agent ya se confirmó | No se puede bloquear una clave que ya se confirmó |
