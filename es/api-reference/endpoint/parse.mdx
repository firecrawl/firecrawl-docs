Convierte cualquier documento local o de acceso público en datos limpios para agentes de IA a través de la API.

* Convierte archivos PDF, DOCX y XLSX en Markdown o JSON
* Conserva el orden de lectura y las tablas
* Compatibilidad con retención de datos cero

<div id="how-to-use-document-parsing">
  ## Cómo usar el procesamiento de documentos
</div>

El procesamiento de documentos en Firecrawl funciona de dos formas:

1. **Procesamiento basado en URL (`/v2/scrape`)**: proporciona una URL que apunte a un tipo de documento compatible. Consulta el endpoint [Scrape](/es/api-reference/endpoint/scrape) para obtener más información.
2. **Procesamiento mediante carga de archivos (`/v2/parse`)**: carga directamente los bytes del archivo con `multipart/form-data`.

En el procesamiento basado en URL, Firecrawl detecta automáticamente el tipo de archivo a partir de la extensión o del tipo de contenido.
