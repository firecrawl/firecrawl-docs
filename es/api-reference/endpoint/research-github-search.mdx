---
title: "Buscar en el historial de GitHub"
openapi: "/es/api-reference/v2-openapi.json GET /search/research/github"
---

Busca en el historial de issues y PR de GitHub, en las discusiones y en los archivos README de los repositorios. Los resultados incluyen metadatos del repositorio, URL, fragmentos y contenido markdown que coincida, cuando esté disponible.

Para ver un resumen del flujo de trabajo, consulta la [guía de Research Index](/es/features/research).
