  ## Novedades en la v2
</div>

<div id="crawl-params-preview">
  ### Vista previa de parámetros de rastreo
</div>

```json
{
  "url": "https://example.com",
  "prompt": "Explora todo el sitio web, pero excluye /admin y /api"
}   
```


<div id="response">
  ### Respuesta
</div>

```json
{
  "url": "https://example.com",
  "prompt": "Rastrea todo el sitio",
  "crawlEntireDomain": true,
  "excludePaths": ["/admin/.*", "/api/.*"],
  "sitemap": "incluir"
}
```
