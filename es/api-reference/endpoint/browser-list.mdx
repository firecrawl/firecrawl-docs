---
title: "Listar sesiones del navegador"
description: "Obtén una lista de todas las sesiones del navegador, con opción de
  filtrarlas por estado."
openapi: '/es/api-reference/v2-openapi.json GET /browser'
---

<div id="headers">
  ## Cabeceras
</div>

| Cabecera | Valor |
|--------|-------|
| `Authorization` | `Bearer <API_KEY>` |

<div id="query-parameters">
  ## Parámetros de consulta
</div>

| Parámetro | Tipo | Requerido | Descripción |
|-----------|------|----------|-------------|
| `status` | string | No | Filtra por el estado de la sesión: `"active"` o `"destroyed"` |

<div id="response">
  ## Respuesta
</div>

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `success` | boolean | Indica si la petición se realizó correctamente |
| `sessions` | array | Lista de objetos de sesión |

<div id="session-object">
  ### Objeto de sesión
</div>

| Campo | Tipo | Descripción |
|-------|------|-------------|
| `id` | string | Identificador único de sesión |
| `status` | string | Estado actual de la sesión (`"active"` o `"destroyed"`) |
| `cdpUrl` | string | URL de WebSocket para conexiones CDP |
| `liveViewUrl` | string | URL para ver la sesión en tiempo real |
| `createdAt` | string | Marca de tiempo ISO 8601 de la creación de la sesión |
| `lastActivity` | string | Marca de tiempo ISO 8601 de la última actividad |

<div id="example-request">
  ### Ejemplo de solicitud
</div>

```bash
curl -X GET "https://api.firecrawl.dev/v2/browser?status=active" \
  -H "Authorization: Bearer $FIRECRAWL_API_KEY"
```


<div id="example-response">
  ### Ejemplo de respuesta
</div>

```json
{

  "sessions": [

      "createdAt": "2025-06-01T12:00:00Z",
      "lastActivity": "2025-06-01T12:05:30Z"
    }
  ]
}
```
