<div id="load-website-with-firecrawl">
  ## Cargar un sitio web con Firecrawl
</div>

Para obtener todos los datos de una página web y asegurarnos de que estén en el formato más limpio, usaremos [Firecrawl](https://firecrawl.dev). Se encarga de sortear sitios bloqueados por JS, extraer el contenido principal y generar una salida en un formato legible por LLM para mejorar la precisión.

Así es como rastrearemos una URL de un sitio web usando Firecrawl. También configuraremos `pageOptions` para extraer solo el contenido principal (`onlyMainContent: True`) de la página, excluyendo la navegación, el pie de página, etc.

```python
from firecrawl import FirecrawlApp  # Importación de FireCrawlLoader

url = "https://about.fb.com/news/2024/04/introducing-our-open-mixed-reality-ecosystem/"

firecrawl = FirecrawlApp(
    api_key="fc-YOUR_FIRECRAWL_API_KEY",
)
page_content = firecrawl.scrape_url(url=url,  # URL de destino para rastrear
    params={
        "pageOptions":{
            "onlyMainContent": True # Ignorar navegación, pies de página, etc.
        }
    })
print(page_content)
```

Perfecto, ahora tenemos datos limpios del sitio web, listos para enviarlos al LLM para extraer datos.

<div id="extraction-and-generation">
  ## Extracción y generación
</div>

Ahora que tenemos los datos del sitio web, usemos Groq para extraer la información que necesitamos. Utilizaremos el modelo Llama 3 de Groq en modo JSON para seleccionar campos específicos del contenido de la página.

En este ejemplo usamos el modelo Llama 3 8B. Puedes usar modelos más grandes para obtener mejores resultados.

```python
import json
from groq import Groq

client = Groq(
    api_key="gsk_YOUR_GROQ_API_KEY",  # Nota: reemplaza 'API_KEY' por tu clave de API real de Groq
)

# Aquí definimos los campos que queremos extraer del contenido de la página
extract = ["summary","date","companies_building_with_quest","title_of_the_article","people_testimonials"]

completion = client.chat.completions.create(
    model="llama3-8b-8192",
    messages=[
        {
            "role": "system",
            "content": "Eres un asesor legal que extrae información de documentos en formato JSON."
        },
        {
            "role": "user",
            # Aquí pasamos el contenido de la página y los campos que queremos extraer
            "content": f"Extrae la siguiente información de la documentación proporcionada:\nContenido de la página:\n\n{page_content}\n\nInformación a extraer: {extract}"
        }
    ],
    temperature=0,
    max_tokens=1024,
    top_p=1,
    stream=False,
    stop=None,
    # Establecemos el formato de respuesta como un objeto JSON
    response_format={"type": "json_object"}
)


# Imprime la respuesta JSON con formato
dataExtracted = json.dumps(str(completion.choices[0].message.content), indent=4)

print(dataExtracted)
```
