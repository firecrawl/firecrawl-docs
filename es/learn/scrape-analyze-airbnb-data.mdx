> Nota: este ejemplo usa la [versión v0 de la API de Firecrawl](/es/v0/introduction). Puedes instalar la versión 0.0.20 para el SDK de Python o la 0.0.36 para el SDK de Node.

> [Repositorio de GitHub (https://github.com/e2b-dev/e2b-cookbook)](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl)

Este ejemplo muestra cómo extraer datos de Airbnb y analizarlos con [Firecrawl](https://www.firecrawl.dev/) y el [Code Interpreter SDK](https://github.com/e2b-dev/code-interpreter) de E2B.

No escribiremos el código de análisis; en su lugar, haremos que la IA de Anthropic lo escriba por nosotros. Usaremos el Code Interpreter SDK de E2B para ejecutar el código de forma segura y obtener los resultados.

<div id="prerequisites">
  ## Requisitos previos
</div>

* Node.js instalado en tu equipo
* Una clave de API de E2B
* Una clave de API de Firecrawl
* Una clave de API de Anthropic

<div id="setup-run">
  ## Configuración y ejecución
</div>

<div id="1-clone-the-repository">
  ### 1. Clona el repositorio
</div>

Este es el [repositorio de GitHub](https://github.com/e2b-dev/e2b-cookbook/tree/main/examples/scrape-and-analyze-airbnb-data-with-firecrawl) para este ejemplo.

```
git clone https://github.com/e2b-dev/e2b-cookbook.git
cd e2b-cookbook/examples/scrape-and-analyze-airbnb-data-with-firecrawl
```

<div id="2-install-dependencies">
  ### 2. Instala las dependencias
</div>

```
npm install
```

<div id="3-set-up-env">
  ### 3. Configura `.env`
</div>

1. Copia `.env.template` en `.env`
2. Obtén la [clave de API de E2B](https://e2b.dev/docs/getting-started/api-key)
3. Obtén la [clave de API de Firecrawl](https://firecrawl.dev)
4. Obtén la [clave de API de Anthropic](https://anthropic.com)

<div id="4-run-the-example">
  ### 4. Ejecuta el ejemplo
</div>

```
npm run start
```
