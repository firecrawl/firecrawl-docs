<div id="welcome-to-firecrawl">
  ## Bienvenido a Firecrawl
</div>

[Firecrawl](https://firecrawl.dev?ref=github) es un servicio de API que toma una URL, la rastrea y la convierte en Markdown limpio. Rastreamos todas las subpáginas accesibles y te entregamos Markdown limpio para cada una. No se requiere sitemap.

<div id="how-to-use-it">
  ## ¿Cómo usarlo?
</div>

Ofrecemos una API fácil de usar con nuestra versión alojada. Puedes encontrar el playground y la documentación [aquí](https://firecrawl.dev/playground). También puedes alojar el backend por tu cuenta si lo prefieres.

<div id="features">
  ### Funcionalidades
</div>

* [**Scrape**](#scraping): extrae una URL y obtiene su contenido en un formato listo para LLM (Markdown, resumen, datos estructurados mediante [modo JSON](#json-mode), captura de pantalla, HTML)
* [**Crawl**](#crawling): extrae todas las URL de una página web y devuelve el contenido en un formato listo para LLM
* [**Map**](/es/features/map): ingresa un sitio web y obtén todas las URL del sitio: extremadamente rápido
* [**Search**](/es/features/search): busca en la web y obtén el contenido completo de los resultados
* [**Extract**](/es/features/extract): obtén datos estructurados de una sola página, múltiples páginas o sitios completos con IA.

<div id="powerful-capabilities">
  ### Capacidades avanzadas
</div>

* **Formatos listos para LLM**: Markdown, resumen, datos estructurados, captura de pantalla, HTML, enlaces, metadatos
* **Lo complejo**: proxies, mecanismos antibot, contenido dinámico (renderizado con JS), análisis del resultado, orquestación
* **Ultrarrápido**: Obtén resultados en segundos—diseñado para la velocidad y casos de uso de alto rendimiento.
* **Personalización**: excluir etiquetas, rastrear detrás de muros de autenticación con encabezados personalizados, profundidad máxima de rastreo, etc.
* **Análisis de medios**: PDF, DOCX, imágenes.
* **Fiabilidad ante todo**: diseñado para obtener los datos que necesitas, por difícil que sea.
* **Acciones**: hacer clic, desplazarse, introducir datos, esperar y más antes de extraer datos

Puedes encontrar todas las capacidades de Firecrawl y cómo usarlas en nuestra [documentación](https://docs.firecrawl.dev/api-reference/v2-introduction)

<div id="open-source-vs-cloud">
  ## Código abierto vs nube
</div>

Firecrawl es de código abierto y está disponible bajo la [licencia AGPL-3.0](https://github.com/mendableai/firecrawl/blob/main/LICENSE).

Para ofrecer el mejor producto posible, proporcionamos una versión alojada de Firecrawl junto con nuestra oferta de código abierto. La solución en la nube nos permite innovar continuamente y mantener un servicio sostenible y de alta calidad para todos los usuarios.

Firecrawl Cloud está disponible en [firecrawl.dev](https://firecrawl.dev) y ofrece una serie de funciones que no están disponibles en la versión de código abierto:
