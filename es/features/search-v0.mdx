<div id="searching-the-web-and-scraping-the-results-with-firecrawl">
  ## Buscar en la web y extraer los resultados con Firecrawl
</div>

Firecrawl integra su API de búsqueda con su sólida infraestructura de scraping para ofrecer una función unificada de búsqueda y extracción mediante un único punto de conexión. Aquí tienes el porqué:

1. **Consulta de búsqueda unificada:**
   Los usuarios envían una consulta de búsqueda a través del punto de conexión de búsqueda.

2. **Extracción automatizada de resultados:**
   Firecrawl procesa automáticamente los resultados de la búsqueda y utiliza sus capacidades de scraping para extraer datos de cada página de resultados.

3. **Entrega de datos:**
   Los datos extraídos de todas las páginas de resultados se compilan y se entregan en un markdown limpio, listo para usar.

Esta integración permite a los usuarios realizar búsquedas web de forma eficiente y obtener datos completos, extraídos de múltiples fuentes con un esfuerzo mínimo.

Para más información, consulta la [documentación del punto de conexión /search](https://docs.firecrawl.dev/api-reference/endpoint/search).

<div id="search-any-query">
  ## Busca cualquier consulta
</div>

<div id="search-endpoint">
  ### punto de conexión /search
</div>

Se usa para buscar en la web, obtener los resultados más relevantes, extraer cada página y devolver el markdown.

```bash
curl -X POST https://api.firecrawl.dev/v0/search \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer TU_API_KEY' \
    -d '{
      "query": "firecrawl",
      "pageOptions": {
        "fetchPageContent": true // false para una API de SERP rápida
      }
    }'
```

```json
{
  "success": true,
  "data": [
    {
      "url": "https://docs.firecrawl.dev",
      "markdown": "# Contenido en Markdown",
      "provider": "web-scraper",
      "metadata": {
        "title": "Firecrawl | Extrae la web de forma fiable para tus LLM",
        "description": "IA para CX y ventas"
        "language": null,
        "sourceURL": "https://docs.firecrawl.dev/"
      }
    }
  ]
}
```

<div id="with-python-sdk">
  ### Con el SDK de Python
</div>

<div id="installing-python-sdk">
  #### Instalación del SDK para Python
</div>

```bash
pip install firecrawl-py
```

#### Realizar una búsqueda

```python
from firecrawl import FirecrawlApp

app = FirecrawlApp(api_key="YOUR_API_KEY")

result = app.search(query="What is firecrawl?")
```

La respuesta será similar a la que se muestra en el comando curl de arriba.


<div id="with-javascript-sdk">

<div id="installing-javascript-sdk">

```javascript
import FirecrawlApp from '@mendable/firecrawl-js';

// Inicializa la FirecrawlApp con tu clave de API

const result = await app.search('What is firecrawl?');
```

La respuesta será similar a la mostrada en el comando curl anterior.


<div id="with-go-sdk">
  ### Con el SDK de Go
</div>

<div id="installing-go-sdk">
  #### Instalar el SDK de Go
</div>

```bash
go get github.com/mendableai/firecrawl-go
```

#### Realizar una búsqueda

```go
import (
  "fmt"
  "log"

  "github.com/mendableai/firecrawl-go"
)

func main() {
  app, err := firecrawl.NewFirecrawlApp("YOUR_API_KEY")
  if err != nil {
      log.Fatalf("Failed to initialize FirecrawlApp: %v", err)
  }

  query := "What is firecrawl?"
  searchResult, err := app.Search(query)
  if err != nil {
    log.Fatalf("Failed to search: %v", err)
  }
  fmt.Println(searchResult)
}
```

<div id="search-a-query">
  #### Ejecutar una consulta de búsqueda
</div>

```rust
async fn main() {
  let api_key = "TU_CLAVE_API";
  let api_url = "https://api.firecrawl.dev";
  let app = FirecrawlApp::new(api_key, api_url).expect("Failed to initialize FirecrawlApp");

  let query = "What is firecrawl?";
  let search_result = app.search(query).await;

  match search_result {
    Ok(data) => println!("Search Result: {}", data),
    Err(e) => eprintln!("Failed to search: {}", e),
  }
}
```
