<Info>
  **Punto de conexión de búsqueda de pago por uso**

  Este punto de conexión te permite usar las potentes capacidades de búsqueda de Firecrawl sin suscripción, pagando por búsqueda con criptomonedas. Ideal para búsquedas puntuales o para probar antes de contratar una suscripción.
</Info>

<Warning>
  **Función experimental**

  Este es un punto de conexión experimental. Las funciones, los precios y la disponibilidad pueden cambiar sin previo aviso. Recomendamos usar el punto de conexión estándar de búsqueda basado en suscripción para casos de uso en producción.
</Warning>

<div id="authentication">
  ## Autenticación
</div>

Este punto de conexión requiere un encabezado de pago especial en lugar de la clave de API estándar:

```
X-Payment: {{paymentHeader}}
```

<div id="how-it-works">
  ## Cómo funciona
</div>

El punto de conexión de Búsqueda bajo demanda combina la búsqueda web (SERP) con las capacidades de scraping de Firecrawl para devolver el contenido completo de la página para cualquier consulta, igual que el punto de conexión `/search` normal, pero con precios de pago por uso.

Incluye `scrapeOptions` con `formats: ["markdown"]` para obtener el contenido completo en Markdown de cada resultado de búsqueda; de lo contrario, por defecto recibirás solo los resultados del SERP (url, title, description).

<div id="request-parameters">
  ### Parámetros de la solicitud
</div>

Se admiten todos los parámetros del punto de conexión estándar `/search`, con la siguiente excepción:

<Note>
  **Limitación importante**: El parámetro `limit` se limita a **un máximo de 10 resultados** para el punto de conexión de wallet (frente a 100 en el punto de conexión estándar).
</Note>

Para ver la lista completa de parámetros disponibles, operadores de consulta admitidos y sus descripciones, consulta la [documentación del punto de conexión de búsqueda estándar](/es/api-reference/endpoint/search).

<div id="example-request">
  ## Ejemplo de solicitud
</div>

```bash
curl -X POST https://api.firecrawl.dev/v1/x402/search \
  -H 'Content-Type: application/json' \
  -H 'Authorization: Bearer <token>' \
  -H 'X-Payment: {{paymentHeader}}' \
  -d '{
    "query": "Calendario de la Premier League para la temporada 2025/26",
    "limit": 10,  # Se permiten como máximo 10 resultados
    "scrapeOptions": {
      "formats": ["markdown"],
      "onlyMainContent": true
    }
  }'
```

<div id="response">
  ## Respuesta
</div>

El formato de la respuesta es idéntico al del punto de conexión `/search` estándar:

```json
{
  "success": true,
  "data": [
    {
      "title": "Título de la página",
      "description": "Descripción de la página del resultado de búsqueda",
      "url": "https://example.com",
      "markdown": "# Contenido de la página\n\nContenido en Markdown si se solicitó la extracción...",
      "metadata": {
        "title": "Título de la página",
        "description": "Descripción meta"
        "sourceURL": "https://example.com",
        "statusCode": 200
      }
    }
  ]
}
```
