<div id="how-it-works">
  ## Cómo funciona
</div>

Usa las capacidades de extracción de Firecrawl para construir sistemas de observabilidad para tus sitios web. Extrae el contenido de las páginas, analiza los cambios en el tiempo, valida las implementaciones y crea flujos de monitoreo que garanticen que tus sitios funcionen correctamente.

<div id="what-you-can-monitor">
  ## Qué puedes monitorear
</div>

* **Disponibilidad**: Tiempo en línea, tiempos de respuesta, tasas de error
* **Contenido**: Cambios de texto, actualizaciones de imágenes, cambios de diseño
* **Rendimiento**: Tiempos de carga de la página, tamaños de recursos, Core Web Vitals
* **Seguridad**: Certificados SSL, encabezados de seguridad, configuraciones erróneas
* **Salud SEO**: Metadatos, datos estructurados, validez del sitemap

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Cómo ayuda Firecrawl con el monitoreo de sitios web?">
    Firecrawl extrae el contenido y la estructura de los sitios web bajo demanda. Crea sistemas de monitoreo que llamen a la API de Firecrawl para comprobar páginas, comparar los datos extraídos con referencias y activar tus propias alertas cuando haya cambios.
  </Accordion>

  <Accordion title="¿Puedo monitorear aplicaciones con uso intensivo de JavaScript?">
    Sí. Firecrawl renderiza JavaScript por completo, lo que lo hace ideal para monitorear SPAs modernas, apps en React y contenido dinámico. Capturamos la página tal como la ven los usuarios, no solo el HTML en bruto.
  </Accordion>

  <Accordion title="¿Qué tan rápido puedo detectar problemas en un sitio web?">
    Firecrawl extrae datos en tiempo real cuando se invoca. Diseña tu sistema de monitoreo para revisar los sitios con la frecuencia que necesites: desde minuto a minuto para páginas críticas hasta diariamente para revisiones rutinarias.
  </Accordion>

  <Accordion title="¿Puedo validar elementos específicos de la página?">
    Sí. Usa el punto de conexión /extract para obtener elementos específicos como precios, niveles de inventario o contenido crítico. Implementa lógica de validación en tu sistema de monitoreo para verificar que la información importante esté presente y sea correcta.
  </Accordion>

  <Accordion title="¿Cómo puedo integrar Firecrawl con sistemas de alertas?">
    Firecrawl ofrece webhooks que puedes usar para crear integraciones con tus herramientas de alertas. Envía los datos extraídos a PagerDuty, Slack, correo electrónico o cualquier plataforma de monitoreo mediante conectores que procesen las respuestas de Firecrawl.
  </Accordion>
</AccordionGroup>
