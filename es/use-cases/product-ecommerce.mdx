<div id="how-it-works">
  ## Cómo funciona
</div>

Convierte sitios de comercio electrónico en datos de productos estructurados. Supervisa los precios de la competencia en tiempo real, controla los niveles de inventario en distintos proveedores y migra catálogos de productos entre plataformas sin complicaciones.

<div id="what-you-can-extract">
  ## Qué puedes extraer
</div>

* **Datos de producto**: título, SKU, especificaciones, descripciones, categorías
* **Precios**: precio actual, descuentos, envío, impuestos
* **Inventario**: niveles de existencias, disponibilidad, plazos de entrega
* **Reseñas**: valoraciones, comentarios de clientes, secciones de preguntas y respuestas

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Cómo puedo seguir los cambios de precios de la competencia?">
    Cree un sistema de monitoreo con la API de Firecrawl para extraer precios a intervalos regulares. Compare los datos extraídos a lo largo del tiempo para identificar tendencias de precios, promociones y posicionamiento competitivo.
  </Accordion>

  <Accordion title="¿Puedo extraer variantes de producto (talla, color, etc.)?">
    Sí, Firecrawl puede extraer todas las variantes de producto, incluidas la talla, el color y otras opciones. Estructure los datos con esquemas personalizados para capturar toda la información de las variantes.
  </Accordion>

  <Accordion title="¿Cómo manejo precios dinámicos o específicos para cada usuario?">
    Para precios dinámicos, puede usar el renderizado de JavaScript de Firecrawl para capturarlos después de que se carguen. Para precios específicos del usuario, configure encabezados de autenticación en sus solicitudes.
  </Accordion>

  <Accordion title="¿Puedo extraer datos de distintas plataformas de comercio electrónico?">
    Sí. Firecrawl puede extraer datos de cualquier sitio de comercio electrónico de acceso público. Los usuarios extraen con éxito de Shopify, WooCommerce, Magento, BigCommerce y tiendas personalizadas.
  </Accordion>

  <Accordion title="¿Puede Firecrawl manejar la paginación y el desplazamiento infinito?">
    Sí. Firecrawl puede navegar por listados de productos paginados y manejar mecanismos de desplazamiento infinito para extraer catálogos de productos completos, asegurando que no se omita ningún producto durante la extracción.
  </Accordion>
</AccordionGroup>
