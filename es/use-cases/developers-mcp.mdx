<div id="how-it-works">
  ## Cómo funciona
</div>

Integra Firecrawl directamente en tu flujo de trabajo de desarrollo con IA. Investiga documentación, obtén especificaciones de API y accede a datos web sin salir de tu entorno de desarrollo mediante Model Context Protocol (MCP).

<div id="why-developers-choose-firecrawl-mcp">
  ## Por qué los desarrolladores eligen Firecrawl MCP
</div>

<div id="build-smarter-ai-assistants">
  ### Crea asistentes de IA más inteligentes
</div>

Dale a tu IA acceso en tiempo real a documentación, APIs y recursos web. Reduce la información desactualizada y las alucinaciones proporcionando a tu asistente los datos más recientes.

<div id="zero-infrastructure-required">
  ### Sin necesidad de infraestructura
</div>

Sin servidores que gestionar ni rastreadores que mantener. Configúralo una sola vez y tu asistente de IA podrá acceder a sitios web al instante a través del Model Context Protocol.

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Qué asistentes de IA son compatibles con MCP?">
    Actualmente, Claude Desktop y Cursor tienen compatibilidad nativa con MCP. Cada vez más asistentes de IA están añadiendo compatibilidad. También puedes usar el SDK de MCP para crear integraciones personalizadas.
  </Accordion>

  <Accordion title="¿Puedo usar MCP en VS Code u otros IDE?">
    VS Code y otros IDE pueden usar MCP mediante extensiones de la comunidad o integraciones de terminal. La compatibilidad nativa varía según el IDE. Consulta nuestro [repositorio de GitHub](https://github.com/mendableai/firecrawl-mcp-server) para guías de configuración específicas por IDE.
  </Accordion>

  <Accordion title="¿Cómo puedo almacenar en caché documentos de acceso frecuente?">
    El servidor MCP almacena automáticamente en caché las respuestas durante 15 minutos. Puedes configurar la duración de la caché en la configuración de tu servidor MCP o implementar una lógica de caché personalizada.
  </Accordion>

  <Accordion title="¿Existe un límite de tasa para las solicitudes de MCP?">
    Las solicitudes de MCP usan tus límites de tasa estándar de la API de Firecrawl. Recomendamos agrupar solicitudes relacionadas y usar caché para documentación de acceso frecuente.
  </Accordion>

  <Accordion title="¿Cómo configuro MCP con mi clave de API de Firecrawl?">
    Sigue nuestra [guía de configuración](https://github.com/mendableai/firecrawl-mcp-server#installation) para configurar MCP. Deberás añadir tu clave de API de Firecrawl al archivo de configuración de MCP. El proceso toma solo unos minutos.
  </Accordion>
</AccordionGroup>
