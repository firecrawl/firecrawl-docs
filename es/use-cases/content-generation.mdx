<div id="how-it-works">
  ## Cómo funciona
</div>

Firecrawl extrae información de sitios web en múltiples formatos —incluidos HTML estructurado, Markdown, JSON y capturas de pantalla—. También puede capturar imágenes y mostrar noticias relevantes como parte de tu solicitud. Esto significa que tu contenido de IA está basado en hechos y enriquecido visualmente con el contexto más reciente.

<div id="what-you-can-create">
  ## Lo que puedes crear
</div>

* **Sales Decks**: Presentaciones comerciales personalizadas con datos de prospectos
* **Email Campaigns**: Campañas de correo personalizadas a escala
* **Marketing Content**: Publicaciones de blog e informes basados en datos
* **Social Media**: Generación de contenido basada en tendencias y noticias
* **Documentation**: Contenido técnico que se actualiza automáticamente
* **Newsletters**: Actualizaciones seleccionadas de noticias del sector y de competidores
* **Visual Content**: Publicaciones e informes enriquecidos con imágenes y capturas de pantalla extraídas

<div id="faqs">
  ## Preguntas frecuentes
</div>

<AccordionGroup>
  <Accordion title="¿Cómo garantiza Firecrawl la precisión de los datos para la creación de contenido?">
    Firecrawl extrae datos directamente de los sitios web de origen, preservando la estructura y el contexto originales del contenido. Todos los datos extraídos incluyen las URL de origen y marcas de tiempo para su verificación.
  </Accordion>

  <Accordion title="¿Qué datos puede proporcionar Firecrawl para la generación de contenido?">
    Firecrawl proporciona markdown limpio, JSON estructurado, HTML, imágenes y capturas de pantalla de sitios web. Estos datos extraídos sirven como base factual para tus flujos de trabajo de generación de contenido.
  </Accordion>

  <Accordion title="¿Puede Firecrawl manejar imágenes y fuentes de noticias?">
    Sí. Firecrawl puede extraer imágenes, tomar capturas de pantalla y obtener contenido de sitios de noticias. Esto te permite crear contenido visualmente atractivo y mantenerte al día con las novedades del sector.
  </Accordion>

  <Accordion title="¿De qué tipos de sitios web puede extraer Firecrawl?">
    Firecrawl destaca en la extracción de sitios web de empresas, sitios de noticias, blogs y documentación. Los sitios con HTML estructurado y jerarquías de contenido claras ofrecen los resultados de extracción más limpios.
  </Accordion>

  <Accordion title="¿Cómo puedo usar Firecrawl para la extracción de datos a gran escala?">
    Usa las API de scraping por lotes y de rastreo de Firecrawl para extraer datos de múltiples sitios web de forma eficiente. Procesa cientos de URL en paralelo para crear conjuntos de datos completos para tus flujos de trabajo de contenido.
  </Accordion>
</AccordionGroup>
