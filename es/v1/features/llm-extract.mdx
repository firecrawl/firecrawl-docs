<div id="scrape-and-extract-structured-data-with-firecrawl">
  ## Raspa y extrae datos estructurados con Firecrawl
</div>

{/* <Warning>Scrape LLM Extract quedará obsoleto en versiones futuras. Usa el nuevo punto de conexión [Extract](/features/extract).</Warning> */}

Firecrawl usa IA para obtener datos estructurados de páginas web en 3 pasos:

1. **Define el esquema:**
   Indica qué datos necesitas definiendo un esquema JSON (usando el formato de OpenAI) junto con la URL de la página.

2. **Realiza la solicitud:**
   Envía tu URL y el esquema a nuestro punto de conexión /scrape. Consulta cómo hacerlo aquí:
   [Documentación del punto de conexión /scrape](https://docs.firecrawl.dev/api-reference/endpoint/scrape)

3. **Obtén tus datos:**
   Recibe datos limpios y estructurados que se ajustan a tu esquema y que puedes usar de inmediato.

Así, obtener datos web en el formato que necesitas es rápido y sencillo.

<div id="json-options-object">
  ### Objeto de opciones JSON
</div>

El objeto `jsonOptions` admite los siguientes parámetros:

- `schema`: El esquema que se utilizará para la extracción.
- `systemPrompt`: La indicación del sistema que se utilizará para la extracción.
- `prompt`: La indicación que se utilizará para la extracción sin esquema.
