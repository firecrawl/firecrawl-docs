<div id="generate-llmstxt">
  ## generate-llmstxt
</div>

Un paquete de NPX sencillo que genera archivos LLMs.txt desde la CLI usando la API de Firecrawl. Este paquete crea dos archivos en el directorio de salida especificado (por defecto, la carpeta 'public'):

- `llms.txt`: Contiene un resumen del contenido relacionado con LLM
- `llms-full.txt`: Contiene el texto completo

<div id="usage">
  ## Uso
</div>

Puedes ejecutar este paquete con NPX sin instalarlo. Hay dos formas de proporcionar tu clave de la API de Firecrawl:

<div id="1-using-command-line-arguments">
  ### 1. Uso de argumentos de la línea de comandos
</div>

```bash
npx generate-llmstxt --api-key TU_API_KEY_DE_FIRECRAWL
```

<div id="2-using-environment-variables">
  ### 2. Uso de variables de entorno
</div>

Crea un archivo `.env` en la raíz de tu proyecto y añade tu clave de API:

```env
FIRECRAWL_API_KEY=your_api_key_here
```

Luego, ejecuta el comando sin la opción --api-key:

```bash
npx generate-llmstxt
```

<div id="options">
  ### Opciones
</div>

- `-k, --api-key <key>` (opcional si está definida en .env): Tu clave de API de Firecrawl
- `-u, --url <url>` (opcional): URL a analizar (por defecto: https://example.com)
- `-m, --max-urls <number>` (opcional): Número máximo de URL a analizar (por defecto: 50)
- `-o, --output-dir <path>` (opcional): Ruta del directorio de salida (por defecto: 'public')

<div id="examples">
  ### Ejemplos
</div>

```bash
# Uso de un argumento de línea de comandos con el directorio de salida predeterminado
npx generate-llmstxt -k your_api_key -u https://your-website.com -m 20

# Uso de un archivo .env con el directorio de salida predeterminado
npx generate-llmstxt -u https://your-website.com -m 20

# Especificar un directorio de salida personalizado
npx generate-llmstxt -k your_api_key -u https://your-website.com -o custom/path/to/output

# Uso de un archivo .env y un directorio de salida personalizado
npx generate-llmstxt -u https://your-website.com -o content/llms
```

<div id="requirements">
  ## Requisitos
</div>

- Node.js 14 o superior
- Una clave de API válida de Firecrawl (mediante la línea de comandos o un archivo .env)

<div id="output">
  ## Salida
</div>

El paquete creará dos archivos en el directorio de salida que especifiques (por defecto, 'public'):

1. `llms.txt`: Contiene un resumen del contenido relacionado con los LLM
2. `llms-full.txt`: Contiene el texto completo
