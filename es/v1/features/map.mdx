<div id="introducing-map">
  ## Presentamos /map
</div>

La forma más sencilla de pasar de una sola URL a un mapa de todo el sitio web. Es especialmente útil para:

- Cuando necesites pedir al usuario final que elija qué enlaces extraer
- Cuando necesites conocer rápidamente los enlaces de un sitio web
- Cuando necesites extraer páginas de un sitio web relacionadas con un tema específico (usa el parámetro `search`)
- Cuando solo necesites extraer páginas específicas de un sitio web

<div id="map-with-search">
  #### Map con búsqueda
</div>

Map con el parámetro `search` te permite buscar URLs específicas dentro de un sitio web.

```bash cURL
curl -X POST https://api.firecrawl.dev/v1/map \
    -H 'Content-Type: application/json' \
    -H 'Authorization: Bearer YOUR_API_KEY' \
    -d '{
      "url": "https://firecrawl.dev",
      "search": "docs"
    }'
```

La respuesta será una lista ordenada de la más relevante a la menos relevante.

```json
{
  "status": "correcto",
  "links": [
    "https://docs.firecrawl.dev",
    "https://docs.firecrawl.dev/sdks/python",
    "https://docs.firecrawl.dev/learn/rag-llama3",
  ]
}
```
