<div id="welcome-to-firecrawl">
  ## Bienvenido a Firecrawl
</div>

[Firecrawl](https://firecrawl.dev?ref=github) es un servicio de API que toma una URL, la rastrea y la convierte en markdown limpio. Rastreamos todas las subpáginas accesibles y te entregamos markdown limpio para cada una. No necesitas un sitemap.

<div id="how-to-use-it">
  ## ¿Cómo utilizarlo?
</div>

Ofrecemos una API fácil de usar con nuestra versión alojada. Puedes encontrar el playground y la documentación [aquí](https://firecrawl.dev/playground). También puedes alojar el backend por tu cuenta si lo prefieres.

Consulta los siguientes recursos para comenzar:

<div id="features">
  ### Funcionalidades
</div>

* [**Scrape**](#scraping): extrae una URL y devuelve su contenido en formato listo para LLM (markdown, resumen, datos estructurados mediante [modo JSON](#json-mode), captura de pantalla, HTML)
* [**Crawl**](#crawling): extrae todas las URL de un sitio y devuelve el contenido en formato listo para LLM
* [**Map**](/es/features/map): ingresa un sitio web y obtén todas sus URL, de forma extremadamente rápida
* [**Search**](/es/features/search): busca en la web y obtiene el contenido completo de los resultados
* [**Extract**](/es/features/extract): obtiene datos estructurados de una sola página, de varias páginas o de sitios completos con IA.

<div id="powerful-capabilities">
  ### Capacidades potentes
</div>

* **Formatos listos para LLM**: markdown, resumen, datos estructurados, captura de pantalla, HTML, enlaces, metadatos
* **Lo difícil**: proxies, mecanismos antibot, contenido dinámico (renderizado con JS), análisis del resultado, orquestación
* **Personalización**: excluir etiquetas, rastrear detrás de muros de autenticación con encabezados personalizados, profundidad máxima de rastreo, etc.
* **Análisis de medios**: PDF, DOCX, imágenes
* **Fiabilidad ante todo**: diseñado para obtener los datos que necesitas, por difícil que sea
* **Acciones**: clic, desplazamiento, entrada de texto, espera y más antes de extraer datos

Puedes encontrar todas las capacidades de Firecrawl y cómo usarlas en nuestra [documentación](https://docs.firecrawl.dev/api-reference/v2-introduction)

<div id="open-source-vs-cloud">
  ## Código abierto vs nube
</div>

Firecrawl es de código abierto y está disponible bajo la [licencia AGPL-3.0](https://github.com/mendableai/firecrawl/blob/main/LICENSE).

Para ofrecer el mejor producto posible, ofrecemos una versión alojada de Firecrawl junto con nuestra versión de código abierto. La solución en la nube nos permite innovar continuamente y mantener un servicio de alta calidad y sostenible para todos los usuarios.

Firecrawl Cloud está disponible en [firecrawl.dev](https://firecrawl.dev) y ofrece una serie de funciones que no están disponibles en la versión de código abierto:
