<div id="whats-new-in-v2">
  ## Novedades de la v2
</div>

<div id="tell-the-crawler-what-you-want">
  ### Indícale al crawler lo que quieres
</div>

Describe qué debe rastrear en inglés sencillo:

```json
{
  "url": "https://example.com",
  "prompt": "Rastrea solo las entradas del blog y la documentación; omite las páginas de marketing"
}
```


<div id="better-sitemap-control">
  ### Mejor control de sitemaps
</div>

v1: activado/desactivado para sitemaps. v2: tres opciones:

- `"include"` - Usar el sitemap + encontrar otras páginas (predeterminado)
- `"skip"` - Ignorar completamente el sitemap
- `"only"` - Solo rastrear las URL del sitemap

<div id="new-crawling-options">
  ### Nuevas opciones de rastreo
</div>

* `crawlEntireDomain` - Rastrear todo el dominio, no solo las páginas secundarias
* `maxDiscoveryDepth` - Controlar la profundidad del rastreo (reemplaza `maxDepth`)

```json
{
  "url": "https://example.com/features",
  "crawlEntireDomain": true,
  "maxDiscoveryDepth": 2,
  "sitemap": "include"
}
```
