<div id="whats-new-in-v2">
  ## Novedades en la v2
</div>

<div id="better-error-messages">
  ### Mensajes de error mejorados
</div>

- Errores claros cuando los prompts en lenguaje natural no funcionan
- Mensajes específicos para problemas con el mapa del sitio y el límite de profundidad
- Mensajes más claros para problemas de rastreo a nivel de dominio

<div id="more-error-details">
  ### Más detalles de errores
</div>

- Errores detallados para problemas en el procesamiento de formatos
- Mensajes específicos para fallos de PDF y del parser
- Mejor depuración para el rastreo basado en ubicación
- Errores claros para problemas de seguimientoDeCambios

<div id="faster-and-easier-debugging">
  ### Depuración más rápida y sencilla
</div>

- Recuperación de errores más ágil en rastreos grandes
- Mejor agrupación de errores
- Contexto de errores más útil

La misma API que en v1, con informes de errores mucho mejores para las funciones de v2.
