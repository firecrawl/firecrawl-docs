<div id="better-sitemap-control">
  ### Mejor control del sitemap
</div>

Tres formas de gestionar los sitemaps:

* `"include"` - Usar el sitemap y encontrar otras páginas (predeterminado)
* `"skip"` - Ignorar el sitemap por completo
* `"only"` - Devolver solo las URL del sitemap

```json
{
  "url": "https://example.com",
  "sitemap": "only"
}
```

<div id="smarter-url-discovery">
  ### Descubrimiento de URL más inteligente
</div>

- Encuentra todas las páginas de un sitio web
- Sin límite de resultados (v1 tenía un tope de 500)
- Más rápido y preciso

La misma API que v1, pero encuentra más páginas y funciona mejor en sitios complejos.
