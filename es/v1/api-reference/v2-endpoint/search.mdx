El endpoint de búsqueda combina la búsqueda web con las capacidades de scraping de Firecrawl para devolver el contenido completo de la página para cualquier consulta.

Incluye `scrapeOptions` con `formats: [{"type": "markdown"}]` para obtener el contenido completo en markdown para cada resultado de búsqueda; de lo contrario, por defecto obtendrás los resultados de búsqueda (url, title, description). También puedes usar otros formatos como `{"type": "summary"}` para contenido resumido.

<div id="supported-query-operators">
  ## Operadores de consulta admitidos
</div>

Ofrecemos una variedad de operadores de consulta que te permiten filtrar mejor tus búsquedas.

| Operador | Funcionalidad | Ejemplos |
---|-|-|
| `""` | Coincidencia exacta con una cadena de texto | `"Firecrawl"`
| `-` | Excluye ciertas palabras clave o niega otros operadores | `-bad`, `-site:firecrawl.dev`
| `site:` | Devuelve solo resultados de un sitio web especificado | `site:firecrawl.dev`
| `inurl:` | Devuelve solo resultados que incluyen una palabra en la URL | `inurl:firecrawl`
| `allinurl:` | Devuelve solo resultados que incluyen varias palabras en la URL | `allinurl:git firecrawl`
| `intitle:` | Devuelve solo resultados que incluyen una palabra en el título de la página | `intitle:Firecrawl`
| `allintitle:` | Devuelve solo resultados que incluyen varias palabras en el título de la página | `allintitle:firecrawl playground`
| `related:` | Devuelve solo resultados relacionados con un dominio específico | `related:firecrawl.dev`

<div id="location-parameter">
  ## Parámetro de ubicación
</div>

Usa el parámetro `location` para obtener resultados de búsqueda geodirigidos. Formato: `"string"`. Ejemplos: `"Germany"`, `"San Francisco,California,United States"`.

Consulta la [lista completa de ubicaciones compatibles](https://firecrawl.dev/search_locations.json) para ver todos los países e idiomas disponibles.

<div id="time-based-search">
  ## Búsqueda por fecha
</div>

Usa el parámetro `tbs` para filtrar los resultados por periodos de tiempo, incluidos rangos de fechas personalizados. Consulta la [documentación de la función de búsqueda](https://docs.firecrawl.dev/features/search#time-based-search) para ver ejemplos detallados y formatos admitidos.
