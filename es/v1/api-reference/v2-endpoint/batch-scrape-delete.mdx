<div id="whats-new-in-v2">
  ## Novedades en v2
</div>

La API de cancelación de scraping por lotes de v2 mantiene la misma funcionalidad que v1, con mayor confiabilidad:

<div id="enhanced-cancellation-process">
  ### Proceso de cancelación mejorado
</div>

- Mejor limpieza y gestión de recursos en operaciones por lotes
- Mejor manejo de la terminación de lotes activos
- Manejo de errores optimizado para operaciones de cancelación de lotes

Los parámetros de la API y el formato de respuesta se mantienen sin cambios respecto a v1, lo que garantiza una migración fluida y una cancelación por lotes más confiable.
