<div id="whats-new-in-v2">
  ## Novedades en la v2
</div>

<div id="better-batch-error-messages">
  ### Mensajes de error por lotes mejorados
</div>

- Errores claros para problemas de procesamiento de formatos en todas las URL
- Mensajes específicos para fallos de PDF y del parser
- Mejor seguimiento de errores para distintos modos de proxy
- Errores claros para problemas de scraping según la ubicación

<div id="more-error-details">
  ### Más detalles de errores
</div>

- Información de depuración para problemas de seguimiento de cambios
- Informes de errores por fallas en la automatización del navegador
- Mejor seguimiento de errores de créditos y límites de recursos
- Errores específicos para problemas en el procesamiento por lotes

<div id="faster-and-easier-debugging">
  ### Depuración más rápida y sencilla
</div>

- Obtención de errores más ágil en lotes grandes
- Mejor agrupación de errores
- Contexto de errores más útil

Misma API que en v1, con reportes de errores mucho mejores para las funciones de v2 en lotes.
