<div id="whats-new-in-v2">
  ## Novedades en la v2
</div>

<div id="smarter-extraction">
  ### Extracción más inteligente
</div>

FIRE-1 (Agente) maneja sitios web complejos:

- Hace clic automáticamente para navegar páginas y paginación
- Funciona en sitios con uso intensivo de JavaScript
- Es mejor encontrando y extrayendo datos

<div id="no-url-required">
  ### No se requiere URL
</div>

Extrae datos sin conocer la URL exacta:

- Describe lo que necesitas; v2 lo encuentra
- Busca en la web y extrae de las mejores páginas
- Más preciso que v1

<div id="better-performance">
  ### Mejor rendimiento
</div>

- Procesamiento más rápido
- Gestión de errores más fiable
- Mayor precisión

La misma API que en v1, fácil de actualizar.
