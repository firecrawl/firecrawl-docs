<div id="whats-new-in-v2">
  ## Novedades en v2
</div>

La API de cancelación de crawl en v2 mantiene la misma funcionalidad que v1, con mayor confiabilidad:

<div id="enhanced-cancellation-process">
  ### Proceso de cancelación mejorado
</div>

- Mejoras en la limpieza y la gestión de recursos
- Mejor manejo de la finalización de rastreos activos
- Manejo de errores optimizado para las operaciones de cancelación

Los parámetros de la API y el formato de respuesta se mantienen sin cambios respecto a v1, lo que garantiza una migración fluida y una cancelación de rastreos más confiable.
