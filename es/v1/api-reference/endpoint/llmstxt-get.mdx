title: "Estado de LLMs.txt"

---

Obtén el estado y los resultados de un trabajo de generación de LLMs.txt. Este punto de conexión te permite comprobar si la generación ya finalizó y obtener el contenido generado.

<div id="response-structure">
  ### Estructura de la respuesta
</div>

La respuesta incluye:

- **success**: Booleano que indica si la solicitud se completó con éxito
- **status**: Estado actual del trabajo de generación:

  - `processing`: El trabajo sigue en ejecución
  - `completed`: El trabajo se completó correctamente
  - `failed`: Ocurrió un error en el trabajo

- **data**: Contenido generado (cuando el estado es `completed`):

  - `llmstxt`: Contenido generado de llms.txt
  - `llmsfulltxt`: Texto completo (si showFullText era true)

- **expiresAt**: Marca de tiempo ISO que indica cuándo vencerán los resultados

<div id="status-examples">
  ### Ejemplos de estado
</div>

1. **Estado: procesando**

   ```json
   {
     "success": true,
     "status": "processing",
     "data": {
       "llmstxt": "# Firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/)...",
       "llmsfulltxt": "# Firecrawl.dev llms-full.txt\n\n"
     },
     "expiresAt": "2025-03-03T23:19:18.000Z"
   }
   ```

2. **Estado: completado**
   ```json
   {
     "success": true,
     "status": "completed",
     "data": {
       "llmstxt": "# http://firecrawl.dev llms.txt\n\n- [Web Data Extraction Tool](https://www.firecrawl.dev/): Transform websites into clean, LLM-ready data effortlessly.\n- [Flexible Web Scraping Pricing](https://www.firecrawl.dev/pricing): Flexible pricing plans for web scraping and data extraction.",
       "llmsfulltxt": "# http://firecrawl.dev llms-full.txt\n\n## Web Data Extraction Tool\nIntroducing /extract - Get web data with a prompt..."
     },
     "expiresAt": "2025-03-03T22:45:50.000Z"
   }
   ```

<div id="error-handling">
  ### Gestión de errores
</div>

Si el trabajo de generación falla o no se encuentra, recibirás una respuesta de error adecuada:

```json
{
  "success": false,
  "error": "No se encontró el trabajo de generación de LLMs.txt"
}
```

<div id="polling-recommendations">
  ### Recomendaciones para el sondeo
</div>

1. Realiza sondeos cada 2–3 segundos al inicio
2. Aumenta el intervalo si sigue en procesamiento después de 30 segundos
3. Deja de sondear si el estado es `completed` o `failed`
4. Implementa backoff exponencial para evitar los límites de tasa
